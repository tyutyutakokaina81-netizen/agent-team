# 案件別 提案テンプレ（クラウドソーシング応募メッセージ）

> 作成：CSO ／ 2026-06-23
> 用途：クラウドソーシング（クラウドワークス・ランサーズ・ココナラ等）で発注者の募集に応募する際の応募文。
> コピペ＋穴埋め（`{{ }}`）で送れる完成品。

---

## 使い方（先に読む・90秒）

1. **冒頭の `{{相手案件への言及}}` は必ず自分の言葉に書き換える。** ここが空欄・テンプレ丸出しだと即スルーされる。募集文を1回読んで「何に困っているか」を1文で書く。
2. `{{ }}` を全部埋めてから送る。埋め残しがある状態で送らない。
3. **やらない約束はしない。**「必ず◯位」「絶対◯◯」は禁止（成果保証NG）。書けるのは「やること・進め方・できる範囲」まで。
4. 文字数は短すぎず長すぎず。スマホで読まれる前提で**5〜8行＋実績1〜2行**が目安。
5. 価格は最後の「価格の伝え方」節を使う。**最初の提示で値下げ前提の安値を書かない。**

### 守るルール（厳守）
- **PIIを書かない**：具体的な住所・町名・個人特定情報を提案文に書かない。実績は「業界・規模・成果の傾向」までに留める。
- **誇張・成果保証しない**：「検索1位確約」「売上◯倍保証」等は書かない。
- **ダンピング・規約違反しない**：相場を壊す安値提示や、プラットフォーム外への即時誘導はしない。連絡先の事前交換は規約に従う。
- **嘘の実績を書かない**：実績欄は事実のみ。書けるものがなければ「サンプル提出可能」で代替する。

### 共通の穴埋め変数
| 変数 | 中身の例 |
|------|---------|
| `{{相手案件への言及}}` | 「{{業界}}の{{媒体}}向け記事を月{{本数}}本、というご依頼拝見しました」 |
| `{{担当者敬称}}` | 「ご担当者様」（個人名が募集に明記されている場合のみ「◯◯様」） |
| `{{自分の強み一言}}` | 「読み手の検索意図を外さない構成設計」など |
| `{{実績1行}}` | 「◯◯ジャンルで月◯本・累計◯本の納品実績」※事実のみ |
| `{{納期感}}` | 「初稿◯営業日／修正◯日」 |
| `{{確認したい点}}` | 「想定読者と1本あたりの文字数」など |

読者の検索意図を外さず最後まで読まれる記事をご用意できます。

【お出しできる価値】

・一次情報・公的資料を確認し、事実と意見を分けて書きます

【進め方と納期感】

## ★ 価格の伝え方（安請けしない・スコープを切る）

提案文の価格欄、または交渉メッセージで使う。**価格は「作業範囲とセット」で出すのが基本。** 範囲を切らずに安値だけ出すと、後から作業が膨らんで疲弊する。

### 原則
1. **範囲（スコープ）を明示してから価格を出す。**「◯◯まで含めて△△円、それ以外は別途」。
2. **修正回数を区切る。**「初稿＋修正2回まで込み、以降は別途」。
3. **安易な値下げをしない。** 値下げを求められたら「価格を下げる」でなく「範囲を狭める」で調整。
4. **相場を壊さない。** ダンピングは自分も市場も傷つける。安値勝負はしない。
5. 予算が募集に明記されている場合は、その範囲で「何ができるか」を提示する。

### A. 価格を提示するとき（提案文に添える）

```
【お見積りについて】
{{作業範囲（例：構成＋本文◯字＋タイトル案、修正2回まで）}}で {{金額}}円を目安に考えております。
ご予算やご希望の範囲に合わせて調整可能ですので、まずは{{お試し範囲（例：1本）}}から
進めて品質をご確認いただく形もご提案できます。
```

### B. 値下げを求められたとき（範囲で調整する）

```
ご予算の件、承知しました。価格を下げるよりも、まずは範囲を絞る形でご提案させてください。
たとえば {{削れる作業（例：修正回数を1回に、図解は別途）}} とすることで {{調整後金額}}円に
収められます。品質を保てる形で進めたいので、ご相談しながら決められれば幸いです。
```

### C. 単価が相場より大きく低い募集への対応（受けない判断も含む）

```
ご依頼内容拝見しました。ご提示の条件ですと、ご期待の品質を保つのが難しい範囲かと思われます。
もし {{現実的な範囲（例：本数を減らす／1本の文字数を抑える）}} で調整いただけるようでしたら、
品質を落とさずお引き受けできます。一度ご希望を伺えればと思います。
```

> ※ ダンピングして受けるくらいなら、上のCで範囲を提案する。それでも折り合わなければ次の「辞退文」へ。

## ★ 辞退・お断り文

信頼を損なわず断る。**理由を正直に・短く・相手を否定せず。** 角を立てず、可能なら代替案や次の機会に触れる。

### D-1. 条件・予算が合わず辞退

```
{{担当者敬称}}

ご連絡ありがとうございます。せっかくお声がけいただいたところ恐縮ですが、
今回はご希望の条件と当方でお出しできる範囲が合わず、お引き受けを見送らせていただきます。
ご期待に沿えず申し訳ありません。条件が合う形がございましたら、改めてご相談いただけますと幸いです。
```

### D-2. スケジュール・キャパオーバーで辞退

```
{{担当者敬称}}

ご依頼ありがとうございます。大変ありがたいお話なのですが、
現在の稼働状況から、ご希望の納期に責任を持って間に合わせることが難しい状況です。
品質を落としてお受けするのは本意でないため、今回は辞退させていただきます。
{{代替（例：◯月以降であれば対応可能です）}}。またの機会にぜひお声がけください。
```

### D-3. 内容が専門外・対応範囲外で辞退

```
{{担当者敬称}}
