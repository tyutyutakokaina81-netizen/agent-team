## 1. 戦略の精緻化 — traffic問題を正面から

新規 github.io は**ドメイン権威が低い**。「AI automation」級の競合語では当面勝てない。だから戦い方を変える。

### 戦略① 勝てる土俵で戦う＝ロングテール特化
- 競合の少ない**超具体キーワード**を狙う（例：「AI automation for [具体業種/具体タスク]」「[具体ツール] 使い方 中小」）。
- ボリュームは小さいが、**新規ドメインでも上位を取れる**。塵が積もって流入になる。これが新規SEOの正攻法。
- → code が継続生成（owner操作0）。

### 戦略② 1訪問を無駄にしない＝転換率最適化
- traffic が細い前提で、来た人を**確実にlead化**。武器＝〔我々がAIで会社を運営という実証〕〔無料サンプル〕〔明確CTA〕。
- 追加でやれる無料施策：**「制作例（サンプル集）」ページ**で信頼の壁を下げる（code制作可・owner操作0）。

### 戦略③ Googleだけに賭けない＝ヘッジ
- SEOはゼロから数ヶ月の賭け。**外れる可能性も正直ある**。だから「借りた集客（既に人がいる場所＝SNS/コミュニティ/ディレクトリ）」を予備に持つ。
- ただしこれは owner の外部行動が要る → **"やる気の日"オプション**（前提にはしない）。
- 肝：**漫然と待たない**。下の運用ラダーで「効いてないと分かったら次へ」を仕組み化する。

## 2. 運用の精緻化（今日の本題）

### A. code の自律サイクル（呼ばれた時に動く）
- 毎回：ロングテール記事を2〜3本／ハブ更新／JSON-LD・canonical自動付与（既存スクリプト）／sitemap追記／commit→自動デプロイ＋IndexNow自動ping。
- **正直：code は常駐しない。** 動くのは owner が「記事足して／最適化して」と言った時。月数回でも複利で効く。

### B. lead対応SOP（来た瞬間に即動く・最重要）
Gmailに問い合わせ → owner が本文を貼る → code が即：
1. **温かい返信案**
2. **その相手の実タスクで無料サンプル**（中小・富裕層ともこれで決まる）
3. **提案**（オファー・価格・進め方・守秘）
→ owner が送る。続けば code が納品物制作、owner が関係管理。台帳＝`tools/retainers.py`（継続）/`tools/jobs.py`（単発）。

### C. 監視ラダー（何を見て、どこが詰まっているか）
Search Console の数字を **週1** で見る（または owner が貼る → code が読む）。段階で詰まりが分かる：

| 段階 | 見る数字 | 詰まっていたら（直す人） |
|---|---|---|
| Googleが拾う | インデックス数 | 0なら待つ／sitemap再確認 |
| 検索に出る | 表示回数 impressions | 増えない＝記事増・ロングテール強化（code） |
| クリックされる | クリック数 | 出るのにクリック0＝title/meta改善（code） |
| 問い合わせ来る | Gmail着 | クリックあるのに0＝CTA/サンプル提示強化（code） |
| 成約する | 契約 | 来るのに決まらない＝サンプル/商談/価格（owner+code） |

- **最初の生命兆候＝impressions が増えること。** これが出たら「見えない所でエンジンが回り始めた」サイン。

### D. エスカレーション（漫然と待たないための判断ルール）
- 4〜6週で impressions ほぼ0 → コンテンツが刺さっていない。**ロングテール総入替** or **借りた集客（owner）**へ。
- impressions あるのにクリック0 → **title を検索意図に寄せる**（code）。
- クリックあるのに問い合わせ0 → **無料サンプルの訴求を強める**（code）。
- これで毎週「どこが悪いか」をピンポイントで潰せる。

## 3. 役割の最終形（運用体制）
- **code**：制作エンジン＋分析。記事・最適化・lead時の即制作・週1の数字読み。呼ばれたら動く。
- **owner**：①週1でSearch Console数字を見る（or codeに貼る）②問い合わせが来たら本文を貼る・最後に送る③やる気の日に借りた集客を1つ。
- **cowork**：任意（借りた集客の実行＝投稿/登録を代行可）。

## 4. 唯一の先行指標
KPIは「初問い合わせ」。だがその**前兆＝Search Console の impressions**。
0→増 に変われば、エンジンが回り始めた証拠。**まずここだけ週1で見る。** 他は追わない。

> 本質：今日 Google の扉は開いた。ここからは「ロングテールで土俵を選ぶ × 来た1人を逃さない × 週1で数字を読み、詰まった所だけ直す」。
> 派手さはないが、ゼロ予算・owner最小行動で**複利が効く唯一の道**。
