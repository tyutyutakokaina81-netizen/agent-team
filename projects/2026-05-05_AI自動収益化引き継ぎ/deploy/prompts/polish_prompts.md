# 無料Web AI ポリッシュプロンプト集

このファイルは、`~/ai-auto/` の生成物を **claude.ai / gemini.google.com / chatgpt.com** の無料枠で
肉付け・整形するためのプロンプト集です。**APIは使わず、すべて無料Web AI で完結**させる前提。

## 使い分け（推奨）

| ツール | 強み | 用途 |
|------|------|------|
| **claude.ai** | 長文・日本語の自然さ・SEO構成 | 3,000字SEO記事、有料note本文、提案書 |
| **gemini.google.com** | 検索連携・最新情報・タイトル発想 | キーワード調査、タイトル候補 |
| **chatgpt.com** | 短文・SNS風表現 | X投稿、Instagramキャプション |

## 1. SEO記事 展開プロンプト（claude.ai 推奨）

```
あなたは月間PV 10万のSEOメディア編集者です。
下記の構成（H2/H3）に沿って、3,000字の本文を執筆してください。

【絶対条件】
- 主キーワードを自然に4〜6回挿入
- 一文40字以内、段落3行以内
- 各H2の冒頭に結論、末尾に次セクションへのつなぎ
- 体験談を1つ以上含める
- 数字を3つ以上含める（具体性）
- AI臭さを消す（「いかがでしたか」「まとめ」を使わない）

【構成】
（ここに ~/ai-auto/outputs/YYYY-MM-DD_seo_article_*.md の構成部分を貼る）

本文のみを Markdown で出力してください。
```

## 2. 有料note 本文展開プロンプト（claude.ai 推奨）

```
あなたは note で月50万円稼ぐ個人発信者の文章コーチです。
下記の有料note骨組みを、購入されやすい本文に書き直してください。

【絶対条件】
- 無料パート（300字）：問題提起→共感→「続きは有料」で締める
- 有料パート（2,000字）：手順・テンプレ・失敗例・チェックリストを具体的に
- 有料部分の冒頭に「ここから先で得られること」を箇条書き3行
- 価格に見合う情報密度（机上論ではなく具体例・数字）
- 「私はこうした」型の体験談を最低1つ

【骨組み】
（ここに ~/ai-auto/outputs/YYYY-MM-DD_paid_note_*.md を貼る）

無料/有料の境界を「----」で明示してください。
```

- 提案を1つ含める（「テストライティング1本でいかがでしょうか」など）
- 全体を400字以内に収める（長すぎは読まれない）

【現在の応募文】
（ここに ~/ai-auto/outputs/YYYY-MM-DD_cw_apply_*.txt を貼る）

【案件詳細】
（CrowdWorksの案件ページから本文をコピペ）
```

## 運用ルール

- **無料枠の範囲**：claude.ai は1日数十回、gemini は実質無制限、chatgpt は1日数十回
- **3つを使い分ければ1日30本ペースまでカバー可能**
- 同じ画面内で複数記事を連続生成すると枠を消費するため、**ブラウザを再起動 or タブ分け**
- 出力をそのまま使わず、**最低1回は人間の目で読み直す**（AI誤情報チェック）
