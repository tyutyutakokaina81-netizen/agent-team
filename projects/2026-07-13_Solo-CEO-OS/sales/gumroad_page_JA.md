## 説明文（本文）

### あなたは「だらしない」んじゃない。「人手が足りない」だけだ。

ひとりで事業をやると、マーケも商品も経理も営業も雑務も、全部を同時に、全部中途半端に、
一日中こなすことになる。AIアシスタントに頼りたいけど、「ねえChatGPT、事業手伝って」では、
記憶もなく担当もない“優秀なインターン”が返ってくるだけ。

**Solo CEO OS はそこを直す。** いつも使っているAIアシスタントの中に、6人のAI幹部を入れる。
それぞれに担当・視点・共通ルールがある。だから“チャットボットと雑談”をやめて、
“チームに任せる”に変わる。

### あなたの幹部チーム

- **CMO（マーケ）** — 集客と共感のための投稿・LP・ローンチ文を書く
- **CPO（商品）** — 散らかった知識を「最後までやりきれる商品」に整える
- **CFO（財務）** — 価格・数字・リスクを曖昧にしない
- **CSO（営業）** — 信頼を壊さずに顧客の反論・提案に対応する
- **CDO（技術）** — 面倒な作業を自動化し、事業が育てば新しい幹部を“採用”する
- **CAO（分析）** — 「先月なぜ伸びたか」を言語化し、意図的に再現できるようにする

各自が自分のレーンを守る。**わざと意見が食い違う** — 本物の経営チームと同じだ。
決めるのはあなた。あなたがCEOだから。

### あなたの事業を「記憶」する

最大の武器は、AIが毎回セッション開始時に読む**永続メモリファイル**。
一度書けば、事業内容を二度と説明し直さなくていい。決定事項・進行中の案件・次の一手を、
日をまたいでも、ツールをまたいでも引き継ぐ。

### あなたと一緒に成長する

誰の担当でもないタスクにぶつかったら、CDOに言えばいい。条件を満たせば、
CDOが新しい幹部（人事・法務・顧客体験など）を専用の作業場ごと立ち上げる。
組織図が事業に合わせて育っていく。

### 手に入るもの

- Solo CEO OS 一式（README＋30分クイックスタート）
- そのまま使える6役職の幹部プロンプト
- 共通ルールブック（`company.md`）
- Claude Code / Claude Desktop / ChatGPT / Cursor 用の導入手順
- すべてを積み上げに変える「永続メモリ」テンプレート
- v1.x内の無料アップデート

### 今あるもので動く

新しいサブスク不要。コード不要。フォルダを読めるAIアシスタント（ClaudeでもChatGPTでも、
デスクトップでもWebでも）があれば、30分で始められる。

### 誇張なしの前提

これはOSであって、魔法のお金ボタンではない。事業を代わりにやってはくれない。
事業に「構造」と「チーム」を与えて、**あなた自身がずっと上手く回せるようにする**もの。
実際にひとりで事業を回している運営者が、毎日使って作った。
