## 0. この資料の読み方（先に3つ）

1. **料金は「日本のクラウドソーシングの一般相場帯（幅）」の目安**であって断定ではない。
   プラットフォーム・時期・難易度で動くため、出品前に**owner が最新相場を [要確認]**。相場帯は「憶測」でなく
   一般に流通している範囲だが、正確な現在値は要検証（諸説あり／[要検証]）。
2. **2段階単価**＝新規で実績ゼロのため、①**レビュー獲得期**（最初の3〜5件・低単価で受注してレビューを積む）
   → ②**通常単価**（レビューが付いてから戻す）。レビュー獲得期は「安く受ける代わりにレビューをお願いする」設計。
3. **手数料**：受注額からプラットフォーム手数料（**おおむね16〜22%程度／[要確認]**）が引かれる。
   表示価格＝請求額。手取りはここから手数料を引いた額（採算判定は 2026-06-23 §2「実効時給」で行う）。

### A5厳守（盛らない）
- 約束するのは**成果物の質と納期のみ**。クライアント側の成果（売上・検索順位・来客・反響）は保証しない。
- 実績ゼロは隠さない。「新規のため実績欄は薄いが、**サンプルを無料でお出しできます**」で正直に補う。
- 「AIで作る」ことを隠さない。**"AIで速く・安く・一定品質で作る"** ことがそのまま強み（done-by-AI）。

## 1. 中核サービス（フック）：商談議事録 → 提案書を最短30分〜当日納品

| 項目 | 内容 |
|---|---|
| 何を | 打ち合わせの**議事録/文字起こし＋要件メモ**を渡してもらい、**提案書（現状整理→解決策→規模別見積り構成）**を作成 |
| 納期 | **最短30分〜当日**（議事録が整っている場合）。ボリューム大・図表多用は翌営業日 |
| 料金・レビュー獲得期 | **¥3,000〜¥5,000/件**（お試し価格・レビュー協力のお願い付き） |
| 料金・通常 | **¥8,000〜¥20,000/件**（分量・図表・スライド化の有無で変動。一般的な提案書作成の相場帯／[要検証]） |
| 当社の強み1行 | **「打ち合わせ当日に提案書が届く速さ」**＝商談の熱が冷める前に出す。属人的な残業でなくAIで即日化するので納期がブレない |

> なぜフックか（かいち §6-5）：「打ち合わせ30分後に提案書が届くと顧客が驚く」＝**即納品の体験そのものが差別化**。
> 観客ゼロの新規でも「速さ」は初回から示せる。※ただし議事録の質に納品品質が依存する点は正直に伝える。

## 2. 資料作成系（当社が実際に出せる4種）

### ② 提案書・企画書作成（議事録なし・要件からゼロ作成）
| 項目 | 内容 |
|---|---|
| 何を | 目的・ターゲット・訴求を伺い、提案書/企画書を構成から作成（10〜20ページ想定） |
| 納期 | **2〜4営業日**（10ページ目安） |
| レビュー獲得期 | **¥5,000〜¥8,000/件** |
| 通常単価 | **¥10,000〜¥30,000/件**（ページ数・調査量・図解量で変動。ココナラ等の提案書作成相場帯／[要検証]） |
| 強み1行 | 論点整理→構成→本文を分けて作るので**「何を言いたいか」がブレない**。急ぎにも当日〜翌日で対応可 |

### ③ スライド作成（パワポ/Googleスライド体裁整え・原稿→スライド化）
| 項目 | 内容 |
|---|---|
| 何を | 原稿やラフを、読めるスライドに構成・文章・体裁を整える（既存テンプレ流用前提） |
| 納期 | **2〜4営業日**（10枚目安） |
| レビュー獲得期 | **¥500〜¥1,000/枚**（初回お試し。10枚で¥5,000〜¥10,000） |
| 通常単価 | **¥1,500〜¥3,000/枚**（テキスト主体。図表多用は上振れ。スライド作成の一般相場帯／[要検証]） |
| 強み1行 | 1枚1メッセージで整理。文章が得意なので**「箇条書きの丸投げ」を伝わる言葉に翻訳**する |
| 含まない | オリジナルデザイン制作・画像生成（A2：当社は画像生成不可）・印刷入稿・登壇代行 |

### ④ SEO記事作成（ブログ・コラム・オウンドメディア）
| 項目 | 内容 |
|---|---|
| 何を | 検索意図に沿った構成＋本文執筆＋推敲。指定トンマナ・NGワード反映 |
| 納期 | **3〜5営業日**（3,000字目安・構成確定後） |
| レビュー獲得期 | **文字単価 ¥0.7〜¥1.0/字**（例：3,000字＝¥2,100〜¥3,000。実績作りの初回のみ） |
| 通常単価 | **文字単価 ¥1.0〜¥2.5/字**（例：3,000字＝¥3,000〜¥7,500。一般的なSEO記事相場帯／[要検証]） |
| 強み1行 | 一次情報・公的資料を確認し**事実と意見を分けて書く**。「あるある羅列」でない読まれる文章 |
| 含まない | 検索順位の保証（A5）・一次取材/撮影・無制限修正・公開作業 |

### ⑤ 議事録作成・整形（文字起こし → 読める議事録）
| 項目 | 内容 |
|---|---|
| 何を | 録音の文字起こしを、要点・決定事項・ToDoに整形（話し言葉→書き言葉） |
| 納期 | **1〜2営業日**（30分相当） |
| レビュー獲得期 | **¥1,000〜¥1,500/件**（30分相当） |
| 通常単価 | **¥2,000〜¥5,000/件**（30分相当。整形のみ。ゼロからの文字起こし込みは上振れ／[要検証]） |
| 強み1行 | 決定事項とToDoを漏らさず拾い、**そのまま関係者に共有できる状態**にする。①の提案書化にも接続 |
| 含まない | 実名・所属の記録（A4：匿名化前提）・録音の品質改善 |

## 3. 商品ラダー（安く入って上げる導線）

```
議事録整形(¥1,000〜) → 議事録→提案書30分(¥3,000〜) → 提案書ゼロ作成(¥5,000〜) → 継続/定額(月◯本パック)
     入口・低摩擦              中核フック                  単価UP                    実効時給が最も上がる本命
```

- **入口**は議事録整形/スライド1枚単位＝発注ハードルが低くレビューが付きやすい。
- **中核**の「議事録→提案書30分」でリピート指名を狙う（速さの体験が刺さる）。
- **本命**は継続・定額化（営業/ヒアリング時間が毎回ゼロになり実効時給が上がる。かいち §7 継続移管の型）。

## 4. レビュー獲得期 → 通常単価に戻す判断（2段階の切替）

| トリガー（どれか1つ） | アクション |
|---|---|
| 高評価レビューが **3〜5件** 付いた | 新規見積から**通常単価**へ戻す（既存客は据え置き） |
| 依頼が**断れないほど**来る（供給＜需要） | 段階値上げ（+10〜20%ずつ・角を立てない） |
| リピート指名が付いた | 実績が単価を裏付ける。ただし**誇張表記はしない（A5）** |

- レビュー獲得期でも**採算の自衛ライン**（実効時給・2026-06-23 §2-2）は意識する。
  「安く受ける＝赤字で受ける」ではない。時短（AI下書き）で総時間を圧縮し、**低単価でも時給割れさせない**設計。
- ただし最初の1〜2件は「レビューという資産を買う投資」＝**多少の時給割れは許容**（回収は通常単価で行う）。

## 5. 全メニュー共通の免責（提案文・納品物にも明記）

- **成果保証しない**：質と納期を約束。売上・順位・反響は保証しない（A5）。
- **PII非掲載**：成果物は匿名化・市レベルまで（A4）。実名・住所・機微情報の記録は受けない。
- **修正回数の上限**：標準2回。3回目以降は別料金 or 別見積（無限修正は受けない）。
- **AI利用を開示**：「AIを活用して速く一定品質で作る」ことを隠さない。最終確認は人（owner）が通す。
