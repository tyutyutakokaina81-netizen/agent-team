# クラウドソーシング 応募文テンプレ（そのまま貼れる・3種）

作成: 2026-07-10 ／ 担当: CSO
対象: ランサーズ／クラウドワークス／ココナラ の「提案書作成」「資料作成」「記事作成」案件
関連: `service_menu.md`（料金・納期の正本） / `projects/2026-06-23_受託_クラウドソーシング/提案テンプレ/案件別提案テンプレ.md`

---

## 使う前に（90秒・厳守）

1. **冒頭の相手案件への言及は必ず自分の言葉で書き換える。** 募集文を1回読み「何に困っているか」を1文にする。
   テンプレ丸出し・空欄のまま送ると即スルーされる。
2. `{{ }}` を全部埋めてから送る。埋め残しがある状態で送らない。
3. **押し売り禁止・成果保証禁止（A5）**：「必ず売れる」「1位保証」「絶対」は書かない。書けるのは
   「やること・進め方・できる範囲・納期・料金」まで。誠実トーンで。
4. **実績ゼロは正直に**：嘘の実績を書かない。書けるものがなければ「**サンプルを無料でお出しできます**」で代替。
5. **AI利用を隠さない**：「AIを活用して短納期・一定品質で作る」と正直に書く（それが強み＝done-by-AI）。
6. スマホで読まれる前提。**本文5〜8行＋料金/納期＋一言**が目安。長文にしない。

### 共通の穴埋め変数
| 変数 | 例 |
|---|---|
| `{{担当者敬称}}` | 「ご担当者様」（募集に個人名があれば「◯◯様」） |
| `{{相手案件への言及}}` | 「◯◯の提案書を短納期で、という募集を拝見しました」 |
| `{{確認したい点}}` | 「想定読者・分量・締切」など |
| `{{お試し価格}}` | `service_menu.md` のレビュー獲得期価格を入れる |

## 応募文① 提案書・資料作成 案件（中核サービスを当てる）

```
{{担当者敬称}}

はじめまして。募集を拝見しご連絡しました。
{{相手案件への言及（例：打ち合わせ内容をもとに提案書を作成、というご依頼、）}}
であれば、打ち合わせの議事録やメモをいただければ、
提案書（現状整理→解決策→ご予算別の見積り構成）を短納期でご用意できます。

【進め方と納期】
1) 議事録・要件メモを共有いただく
2) 構成をお出しして方向をすり合わせ
3) 提案書に仕上げてお渡し
議事録が整っていれば、当日〜翌営業日でお出しできます。

【正直にお伝えすること】
・私はAIを活用して短納期・一定品質で作成します（最終確認は人が行います）。
・当プラットフォームでは新規のため評価はまだ少ないですが、
　ご不安であれば簡単なサンプルを無料でお出しできます。
・成果（受注や売上）はお約束できませんが、資料の質と納期は責任を持ちます。

【料金】
今回は実績づくりのため {{お試し価格（例：¥5,000〜）}} でお受けできます。
まず{{確認したい点（例：提案の相手・盛り込みたい要点・締切）}}を伺えれば、
分量と納期を確定してお見積りします。ご検討よろしくお願いいたします。
```

## 応募文② スライド・パワポ資料作成 案件

```
{{担当者敬称}}

募集を拝見しました。
{{相手案件への言及（例：原稿をスライドに整えてほしい、とのこと、）}}

箇条書きのラフや原稿をいただければ、伝わる言葉と構成に落とし込みます。

【できること】
・原稿/ラフ → 構成・文章・体裁の整ったスライド化（既存テンプレに合わせます）

・AIを活用して短納期で作ります（最終チェックは人が行います）。
・新規のため評価は少ないですが、1〜2枚のサンプルを無料でお出しできます。
・オリジナルのデザイン制作や画像生成は範囲外です（文章・構成・体裁が担当領域です）。

【料金・納期】
実績づくりのため {{お試し価格（例：¥500〜/枚）}} でお受けできます。
枚数と締切、参考にされたい体裁を伺えれば確定でお見積りします。
どうぞよろしくお願いいたします。
```

```
{{担当者敬称}}

はじめまして。募集を拝見しご連絡しました。
{{相手案件への言及（例：◯◯ジャンルの記事を継続で、とのこと、）}}
読者の検索意図を外さず、最後まで読まれる記事をご用意できます。

【進め方】
1) 構成案を先にお出しして方向確認 → 2) 本文執筆 → 3) ご指摘を反映

【正直にお伝えすること】

・新規のため評価は少ないですが、テーマをいただければ短いサンプルを無料でお出しできます。
・検索順位はお約束できません（アルゴリズム依存のため）。責任を持つのは記事の質と納期です。

【料金】
実績づくりのため、今回は {{お試し価格（例：文字単価¥0.7〜/字）}} でお受けできます。
{{確認したい点（例：想定読者・1本の文字数・参考記事）}}を伺えれば、
お試し1本からご提案します。ご検討よろしくお願いいたします。
```

## 送信後のふるまい（押し売りしない）

- 返信が来たら、まず**確認したい点を1〜2問**聞いてスコープを固める（分量・締切・回数）。
- 断られても追撃しない。1案件1提案。**プラットフォーム外への即時誘導はしない**（規約遵守）。
- 受注可否は必ず 2026-06-23「メニュー料金_受注判断.md」§3チェックリストに戻す（安すぎ/無限修正/スコープ不明/PII過多/成果保証要求は辞退）。
