## 3. 各ツールに足す最小要素（code が実装する粒度）

各ツール共通で足すもの（1ツールあたり最大でこれだけ。増やさない）：
1. **`<title>` / `<meta name="description">` の検索語最適化**（②③は既に良好、①④⑤は下記に沿って調整）。
2. **「使い方」1ブロック**（3ステップ程度。既に説明があるものは不要）。
3. **フッターCTA 1つ**（主CTAのみ。文面は §後半の確定コピーを使用）。
4. **関連ツール相互リンク**（フッターに「他の無料ツール」小リスト＝5本を相対リンク `../<tool>/`）。
5. プレースホルダ（`【商品リンク】`・`href="#"`）を**確定URL**に差し替え。

### ツール別・具体差分

**① freelance-rate-calculator**
- `<title>`：現状可。`description` に「フリーランス 単価 時給」を自然に含める微調整。

- フッターに関連ツール5本リンクを追加。
- 使い方：現状のリード文で足りる（追加不要）。

**② invoice-generator**（最優先・SEO母数最大）
- `<title>`・`description` は既に「請求書 テンプレート 無料 PDF」を含み良好＝**変更最小**。
- `.promo` 内 `<a href="#" id="productLink">` → 実務テンプレ集/収支シートの確定URL（暫定 `../store.html#products`）。
- 関連ツール5本リンクをフッター（`footer.site`、`print-hide` 側）に追加＝印刷時は出さない。

**④ ai-shindan**

## 6. 判定指標（30日 / 90日）

ファネル段：**ツール利用（PV/操作）→ 商品ページ遷移（CTAクリック）→ 初売上**。
数値目標は控えめ・実測ベースで見直す前提（A5：達成保証ではない）。

### 30日（立ち上げ・"導線が生きているか"の確認）
| 指標 | 見るもの | 合格ライン（初期目安） |
|---|---|---|
| ツール到達 | 5ツール合計の月間PV | GA4で**計測が取れている**こと自体が第一（現状0導線からの脱却） |
| 操作率 | ①②③で「計算/生成/印刷」まで到達した割合 | PVの30%以上が何らか操作 |
| CTA遷移 | ツール→商品ページ/mailtoのクリック率 | PVの**1〜3%**（低くても導線が動作すれば◎） |
| 初売上 | ツール経由の商品購入 or 受託問い合わせ | **1件以上**（金額不問）。受託問い合わせ(④⑤)は1件でも価値大 |
| SEO土台 | Search Console で各ツールURLが**インデックス済み** | 5本中3本以上 index |

### 90日（成長・"どのツールが稼ぐか"の選別）
| 指標 | 見るもの | 目安 |
|---|---|---|
| 検索流入 | ②請求書・③プロンプトの検索クリック | 右肩上がり（母数大の②が牽引想定） |
| CTA遷移率 | 各ツールの遷移率 | 3〜5%へ改善（コピー/配置の最適化後） |
| 転換 | 遷移→購入/問い合わせの成約率 | デジタル商品数%・受託は問い合わせ→受注で判断 |
| 売上/リード | ツール経由の合計 | **デジタル商品 数件〜／受託リード 数件**を目標に実測で調整 |
| 選別判断 | ツール別ROI（PV×遷移×成約） | 効くツールに相互リンク/SEOを寄せ、効かないツールは現状維持（放置でも¥0で無害） |

判定ルール：**90日で「ツール経由の売上/受託リードが1件も出ない」場合**、収益化リンクは残しつつ（¥0・無害）、
新規の追加投資（EN版・新ツール）は保留。**②請求書 or ④⑤受託のどちらかが1件でも刺されば**、そこへ集中投下する。
