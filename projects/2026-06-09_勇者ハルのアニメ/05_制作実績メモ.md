## このサンドボックスで分かった制約（重要・次回の前提）
外向き通信はプロキシ経由で、**ほとんどの外部TTS/画像サービスが遮断**されていた：

| 種別 | サービス | 可否 |
|---|---|---|
| TTS | Google(gTTS) / MS Edge / Amazon Polly(StreamElements) | ❌ 遮断 |
| TTS | ローカル: pyopenjtalk | ✅ 動くが声がロボット寄り |
| TTS | **ローカル: VOICEVOX engine** | ✅ **採用（声優級）** |
| 画像 | HuggingFace / api.github.com / download.pytorch.org | ❌ 遮断 |
| 画像 | PyPI / github.com / raw.githubusercontent | ✅ 到達可 |
| 画像 | **AnimeGANv2 (animegan2-pytorch)** | ✅ **採用** |

- プロキシはAnthropicのegress CAで再署名。`SSL_CERT_FILE=/etc/ssl/certs/ca-certificates.crt` を使えば検証は通る。
- torchは `pip install torch`（PyPI本体, CUDAビルド)でCPU実行。
- AnimeGANの重みは `raw.githubusercontent.com/bryandlee/animegan2-pytorch/main/weights/*.pt` から直接取得（torch.hubはapi.github.com遮断で不可）。

## 実際のパイプライン（制作スクリプト/ に保存）
1. `stylize.py` — 写真3枚を **AnimeGANv2 (celeba_distill)** で本物のアニメ絵に変換
   - モデル比較結果：celeba_distill=フラットで一番アニメらしい（採用）／face_paint_512_v2=絵画寄り／paprika=実写寄り
2. `make_audio.py` — **VOICEVOX**(localhost:50021) でキャラ別アフレコを合成
   - ハル=白上虎太郎 / ミラ先生=九州そら / チビ=ずんだもん / ぷるん=四国めたん / ネムリン=青山龍星しっとり / ナレ=青山龍星 / みんな=春日部つむぎ
   - 速度・ピッチをキャラごとに調整
3. `build_video.py` — 写真にKen Burns(ズーム/パン)＋**ドラクエ風メッセージウィンドウ字幕**＋**チップチューンBGM**を合成し ffmpeg でmp4化
   - 音声尺に合わせてシーン尺を自動決定（完全同期）

## 再現手順（コンテナ再構築時）
```bash
pip install gTTS pillow imageio-ffmpeg py7zr edge-tts pyopenjtalk torch numpy  # 必要分
# VOICEVOX engine 0.25.2 を github releases から取得・展開し ./run で起動
# raw.githubusercontent から animegan2 の model.py と weights/*.pt を取得
python3 stylize.py && python3 make_audio.py && python3 build_video.py
```
