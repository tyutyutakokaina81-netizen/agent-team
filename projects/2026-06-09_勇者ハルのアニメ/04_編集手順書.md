# 04_編集手順書 — 15分アニメの組み立て（音声あり）

オーナーのMac側での作業手順。無料〜安価ツール想定。

## 必要なもの
| 用途 | ツール（例） | 料金 |
|---|---|---|
| 音声合成 | **VOICEVOX**（Mac版） | 無料 |
| 映像生成 | Runway / Kling / Luma / Sora いずれか | 一部無料枠あり |
| 静止画生成 | Midjourney / NijiJourney（任意） | 有料 |
| 編集 | **CapCut**（簡単）or DaVinci Resolve（高機能・無料） | 無料 |
| BGM/SE | DOVA-SYNDROME / 甘茶の音楽工房（フリー） | 無料 |

## 手順
### STEP 1: 音声を作る（先に音声＝尺の基準）
1. `02_音声台本_VOICEVOX.md` の各行を VOICEVOX に貼り、話者を割り当てて1行ずつ書き出す。
2. ファイル名を `01-00_narration.wav` のようにカット番号順で保存。
3. 全部の長さを足すと「実尺」が出る。**音声尺に映像を合わせる**のが楽。

### STEP 2: 映像を作る
1. `03_映像生成プロンプト.md` の共通スタイル＋キャラ設定＋各カット文を結合して生成。
2. まず**静止画**で全カットの絵柄を固め、OKカットだけ i2v で動かす（破綻・コスト対策）。
3. 各クリップを `01-01.mp4` のようにカット番号で保存。

### STEP 3: 並べる（CapCut）
1. 新規プロジェクト 16:9 / 24fps。
2. **音声トラック**にSTEP1のwavをタイムコード順に配置（台本の `[0:00]` 等が目安）。
3. **映像トラック**に各クリップを音声に合わせて配置。短ければ速度調整 or 静止画パンで延長。
4. 章の変わり目に**クロスフェード／ホワイトアウト**を入れると15分でもダレない。

### STEP 4: 仕上げ
1. BGMトラックを章ごとに敷く（音量はセリフの -12〜-18dB 下げる＝ダッキング）。
2. SE（キラキラ・ぴょん・かけら発見の「ぱぁ」）を要所に。
3. 字幕（任意・子ども向けならひらがな大きめ）。
4. タイトル「青き勇者ハルのぼうけん」とラストのスタッフロールを追加。
5. 書き出し: 1080p / mp4 / 24fps。

## チェックリスト（公開する場合）
- [ ] 実在児童の顔をそのまま再現していない（オリジナルキャラ化済み）
- [ ] ドラクエ等の固有ロゴ・キャラ・楽曲を使っていない（フリー音源のみ）
- [ ] 実名・園名・地名が映像/字幕に出ていない
- [ ] 家庭内視聴のみなら上記は緩めてOK（公開時のみ厳守）

## 尺の目安
音声 約3〜4分（セリフ/ナレ実尺）＋ 映像の間・BGM・章転換 で **約15分** に伸ばす。
セリフが詰まりすぎないよう、各カットに「絵を見せる無音の間」を1〜2秒入れると自然。
