# Phase 0：販売検証実験計画

**目的**: 「本当に売れるのか」を実地で確かめる（3週間）  
**前提**: 「月¥30万戦略」はその後に。まず「売上が出るか」を検証してからスタート

---

## 現在の問題

| 項目 | 状態 | 問題 |
|------|------|------|
| **販売実績** | ゼロ | テンプレも記事も、1個も売ったことがない |
| **顧客の声** | なし | 「このテンプレ欲しい」という実際の声がない |
| **購買可能性** | 未検証 | 「¥980で買う人がいるのか」が仮説のまま |
| **ターゲット** | 曖昧 | 「誰が買い手なのか」の仮説しかない |
| **価値検証** | 未済 | 「このテンプレは本当に役立つのか」を実ユーザーで試していない |

**これらが全てゼロのまま「月¥30万戦略」を立てても、意味がない。**

---

## やるべきことの順序

```
❌ 今までのやり方：
   「テンプレができた」
   → 「メンバーシップ開店しよう」
   → 「自動化スクリプト書こう」
   → 「月¥30万戦略を立てよう」
   
   結果：売上 ¥0

✅ 正しい順序：
   「テンプレができた」
   → 「本当に売れるのか検証する」
   → 「売れることが確認できた」
   → 「スケール方法を考える」
   → 「自動化・戦略を立てる」
```

---

## Phase 0：3週間の販売検証実験

### 目標

**「¥980のテンプレが月 5-10個売れるのか」を確認する**

- 売上：¥5,000-10,000
- 購入者数：5-10人
- 購入者フィードバック：「実際に使えるか」「価値があるか」

この 3 つが確認できたら、その時点で初めて「スケール戦略」を立てる。

---

## 実験方法：「リアルな販売」をシミュレートする

### ステップ 1：note無料読者に直接聞く（Day 1-3）

#### 方法 A：最新の無料note記事に「リサーチDM」を貼る

```
記事の最後に追加：

---

【読者向けアンケート】
この記事を読んで、「こんなテンプレがあったら、今日から使いたい」と思うことはありますか？

例えば：
- 請求書を 3分で作るプロンプト
- 売上目標を逆算するスプレッドシート
- 単価交渉のメール文例

「欲しい」と思ったら、コメント欄で教えてください。

（具体的にどれが欲しいか、いくらなら買うか、を知りたいです）

---
```

**目的**：
- 無料読者が「何を欲しいのか」を直接聞く
- 「¥980なら買うか」という購買意欲を確認

**期待**：コメント 5-20件

---

#### 方法 B：Twitter で投票（Day 1-3）

```
【アンケート】個人事業主の皆さんへ

今月、時間を奪われている業務は？
(複数選択可)

□ 請求書作成（毎月 1-2時間）
□ 売上・経費の整理（毎月 2-3時間）
□ 単価・値上げの提案書作成（毎案件 30分-1時間）
□ 外注指示書の作成（毎回 30分）
□ その他

この中で「テンプレがあれば解決するなら ¥980で買う」ものはありますか？
```

**目的**：
- Twitter フォロワー（まだいるなら）のニーズを把握
- 「何が最も欲しいのか」を投票で確認

**期待**：投票 10-50票

---

### ステップ 2：実際に 1-2 個のテンプレを「試験販売」（Day 4-10）

#### 販売方法 A：note「単発有料記事」で試す（最も簡単）

```
現状：有料記事 1-2本は無料メンバーシップ限定で投稿済み

新規実験：有料記事（非メンバーシップ）を 1 本作成
- タイトル：「請求書を 3分で作るプロンプト【テンプレ付き】」
- 価格：¥980
- 内容：完全版プロンプト + 使用例 3パターン
- 公開：Day 4（note から公開）

計測：
- PV 数
- 購入数
- 購入率（PV ÷ 購入）
```

**理由**：
- BOOTH・Gumroad と違い、note内で完結
- 既存の無料読者にリーチしやすい
- 実装が簡単（テンプレをnote記事として投稿するだけ）

**期待**：
- PV：100-500
- 購入数：2-10個
- 購入率：2-5%

---

#### 販売方法 B：BOOTH に 1 個出品（Day 4-10）

```
BOOTH に「請求書プロンプト完全版」を出品
- 価格：¥980
- 説明文：note無料読者向けに「5年間の実運用で改善」を強調
- サムネ：簡潔なテキストデザイン

計測：
- アクセス数
- 購入数
```

**理由**：
- BOOTH は「テンプレ販売プラットフォーム」として Google で上位
- 新規顧客の可能性がある

**期待**：
- アクセス数：5-20
- 購入数：0-2個（初期段階なので低い）

---

### ステップ 3：購入者に「本当に価値があるか」をヒアリング（Day 11-14）

#### ヒアリング質問

購入してくれた人に、メールで以下を聞く：

```
【ご購入ありがとうございます】

テンプレをご使用いただいて、以下を教えていただけますか？

Q1. 実際に使ってみて、どんな感じでしたか？
  - 「本当に 3分で終わった」
  - 「2回目以降、もっと早くなった」
  - 「微調整が必要だった」
  など

Q2. 「¥980の価値」があると思いますか？
  - 「安い」
  - 「妥当」
  - 「ちょっと高い」

Q3. 他に「テンプレ化してほしいもの」ありますか？
  - 「スプレッドシート版」
  - 「毎月の更新版」
  など

Q4. 今後、こうしたテンプレを購入する予定ありますか？
  - 「月 1-2個は欲しい」
  - 「必要に応じて」
  - 「もういい」
```

**目的**：
- 「本当に価値があるか」の生の声を聞く
- 「他に何が欲しいのか」のアイデアを得る
- 「継続購入するか」の意思を確認

## 判定基準：「検証成功」の定義

### ✅ 成功パターン

```
以下の条件を **1つ以上**満たす：

1. 販売実績
   - note単発記事が 3個以上売れた
   - BOOTH で 1個以上売れた
   - 合計 ¥3,000以上の売上が出た

2. 顧客反応
   - 購入者の 50%以上が「¥980の価値がある」と答えた
   - 購入者の 50%以上が「また買う」と答えた
   - コメント / 投票で「欲しい」の声が 10件以上あった

3. 需要の明確化
   - 「最も欲しいテンプレ」が明確に浮かび上がった
   - 複数の購入者から同じテンプレのリクエストがあった
```

**判定：「検証成功」→ Phase 1（スケール計画）へ進む**

---

### 🔴 失敗パターン

```
以下の場合：

1. 販売実績がゼロ
   - note単発記事：PV は来ているが 0 購入
   - BOOTH：アクセスが 5以下 or 0 購入
   - note リサーチアンケート：コメント 2件以下

2. 顧客反応が冷淡
   - 購入者の 80%以上が「ちょっと高い」
   - 購入者の全員が「もう買わない」
   - Twitter 投票で「興味なし」が 60%以上

3. ニーズが不明確
   - 「欲しい」という声が 5件以下
   - リクエスト内容がバラバラで傾向がない
```

**判定：「検証失敗」→ ビジネスモデル全体の再考**

---

## 「失敗時」の対応

もし販売実績がゼロだった場合、以下を検討：

### 案 A：「テンプレが悪い」仮説

```
→ 対応：購入者に直接聞いた「ニーズ」を反映したテンプレを作り直す
  例）「スプレッドシート版が欲しい」→ 上記を作成・再販売テスト
```

### 案 B：「ターゲットが合致していない」仮説

```
→ 対応：ターゲットを変更してテスト
  現在：「個人事業主全般」
  新案A：「個人コンサル・コーチ層」に絞る
  新案B：「飲食店・美容院」に変更
  
  新ターゲット向けの文案・メッセージを作成し、再販売テスト
```

### 案 C：「価格が高い」仮説

```
→ 対応：価格を下げて再テスト
  現在：¥980
  新案：¥500 or ¥298
  
  「¥500なら買う」という層がいるか確認
```

### 案 D：「認知がない」仮説

```
→ 対応：マーケティング施策を強化して再テスト
  現在：note 内部告知のみ
  新案：
  - Twitter 発信強化（毎日 2-3投稿）
  - ブログ記事で SEO（Google で「請求書 プロンプト」狙い）
  - YouTube で「テンプレ使用方法」動画
  
  の 3ヶ月後に再販売テスト
```

## Phase 0 の成果物

このテストが終わった時点で、以下が明確になる：

```
□ 「売れるのか売れないのか」が実データで明確
□ 「いくらなら買うのか」が顧客の口から聞ける
□ 「何が最も欲しいのか」がわかる
□ 「購入者は誰なのか」がペルソナ化できる
□ 「月¥30万は現実的か、それとも別の数字か」が判断できる
```

**この 5つが揃ってから、初めて「戦略」を立てる価値がある。**

## 最後に：販売検証を甘く見ないでください

多くのオーナーが同じ罠に陥ります：

```
❌ 計画の罠：
   「このテンプレは絶対売れるはず」という信念で、
   「市場検証」を飛ばして戦略を立ててしまう
   
   結果：月¥30万どころか、売上 ¥0 で終わる

✅ 正しい進め方：
   「こんなテンプレがあったら役立つと思うけど、
    本当に売れるかはわからない」という謙虚さで、
   「小さく売ってみる」を先にやる
   
   結果：「月 5個売れた」という実績が出る
   → その時点で初めて「月¥30万戦略」を立てる価値がある
```

**販売実績ゼロのまま、戦略を立ててはいけません。**

まず売ってください。1個でいいから。

その時点で、すべてが変わります。
