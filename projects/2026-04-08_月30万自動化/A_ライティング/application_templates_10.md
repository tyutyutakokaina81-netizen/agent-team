# A柱：クラウドソーシング応募文テンプレート10選

> 業種別・案件タイプ別の応募文バリエーション。コピペ後、`[ ]`箇所を案件情報に書き換えて使用。

---

## 応募文の鉄則（全テンプレ共通）

1. **冒頭3行で読ませる** → クライアントは1日に数十件読む
2. **「私たち」より「あなたの案件」を主語に**
3. **AI使用は冒頭で明示**（規約遵守・信頼）
4. **実績がなければ「準備の本気度」を示す**
5. **質問1つで終わる**（返信ハードルを下げる）
6. **長さは350〜500字**（短すぎず長すぎず）

## テンプレ7：医療・健康メディア案件（YMYL警戒型）

```
[クライアント名]様

健康・医療領域の案件、興味深く拝見しました。

【お引き受け可能な範囲】
・医師監修記事の構成・骨格作成

・予防・生活習慣など、医療行為に該当しない領域

【お引き受け不可】
・特定の医薬品・サプリメント推奨
・診断・治療を促す内容

監修者様との連携が前提となれば、構成〜執筆をお引き受け可能です。
ご相談承ります。
```

## ボツ案件の見分け方（応募前チェック）

| 兆候 | 対応 |
|------|------|
| 単価が文字単価0.5円未満 | 応募しない（時間の無駄） |
| 「未経験者大歓迎」+ 単価極端に低い | 警戒 |
| 評価0・本人確認なし | 警戒 |
| 大量発注を匂わせて単価交渉NG | 警戒 |
| 著作権買い取り条件が異常 | 確認必須 |
| 連絡が深夜・休日に急かす | 警戒 |
| 案件詳細が極端に少ない | 質問してから応募 |
