# 応募文テンプレ集（A：SEOライティング向け）

クラウドワークス・ランサーズで案件に応募する際の文面テンプレ。  
AI 補助使用を明示しつつ、差別化要素を伝える。

---

## 基本テンプレ（汎用）

```
○○様

はじめまして。SEO記事ライティングを専門に活動しております○○と申します。

「【案件タイトル】」を拝見し、御社にお力添えできると考えご応募いたしました。

【ご提案】
ご要件の「【キーワード or テーマ】」について、
SEO観点を意識した構成設計から本文執筆まで一貫して対応いたします。

【作業フロー】
1. キーワード・ペルソナ・競合記事の確認
2. 構成案（H2/H3）を初稿として提示（承認後に本文執筆）
3. AI補助＋人間レビューで本文執筆
4. 納品前の事実確認・読みやすさ調整

【AI使用について】
Claudeを補助ツールとして使用し、構成設計・初稿生成を行います。
最終的な品質管理・事実確認・トーン調整は人間が行います。
クラウドワークス（ランサーズ）のAI使用ガイドラインに準拠しています。

【お見積り】
・文字数：【ご希望の文字数】字
・料金：【金額】円
・納期：【日数】日（ご相談に応じます）

【実績】
・【過去実績ジャンル】（受注後随時更新）

ご質問・ご要望があればお気軽にお知らせください。
よろしくお願いいたします。

【屋号 / 氏名】
```

## 応募前チェックリスト

応募ボタンを押す前に確認：

- [ ] AI 使用の明示が入っている
- [ ] 規制ジャンル（医療・薬機・金融）でない
- [ ] 納期・料金・文字数が現実的
- [ ] クライアントの過去評価を確認した
- [ ] 「テスト記事無償」を要求されていない（搾取案件の可能性）
- [ ] 応募文に競合他社の名前を出していない

## 応募後フォロー

応募から 24 時間返信がなければ、丁寧にフォロー：

```

もし追加のご質問・ご要望があれば、お気軽にお知らせください。

よろしくお願いいたします。
```

## NG 案件の見分け方

応募 **しない** べき案件：
- 「テスト記事を無償で書いてください」と要求してくる
- 単価が異常に低い（¥500/記事 など）
- クライアントの評価が極端に低い／レビュー 0 件
- 「医薬品の効果」「金融商品の推奨」「実体験風広告」など規制 NG ジャンル
- 「他社サイトをほぼコピーして」など著作権侵害指示
- 連絡先を SNS／メール直接交換に誘導してくる（規約違反）
