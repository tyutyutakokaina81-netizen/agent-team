## 公開・運用フロー

1. オーナーが本ファイルを **Googleドキュメント** にコピーする
2. 各ドキュメントの共有設定を「リンクを知っている全員：閲覧のみ」に設定
3. 共有リンクを `CSO/outputs/2026-05-20_クラウドワークスプロフィール.md` のサンプル欄に記載
4. 応募テンプレ③（初心者歓迎系）の「サンプル記事3本」言及部分でリンクを提示

## カスタマイズ・運用方針

- 本記事はあくまで **初期ポートフォリオ**。実案件を1〜2件こなしたら、
  クライアントの許諾を得た上で実績記事と差し替える
- 本ファイル冒頭のメタ情報（日付・想定読者・想定メディア）は
  Googleドキュメントへコピーする際に削除する（読者向けには不要）
- 「メタディスクリプション」「タイトル」は記事末や別ドキュメントで提示するか、
  記事本体には載せずに納品物の構成として説明用に使う
