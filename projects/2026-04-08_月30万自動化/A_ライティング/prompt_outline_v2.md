# A：SEO構成案プロンプト v2（実例付き）

担当：CMO（プロンプト設計）× CDO（品質）× CSO（フィードバック）
作成日：2026-05-19
バージョン：v2（v1は `order_workflow.md` 内 Step3 に記載）

> v1からの主な改善：
> 1. 検索意図を「Know / Do / Buy / Go」の4分類で明示させる
> 2. 想定読者の「悩み」「ゴール」を別フィールドで分離
> 3. 競合構成の重複回避を明示指示
> 4. メタディスクリプションの構造を3要素（共感→ベネフィット→CTA）に
> 5. 出力前に「自己チェック5項目」を Claude に実行させる

---

## プロンプト本文（v2）

```
あなたは10年以上のSEOライティング経験を持つ編集者です。
以下の条件で日本語のSEO記事の構成案を作成してください。

━━━━━━━━━━━━━━━━━━━━━━
【メインキーワード】
[例：副業 おすすめ]

【サブキーワード（3〜5個）】
[例：副業 在宅／副業 月5万／副業 始め方]

【検索意図の分類】
以下4分類のうち、このKWに最も該当するものを選んでください。
- Know（情報を知りたい）
- Do（具体的にやりたい）
- Buy（買いたい）
- Go（場所に行きたい）

【想定読者ペルソナ】
- 属性：[例：30代会社員・男性・年収400万]
- 抱えている悩み：[例：給料が上がらない・将来不安]
- 達成したいゴール：[例：月5万円の副収入]
- 知識レベル：[初心者／中級者／上級者]

【参考競合URL（最大3本）】
[URL1]
[URL2]
[URL3]
※ 同じ見出し構成は採用せず、独自の切り口を1つ以上加える

【目標文字数】
[例：4,000字 ±10%]

【トーン】
[です・ます調／だ・である調／フランク]

【絶対NG】
[クライアント指定NGワード・触れてはいけない話題]
━━━━━━━━━━━━━━━━━━━━━━

【出力フォーマット】

# 構成案

## 1. タイトル案（3パターン）
- A案：[32字以内・KW前方配置・数字を含む]
- B案：[ベネフィット訴求型]
- C案：[疑問形・好奇心訴求型]

## 2. リード文（150〜200字）
構成：①読者の悩みへの共感 → ②本記事で得られるベネフィット → ③記事の信頼性（誰が書いたか／何を根拠に）

## 3. 見出し構成

| 階層 | 見出し | 想定文字数 | 含めるサブKW | 想定読者の疑問 |
|------|--------|-----------|------------|--------------|
| H2 | (例) | 600字 | (例) | (例) |
| H3 |  |  |  |  |
| H3 |  |  |  |  |
| H2 |  |  |  |  |
| ...|  |  |  |  |

【ルール】
- H2は5〜6本（記事構造として標準的）
- 各H2にH3を2〜3本（4,000字なら）
- 各H2の冒頭に「読者の疑問」を明示
- メインKWはH1・リード・最初のH2に含める
- サブKWは各H2に1つずつ自然に配置
- 競合と完全一致のH2タイトルは避ける

## 4. まとめ（200字）
構成：①記事の要点3つを箇条書き → ②読者への行動提案（CTA）

## 5. メタディスクリプション（120字）
構造：①共感（読者の悩み）→ ②ベネフィット（本記事で得られる答え）→ ③CTA（クリック動機）

## 6. 独自の切り口（競合と差別化する視点）
[例：「副業 おすすめ」の場合、競合は「種類別ランキング」が多いので、
本記事では「平日1時間でできる副業」という時間軸での切り口を採用]

## 7. 内部リンク候補
- このH2セクションから他記事へのリンク提案（任意）

━━━━━━━━━━━━━━━━━━━━━━

【出力前の自己チェック5項目】
最後に以下5点を自分で確認し、問題があれば構成を修正してから出力してください。
- [ ] メインKWがH1・リード・最初のH2に含まれているか
- [ ] サブKWが各H2に1つ以上含まれているか
- [ ] 競合と完全一致するH2タイトルがないか
- [ ] 「読者の疑問」がH2ごとに具体的に書かれているか
- [ ] 独自の切り口が1つ以上明示されているか
```

## 実例：「副業 おすすめ」のテスト入力

```
【メインキーワード】副業 おすすめ
【サブキーワード】副業 在宅／副業 月5万／副業 初心者
【検索意図の分類】Know（情報を知りたい）→ Do（やりたい）への移行型
【想定読者ペルソナ】
- 属性：30代会社員・男女問わず・年収400万
- 抱えている悩み：給料の上昇が見込めず将来不安
- 達成したいゴール：副業で月5万円の収入を作る
- 知識レベル：初心者（副業未経験）
【参考競合URL】（実際のURLを3本入れる）
【目標文字数】4,000字 ±10%
【トーン】です・ます調
【絶対NG】特定の投資案件名／断定的な収入額の約束
```
