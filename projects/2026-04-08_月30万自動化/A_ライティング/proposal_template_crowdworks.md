# A：クラウドソーシング応募文テンプレート

担当：CSO
作成日：2026-04-17

応募ボタン後の「メッセージ欄」に貼付して使用。
案件内容に応じて【】部分をカスタマイズする。

---

## 標準テンプレ（汎用）

```
はじめまして、ライターの○○と申します。
【案件タイトルから1要素】の記事作成案件、興味深く拝見いたしました。

私はAI執筆支援ツール（Claude）を活用し、
通常の2〜3倍の速度で高品質な記事を納品しております。
AI使用については案件ご依頼前に必ずご相談・ご了承を得ております。

【ご提案内容】
・文字数：【ご希望文字数】に対応可能
・納期：【ご希望納期】より1日短縮可能
・修正：1回まで無料対応

【関連実績】
・【ジャンル】記事 ○本（サンプルURL：●●●）
・【ジャンル】記事 ○本（サンプルURL：●●●）

【ご質問】
1. 想定読者・ペルソナのご共有は可能でしょうか
2. 競合記事・参考記事のURLはございますか
3. NGワード・トーンの指定はございますか

ご検討のほど、よろしくお願いいたします。
```

---

## 業種別カスタマイズ例

### SEO記事（中小企業オウンドメディア向け）

差し込み追加：
```
御社の【業界】における集客課題を理解した上で、
検索意図を満たしつつコンバージョンに繋がる構成をご提案いたします。
```

### 専門ジャンル（金融・医療・法律など）

差し込み追加：
```
専門ジャンルの記事は、信頼できる一次情報（公的機関・公式サイト）を
必ず参照し、引用元を明記いたします。
監修者がいる場合はその指示に沿って執筆いたします。
```

### コラム・ブログ系（読み物）

差し込み追加：
```
読み手が「最後まで読みたくなる」構成・文体を意識し、
SEOと読みやすさの両立を心がけております。
```

## NGパターン（応募してはいけない案件）

- AI使用を明確に禁止している案件
- 単価 ¥3円/字 未満（時給換算で割に合わない）
- 検収条件が曖昧（「気に入らなければ報酬なし」など）
- 著作権を完全譲渡しない条件で買い叩く案件

---

## 応募後のフロー

1. 応募 → スプレッドシートに記録（日付・案件名・単価・応募文URL）
2. 24時間以内に返信なし → 別案件に注力（追いかけない）
3. 採用通知 → 1時間以内に契約承諾＋ヒアリング送付
4. 不採用通知 → 「またの機会によろしくお願いします」と返信のみ
