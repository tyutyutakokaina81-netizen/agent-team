# SEOライティング：案件評価チェックリスト

**用途**：クラウドワークス・ランサーズで案件を見つけたとき、応募する前に「GO/CAUTION/NO-GO」を3秒で判断するための基準。
**運用**：時間ロスと地雷案件を避けるための最重要ツール。
**費用**：¥0（既存ブラウザ・既存スプレッドシート内で完結）

---

# 評価フロー

```
1. 案件タイトルを読む（5秒）
   ↓
2. 募集詳細を読む（30秒）
   ↓
3. 案件主のプロフィールを開く（15秒）
   ↓
4. 下のチェックリストで採点
   ↓
5. 80点以上なら応募／40〜80点はCAUTION／40点未満はNO-GO
```

総所要時間：1案件あたり 1分以内。

---

# スコアリング基準（100点満点）

## A. 単価評価（30点）

| 単価 | 点数 |
|------|------|
| 1文字 1円以上（または 4,000字 4,000円以上） | 30点 |
| 1文字 0.7〜1円 | 20点 |
| 1文字 0.5〜0.7円 | 10点 |
| 1文字 0.3〜0.5円 | 5点 |
| 1文字 0.3円未満 | **0点（即NO-GO）** |

## B. 案件主の信頼度（25点）

| 項目 | 点数 |
|------|------|
| プロフィール認証あり（本人確認・電話） | +10 |
| 過去発注実績10件以上 | +10 |
| 過去発注の評価★4.5以上 | +5 |
| プロフィール画像あり | +3 |
| 本人確認なし | -10 |
| 過去発注ゼロ（新規案件主） | -5 |

## C. 案件内容の明瞭さ（20点）

| 項目 | 点数 |
|------|------|
| キーワード・文字数・納期が明記されている | +10 |
| 構成指示・参考記事URLが提示されている | +5 |
| 検収基準・修正回数が明記されている | +3 |
| 著作権・継続発注の方針が明記されている | +2 |
| 内容が曖昧（「いい感じに書いて」等） | -10 |

## D. 案件主のコミュニケーション姿勢（15点）

| 項目 | 点数 |
|------|------|
| 質問への返信が早い／返信実績が見える | +10 |
| 募集文の言葉遣いが丁寧 | +5 |
| 急かす表現・脅し表現がある（「即対応」「他にも候補多数」等） | -10 |

## E. 案件規模・継続性（10点）

| 項目 | 点数 |
|------|------|
| 継続発注の可能性がある（または明記） | +10 |
| 1記事だけ・テスト発注 | +3 |
| 100記事一括の大量発注（品質基準が緩い兆候） | -5 |

# 即 NO-GO の地雷案件パターン

下記のいずれか1つでも該当すれば、応募しない（点数不問）。

```
□ 1文字 0.3 円未満の超低単価
□ 「テスト記事を無料で書いて」要求がある
□ 案件主の過去評価が ★3.5 未満
□ 募集文が異常に短い（10行未満で詳細不明）
□ 「初心者歓迎」かつ「単価1文字 0.5 円未満」
□ 「文字数指定なし」「納期1日」「修正回数無制限」のいずれか
□ 連絡先が SNS DM や LINE 等に誘導される（プラットフォーム外取引の兆候）
□ 機密保持契約（NDA）なしで詳細を全公開してくる
□ 案件主の他案件にトラブル評価が複数ある
□ アダルト・違法・誇大広告・薬機法違反の可能性
```

---

# 即 GO の優良案件パターン

下記の条件をすべて満たす案件は、即応募する（30秒以内）。

```
✅ 1文字 1円以上
✅ 案件主が認証済みで過去評価★4.5以上
✅ キーワード・文字数・納期が明記
✅ 構成指示または参考記事URLあり
✅ 継続発注の可能性が記載
```

このパターンの案件は、応募から24時間以内で発注決定するケースが多い。
午前9〜11時の応募が採用率トップ。

---

# CAUTION 案件への対応方針

40〜80点の「微妙な案件」は、応募時に **質問で1点だけ確認** すると地雷を回避できる。

## CAUTION 案件への質問例

```
■ 単価が曖昧な場合
「文字単価について確認させてください。
本案件は1文字 X 円のご認識でよろしいでしょうか？」

■ 構成指示がない場合
「執筆前に簡易な構成案を別途ご提案させていただいてよろしいでしょうか？
方向性のすり合わせを早めにできれば、修正回数を減らせます」

■ 修正回数が不明な場合
「納品後の修正対応について、目安となる回数・範囲を
事前にお聞かせいただけますと幸いです」

■ 継続発注の可能性
「ご好評いただけた場合、継続的にご依頼いただける可能性は
ございますでしょうか？」
```

質問への返答の早さ・丁寧さで、案件主の質を見極める。
24時間以内に丁寧な返信があれば応募、なければ別案件へ。
