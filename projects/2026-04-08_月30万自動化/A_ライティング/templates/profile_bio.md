```
ご覧いただきありがとうございます。
SEOライティングを承っております○○と申します。

【提供できる価値】
✅ AI執筆支援を活用した「高速」かつ「人間品質」の記事作成
✅ キーワード選定〜構成〜本文〜メタディスクリプションまで一貫対応
✅ 納品後1回までの修正は無料

【対応可能ジャンル】
- フリーランス・働き方・副業
- 美容・コスメ・スキンケア
- 飲食・グルメ・レシピ
- 不動産・住まい
- 金融・家計・節約
- 医療・健康・サプリ（一次情報の引用は専門家監修推奨）
- IT・ガジェット・ソフトウェア

【記事構成のスタンダード】
- 文字数：2,000〜8,000字
- 構造：H1/H2/H3 階層、比較表、箇条書き、まとめ
- メタディスクリプション、サジェストキーワード対応

【料金プラン】
- ライト    2,000字   8,000円  納期2日
- スタンダード 4,000字  15,000円 納期3日

※継続契約・大量発注はご相談で割引対応します

1. キーワード・構成指示を確認（必要に応じて競合分析も対応）
2. AI 執筆支援でドラフト作成
3. 人間レビューで事実確認・トーン調整・文章リライト
4. メタディスクリプション・サジェスト対応
5. Word / Google Docs / テキストで納品

【AI使用について】
AI 執筆支援を使用していることを、応募時・契約時に必ずお伝えします。
出力された文章は **必ず人間がレビュー・事実確認・調整** を行います。
クライアント様のトーン・ブランドに合わせる工夫も併せて行います。

【強み】
- 1日あたり最大 4,000〜6,000字の執筆対応が可能
- 急ぎ案件（24時間納品）にも対応実績あり（要相談）
- 構成案 + 本文セットでの提案が得意

【NGとさせていただく案件】
- 違法性のある内容、誇大広告、虚偽表記
- 著作権を侵害する転載作業
- 1文字 0.3 円未満の超低単価案件

【ポートフォリオ】
サンプル記事を Google Docs で共有いたします。
応募時にメッセージでお伝えください。

何卒よろしくお願いいたします。
```
