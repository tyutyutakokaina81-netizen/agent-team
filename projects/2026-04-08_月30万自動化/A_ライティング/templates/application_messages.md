# SEOライティング：応募メッセージテンプレ集（業種別6種）

**用途**：クラウドワークス・ランサーズの個別案件への応募メッセージ。業種に合わせて選び、[ ]部分のみ書き換えて使う。
**運用方針**：1日5〜10件応募 × 平日のみ。応募から24時間以内に発注決定する案件主が多いため、午前中の応募を優先する。
**費用**：¥0（既存ツール内で完結）

---

## 共通の応募メッセージ構造（全テンプレに適用）

```
1. 冒頭の挨拶（1行）
2. 案件への共感・読み込みアピール（2〜3行）
3. 提供できる価値（箇条書き3つ）
4. 強み・実績（1〜2行）
5. 質問または確認事項（任意・1行）
6. 締めの挨拶（1行）
```

文量目安：300〜500字（長すぎず、短すぎず）。

# 共通：差別化フレーズ集

応募メッセージに1〜2フレーズ織り交ぜると、テンプレ感が薄まる。

| 場面 | フレーズ例 |
|------|----------|
| AI使用の透明性 | 「AI執筆支援を使用しますが、必ず人間がレビュー・事実確認を行います」 |
| 納期へのコミット | 「納期は厳守、トラブル時は事前にご連絡を徹底いたします」 |
| 修正対応 | 「納品後1回までの修正は無料で承ります」 |
| 数値根拠 | 「主張には必ず一次情報の出典を添えます」 |
| 構成提案 | 「ご希望に応じて、執筆前に構成案を別途ご提案できます」 |
| 急ぎ対応 | 「お急ぎ案件にも対応した実績がございます（要相談）」 |
| 大量発注 | 「継続・大量発注の場合は、別途お見積りでご対応します」 |

# 応募運用ルール

## 応募ペース

| 期間 | 1日あたりの応募数 | 累計目標 |
|------|-----------------|---------|
| Week 1 | 5件 | 25件 |
| Week 2 | 8件 | 65件 |
| Week 3 | 10件 | 115件 |
| Week 4 | 10件 | 165件 |

## 応募タイミング（重要）

- **平日 9〜11時** が最も発注決定が早い時間帯（案件主が朝のメールチェックをする時間）
- 土日応募は反応率が下がる（避ける）
- 案件公開から **6時間以内の応募** が採用率トップ

## 応募メッセージのカスタマイズ範囲

最低限カスタマイズすべき箇所：
1. 案件主名（[案件主名]）
2. 案件タイトル抜粋（[案件タイトル]）
3. 案件主の媒体・サイト URL に触れる1行（あれば）

それ以上のカスタマイズは時間効率が悪い。テンプレベースで6種類を使い分け。
