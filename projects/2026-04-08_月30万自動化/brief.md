# プロジェクト：月30万自動化 ／ 自立型AI構築月への方針転換

## 現在のミッション（2026-04-11 更新）

**「今月勝つ」を捨て、「勝てる時代が来た瞬間に、既に勝てる状態で待機している」**
を4月の目標にする。

### なぜ方針転換したか

- オーナー条件：完全自立型AI／リスクゼロ／作業ゼロ／新規口座ゼロ
- 2026年4月時点の現実：決済アカウント・本人確認・外部プラットフォーム ToS の壁により、
  上記全条件を同時に満たしたままの収益化は物理的・法的に不可能
- したがって4月は **収益月ではなく基盤構築月** と位置づけ直す

## リスク封じ込め（ハード制約）

| カテゴリ | 制限 |
|---------|------|
| 新規金銭支出 | Claude API のみ、上限 ¥2,000/月（Haiku 中心・batch API 活用） |
| 新規口座開設 | **一切行わない** |
| 外部サービスへの自動投稿・応募 | **禁止**（ToS リスク回避） |
| 顧客との直接接触 | **禁止** |
| 本人情報の露出 | **ゼロ**（リポジトリ内部で完結） |
| 暴走防止 | 1日あたりの Claude 呼び出し回数／トークン／コミット数に hard limit |
