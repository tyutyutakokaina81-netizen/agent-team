# 反証ノート：動画「AI社員で月20万外注ゼロ」を盲信しないために

## 目的
本日5役職分の知恵帳に動画から得た学びを記録した。各役職が「**動画の主張は当社では崩れる可能性**」を1〜2件ずつ能動的に書き、試作の歯止め判断を早くする。

## 運用
- 反証は「予言」ではなく「仮説」。実機検証で覆ってもよい
- 反証が当たった場合は知恵帳に「学び」として昇格させる
- 楽観だけに偏らないための定期セルフチェックとして、月次で見直す

参考：`context/references/2026-05-13_YouTube_AI社員で月20万外注ゼロ.md`

## CDO（技術・自動化）

### 反証1：「Claude Code = AI社員」のメタファーは過大評価
- **動画の主張**: Claude Code を起動すれば「常駐スタッフ」のように働く
- **当社で崩れる点**: Claude Code はセッション単位のためコンテキストが永続化しない。中断・再起動時に取引先・単価感を再学習させる必要あり。本物の「社員」とは違う
- **対策**: 「AI社員」ではなく「即起動できる外注ワーカー」と内部呼称する。コンテキスト永続化の仕組み（プロンプトテンプレ常備）を必須化

### 反証2：gemini-cli/MCP 連携は環境依存が大きい
- **動画の主張**: gemini-cli（GOG）で Gmail を自由操作
- **当社で崩れる点**: 動画は Mac 前提の可能性。OS・Node/Python 版・OAuth スコープの組み合わせで動かないケース多発の可能性
- **対策**: 公式 MCP が動かなければ自作シェル（候補C）に即切り替え。1日試して動かなければ撤退

## CFO（財務・事務）

### 反証1：「請求書AI化で月3万削減」は当社には効かない
- **動画の主張**: 事務代行を AI に置き換えて月3万削減
- **当社で崩れる点**: 当社は事務代行を使っていない（オーナー自前）。削減効果は0、むしろ AI 構築工数（数時間〜数日）が**純増コスト**
- **対策**: コスト削減目的では試作しない。「時間削減」（オーナーの請求書発行時間 30分→3分）を KPI に置き換える

### 反証2：AI 生成請求書はインボイス税務リスクを孕む
- **動画の主張**: インボイス番号・税率を自動採番
- **当社で崩れる点**: 税率・登録番号の誤りは税務調査対象。過去分の遡及課税・延滞税リスク
- **対策**: 発行前 CFO 目視チェックを**必須化**（チェックリスト5項目を遵守）。AI 単独完結させない

## CMO（マーケ・SNS）

### 反証1：「Threads 29万再生」は再現性に疑問
- **動画の主張**: 1ヶ月で Threads 29万再生
- **当社で崩れる点**: 杉田氏のフォロワー基盤・業界内既存著名度・タイミング運が前提の可能性。当社（フォロワー基盤なし）が同ワークフローで再現できる保証なし
- **対策**: 当社初動は「再生数」ではなく「投稿継続率（週N回）」を指標に。再生数は副次指標

### 反証2：媒体側仕様変更で半年内に陳腐化
- **動画の主張**: マルチメディア戦略は強力
- **当社で崩れる点**: X有料化・Threads/Instagram の API 制限・各社ガイドライン変更（自動投稿規制）で半年内に陳腐化する可能性
- **対策**: 3媒体集中＋手動投稿の比率を一定維持。API 全面依存のオペレーションは構築しない

## CPO（プロダクト・教材）

### 反証1：「AI社員化スターターキット」は積読率が高い
- **動画の主張**: 教材＋プロンプト集は需要が高い
- **当社で崩れる点**: 教材ビジネスの宿命として購入者の70-80%は「買って満足」で終わる。Vol.5 の販売数=実使用数にはならない
- **対策**: 価格設定で「使い始めないと損」と思わせる構造（早期実践特典・1週間後フォローメール等）を組み込む

### 反証2：AI トレンドは半年で陳腐化
- **動画の主張**: Claude Code は2025年12月から業務レベル
- **当社で崩れる点**: 3ヶ月後にはより新しい AI が話題化し、Vol.5 のメッセージが古く見える可能性
- **対策**: 「2026年版」と版数を明示。半年ごとに改訂版を出す前提で初版を軽量化（フル投資しない）

## CSO（営業・顧客対話）

### 反証1：「年234万削減」訴求はターゲット層にミスマッチ
- **動画の主張**: 年234万削減は強力なフック
- **当社で崩れる点**: 当社ターゲット（中小事業者・1人社長）の外注費は月数千〜数万円規模。234万という数字は現実離れして響かない可能性
- **対策**: 当社版は「月3万→月0円（年36万削減）」など現実的レンジに**翻訳**して訴求。CSO 営業フック v2 を確認・調整

### 反証2：「AI社員」というメタファーは初回商談で警戒される
- **動画の主張**: AI社員という擬人化が顧客説明に有効
- **当社で崩れる点**: 「AI に任せると品質が落ちる」という偏見を持つ顧客層が一定数存在。初回商談で出ると深掘りされづらい
- **対策**: 初回商談では「AI社員」を出さず「効率化された運用」と表現。2回目以降の信頼関係が出来てから AI 文脈を開示する2段階戦術

## 横断的な気付き

【事実】5役職とも、動画の主張は「当社固有の前提」と擦り合わせると弱まる箇所がある。

【考察】動画は **発信者（杉田氏）の事業構造・顧客層・ブランド基盤** を前提にしている。同じワークフローでも当社が再現できる保証はない。

【提案】試作段階の今期は「動画の真似」ではなく「**動画から要素を抽出して当社用に再設計する**」スタンスを徹底する。各試作メモには「動画ではこうだが当社では」の対比セクションを v2 以降で追加することを推奨。
