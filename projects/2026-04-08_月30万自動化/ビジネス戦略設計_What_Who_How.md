# ビジネス戦略の再設計：What・Who・How の明確化

**作成日**: 2026-06-22  
**目的**: 「何を売るのか」「誰に売るのか」「どう売るのか」を根本から定義し直す

---

## 現在の問題点

| 要素 | 現状 | 問題 |
|------|------|------|
| **What（何を売るか）** | テンプレ＋記事＋プロンプトをセット販売 | セットにする理由が不明。単品の方が売れるのでは？ |
| **Who（誰に売るか）** | 「ひとりで会社やる人」（約100万人） | 範囲が広すぎる。具体的なペルソナが不明 |
| **How（どう売るか）** | noteメンバーシップのみ | 他の販路（広告・SNS・メルマガ）がない。露出ゼロ |
| **Marketing（マーケティング）** | なし | 告知戦略・ターゲティング・クリエイティブなし |

---

## ステップ1：「What」を定義し直す

### 仮説1：テンプレ単品が最強では？

**現在の構成**：
```
月¥980メンバーシップ
├─ 毎週の深掘り記事（読む）
├─ テンプレ配布（使う）
└─ プロンプト公開（実行）
```

**問題**：
- 「毎週記事を読む」習慣がある人は少数派
- 実は「テンプレが欲しい」だけかもしれない
- 記事の執筆負荷が続かない可能性が高い

**代替案1A：テンプレを単品で売る**
```
BOOTH・Gumroadで：
- 「請求書プロンプト完全版」：¥980（単品・買い切り）
- 「売上目標スプレッドシート」：¥980（単品・買い切り）
- 「単価交渉メール30パターン」：¥1,480（単品）
- バンドル：「3点セット」¥2,480（割引）
```

**メリット**：
- 毎月の執筆圧力なし
- 顧客が「好きな時に買える」（気軽）
- 1個売れれば利益確定（継続率の心配なし）

**期待売上**：月10-30個 × ¥980 = 月¥10-30万円

---

### 仮説2：記事こそが本体では？

**代替案1B：有料noteコース（テンプレなし）**
```
無料記事：「売上逆算の概要」「請求書の手順」
↓
有料記事（単発）：「実装テンプレ付き完全版」¥1,500～¥2,000
```

**メリット**：
- 「記事ごとに単発購入」なので、継続率の圧力がない
- 読者が「今月は買う、来月は買わない」と自由に選べる
- 単発記事の方が実装されやすい（「今月の課題」に特化）

**期待売上**：月8-12記事 × 平均¥1,500 × 20%購入率 = 月¥24-36万円

---

### 仮説3：コース化（Udemy等）が正解では？

**代替案1C：オンラインコース化**
```
「ひとりで会社をやる人の、数字・判断・AI活用を8週間で身につけるコース」
- 動画8本（各15-30分）
- テンプレ・プロンプト配布
- 売上：¥4,980～¥9,980（買い切り）
```

**メリット**：
- 高単価（¥5,000以上）で少数販売可
- 「完成度の高い教材」なので、一度作ったら何度も売れる
- 継続率の概念がない

**期待売上**：月3-5名 × ¥5,000 = 月¥15-25万円

## ステップ2：「Who」を定義し直す

### 現在のターゲット：曖昧

**「ひとりで会社やる人」の内訳**：
- フリーランス（エンジニア・ライター・デザイナー）
- 個人事業主（コンサル・コーチ・美容師）
- 小規模企業経営者（年商¥500万～¥5,000万）
- 会社員の副業層
- etc...

**問題**：範囲が広すぎて、マーケティングの打ち手が定まらない

---

### ターゲット再定義：3案

#### **案2A：個人コンサル・コーチ層（推奨）**

**ペルソナ**：
- 年代：30-50才代
- 年商：¥500万～¥3,000万
- 課題：「売上は安定しているが、数字管理と単価交渉に悩み」
- ITリテラシー：中程度（スプレッドシート・AI基本的な使い方はできる）
- 購買力：月¥2,000-5,000のコンテンツに課金可能

**理由**：
- 数字・判断が直結する職種
- テンプレ・プロンプトの必要性が高い
- 月¥980が「安い」と感じるレベルの購買力がある
- noteにこのターゲットが多い可能性

**マーケティング経路**：
- Twitter/X で「コンサル・コーチ向け」の発信
- note の「おすすめ」アルゴリズムに乗りやすい

---

#### **案2B：小規模企業の管理職（月商¥1,000万～¥5,000万）**

**ペルソナ**：
- 年代：40-55才代
- 職種：事業部長・営業管理職・CFO級
- 課題：「現場数字の見方」「部下への給与・時給交渉」「外注判断」
- 購買力：月¥3,000-10,000のコンテンツに課金可能
- ITリテラシー：高い（Excel・Google Sheets 自在）

**理由**：
- B2B市場は「継続率が高い」（組織継続性）
- 単価を2-3倍に引き上げられる
- テンプレは「組織資産」として評価される

**マーケティング経路**：
- LinkedIn（法人ターゲティング）
- 業界紙・業界メディア
- 法人向け広告（Google Ads・Facebook Ads）

---

#### **案2C：飲食店・美容院などの現場経営者**

**ペルソナ**：
- 年代：25-45才代
- 課題：「従業員の時給計算」「原価管理」「現金管理」
- 購買力：月¥500-1,500（低価格帯）
- ITリテラシー：低い（スマホメインの操作）

**理由**：
- 数字が経営に直結する業種
- テンプレのニーズが高い
- 月¥500単品販売の方が合致

**マーケティング経路**：
- Instagram（飲食・美容向け）
- LINE公式アカウント（店舗向け）
- 業界団体・商工会議所との提携

---

## ステップ3：「How」を定義し直す

### 現在：noteメンバーシップのみ

**問題**：
- noteは「既に認知している読者」にしかリーチしない
- 新規顧客獲得ができていない
- 広告・PR・SNSを使わない = 露出ゼロ

---

### 販路の再設計

#### **販路1：単品販売（低リスク・即実装可）**

```
BOOTH で「テンプレ単品」販売
├─ 請求書プロンプト：¥980
├─ スプレッドシート：¥980
├─ メール交渉文：¥1,480
└─ 3点バンドル：¥2,480

Gumroad でも同時販売
```

**利点**：
- セットアップが簡単（既存テンプレを流用）
- 即実装可能（今週中）
- 毎月の執筆圧力なし

**期待**：月10-30個売上

---

#### **販路2：noteメンバーシップ（高継続率狙い）**

```
現状：月¥980（1プラン）
改善：複数プラン化

梅 ¥500：テンプレ配信のみ
竹 ¥1,980：テンプレ＋解説記事＋月末相談（30分）
松 ¥4,980：竹＋個別コンサル2回/月
```

**条件**：
- 「権威性」を先に確立してから
- BOOTHで月¥10-20万円の単品売上が出た後に

**期待**：月3-10人の登録

---

#### **販路3：SNS発信経由（新規顧客獲得）**

```
Twitter/X：
- 「個人事業主の数字管理の失敗事例」を毎日発信
- 「テンプレを使うと3分で終わる」の価値を見せる
- BOOTHリンクを貼る

LinkedIn（B2B向け）：
- 「管理職向けの原価・時給計算」の具体例
- 企業研修・コンサルへの導線

Instagram（飲食・美容向け）：
- ビジュアル化されたテンプレ活用例
- Before/After（手計算 vs テンプレ）
```

**期待**：月50-200PV → 月2-5件の購入

---

#### **販路4：有料広告（テスト運用）**

```
Google Ads：
- キーワード：「請求書 テンプレ」「Excel 売上管理」
- ターゲット：¥980単品購入者（低CPA）
- 予算：月¥5,000-10,000でテスト

Facebook Ads（B2C向け）：
- ターゲット：個人コンサル・コーチ（興味関心）
- 予算：月¥5,000-10,000でテスト

LinkedIn（B2B向け）：
- ターゲット：管理職・事業部長
- 単価高い（CPA ¥3,000-5,000）
```

**期待**：CPA ¥500-2,000、ROAS 200-300% で月¥10-30万円

## ステップ4：「What to Sell」の仮説検証プラン

### 3週間の検証フェーズ

#### **Week 1：単品テスト（テンプレのみ）**

```
Day 1-3：BOOTH に「請求書プロンプト」¥980 を出品
         → 広告なし、無料読者頼み
Day 4-7：結果を計測
         - 売上 0本？ 1本？ 5本？
         - クリック数 / コンバージョン率
```

**判定基準**：
- 3本以上 → 単品販売の可能性あり
- 0本 → 権威性不足 or ターゲットが曖昧

---

#### **Week 2：記事テスト（有料noteの単発化）**

```
Day 8-10：有料記事「売上逆算テンプレ解説」¥1,500 を試験公開
          無料読者に告知
Day 11-14：購入数を計測
          - 購入者数
          - 記事の滞在時間
```

**判定基準**：
- 5人以上購入 → 記事販売のポテンシャルあり
- 0人 → メンバーシップ継続の必要性が疑問

---

#### **Week 3：Twitter発信テスト（認知）**

```
Day 15-21：Twitter で「個人事業主の数字管理」テーマで毎日発信
          テンプレへのリンク付き
          
計測：
- リーチ数 / エンゲージメント
- クリック数
- 購入への転換
```

**判定基準**：
- 1日平均 500リーチ、2-3クリック/日 → Twitter の効果あり
- 100リーチ未満 → フォロワー数・発信力の不足

```
  → SNS 発信の効果あり

```

## ステップ6：何をやめるべきか

### 「メンバーシップ」の継続判断

**メンバーシップを続けるには、以下が必須**：

- [ ] noteの無料読者が **月1,000PV以上** ある
- [ ] 実行フェーズで **月3人以上の登録** が出た
- [ ] **毎週の記事執筆を12ヶ月継続** できる体力がある
- [ ] Twitter・LinkedIn などで **月500フォロワー以上** に成長する見込みがある

**これらが揃わないなら**：
- メンバーシップは一旦終了
- 単品販売（BOOTH・Gumroad）に切り替え
- SNS で「テンプレの使い方」を無料発信
- 需要が見えてから「再開」を判断

## 最終判断：3つの選択肢

### **選択肢A：メンバーシップを本気で続ける**
- 条件：上記の「検証データ」で月3人以上確保できたら
- アクション：複数プラン化（梅¥500・竹¥1,980・松¥4,980）
- 覚悟：年52本の記事執筆が必須

### **選択肢B：単品販売に転換**
- 条件：Week 1-3 のテストで単品が3本以上売れたら
- アクション：BOOTH・Gumroad を本格展開
- 利点：毎月の執筆圧力ゼロ

### **選択肢C：ハイブリッド（推奨）**
- 単品販売を主体（月¥10-30万円）
- メンバーシップは高額プラン（¥4,980・相談付き）で少数運用
- 無料SNS発信で認知を広げる

## 最後に：「売りたいもの」の定義

オーナーが本当に売りたいのは、おそらく：

> **「ひとりで事業をやる人が、判断を数字で下せるようになるための、実用ツールとノウハウ」**

これは：
- テンプレ単品でいいのか？
- 毎週1本の記事セットが必須なのか？
- 個別相談も含めるべきなのか？

その答えは、**データが教えてくれます**。

テストを回して、買い手の行動を見ましょう。
