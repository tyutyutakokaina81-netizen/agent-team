# 応募文テンプレ集（D：データ入力・スクレイピング向け）

クラウドワークス・ランサーズで案件に応募する際の文面テンプレ。  
規約遵守・AI 補助・人間チェックを軸に差別化する。

---

## 基本テンプレ（スクレイピング）

```
○○様

はじめまして。データ収集・スクレイピングを専門にしている○○と申します。

「【案件タイトル】」について、対応可能と考えご応募いたしました。

【ご提案・進め方】
1. 対象サイトの利用規約・robots.txtを事前確認
2. Pythonでデータ取得、AI補助でクレンジング
3. 納品前に件数・フォーマット・欠損を人間が確認
4. Excel／CSVで納品

【確認させていただきたい点】
・対象サイト：【サイト名・URL】
・取得項目：【店舗名／住所／電話／公開情報のみ等】
・件数：【〇〇件】
・納期：【希望日】
・出力形式：【Excel／CSV／JSON】

【お見積り】
・料金：【金額】円（規約確認結果により変動の可能性あり）
・納期：【日数】日（ご相談に応じます）

【AI使用について】
本案件はAI（Claude）を補助ツールとして使用します。
最終的な品質確認は人間が行い、
クラウドワークス（ランサーズ）のAI使用ガイドラインに準拠しています。

【お受けできない場合】
・対象サイトがスクレイピングを規約で禁止している場合
・ログイン認証の突破を要する場合
・個人情報を含み、利用目的が不明な場合

事前に対象サイトの規約を確認のうえお見積り確定いたします。
よろしくお願いいたします。
```

## 応募前チェックリスト

応募ボタンを押す前に確認：

- [ ] 対象サイトの利用規約に明示的な「スクレイピング禁止」記述がない
- [ ] ログイン突破・パラメータ改竄を要求していない
- [ ] 個人情報の収集を含む場合、利用目的が明示されている
- [ ] 編集著作物サイト（食べログ等）の全件取得ではない
- [ ] AI使用の明示が応募文に入っている
- [ ] クライアントの過去評価が良好（評価0件の場合は要警戒）
- [ ] 単価が時給換算で適正（¥2,000/h 以上）

## 受注辞退のテンプレ

法務リスクが判明した場合：

```
○○様

ご相談ありがとうございます。
詳細をお伺いし、検討させていただきました。

恐れ入りますが、本案件は以下の理由により当方では
お引き受けが難しい状況です。

【理由】
・対象サイトの利用規約上、自動収集が制限されているため
・取得項目に個人情報が含まれており、利用目的の確認が必要なため
・対象サイトの編集著作物性が高く、大量取得にリスクが想定されるため
（該当する理由のみ記載）

【代替案】
・公式API利用（提供されている場合）
・対象サイト運営者への直接問い合わせ
・取得範囲を限定する形での再見積り

ご希望に沿えず申し訳ございません。
別の案件でお力添えできる機会がございましたら、
お気軽にお声がけください。
```

## NG 案件の見分け方

応募 **しない** べき案件：
- ログイン必須サイトのスクレイピング
- 食べログ・ぐるなび・タウンページ等の全件複製
- 個人情報の収集を含むが利用目的が不明
- 「副業」「初心者歓迎」「LINE登録」など詐欺の臭い
- 受け取り・送金・転送系の案件
- クライアントの評価 0 件かつ募集額が異常に高い
