# 柱D：エクセル入力・Webスクレイピング自動受注パイプライン

## 概要

クラウドソーシングでデータ入力・スクレイピング案件を受注し、
Claude が実行・念査・納品まで一気通貫で自動化する。

## ゴール

- 月収目標：¥100K〜¥300K
- 完全自動（人手介入ゼロ）
- 初期コスト：Claude API費用のみ

## パイプライン

```
① 案件検索  →  ② 評価・選別  →  ③ 応募文生成
→  ④ 作業実行  →  ⑤ 念査  →  ⑥ 納品
```

## フォルダ構成

```
D_エクセル入力スクレイピング/
├── brief.md               ← このファイル
├── pipeline/              ← 自動化スクリプト群
│   ├── 01_search.py       ← 案件検索
│   ├── 02_evaluate.py     ← 案件評価・選別
│   ├── 03_apply.py        ← 応募文生成（API利用 / シングルプロンプト）
│   ├── 03b_sansimai_apply.py ← 応募文生成（三姉妹AI統合：りん/しおり/あかね）
│   ├── 04_execute.py      ← 作業実行（Excel/スクレイピング）
│   ├── 05_review.py       ← 念査
│   ├── 06_deliver.py      ← 納品
│   └── inputs/            ← 03bへの手動コピペ案件入力（jobs.md は gitignore対象）
├── templates/             ← 応募文テンプレート・プロンプト
└── outputs/               ← 納品前成果物（gitignore対象）
```

## 関与役職

- CDO：パイプライン設計・実装
- CSO：受注戦略・顧客対応文面

## 注意事項

- 各プラットフォームの規約確認が必須（自動応募の可否）
- 規約NGの場合は「半自動」（Claude実行・人手で送信）に変更
