✅ 問い合わせは減った（[件数]→[件数]）

## Googleスプレッドシート 実装仕様

### Sheet 構造

```
A列: 週 (Week 1-4)
B列: 月曜日（テンプレ A1 の文章）
C列: 火曜日（テンプレ B1 の文章）
D列: 水曜日（テンプレ C1 の文章）
E列: 木曜日（テンプレ A2 の文章）
F列: 金曜日（テンプレ C2 の文章）
G列: 土曜日（テンプレ B2 の文章）
H列: 日曜日（テンプレ まとめ投稿）
```

### セルフォーマット

```
1. テキストのみ（修正ポイント [〇〇] をハイライト）
2. セルの高さ：テンプレ投稿文（150-200 文字）に合わせる
3. 背景色：テーマ別に色分け
   - 失敗ログ系：赤
   - 実績系：緑
   - Tips系：青
   - 質問系：黄
   - ニュース系：紫
   - プロモ系：オレンジ
```

### テンプレ参照方法

```
Googleスプレッド内に「テンプレ参照」シートを作成
→ 各セルのテンプレ番号（A1, B1, C1 等）をクリック
→ 「テンプレ参照」シートで詳細文を確認
→ 修正点を入れてコピー＆ペースト
```
