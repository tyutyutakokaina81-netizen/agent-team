# テンプレVol.2：SNS Content Calendar
## 仕様書 + 制作ロードマップ

> **目標**: 月間 SNS 投稿を自動化するテンプレ。毎月 30 分で企画・カレンダー完成。

**販売開始**: 2026-06-30（来週）  
**プロダクト価格**: ¥2,980  
**販売チャネル**: note / BOOTH  
**月収目標**: ¥20K（初月目安）  
**優先度**: 🔴 Stage 1 最優先（+12点）  

---

## コンセプト

**「毎日『何を投稿するか悩む時間』を、ここで一括決定する」**

Solo CEO OS（月商管理）と組み合わせると：
- 売上目標が決まった
- → それを達成するための集客戦略が必要
- → Instagram / X / note の「何を投稿するか」を決めるテンプレ

**ユーザー像**: フリーランス・ひとり起業家（Solo CEO OS Vol.1 購入者）

## Sheet 仕様：SNS Content Calendar

### レイアウト

```
        月        火        水        木        金        土        日
Week1  テーマA   テーマB   Tips1    失敗A    Tips2    実績1   まとめ
Week2  テーマC   テーマA   Tips3    失敗B    質問1    実績2   まとめ
Week3  テーマD   テーマB   Tips4    失敗C    質問2    実績3   まとめ
Week4  テーマE   テーマC   Tips1    失敗A    実績4    質問3   まとめ

備考:
- 網掛けセル = テンプレ集から「そのまま貼れる投稿文」
- 薄い背景 = カスタマイズ必須（[自分の実績]などを入れる）
```

### 提供データ

1. **カラーコード**：RGB で投稿テーマ別に色分け
   - 青: ノウハウ系
   - 赤: 失敗ログ系
   - 緑: 実績・事例系
   - 黄: プロモーション系
   - グレー: まとめ・メタ投稿

2. **フォント**：Noto Sans JP（デフォルト）

3. **セルサイズ**：テンプレ投稿文（最大200文字）に合わせた高さ

### 自動化機能（オプション）

```
IF: [売上目標] = [条件]
THEN: [推奨テーマ] = [カテゴリ]
```

※ 初期版は手選択。バージョン2で自動化予定

## 販売ページ コピー案（note 商品説明）

### タイトル
```
SNS Content Calendar — 毎月の投稿を 30 分で企画する
```

### 小見出し
```
Instagram / X / note の「何を投稿するか」で迷わない。
毎月の投稿スケジュールを、30 分で自動決定するテンプレ。
```

### 本文

```
こんにちは。つつです。

フリーランスになると、毎日が営業になります。

Instagramで何を投稿するか、Xで誰に話しかけるか、noteでどんなネタを書くか。
いちいち考えていたら、本来の仕事の時間が消えてしまいます。

だから、売上目標が決まったら、その目標を達成するために「今月は何を投稿するか」を先に決めてしまう。

このテンプレは、そのために作りました。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**何が入っているのか？**

1. **月間カレンダー（Googleスプレッドシート）**
   - あなたの売上目標に合わせた、投稿テーマの自動決定
   - 4週分 × 7日間 = 28 マスに最適なテーマを配置
   - コピペして使える投稿テンプレ URL 自動挿入

2. **セットアップガイド（PDF / 8ページ）**
   - 「このテンプレで何が解決するのか」の3つのメリット
   - Googleドライブへのコピー方法（初心者向け）
   - 自分の売上目標から「投稿テーマ」を決める方法

3. **投稿テンプレ集（20パターン）**
   - そのままコピペできる投稿文
   - ハッシュタグ（Instagram / X 両対応）
   - 修正ポイント（[自分の実績]など）を明記

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**このテンプレが特別な理由**

❌ 一般的なコンテンツカレンダーテンプレ
→ 「何を投稿すればいいか」を自分で考える必要がある
→ 時間がかかる / テーマのブレが出る / 継続しない

✅ このテンプレ
→ 「売上目標から逆算して、投稿テーマが決まっている」
→ 毎日の迷いが消える / 一貫性が出る / 3ヶ月継続できる

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**こんな人に合っています**

✅ Instagram / X で集客を増やしたい
✅ 毎日の投稿ネタに悩まない設計を欲しい
✅ 売上目標に合わせた発信戦略を実行したい
✅ テンプレ形式で「すぐに実装したい」

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**使用例**

例1: 月商 ¥50万の目標なら
→ 「新規顧客獲得」テーマを 週2回 挿入
→ 実績・事例の投稿を 週2回 配置
→ まとめ投稿を 日曜定期

例2: サービス単価アップ中なら
→ 「高単価ノウハウ」を週3回
→ 失敗ログで信頼構築を週1回
→ 質問投票で顧客の声を週1回

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**よくある質問**

Q: Googleスプレッドシートの操作は簡単ですか？
A: はい。コピー＆ペーストと色の変更だけです。VBA やマクロは不要。

Q: テンプレ投稿文をそのまま使っていいですか？
A: はい。そのままコピペ OK です。修正ポイントだけ入れ替えてください。

Q: 複数の SNS に同時投稿できますか？
A: Googleスプレッドシートで一括決定 → 各 SNS プラットフォームで手動投稿（または Buffer / Later など自動投稿ツールと連携）

Q: 毎月、新しいテンプレをダウンロードする必要がありますか？
A: いいえ。1つのテンプレを毎月コピーして使用できます。

Q: Solo CEO OS と組み合わせると何が変わりますか？
A: 「月商目標」が決まると、その達成に必要な投稿戦略がここで自動決定されるので、迷いなく実行できます。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**このテンプレを使う流れ**

1. このページの「ファイルをコピー」ボタンを押す
   → あなたの Googleドライブにコピーされます

2. PDF ガイドを読みながら、3つのステップを実行
   - ステップ1: 売上目標を入力
   - ステップ2: 投稿テーマを決定
   - ステップ3: テンプレ投稿文を週別に配置

3. 月曜日から、カレンダーに沿って投稿するだけ
   - テンプレから投稿文をコピー
   - [自分の実績] など を置き換え
   - Instagram / X / note に投稿

4. 1ヶ月後、振り返り
   - 売上が増えたか？
   - どのテーマの投稿が反応良かったか？
   - 来月のテーマ比率を調整

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**Solo CEO OS Vol.1 購入者へのメッセージ**

もし あなたが Solo CEO OS で「月¥50万」の目標を立てたなら、
その目標を達成する投稿戦略が、このテンプレに詰まっています。

売上＝集客の質 × 数 × 単価

集客の「質」と「数」を高めるために、
毎日の投稿を迷わず実行する仕組みを、ここに作りました。

つつより

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```
