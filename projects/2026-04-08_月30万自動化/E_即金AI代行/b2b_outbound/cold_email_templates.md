# コールドメール テンプレート集（5通セット）

## 共通ルール

### 宛名
- 店主名がわかる場合：「[店舗名] [店主名] 様」
- わからない場合：「[店舗名] ご担当者様」
- 「店長様」は避ける（個人店では店主＝オーナーが多く、距離が遠く感じる）

### 署名（毎回末尾に固定）
```
――――――――――――――――――
[氏名]（フリーランス／Googleマップ口コミ返信代行）
メール：[your-email@example.com]
ポートフォリオ：[URL or note]
※本メールは Googleマップで公開されている店舗様向けに、
　業務改善のご提案としてお送りしています。
　ご不要の場合は「不要」とご返信いただければ二度と送信しません。
――――――――――――――――――
```
※特商法・迷惑メール対策：オプトアウト導線を必ず明記。

### 送信タイミング
- 火曜〜木曜の **10:00〜11:00 / 14:00〜15:00**（飲食店の仕込み前後）
- 金土日・連休前夜は避ける

## ① 最初のアプローチ（問題提起型）

### 件名案（3つ）
1. 「[店舗名]様の口コミ返信を、月10件分お手伝いできます」
2. 「Googleマップの口コミ、返信が止まっていませんか？」
3. 「[店舗名]様｜口コミ返信を24時間以内に代行するご提案」

### 本文（約280字）
```
[店舗名] ご担当者様

突然のメール失礼いたします。
フリーランスで Googleマップ口コミ返信代行をしている [氏名] と申します。

[店舗名]様の口コミを拝見し、評価[X.X]・[XX]件と多くの来店客に
支持されている一方、お忙しさからか返信が追いついていない様子を
お見かけしました。

口コミへの返信は来店検討客の信頼感に直結すると言われており、
返信文の作成だけを月額¥3,000〜で代行しております。
24時間以内納品・修正2回まで・最低契約期間なしです。

ご興味があれば、まずは [口コミ3件分の無料サンプル] をお作りします。
ご不要であればこのメールは破棄してください。

[署名]
```

## ② 2回目リマインド（事例提示型 / 7日後）

### 件名案
1. 「先日のご提案の補足です｜[店舗名]様」
2. 「[店舗名]様 — 無料サンプル3件、ご希望ありますか？」

```
[店舗名] ご担当者様

先週、口コミ返信代行のご案内をお送りした [氏名] です。
ご多用中と存じますので、補足のみお伝えいたします。

ご不要の場合は「不要」とだけご返信ください。
それ以降のご連絡は控えます。

```

## ③ 3回目（無料トライアル提示型 / さらに7日後）

### 件名案
1. 「最後のご提案です｜[店舗名]様 無料トライアル1週間」
2. 「[店舗名]様｜口コミ返信を1週間タダでお試しいただけます」
3. 「これ以降ご連絡しません｜無料トライアルのご案内」

### 本文（約290字）
```
[店舗名] ご担当者様

度々失礼いたします、[氏名] です。
本メールを最後とし、以降のご連絡は控えますのでご安心ください。

最後のご提案として、**1週間 完全無料トライアル** をご案内します。

【内容】
・期間中の新規口コミ全件の返信文を作成
・ご納品はGoogleドキュメントで共有
・トライアル後の継続は任意（自動課金なし）

お試しいただいたうえで「不要」と感じられた場合、
費用は一切発生しません。

ご関心があれば「トライアル希望」とだけご返信ください。
30秒で開始のご案内をお送りします。

[署名]
```

## ④ 返信あり → 商談設定

### 件名案
1. 「Re: [店舗名]様｜次のステップのご案内」
2. 「ありがとうございます｜サンプル納品と簡単な打合せのご相談」
3. 「[店舗名]様｜お打合せ日程のご相談」

### 本文（約280字）
```
[店舗名] [店主名] 様

ご返信ありがとうございます。[氏名] です。
ご興味をお寄せいただき大変嬉しく存じます。

次のステップとして、以下2点をご案内いたします。

1. **無料サンプル3件**
　貴店の既存口コミから3件を選び、48時間以内にお送りします。

2. **15分のオンライン or お電話での打合せ（任意）**
　返信のトーン・避けたい表現などをヒアリングします。
　以下からご都合の良い時間をお知らせください。
　・4/[XX] 10:00〜 / 14:00〜 / 17:00〜
　・4/[XX] 10:00〜 / 14:00〜 / 17:00〜

打合せなしでサンプルだけご確認いただくことも可能です。
ご希望をお知らせください。

[署名]
```
