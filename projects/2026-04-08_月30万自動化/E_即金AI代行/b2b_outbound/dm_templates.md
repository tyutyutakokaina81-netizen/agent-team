# Instagram DM テンプレート集（3パターン）

## 共通ルール

### 送信前チェック
- 店舗の最新投稿に **いいね** を押す（DM受信通知時の心象が変わる）
- 直近1ヶ月以内の投稿があるアカウントのみ送信（休眠店は除外）
- フォロー数より **投稿頻度** を見る（週1以上が理想）

### 送信タイミング
- 火〜木の **15:00〜17:00**（仕込み終わり〜開店前）
- 営業時間中（18:00〜23:00）と早朝は避ける

### NG事項
- いきなり料金提示しない（フォーマル型のみ料金OK）
- 「DM失礼します」より「投稿拝見しました」のほうが通読率が高い
- スタンプ・絵文字は1〜2個まで（多用は迷惑業者と区別がつかない）

### 共通注意
- DMは「反応がなければ追撃しない」（Instagram規約・嫌がらせ防止）
- 1日あたり送信上限：**20件**（スパム判定回避）

## ① フォーマル型（200字）

```
[店舗名]様

はじめまして。[氏名]と申します。
[店舗名]様のInstagram投稿を拝見し、
[料理写真／世界観／店主の言葉]の表現にとても惹かれてご連絡しました。

私はGoogleマップの口コミ返信を月額¥3,000〜で代行しております。
24時間以内納品・修正2回まで・契約期間縛りなしです。

[店舗名]様の口コミを3件分、無料でサンプル作成いたします。
ご興味があればお返事ください。突然のご連絡失礼いたしました。
```

**使いどころ**：高単価業態（和食・割烹・ワインバル等）／お堅めの店主向け。

## ② フレンドリー型（195字）

```
[店舗名]さん、こんにちは！
[氏名]と申します。

[店舗名]さんの[名物メニュー名]の投稿を見て
思わずお腹が鳴ってしまいました…！

突然のご連絡で恐縮なのですが、
私はGoogleマップの口コミ返信を代行している者です。
「返信したいけど時間が…」というお店向けに
24時間以内に返信文をお作りしています（月¥3,000〜）。

無料でサンプル3件お作りできます。
お忙しい中失礼しました！
```

**使いどころ**：カフェ／ラーメン店／若い店主の個人店。投稿のトーンが砕けているアカウント向け。

## ③ 事例提示型（200字）

```
[店舗名]様

[氏名]と申します。Googleマップ口コミ返信代行をしております。

[店舗名]様の口コミを拝見し、[★4以上の高評価コメントが続いていて]
返信のしがいがあるなと感じご連絡しました。

導入事例は現在準備中のため、
**初月50%OFF（¥3,000→¥1,500）でモニター価格** をご提示しています。
4月末まで先着3社限定です。

まず無料サンプル3件、お送りします。
ご関心あればお返事ください。
```

**使いどころ**：価格の透明性を重視する店主／決済までが早そうなアカウント向け。

## 返信があった場合の即応テンプレ

### A. 「サンプル希望」と返信が来たとき
```
ご返信ありがとうございます！
それでは48時間以内にサンプル3件をお送りします。

確認事項：
1. 既存口コミから3件をこちらで選定してOKですか？
　（特定の口コミを指定いただくことも可能です）
2. 納品形式は Googleドキュメント / PDF どちらが良いですか？

お急ぎのご返信は不要です。お手すきの際にお願いいたします。
```

### B. 「料金は？」と返信が来たとき
```
ご質問ありがとうございます！

3プランございます：
・ライト ¥3,000／月10件まで
・スタンダード ¥5,000／月20件まで（推奨）
・プレミアム ¥9,800／月50件＋月次レポート

すべて初月50%OFFモニター価格対応中（4月末まで先着3社）。
最低契約期間なし・修正2回まで・24時間以内納品です。

無料サンプル3件もお作りできますので、
まずはご確認いただいてからのご判断で大丈夫です。
```

### C. 「不要です」と返信が来たとき
```
ご返信いただきありがとうございます。
承知いたしました。今後ご連絡は控えます。
[店舗名]様のますますのご繁盛を祈っております。
失礼いたしました。
```
※必ず返信。無視せずクローズすることが信頼担保になる。
