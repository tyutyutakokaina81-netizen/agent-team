## ページ2：プラン・導入イメージ・Q&A

### ◆ 3つのプラン

```
┌──────────┬──────────┬──────────┬──────────────────┐
│   プラン   │ 月額(税込) │ 件数上限 │     主な対象      │

└──────────┴──────────┴──────────┴──────────────────┘

★ 初月モニター価格：上記の50%OFF（4月末まで先着3社）
```

### ◆ 導入から納品までのイメージ

```
[Day 0]  契約 → オンボーディングシート送付
   ↓
[Day 1-2] 店主様：トーン・NG表現をシート記入（10分）
   ↓
[Day 3]  既存口コミ3件で試作 → 店主様確認
   ↓
[Day 4]  フィードバック反映
   ↓
[Day 5-7] 本稼働開始（新規口コミを24時間以内に返信案納品）
```


### ◆ 想定Q&A（提案書内に5つだけ抜粋）

```
Q1. 個人情報（口コミ投稿者名）はどう扱われますか？
A1. 契約期間中のみ作業用フォルダに保管し、解約後30日以内に
    完全削除します。第三者への提供は一切ありません。

Q2. 投稿はそちらでしてくれますか？
A2. いいえ。Googleビジネスプロフィールへの直接投稿は権限委譲
    リスクがあるため、原則は店主様にコピペで投稿いただきます。
    （※プレミアムプランでは別途相談可）

Q3. 低評価★1の口コミにはどう対応しますか？
A3. プレミアムプランでは6時間以内の優先対応、その他プランは
    通常24時間以内です。どのプランも、店主様確認後に投稿いただく
    形のため、最終判断は店主様に委ねられます。

Q4. AIが書くと不自然な文になりませんか？
A4. AIで初稿を作成後、必ず人間がトーン・固有名詞・違和感を
    チェックし納品します。修正は2回まで無料です。

Q5. やめたい場合は？
A5. 次月開始の7日前までにメールで「解約」とご連絡いただければ
    OKです。違約金・引き止めは一切ありません。
```

## レイアウト指示（Canva実装時）

| 要素 | スタイル |
|------|--------|
| 表紙背景 | 白／薄ベージュ。店舗写真は使わない（無断使用回避） |
| メインカラー | 深緑（#2E5D3B）または濃紺（#1A3A5C）の信頼感色 |
| フォント | 本文：Noto Sans JP Regular／見出し：Noto Sans JP Bold |
| ロゴ | 不要（フリーランスのため、署名のみで代用） |
| 写真 | フリー素材1点（飲食店のテーブル風景など）／表紙のみ |
| ページ数 | 2ページ厳守（長くしない） |

## 提案書バージョン管理
- ファイル名：`proposal_[店舗名]_2026-04-XX.pdf`
- 送付前に **店舗名・評価数・口コミ数** を必ず差し替える（差し替え忘れは失礼の極み）
- 無料サンプル3件は **PDFまたはGoogleドキュメント** で別添付
