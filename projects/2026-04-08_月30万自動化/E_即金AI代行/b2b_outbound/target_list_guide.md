# ターゲットリスト作成ガイド（19日で1社契約を取るための集中設計）

## 1. 攻めるべき地域（優先順）

### 第1優先：首都圏のローカル繁華街（オフィス街ではなく住宅×飲食混在エリア）
- 東京：高円寺、中野、三軒茶屋、下北沢、武蔵小山、赤羽、北千住、十条
- 神奈川：鶴見、藤沢、横須賀中央
- 埼玉：大宮、川口、所沢
- 千葉：船橋、柏、松戸

理由：個人経営店の密度が高く、競争意識が強い。SNS・Web施策に積極的だが手は回っていない。

### 第2優先：地方政令市の駅前商店街
- 仙台、新潟、金沢、岡山、熊本、鹿児島
- 競合代行業者が少なく、価格競争に巻き込まれにくい

### 除外：銀座／表参道／六本木／新宿歌舞伎町中心部
理由：単価高すぎ・既に代理店が入っている・反応が遅い。

## 2. 狙う業態

### ◎ 最優先（反応率が高いと想定）
- **個人経営の居酒屋**（席数30〜80）：店主が現場に出ており即決可能
- **個人経営のラーメン店**：口コミ件数が多く、悪評対応に困っている店主が多い
- **地域密着のカフェ／喫茶店**：オーナーの世界観表現にこだわりがあり、文章代行ニーズが顕在化しやすい

### ○ 次点
- 焼肉店（個人経営、50〜150席）
- 和食・小料理屋
- ベーカリー、洋菓子店

### × 除外
- チェーン店（スシロー、サイゼリヤ等）：本部承認が必要で決裁が遅い
- 大手居酒屋ブランド：自社マーケ部門あり
- 既にInstagramフォロワー1万人超の店：自前で運用済み
- 評価4.5以上の超人気店：困っていない
- 評価3.0以下の店：根本問題が口コミ返信ではない

## 3. リサーチ条件（「困っていそうな店」の定義）

| 指標 | 条件 | 理由 |
|------|------|------|
| Google評価 | **3.5〜4.2** | 改善意欲があり、かつ余地がある |
| 口コミ件数 | **20〜100件** | 動いているが運用しきれていない |
| オーナー返信率 | **20%以下** | 明確な未対応問題が存在する |
| 直近3ヶ月の口コミ | **5件以上** | 定期的に新規口コミが入っている |
| 公式サイト | **あり or インスタあり** | 連絡手段が取れる |
| 席数 | **30〜200席** | 店主決裁の規模 |

## 4. リサーチ手順（1時間で50件集める方法）

### Step 1：Googleマップで業態×エリアで検索（10分）
1. Googleマップを開く
2. 検索窓に「居酒屋 高円寺」「ラーメン 中野」など `[業態] [駅名]` で入力
3. リスト表示に切り替え
4. 上位50件をスクロールして全件表示

### Step 2：条件フィルタ（20分）
各店舗ページで以下を順番にチェック：
1. 評価が3.5〜4.2の範囲か
2. 口コミ件数が20〜100か
3. 「オーナーからの返信」を5件サンプリング → 返信率20%以下か（口コミ20件中4件以下に返信）
4. 直近の口コミ日付が3ヶ月以内か

→ 合致した店のみ次工程へ。1検索あたり10〜15件は条件に合致するはず。

### Step 3：連絡先取得（20分）
1. 店舗の公式サイトURLをクリック → 「お問い合わせ」「contact」ページから **メールアドレス** を取得
2. メールがなければ **問い合わせフォーム** のURLを記録
3. **Instagramアカウント** のURLも必ず記録（DM経路）
4. 電話番号も記録（最終手段）

### Step 4：スプレッドシートに記録（10分）

## 5. スプレッドシート設計（コピペ用ヘッダー）

```
A列：取得日（2026-04-11）
B列：店舗名
C列：エリア／最寄駅
D列：業態（居酒屋／ラーメン／カフェ／etc）
E列：席数（推定可）
F列：Google評価
G列：口コミ件数
H列：オーナー返信率（%）
I列：直近口コミ日
J列：公式サイトURL
K列：問い合わせ手段（メール／フォーム／DM／電話）
L列：連絡先（メアド or DMリンク）
M列：店主名（わかれば）
N列：優先度（A／B／C）
O列：アプローチ済みフラグ（未／メール送信済／DM送信済／返信あり／商談中／契約／NG）
P列：最終アクション日
Q列：メモ（口コミの特徴、店の強み、刺さりそうな切り口）
```
