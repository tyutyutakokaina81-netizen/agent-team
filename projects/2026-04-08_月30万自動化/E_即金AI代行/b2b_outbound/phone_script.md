# 電話アプローチ台本（情報提供型・3分以内）

## 基本方針

- **「営業電話」と思わせない**：最初の10秒で「営業お断り」と言われない設計が9割
- **情報提供→相手の反応→提案** の順を絶対に崩さない
- **アポ取りより資料送付**：その場で契約させない。メール／サンプル送付を着地点にする
- 電話は **メール／DM送信後の3〜5日後** に「補足」として掛けるのが最も通る
- **3分以内**で必ず終わる構成。長電話は嫌われる

## 掛けるタイミング
- 火〜木の **14:30〜16:30**（ランチ後・ディナー仕込み前）
- 月曜・金曜・日曜・祝日は避ける
- 雨天は反応率が高い（暇な時間が増えるため）

## メイン台本（約2分30秒構成）

### 【0:00 - 0:15】挨拶＋名乗り＋目的の透明化

```
お忙しいところ恐れ入ります。
私、Googleマップの口コミ運用をサポートしている [氏名] と申します。
お電話3分だけ、情報提供のご案内でご連絡しました。
セールスではなく、無料の改善サンプルをお送りしたいというご相談です。
店主様、もしくは店舗運営をされている方はいらっしゃいますか？
```

**ポイント**
- 「営業」と言わず「情報提供」「無料サンプル」を必ず最初に出す
- 「3分だけ」と時間枠を提示する
- 名乗りは会社名ではなく **個人名** で行う（フリーランスの誠実さを出す）

---

### 【0:15 - 0:45】相手の状況確認（質問→共感）

```
[店主に変わったあと]

[店主名] 様でいらっしゃいますか。お忙しいところ失礼いたします。

実は先週、[店舗名] 様のGoogleマップを拝見しまして、
口コミが [XX] 件、評価 [X.X] と多くのお客様に支持されている一方で、
「返信が追いついていないのかな」と感じる箇所があり、
お役に立てるかもしれないと思いお電話しました。

口コミ返信、最近はどんな状況ですか？
お時間の関係で滞っていたり、文章を考えるのが大変だったりしませんか？
```

**ポイント**
- 「口コミ件数・評価」を具体的に言うことで「リサーチして掛けている」と伝わる
- 質問は **YESを引き出しやすい二択** にする
- 共感の前に「お忙しいところ失礼します」を必ず2回入れる

---

### 【0:45 - 1:30】解決策のさわりだけ提示

```
私は、Googleマップの口コミ返信文を24時間以内に作成し、
店主様にコピペでお使いいただける形でお届けする
代行サービスをしております。

料金は月額¥3,000からで、最低契約期間もなく、
今月は導入事例づくりのため初月半額のモニター価格でご案内しています。

ただ、いきなり契約の話ではなく、
まずは [店舗名] 様の既存口コミから3件、
無料で返信案をお作りしてお送りしたいんです。
品質を見ていただいて、ご不要であれば破棄してください。
```

**ポイント**
- 価格は **早めに開示** する（隠すと不信感）
- 「まずは無料サンプル」を必ず最後に置く
- 「契約の話ではない」を明言

---

### 【1:30 - 2:30】着地点：メールアドレスの取得

```
もし「サンプルを見るだけならOK」ということでしたら、
本日中にメールでサンプルをお送りします。

メールアドレスをお伺いしてもよろしいでしょうか。
（あるいは、お店の公式アドレスにお送りしても大丈夫です）

[メアド取得後]

ありがとうございます。
本日中に[氏名]の名前で「Googleマップ口コミ無料サンプル」という件名で
お送りします。
ご不要であればそのまま破棄していただいて構いません。
お忙しいところお時間ありがとうございました。
```

**ポイント**
- ゴールは **メアド取得＋送信予告** のみ
- 「本日中」と納期を切る
- 件名を伝えておく（迷惑メールフォルダ回避）

## 切り返し集（10パターン）

### ① 「間に合ってます」
```
承知しました、お忙しいところ失礼しました。
ひとつだけお伝えして失礼します。
口コミ返信のサンプルをメールで1件だけ送ることもできます。
必要ないと感じたら破棄していただいて大丈夫です。
お送り先のメアドだけ伺ってもよろしいですか？
…ダメなら諦めます！
```

### ② 「予算ない」
```
そうですよね、ご事情よく分かります。
今回は **完全無料のサンプル提供のご案内** で、
契約をお勧めするものではないんです。
お時間あるときに、品質だけご確認いただきたく…
それでもご不要でしたら、また機会があればで大丈夫です。
```

### ③ 「信頼性は？／怪しい」
```
ごもっともです、突然のお電話で恐縮です。
私はフリーランスで個人で運営しており、
ホームページとnote（[URL]）で情報を公開しております。
契約書もございますし、ご不審な場合は無料サンプルだけ
ご覧いただければと存じます。
```

### ④ 「自分で返信してる」
```
それが理想だと思います、素晴らしいです。
ただ、忙しい日が続いた時だけ「数件まとめて代行」という
スポット利用もできますので、もし良ければ
情報だけメールでお送りしておいてもよろしいですか？
必要なときに思い出していただければ十分です。
```

### ⑤ 「AIなんでしょ？信用できない」
```
おっしゃる通り、AIで初稿を作っています。
ただ、最終的に人間（私）が必ず読み直して、
店舗のトーンに合うか、固有名詞が間違っていないかを
チェックして納品しています。
無料サンプルでその品質をご確認いただければと思います。
```

### ⑥ 「店主は今いない」
```
承知しました、お忙しいところ失礼しました。
店主様にお伝えいただける場合のみで結構ですので、
「Googleマップ口コミ返信代行の[氏名]から電話があった」と
お伝えいただけると幸いです。
改めてかけ直すのは控えますので、ご興味あれば
公式メールに資料をお送りすることもできます。
```

### ⑦ 「営業電話お断り」
```
失礼いたしました、すぐに失礼します。
営業ではなく **無料サンプル提供のご案内** だったのですが、
ご不快な思いをさせてしまい申し訳ありません。
今後ご連絡は控えます。失礼しました。
```
※即時撤退。粘らない。

### ⑧ 「効果あるの？／成果保証は？」
```
正直に申し上げますと、口コミ返信だけで売上が
劇的に上がる保証はできません。
ただ、Googleの公式情報でも「返信のあるビジネスは
ユーザーの信頼を得やすい」と紹介されており、
来店検討中のお客様に与える印象は確実に変わります。
無料サンプルで雰囲気だけご覧いただければ幸いです。
```

### ⑨ 「もう他社でやってる」
```
そうでしたか、失礼しました。
もしご参考までに比較されたい場合は、
無料サンプルだけお送りすることもできますが、
不要でしたらこのまま失礼いたします。
お時間ありがとうございました。
```

### ⑩ 「資料を送って」
```
ありがとうございます！
本日中にメールでサンプル＋簡単な料金表をお送りします。
件名は「Googleマップ口コミ無料サンプル」でお送りしますので、
迷惑メールフォルダもご確認いただければ幸いです。
お送り先のメアドを教えていただけますか？
```
※最も理想的な着地。必ずメアドを取って終話。

## 電話後の処理（必須）

1. スプレッドシートのO列を **「電話済」** に更新
2. 取得したメアドをL列に追記
3. 30分以内にサンプルメールを送信（鮮度命）
4. 翌日に「昨日お電話した者です」として軽い確認メッセージを送る

## 1日の電話上限
- **15件まで**（連続架電は喉と精神を消耗するため）
- 通電率：約30%（実際に店主と話せるのは5件）
- メアド取得率：通電の20-30% → **1-2件/日**
