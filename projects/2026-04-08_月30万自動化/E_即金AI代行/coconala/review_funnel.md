# レビュー獲得＋リピート誘導 動線設計

作成日：2026-04-11
担当：CSO
目的：初受注からレビュー獲得→2件目購入（リピート）→値上げの軌道に乗せるための運用ファネル

---

## ファネル全体像

```
[出品公開]
   ↓
[初インプレッション獲得：~3日]
   ↓
[初受注（¥1,500スターター）：Day 3〜5想定]
   ↓
[ヒアリング → 制作 → 納品メッセージ]
   ↓
[クライアント正式承認]
   ↓
[24時間以内：レビュー依頼メッセージ送信]
   ↓
[レビュー獲得（★4以上目標）]
   ↓
[3件達成 → 値上げ準備]
   ↓
[¥3,000〜¥5,000帯に移行]
   ↓
[リピート購入の自然誘導]
```

## ステップ1：初受注獲得施策（Day 1〜5）

### 出品時の差別化ポイント
- 「新規出品キャンペーン｜先着3名様限定特別価格」と説明文に明記する
- 「最短当日納品可能」と強調する
- プロフィールで「AI活用＋人力チェックの品質保証」を打ち出す
- 出品ページのサムネイルに「先着3名｜特別価格」と入れる

### 初受注を呼び込む追加アクション
- ココナラ内のおすすめ・新着出品者枠に表示されるよう、出品後すぐ複数ギグを公開する
- 「ココナラ公式ブログ／TwitterのRT施策」をフォロー
- TwitterやInstagramで「ココナラ始めました」投稿（owner SNS活用）

## ステップ2：レビュー依頼メッセージ

### タイミング
- クライアントの正式納品承認後、24時間以内
- 早すぎず遅すぎず、納品物の確認余韻が残るうちに

### メッセージ本文（共通テンプレ）

```
[クライアント名]様

このたびはご利用いただきまして誠にありがとうございました。
納品物にご満足いただけておりましたら幸いです。

もしよろしければ、ご利用いただいた感想を
レビューでお聞かせいただけませんでしょうか。

新規出品して間もないため、
お客様のお声が今後の励みになりますし、
他の検討中のお客様にとっても安心材料となります。

お時間のあるときで構いません。
一言だけでも大変助かります。

また、今回の納品物でご活用できそうな点や
さらに改善したい点などございましたら、
今後のサービス改善の参考にさせていただきます。

引き続き何卒よろしくお願いいたします。
```

### レビュー依頼時のNGポイント
- 「★5でお願いします」と高評価を強要する → ココナラ規約違反
- 報酬やサービスを対価にレビューを依頼する → 違反
- 何度も催促する（1回のみで止める） → 印象悪化

## ステップ3：低評価リスク回避

### 納品時の予防策
- 納品前にClaudeで誤字脱字・薬機法・景表法を再チェック
- ヒアリング内容と納品物の整合性を確認
- 「軽微な修正は1回まで無料対応」を必ず明記
- 納品メッセージで「ご不明点があればお気軽に」と再送ハードルを下げる

### 万一クレーム発生時の対応フロー

1. **24時間以内に必ず一次返信** — 「ご連絡ありがとうございます。すぐに確認いたします」
2. **クレーム内容の事実確認** — どの部分が問題か、具体的に質問
3. **即時無料修正対応** — 1回無料の範囲で対応する
4. **解決しない場合** — 部分返金 or 全額返金を提案する
5. **エスカレーション** — ココナラ事務局に相談

### 返金提案テンプレート

```
[クライアント名]様

このたびはご期待に沿えず申し訳ございません。
ご指摘いただいた点を真摯に受け止めております。

修正対応としては以下をご提案させていただきます。

【ご提案】
1. ご指摘箇所を全面的に作り直す
2. 一部返金（[金額]円）でご了承いただく
3. 全額返金で取引をクローズ

どの選択肢でもお客様のご希望を最優先で対応いたします。
ご都合のよろしいものをお知らせください。

今後ともどうぞよろしくお願いいたします。
```

## ステップ4：リピート購入誘導

### 自然な誘導タイミング
- 納品メッセージの末尾（過剰にならず1〜2文程度）
- レビュー依頼メッセージの後に余裕があれば
- 1ヶ月後にリピートご案内メッセージ（1回限り）

### リピートご案内メッセージ（1ヶ月後）


## ステップ5：値上げのタイミングと進め方

### 値上げ判定基準
- レビュー数が3件以上、★4.5以上
- 直近1週間の問い合わせが安定（週2件以上）
- 価格を上げてもインプレッションが落ちない見込み

### 値上げ手順
1. レビュー3件達成を確認
2. ギグ説明文を更新（「新規出品キャンペーン終了」表記に変更）
3. 価格を¥1,500 → ¥2,500 に引き上げ
4. 数日インプレッションを観察
5. 安定すれば¥3,000、¥3,500、¥5,000 へ段階的に上げる

### 値上げ告知例

```
■ 価格改定のお知らせ
新規出品キャンペーンは終了いたしました。
[日付]より通常価格での提供となります。
ご検討中の方はお早めにご相談ください。
```
