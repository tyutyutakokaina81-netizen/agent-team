担当：CSO（営業実行）× CMO（提案資料）× CDO（自動化）
作成日：2026-05-19
目的：Month 2 で1社目契約（¥50K）、Month 3 で計6社（¥300K）

```
[1] ターゲットリスト作成
```

## Step 1：ターゲットリスト作成

### 業種優先順位（Month 2-3）

| 優先度 | 業種 | 理由 | 想定単価 |
|------|------|------|---------|
| ★★★ | 地域の美容院・ネイル・サロン | Instagram必須・経営者が忙しい | ¥50K |
| ★★★ | 個人飲食店（カフェ・居酒屋） | Instagram集客に困っている | ¥50K |
| ★★ | 士業（税理士・社労士・行政書士） | 専門性訴求しやすい | ¥30K |
| ★★ | オンラインコーチ・コンサル | リテラシー高く話が早い | ¥50K |
| ★ | 整体院・整骨院 | 競合多く差別化必要 | ¥30〜50K |

### リスト作成手順（週次・90分）

1. **検索ソース**（業種別に下記を巡回）
   - Google マップ（「[地域名] [業種]」で50件抽出）
   - Instagram ハッシュタグ検索（地域タグ＋業種タグ）
   - Instagram「○○ 店舗」検索 → フォロワー500〜5,000のアカウント

2. **抽出条件（受注しやすい店）**
   - 最終投稿が2週間以上前 → 投稿に困っている可能性
   - フォロワー500〜5,000 → 一定の規模＋伸び悩み
   - 公式サイトまたはGoogleビジネスにメールアドレス記載あり
   - 個人経営（チェーン店は本部承認が必要なので除外）

3. **記録項目**（`sales_pipeline.csv` に入力）
   - 業種 / 店舗名 / 地域 / Instagram URL / 公式サイト / 連絡先 / 最終投稿日 / フォロワー数 / 備考

4. **NGリスト**
   - すでにSNS運用代行を契約している（投稿に「@◯◯ 制作」表記あり）
   - 全国チェーン
   - 1年以上更新停止

## Step 2：初回アプローチ

### 送信方法の優先順位

| 方法 | 送信時間／件 | 返信率目安 | 備考 |
|------|-----------|-----------|------|
| 公式サイト問い合わせフォーム | 3分 | 8〜12% | 最も丁寧、優先 |
| Google ビジネス記載メール | 2分 | 5〜8% | 開封率高い |
| Instagram DM | 1分 | 3〜5% | 量で勝負 |

### 送信ルール
- 1日10件まで（スパム判定を避ける）
- 同一文面の連投は禁止（業種別テンプレを最低3パターン用意）
- 送信時間：火〜金の10:00〜11:00 または 14:00〜16:00
- 送信後 `sales_pipeline.csv` の「初回送信日」を記録

### 業種別パーソナライズ要素

`proposal_templates.md` の業種別テンプレをベースに、以下を必ず1文加える：

| 業種 | パーソナライズの観点 |
|------|------------------|
| 美容院 | 「○○さんのカット写真の構図が魅力的」 |
| 飲食店 | 「○○の料理写真の色合いが素敵」 |
| 士業 | 「最近のブログ記事『○○』を拝読しました」 |
| コーチ | 「○○さんの△△の発信に共感しました」 |

→ 「相手のアカウントを実際に見た」ことが伝わる1文が返信率を2倍にする。

## Step 3：無料相談（オンライン15〜30分）

### 事前準備（相談15分前）

- 相手のSNS最新10投稿を確認
- 投稿頻度・エンゲージメント率を計算
- 改善ポイントを3つメモ
- 同業界の参考事例を1つ用意

### 相談アジェンダ（30分版）

| 時間 | 内容 |
|------|------|
| 0〜5分 | アイスブレイク・自己紹介 |
| 5〜10分 | 現状ヒアリング（投稿頻度・困りごと・目標） |
| 10〜20分 | 現状分析・改善ポイント3つを提示 |
| 20〜25分 | サービス提案（プラン・料金・初月お試し） |
| 25〜30分 | Q&A・次アクション確認 |

### ヒアリング必須項目

```
1. SNSを始めた理由・目的
2. 現在の投稿頻度・かかっている時間
3. 投稿で困っていること
4. ターゲット顧客（年代・性別・地域）
5. 月のSNS経由問い合わせ数
6. 今後3ヶ月で達成したい目標
7. SNS運用に使える月予算
8. NG事項（出したくない情報・避けたい表現）
```

### クロージングトーク

```
「ここまでお話を伺って、私からご提案できるのは月15投稿のスタンダードプランです。
通常月額5万円ですが、初月は半額の25,000円でお試しいただけます。
初月の成果を見て継続するかどうかをご判断いただければと思います。

もしご検討いただけるなら、来週月曜から運用開始できます。
いかがでしょうか？」
```

### 異論処理（よくある反論と返し）

| 反論 | 返し |
|------|------|
| 「自社でやれそう」 | 「もちろん可能です。ただ月15投稿だと約15時間かかります。本業の時間との比較でご検討ください」 |
| 「効果が出るか不安」 | 「だからこそ初月半額のお試しがあります。1ヶ月運用してエンゲージメント率の変化を見てから判断いただけます」 |
| 「他社と比較したい」 | 「ぜひ比較してください。私の強みはAI活用での投稿スピードと、初月お試しを設けている柔軟性です」 |
| 「予算が厳しい」 | 「スモールプラン（月8投稿・¥30K）もご用意しています。1媒体に絞って始める形ではいかがでしょうか」 |
| 「もう少し考えたい」 | 「もちろんです。1週間後にもう一度ご連絡してもよろしいですか？」（必ず再連絡日を確定する） |

## Step 4：初月お試し契約

### 契約締結フロー

1. 相談時に口頭合意 → 当日中に確認メール送信
2. 契約書（CFOに依頼・電子サイン or PDF）
3. 初回ヒアリングシート送付
4. 振込確認 or クレカ決済確認
5. 運用開始日確定

### 確認メールテンプレ

```
件名：【ご契約確認】SNS運用代行 初月お試しプランのご案内

○○様

本日はお時間いただきありがとうございました。
お申し込みいただいた内容を以下にまとめます。

▼ ご契約内容
・プラン：スタンダード（月15投稿・2媒体）
・初月料金：25,000円（税込）／お試し価格
・通常料金（2ヶ月目以降）：50,000円／月
・運用開始日：2026-XX-XX
・支払方法：銀行振込（請求書発行）

▼ 次のステップ
1. 別途お送りするヒアリングシートにご記入ください（5分）
2. 初回投稿は運用開始日翌日に予定しています
3. 月末に1ヶ月のレポートをお送りします

ご不明点があればお気軽にご連絡ください。
よろしくお願いいたします。
```

### 初回ヒアリングシート（クライアントに送付）

```
■ 基本情報
1. 店舗名・サービス名
2. URL・SNSアカウント
3. 営業時間・定休日
4. キャッチコピー（あれば）

■ ターゲット
5. 主な顧客の年代・性別
6. 顧客の悩み・ニーズ
7. 競合店（参考にする・差別化したい店）

■ 今月のテーマ
8. 今月の重点訴求（新商品・キャンペーン等）
9. 今月避けたい話題

■ ブランド・トーン
10. 投稿の口調（です・ます／カジュアル）
11. 使ってほしいキーワード
12. NGワード・禁止表現

■ 素材
13. 使える画像・動画の有無（クラウドストレージ共有可？）
14. 投稿に使える「お客様の声」「ビフォーアフター」の有無

■ 連絡
15. 確認・承認のレスポンス時間（即日／翌日／週末まとめて）
```

## Step 5：継続→正規契約

### 初月運用中のタッチポイント

| 時期 | アクション |
|------|---------|
| 運用開始3日後 | 「初投稿のリアクション、確認しました」連絡 |
| 運用開始2週目 | 中間レポート（簡易・3行） |
| 運用開始3週目 | 継続意向ヒアリング |
| 月末 | フルレポート＋翌月契約提案 |

### 継続提案メール

```
件名：【1ヶ月のご報告】継続プランのご案内

○○様

おかげさまで1ヶ月の運用が完了しました。
レポートを添付いたします。

▼ 主な成果
・投稿数：15本完了
・フォロワー変化：+XX人
・エンゲージメント率：X.X%（業界平均X.X%）
・特に反応の良かった投稿：[投稿URL]

▼ 翌月以降のご提案
継続いただける場合、通常料金の50,000円／月となります。

3ヶ月契約の場合は1ヶ月分割引（45,000円／月）も可能です。
いかがでしょうか？

引き続きよろしくお願いいたします。
```

### 失注時のヒアリング

「今後の改善のため、契約を見送られた理由を1つだけ教えていただけますか？」
→ 失注理由を `sales_pipeline.csv` の「失注理由」欄に記録
→ 月次で集計し、提案内容を改善する
