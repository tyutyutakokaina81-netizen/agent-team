## 業種別 検索クエリ集

### 1. 美容院・ネイル・サロン（15件）

#### Google マップ検索
```
[地域名] 美容院
[地域名] ネイルサロン
[地域名] エステサロン
[地域名] ヘアサロン 個人
```

#### Instagram ハッシュタグ検索
```
#[地域名]美容院
#[地域名]ネイル
#[駅名]美容院
#プライベートサロン
```

#### 抽出条件
- フォロワー500〜5,000
- 最終投稿が2週間以上前
- 個人経営（公式サイトに代表者氏名）
- 既存SNS運用代行なし（投稿に @運用代行 表記なし）

### 3. 士業（8件）

#### Google マップ検索
```
[地域名] 税理士事務所
[地域名] 社労士
[地域名] 行政書士
[地域名] 司法書士
```

#### 探し方の工夫
- Google検索：`"[地域名] 税理士" -大手 -法人`（個人事務所に絞る）
- HP記載のメールアドレスは開封率高い
- ブログを書いている士業 = 情報発信に前向き → 提案受容性高い

#### 抽出条件
- 個人事務所（所員5名以下）
- ブログ or X アカウントあり（情報発信意欲がある証拠）
- 集客に課題がありそう（HPで「お客様募集」明記等）

---

### 4. オンラインコーチ・コンサル（7件）

## リスト記録テンプレ（`sales_pipeline_template.csv` を流用）

CSO は `CSO/outputs/2026-05-19_B営業ターゲットリスト.csv` を新規作成し、
以下の列で50件分の実データを収集する。

```csv
リードID,業種,店舗名,地域,Instagram_URL,連絡先,初回送信日,送信方法,返信日,商談日,ステータス,失注理由,受注金額,契約開始日
LEAD-001,美容院,（実店舗名）,東京都世田谷区,（実URL）,（実メール）,,,,,未送信,,,
LEAD-002,...
```

### IDの振り方
- `LEAD-001` 〜 `LEAD-050`
- 業種に関わらず通し番号
- 失注後も再利用しない

## リスト作成 1日のタスクフロー（90分）

| 時間 | アクション |
|------|---------|
| 0〜10分 | 業種を1つ決める（今日は美容院） |
| 10〜50分 | Google マップ＋Instagram で10件抽出（条件チェック） |
| 50〜80分 | CSVに記入（店名・URL・連絡先・備考） |
| 80〜90分 | NGリスト照合（既存運用代行・全国チェーン除外） |

→ 1日10件 × 5日（平日のみ）= **1週間で50件完成**

## NG判定の早見表

| 判定基準 | 該当する場合 | 対応 |
|---------|------------|------|
| 投稿に @○○制作 / @○○運用 表記 | あり | スキップ |
| Instagram bio に「PR担当：」 | あり | スキップ |
| 全国チェーン店 | あり | スキップ（本部承認が必要） |
| フォロワー10,000超 | 既に運用代行ありの可能性高 | スキップ |
| 最終投稿1年以上前 | 既に閉店リスク | スキップ |
| HPに「現在新規受付停止」 | あり | スキップ |

## アプローチ送信時のローテーション

50件を一気に同日送信せず、業種ローテで毎日10件ずつ送る。

```
Day 1：美容院 ×5・飲食店 ×5
Day 2：飲食店 ×5・士業 ×3・コーチ ×2
Day 3：美容院 ×5・整体 ×3・コーチ ×2
Day 4：飲食店 ×5・士業 ×3・コーチ ×2
Day 5：美容院 ×5・整体 ×2・コーチ ×3
```

→ 5日で50件送信完了。
