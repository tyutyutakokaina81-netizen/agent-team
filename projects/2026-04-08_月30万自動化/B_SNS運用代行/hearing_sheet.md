# SNS運用代行：クライアントヒアリングシート

**用途**：受注前の30〜60分オンライン商談で使う。完了後そのままClaudeに渡せば、初月15投稿が自動生成できる構造になっている。
**運用**：Google ドキュメントまたは Microsoft Word（既存契約 ¥1.3K/月内）にコピペして使用。Zoom 共有画面で記入しながら進行。
**費用**：¥0（既存ツール内で完結）

```

```

# Tips（CSO 営業時の注意）

- ヒアリング中、6つ以上の項目で「わからない」が出たら、契約前にもう1回相談を提案する
- 「写真素材を毎月送る運用が回せるか」が最大のドロップアウト要因。契約前に必ず合意する
- 「投稿前承認フロー」をどうするかで作業量が3倍変わる。CMO の作業負担に直結する点を本人にも理解してもらう
- 価格交渉は受けない（初月半額のみ。それ以上の割引は契約期間6ヶ月以上の場合に限る）
