## アウトリーチ文面（チャネル別）

### Instagram DM（飲食店向け）

```
はじめまして、SNS運用代行の○○と申します。
○○様のお料理写真が素敵で、思わずDMさせていただきました。

もし投稿頻度を増やしたいなどのご希望があれば、
月15投稿を月5万円で代行しております。
初月は半額でお試しいただけます。

ご興味あればお気軽にご返信ください。
```

### 問い合わせフォーム（士業向け）

`proposal_templates.md` の「士業向け」テンプレを使用。
件名・敬称（先生）・専門知識訴求の3点を必ず守る。

### X DM（コーチ・コンサル向け）

```
はじめまして。SNS運用代行の○○です。
○○さんの発信内容、いつも参考にさせていただいています。

もし発信を増やしたいけど時間が足りないという場合、
月15投稿を5万円で代行しております。
初月半額です。

15分のオンライン相談はいかがでしょうか？
```

## NGアプローチ

- 同一テンプレを大量送信（Instagram のスパム判定リスク）
- フォロー外し前提のフォロー（信頼を損なう）
- 返信ない相手への複数回送信（1回フォローのみ）
