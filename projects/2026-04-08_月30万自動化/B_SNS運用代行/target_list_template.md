# B：ターゲットリスト管理テンプレ＆ソーシング手順

担当：CSO
作成日：2026-04-17

> **重要：** 実在企業の社名・連絡先・SNSアカウント等のリストは PII・営業情報のため
> **Git にコミットしない**。本ファイルは管理フォーマットと探索手順のみを定義する。
> 実データは Google Sheets またはローカル CSV で管理する。

## ターゲットリスト管理表（Google Sheets 推奨）

### 列構成

| 列 | 項目 | 内容 | 入力形式 |
|----|------|------|---------|
| A | ID | 連番（001〜） | 数値 |
| B | 業種 | 飲食／美容／士業／コーチ等 | プルダウン |
| C | 会社名・店舗名 | 正式名称 | テキスト |
| D | 所在地 | 都道府県・市区町村 | テキスト |
| E | 公式サイトURL | あれば | URL |
| F | Instagram | @ハンドル | テキスト |
| G | X（Twitter） | @ハンドル | テキスト |
| H | Facebook | ページURL | URL |
| I | 現在のフォロワー数 | 直近確認時点 | 数値 |
| J | 月間投稿数 | 直近1ヶ月 | 数値 |
| K | 評価スコア | 1〜5（後述） | 数値 |
| L | 接触日 | YYYY-MM-DD | 日付 |
| M | 接触チャネル | DM／フォーム／メール | プルダウン |
| N | 反応 | 未読／既読／返信／断り／契約 | プルダウン |
| O | 次アクション | 1週間後フォロー等 | テキスト |
| P | 備考 | 担当者名・特記事項 | テキスト |

## 評価スコア基準（K列）

| スコア | 基準 |
|-------|------|
| 5 | フォロワー1,000+／月3投稿以下／公式サイトあり／更新意欲あり |
| 4 | フォロワー500+／月3投稿以下／代行ニーズが見込める |
| 3 | フォロワー100+／投稿頻度低／要ヒアリング |
| 2 | アカウントはあるが放置気味／予算不明 |
| 1 | アクティブで自走できている／代行ニーズ薄 |

**接触優先順位：** スコア5・4を最優先、3を二番手、2-1は対象外。

## 業種別ソーシング手順

### 飲食店（目標20社）

**探索手順：**
1. Google Maps で「[地域名] 飲食店」を検索
2. 評価4.0以上・口コミ50件以上の店舗をピックアップ
3. Instagram で店名検索、アカウント確認
4. 月3投稿以下の店舗を候補化

**フィルター条件：**
- チェーン店は除外（本部承認が必要で時間がかかる）
- 個人経営・小規模法人を優先
- ペット可・テラスあり等の写真映え要素がある店

### 美容院・整体院（目標15社）

**探索手順：**
1. ホットペッパービューティー or Google Maps で検索
2. Instagram アカウント保有店舗をピックアップ
3. 投稿頻度が月2回以下の店舗を候補化

**フィルター条件：**
- 個人サロン・小規模店舗を優先
- 既に投稿はしているが頻度・質に課題がある店

### 士業（目標10社）

**探索手順：**
1. 「[地域名] 税理士／社労士／行政書士」で検索
2. 公式サイトを持つ事務所をピックアップ
3. SNSアカウントの有無を確認

**フィルター条件：**
- 大手事務所は除外
- 1〜3名規模の個人事務所を優先
- ブログ更新があるが SNS は弱い事務所

### コーチ・コンサル（目標15社）

**探索手順：**
1. X（Twitter）で「コーチ」「コンサル」「フリーランス」関連ハッシュタグを検索
2. フォロワー500〜5,000の中堅層をピックアップ
3. 投稿頻度が低下傾向の人を候補化

**フィルター条件：**
- インフルエンサー級（フォロワー1万+）は除外（自走できている）
- LP・公式サイトを持っている人を優先（事業として真剣な層）

## 接触チャネル選定ルール

| 業種 | 第1候補 | 第2候補 | 避けるチャネル |
|------|--------|--------|--------------|
| 飲食店 | Instagram DM | 公式サイト問い合わせ | 電話（営業時間中の迷惑） |
| 美容院 | Instagram DM | ホットペッパー経由 | 電話 |
| 士業 | 公式サイト問い合わせ | メール | DM（カジュアルすぎ） |
| コーチ | X DM | LinkedIn | Instagram DM（業務外と認識される） |

## NG行動

- フォローした直後にDM送信（スパム判定リスク）
- 同一文面の連続送信
- 返信がない相手への複数回フォロー（1回まで）
- 競合代行サービスを利用中の相手への横取り提案
