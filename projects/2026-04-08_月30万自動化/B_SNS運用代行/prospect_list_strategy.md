# 柱B 営業リスト生成戦略（2026-04-13）

SNS運用代行の見込み客を100社リストアップする手順。

---

## ターゲット業種（優先順）

1. **飲食店（個人店）** — Instagram投稿が少ない店が多い・月5万の予算OK
2. **美容室・整体院・サロン** — リピート促進ニーズ・SNS重要度高い
3. **税理士・社労士・行政書士** — 情報発信で差別化したい
4. **ヨガ・フィットネススタジオ** — 地域密着・定期集客が課題
5. **オンラインスクール・コーチ** — 個人ブランド構築が必要

---

## リスト生成の3ルート

### ルート1：Google Maps検索（推奨）

**手順：**
1. Google Mapsで「地域名 + 業種」で検索
   例：「渋谷 個人経営 イタリアン」「三軒茶屋 カフェ」
2. 上位表示の店舗をリストアップ
3. **Instagram欄がない / フォロワー少ない / 投稿が3ヶ月以上停滞している店** を選別
4. 店名・住所・電話番号・Web・Instagramをスプレッドシートに記録

**1時間で20件ピックアップ可能。**

**ターゲット条件：**
- 食べログ3.0以上（一定の集客力あり）
- Instagram投稿頻度：月3投稿以下
- フォロワー：500〜5,000人（小規模〜中規模）

### ルート2：Instagram直接検索

**ハッシュタグ検索：**
- `#渋谷グルメ` `#三軒茶屋カフェ` `#新宿ランチ`
- `#地域名+業種` で最新投稿をチェック
- 投稿が月2〜3本で止まっているアカウントを狙う

### ルート3：食べログ・ホットペッパーからの収集

**食べログ：**
- 地域×業種で絞り込み
- 各店舗ページの「お店のSNS」欄をチェック
- Instagramアカウントがあるが運用が止まっている店が狙い目

**ホットペッパービューティー（美容室系）：**
- 地域×業種で絞り込み
- 予約サイトは充実してるがSNS発信が弱い店が多い

## リスト管理スプレッドシート（テンプレ）

### 列構成

```
A: No.
B: 店名
C: 業種
D: 地域
E: 住所
F: 電話番号
G: Webサイト
H: Instagramアカウント
I: フォロワー数
J: 直近投稿日
K: 優先度（A/B/C）
L: アプローチ方法
M: アプローチ日
N: 返信状況
O: 契約/失注
P: メモ
```

### 優先度の基準

- **A（優先）**：Instagram停滞・フォロワー1000〜5000・食べログ3.3以上
- **B（中）**：フォロワー500〜1000・投稿月2回以下
- **C（低）**：すでに運用されている・フォロワー多すぎる

## アプローチ順序（5つの段階）

### Day 1：Instagram DM
```
はじめまして、○○と申します。
[店名]様の[料理名/写真/雰囲気]が素敵で拝見しました。

SNS運用代行のご案内で失礼します。
月15投稿（投稿文・ハッシュタグ・予約投稿）を
月5万円で代行しております。

初月は半額の2.5万円でお試し可能です。
15分の無料相談はいかがでしょうか？

ご都合よろしい時にご返信いただければ幸いです。
```

### Day 3：ストーリーのいいね・足跡
- DM返信がなくてもストーリーに反応を残す
- 認識してもらう効果

### Day 7：メール（問い合わせフォームから）
```
件名：【SNS運用代行のご提案】月5万円で投稿を代行します

[店名] 様

先日InstagramでもDMをお送りした○○と申します。
SNS運用代行を専門としております。

御店のInstagramを拝見し、お料理や雰囲気の素晴らしさが
もっと多くの方に届けば、集客にも直結すると感じました。

【ご提案】
・月15投稿の投稿文・ハッシュタグ作成
・予約投稿・月次レポート
・月額50,000円（初月半額25,000円でお試し可）

15分の無料オンライン相談はいかがでしょうか？
ご都合の良い日時をお知らせいただければ、調整いたします。

お忙しいところ恐縮ですが、ご検討いただけますと幸いです。
```

### Day 14：電話（ランチ営業の合間が狙い目）
- 昼営業と夜営業の間（14:00〜16:00）が繋がりやすい
- 「DM・メール差し上げた○○です」と切り出す

### Day 21：最終フォロー
- まだ返信なければ諦めて次のリストへ
- 3ヶ月後にリトライする
