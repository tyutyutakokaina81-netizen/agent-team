## 1. 営業開始前チェックリスト

営業活動を始める前に、以下のすべてが完了していることを確認する。

### プロフィール・ポートフォリオ準備

- [ ] **自社SNSアカウントの整備**
  - [ ] Instagram ビジネスアカウントを開設し、プロフィールを整備した
  - [ ] X（Twitter）プロフィールに「SNS運用代行」を明記した
  - [ ] 各アカウントに最低10投稿のサンプルを公開した
- [ ] **ポートフォリオ準備**
  - [ ] サンプル投稿を3業種分（飲食・士業・コーチ）作成した
  - [ ] 各業種につき投稿文5本＋画像テキスト案を用意した
  - [ ] Canvaでサンプル画像を3〜5枚作成した
- [ ] **営業ツール準備**
  - [ ] サービス概要資料（PDF 1枚）を作成した
  - [ ] 料金表を作成した（スモール/スタンダード/フルの3プラン）
  - [ ] 営業メールテンプレを3業種分用意した（proposal_templates.md 参照）
  - [ ] ヒアリングシートを準備した（本書セクション4参照）

### インフラ・ツール準備

- [ ] **投稿管理ツール**
  - [ ] Buffer または Meta Business Suite のアカウントを開設した
  - [ ] 予約投稿の操作手順を確認した
- [ ] **デザインツール**
  - [ ] Canvaのテンプレートを3〜5パターン用意した
  - [ ] 業種別の配色・フォントパターンを決めた
- [ ] **コミュニケーションツール**
  - [ ] Zoom/Google Meetの無料プランを確認した
  - [ ] オンライン面談の予約リンク（Calendlyなど）を用意した
- [ ] **レポートツール**
  - [ ] 月次レポートテンプレを用意した（本書セクション6参照）

### 契約関連準備

- [ ] **書類準備**
  - [ ] 業務委託契約書のテンプレートを用意した（CFO管轄）
  - [ ] 秘密保持契約（NDA）のテンプレートを用意した
  - [ ] 請求書テンプレートを用意した

## 2. 営業リスト作成方法

### 2-1. Google Maps を使った地域ビジネスの発掘

**手順**

1. Google Maps で以下のキーワードを検索する
   - 「[地域名] カフェ」「[地域名] 美容院」「[地域名] 整体」
   - 「[地域名] 税理士」「[地域名] 行政書士」
2. 検索結果から以下の条件に当てはまる店舗をリストアップする
   - 口コミ評価3.5以上（サービス品質がある程度保証される）
   - WebサイトまたはSNSアカウントが存在する
   - SNSの投稿頻度が低い（月2〜3回以下）または更新が止まっている
3. スプレッドシートに以下を記録する

| 店名 | 業種 | エリア | Instagram | X | Webサイト | 投稿頻度 | 連絡先 | 備考 |
|------|------|--------|-----------|---|-----------|---------|--------|------|
| | | | | | | | | |

### 2-2. Instagram 検索を使った見込み客の発掘

**手順**

1. Instagram で以下のハッシュタグを検索する
   - `#[地域名]カフェ` `#[地域名]ランチ` `#[地域名]美容室`
   - `#個人サロン` `#自宅サロン` `#整体院`
2. 投稿からビジネスアカウントを見つける
3. 以下の条件に該当するアカウントをリストアップする
   - フォロワー100〜5,000人（中小規模＝外注ニーズあり）
   - 投稿頻度が低い（週1回以下）
   - 投稿内容に改善余地がある（写真だけでテキストが薄い等）
4. DMを送れるようフォローしておく

### 2-3. 地域ビジネスディレクトリの活用

**活用するサイト**

| サイト | 使い方 |
|--------|--------|
| 食べログ / ホットペッパー | 飲食店・美容院のリストアップ |
| エキテン | 地域の各種サービス業 |
| ミツモア | 士業・専門家のリスト |
| ジモティー | 地域の個人事業主 |
| タウンページ（iタウンページ） | 業種別の地域ビジネス検索 |

**手順**

1. 各サイトで業種・地域を絞り込む
2. 掲載されている事業者のSNSアカウントを確認する
3. SNSが未整備または更新頻度が低い事業者をリストに追加する

### 2-4. 営業リストの管理目標

| 期間 | 週間リスト追加数 | 累計リスト数 |
|------|-----------------|-------------|
| 1週目 | 30社 | 30社 |
| 2週目 | 30社 | 60社 |
| 3週目 | 20社 | 80社 |
| 4週目 | 20社 | 100社 |

※毎月最低100社をリストアップし、継続的にアプローチする

## 3. アプローチ手順

### 3-1. アプローチの優先順位

```
第1優先：Instagram DM（返信率が最も高い）
  ↓
第2優先：メール（問い合わせフォーム経由）
  ↓
第3優先：電話（飲食店など対面ビジネス向け）
```

### 3-2. Instagram DM アプローチ

**手順**

1. 対象アカウントをフォローする
2. 最新の投稿にいいね・コメントを残す（3〜5日間）
3. 関係構築後にDMを送る

**DM テンプレ**

```
○○様

はじめまして。
SNS運用のサポートをしている○○と申します。

○○様の投稿を拝見しました。
[具体的な投稿内容に触れる一言]がとても素敵ですね。

もし投稿作成の手間を減らしたいとお感じでしたら、
月5万円で投稿文の作成から予約投稿まで
すべてお任せいただけるサービスがございます。

初月は半額の25,000円でお試しいただけます。

ご興味があれば、15分ほどお話しさせていただけますか？
お気軽にお返事ください。
```

### 3-3. メール/問い合わせフォーム アプローチ

**手順**

1. Webサイトの問い合わせフォームまたは公開メールアドレスを確認する
2. 業種別のメールテンプレを使って送信する（proposal_templates.md 参照）
3. 3日後に返信がなければフォローメールを送る

**フォローメール テンプレ**

```
件名：Re: SNS運用のご提案（先日のフォローアップ）

○○様

先日はSNS運用代行のご提案をお送りいたしました。
お忙しいところ恐れ入りますが、ご確認いただけましたでしょうか。

無料で15分のオンライン相談を実施しておりますので、
「SNSは気になるけど時間がない」という場合でも
お気軽にご相談いただければ幸いです。

ご都合の良いお日にちをお知らせいただければ、
すぐにご案内いたします。
```

### 3-4. 電話アプローチ（飲食店向け）

**手順**

1. 14:00〜16:00のアイドルタイムに電話する（ランチ後〜ディナー前）
2. 以下のスクリプトで30秒以内に要点を伝える

**電話スクリプト**

```
お忙しいところ失礼いたします。
SNS運用のサポートをしている○○と申します。

御店のInstagramを拝見して、お料理の写真が素敵だったので
お電話させていただきました。

投稿を週3〜4回に増やすと来店数が上がるケースが多いのですが、
毎日の営業でお忙しいオーナー様に代わって
投稿の作成から予約投稿まで月5万円で代行しております。

もしご興味があれば、15分だけオンラインでご説明できますが、
いかがでしょうか？

→ はい → 日時を調整
→ いいえ → 「承知しました。資料だけメールでお送りしてもよろしいですか？」
```

### 3-5. アプローチ数の目標

| 方法 | 日次目標 | 週間目標 | 月間目標 |
|------|---------|---------|---------|
| Instagram DM | 5件 | 25件 | 100件 |
| メール | 3件 | 15件 | 60件 |
| 電話 | 2件 | 10件 | 40件 |
| **合計** | **10件** | **50件** | **200件** |

## 5. 契約〜開始フロー

### フロー全体図

```
Step 1：初回面談（15〜30分・オンライン）
  ↓
Step 2：ヒアリングシート回収・整理
  ↓
Step 3：提案書＋見積書の送付（面談から24時間以内）
  ↓
Step 4：契約書の締結
  ↓
Step 5：アカウント引き継ぎ・アクセス権の取得
  ↓
Step 6：初月投稿カレンダー作成（Claude使用）
  ↓
Step 7：投稿カレンダーをクライアントに確認
  ↓
Step 8：予約投稿の設定・運用開始
```

### 各ステップ詳細

**Step 1：初回面談**
- Zoom/Google Meetで15〜30分実施する
- ヒアリングシートに沿って質問する
- 「まずは初月半額でお試しいただけます」で背中を押す
- 面談後に御礼メールを送る

**面談後 御礼メールテンプレ**

```
○○様

本日はお時間をいただきありがとうございました。

お話を伺い、○○様の[業種]における
SNS活用には大きな可能性があると感じました。

本日のお話を踏まえて、
明日中に正式なご提案書をお送りいたします。

ご不明な点がございましたら、
いつでもお気軽にご連絡ください。
```

**Step 2：ヒアリングシート整理**
- ヒアリング内容をClaude用のプロンプトに変換する
- 業種・ターゲット・トーン・NG事項を整理する

**Step 3：提案書＋見積書送付**
- proposal_templates.md の業種別テンプレをベースに作成する
- 見積書には以下を明記する
  - プラン名・月額・投稿数
  - 契約期間（最低3ヶ月推奨）
  - 初月半額キャンペーン
  - 含まれるサービス内容

**Step 4：契約書締結**
- 業務委託契約書を送付する（CFO管轄）
- 含めるべき条項：
  - 業務範囲（投稿文作成・予約投稿・月次レポート）
  - 月額報酬・支払いサイクル
  - 契約期間・解約条件（1ヶ月前通知）
  - 秘密保持条項
  - 知的財産権（投稿の著作権はクライアントに帰属）
- 電子契約（クラウドサイン等）で締結する

**Step 5：アカウント引き継ぎ**
- 以下のアクセス権をクライアントから取得する

| 媒体 | 必要なアクセス権 |
|------|----------------|
| Instagram | ビジネスアカウントの共同管理者 or Meta Business Suite招待 |
| X (Twitter) | TweetDeck のチーム招待 or アカウント情報共有 |
| Facebook | ページの編集者権限 |

- パスワードを直接共有する場合は、セキュリティリスクを説明し共同管理者権限を推奨する

**Step 6：初月投稿カレンダー作成**
- ヒアリング情報をもとにClaude で投稿文を一括生成する
- proposal_templates.md のClaudeプロンプトテンプレを使用する
- 投稿カレンダーをスプレッドシートにまとめる

| 日付 | 曜日 | 媒体 | 投稿文 | ハッシュタグ | 画像テキスト案 | ステータス |
|------|------|------|--------|-------------|---------------|----------|
| | | | | | | 下書き/承認済/投稿済 |

**Step 7：クライアント確認**
- 投稿カレンダーをクライアントに送付する
- 修正点があれば反映する
- 承認を得てから予約投稿を設定する

**Step 8：運用開始**
- Buffer / Meta Business Suite で予約投稿を設定する
- 投稿が正常に公開されたことを確認する
- 初週は毎日投稿結果をクライアントに報告する
