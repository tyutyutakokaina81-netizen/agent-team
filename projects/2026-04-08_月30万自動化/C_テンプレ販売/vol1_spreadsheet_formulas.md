# Vol.1 フリーランス収支管理スプレッドシート
# Googleスプレッドシート 完全設計書（そのまま作れる）

---

## 作り方（所要時間：約30分）

1. Googleスプレッドシートを新規作成
2. 以下の4シートを作る：「収入」「経費」「案件」「サマリー」
3. 下記のセル設定・数式をそのままコピーする
4. 完成したらリンクを「コピーを作成」で共有設定にする

---

## シート1：収入

### 列構成（1行目：ヘッダー）
```
A1: 日付
B1: クライアント名
C1: 案件名
D1: 請求金額（税抜）
E1: 消費税
F1: 請求金額（税込）
G1: 入金日
H1: 入金確認
I1: 請求書番号
J1: 備考
```

### 数式
```
E2: =D2*0.1                        （消費税10%自動計算）
F2: =D2+E2                         （税込自動計算）
H2: チェックボックス挿入（挿入→チェックボックス）

月合計行（例：34行目）
D34: =SUMIF(A2:A33,">="&DATE(YEAR(TODAY()),MONTH(TODAY()),1),D2:D33)
```

## シート2：経費

### 列構成
```
A1: 日付
B1: 科目
C1: 金額
D1: 支払方法
E1: 摘要
F1: レシートあり
```

### B列：科目プルダウン設定
```
データ→データの入力規則→リストから項目を選択：
通信費,交通費,ソフトウェア,書籍・研修,外注費,消耗品,接待交際費,その他
```

### 科目別集計（H列以降に配置）
```
H1: 科目名       I1: 合計
H2: 通信費       I2: =SUMIF(B:B,"通信費",C:C)
H3: 交通費       I3: =SUMIF(B:B,"交通費",C:C)
H4: ソフトウェア  I4: =SUMIF(B:B,"ソフトウェア",C:C)
H5: 書籍・研修   I5: =SUMIF(B:B,"書籍・研修",C:C)
H6: 外注費       I6: =SUMIF(B:B,"外注費",C:C)
H7: その他       I7: =SUMIF(B:B,"その他",C:C)
H8: 合計         I8: =SUM(I2:I7)
```

## シート3：案件管理

### 列構成
```
A1: 案件名
B1: クライアント
C1: 単価（円）
D1: 稼働時間（h）
E1: 時給（自動）
F1: 開始日
G1: 終了日
H1: ステータス
I1: 備考
```

### 数式・設定
```
E2: =IF(D2=0,"",C2/D2)            （時給自動計算）
E列: 書式→数値→￥#,##0            （円表示）

H列プルダウン：
提案中,進行中,納品済,請求済,入金済,完了
```

## シート4：サマリー（自動集計・グラフ）

### 月別収支テーブル
```
A1: 月
B1: 収入（税抜）
C1: 経費
D1: 手取り
E1: 前月比

A2: 1月
B2: =SUMPRODUCT((MONTH(収入!A$2:A$200)=1)*(YEAR(収入!A$2:A$200)=YEAR(TODAY()))*収入!D$2:D$200)
C2: =SUMPRODUCT((MONTH(経費!A$2:A$200)=1)*(YEAR(経費!A$2:A$200)=YEAR(TODAY()))*経費!C$2:C$200)
D2: =B2-C2
E2: （2月以降）=D2-D1
```

### 年間サマリー（自動）
```
年間収入:  =SUM(B2:B13)
年間経費:  =SUM(C2:C13)
年間手取り: =SUM(D2:D13)
最高月:    =MAX(B2:B13)
最低月:    =MIN(B2:B13)
```

### グラフ設定
```
月別収入グラフ：A1:B13を選択→挿入→グラフ→棒グラフ
月別経費グラフ：A1:A13,C1:C13を選択→挿入→グラフ→折れ線
```
