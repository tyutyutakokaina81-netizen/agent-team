## 契約書1：業務委託契約書

```
業務委託契約書

委託者 [甲：会社名／氏名] と 受託者 [乙：氏名／屋号]（以下それぞれ「甲」「乙」という）は、
下記業務の委託について次のとおり契約を締結する。

第1条（業務内容）
乙は甲に対し、以下の業務を遂行する：
[業務内容を具体的に記入：例 SEO記事執筆 月10本 / SNS運用月15投稿 等]

第2条（期間）
本契約の期間は [開始日：YYYY年MM月DD日] から [終了日：YYYY年MM月DD日] までとする。
ただし、契約期間満了の1ヶ月前までに双方から異議がない場合、自動的に1ヶ月間延長されるものとし、以後同様とする。

第3条（報酬）
1. 甲は乙に対し、本業務の対価として金 [金額] 円（消費税別）を支払う。
2. 支払いは [月末締め翌月末払い／納品後7日以内 等] とする。
3. 振込手数料は [甲／乙] が負担する。

第4条（成果物の権利）
1. 本業務の成果物に係る著作権（著作権法27条・28条の権利を含む）は、報酬完済時に甲に移転する。
2. 乙は成果物について著作者人格権を行使しない。
（※ 著作権を乙に残したい場合は本条を削除し、別途使用許諾契約を結ぶ）

第5条（秘密保持）
1. 双方は、本契約遂行上知り得た相手方の業務上の情報を秘密として保持し、第三者に開示・漏洩しない。
2. 本義務は契約終了後も2年間継続する。

第6条（契約解除）
1. 双方は1ヶ月前の書面通知により本契約を解除できる。
2. 一方が本契約に違反し、催告後14日以内に是正されない場合、相手方は本契約を即時解除できる。

第7条（損害賠償）
本契約違反により相手方に損害を与えた場合、賠償責任は本契約上の報酬月額の3倍を上限とする。
（※ 相互の責任を限定する条項。削除可）

第8条（協議事項）
本契約に定めなき事項、または解釈に疑義が生じた事項は、双方誠実に協議の上、決定する。

第9条（管轄）
本契約に関する紛争は、[甲または乙の所在地] を管轄する地方裁判所を第一審の専属的合意管轄裁判所とする。

以上、本契約締結の証として本書2通を作成し、甲乙署名押印の上、各自1通を保有する。

[締結日：YYYY年MM月DD日]

甲：[住所]
   [氏名／会社名]                印

乙：[住所]
   [氏名／屋号]                  印
```

## 契約書2：秘密保持契約書（NDA）

```
秘密保持契約書

[甲：会社名／氏名] と [乙：氏名／屋号]（以下それぞれ「甲」「乙」という）は、
両当事者間の業務に関連して開示される秘密情報の取扱いについて、次のとおり契約を締結する。

第1条（秘密情報の定義）
本契約における「秘密情報」とは、以下のいずれかに該当する情報をいう：
1. 開示当事者が「秘密」と明示した情報
2. 顧客情報、取引先情報、技術情報、財務情報、人事情報
3. 製品・サービスの開発・運用に関する情報
4. その他通常秘密として扱われる情報

ただし、以下の情報は秘密情報に含まれない：
・既に公知の情報
・受領前から自己が保有していた情報
・正当な権利を有する第三者から守秘義務なく取得した情報
・独自に開発した情報

第2条（秘密保持義務）
1. 受領者は秘密情報を厳重に管理し、第三者に開示しない。
2. 秘密情報は本契約の目的達成に必要な範囲でのみ使用する。
3. 受領者の役員・従業員・委託先に開示する場合、本契約と同等以上の守秘義務を課す。

第3条（複製・記録）
1. 秘密情報の複製は本契約目的達成に必要な範囲に限る。
2. 受領者は秘密情報の管理状況について開示者の求めに応じ報告する。

第4条（返却・廃棄）
本契約終了時または開示者の要求時、受領者は秘密情報の記録媒体を速やかに返却または廃棄する。

第5条（有効期間）
1. 本契約の有効期間は締結日から [1〜3] 年とする。
2. 本契約終了後も、第2条の秘密保持義務は契約終了後 [2〜5] 年間有効とする。

第6条（違反時の措置）
本契約違反により開示者に損害が生じた場合、受領者は損害賠償責任を負う。
ただし、賠償額は秘密情報の経済価値および違反の程度を勘案し決定する。

第7条（管轄）
本契約に関する紛争は、[甲または乙の所在地] を管轄する地方裁判所を第一審の専属的合意管轄裁判所とする。

以上、本契約締結の証として本書2通を作成し、甲乙署名押印の上、各自1通を保有する。

[締結日：YYYY年MM月DD日]

甲：[住所／氏名]                印
乙：[住所／氏名]                印
```

## 契約書3：月額顧問契約書

```
月額顧問契約書

[甲：会社名／氏名] と [乙：氏名／屋号] は、乙が甲に対して継続的に専門的助言・支援を提供することについて、次のとおり契約を締結する。

第1条（顧問業務の内容）
乙は甲に対し、毎月以下の業務を提供する：
1. [専門分野] に関する相談対応（月 [回数] 回、1回あたり [時間]）
2. [具体業務：例 SNS運用方針提案／月次レポート作成／施策レビュー 等]
3. 緊急事態における初期対応助言

第2条（顧問業務の範囲外）
以下は本契約の対象外とし、別途見積もりにて対応する：
・実装作業（コーディング・デザイン制作 等）
・第三者対応（クライアント対応・取材対応）
・出張を伴う業務

第3条（顧問料）
1. 月額顧問料は金 [金額] 円（消費税別）とする。
2. 支払いは毎月末日締め翌月末日払いとする。
3. 業務量にかかわらず月額固定とする（ただし第2条の範囲外業務は別途）。

第4条（期間）
1. 本契約の期間は [開始日] から [終了日] までの [N] ヶ月とする。
2. 期間満了の1ヶ月前までに双方から異議がない場合、自動的に1ヶ月間延長される。

第5条（解約）
1. 双方は1ヶ月前の書面通知により本契約を解約できる。
2. 解約月の顧問料は満額発生する（日割り計算なし）。

第6条（秘密保持）
乙は顧問業務遂行上知り得た甲の情報を、契約終了後3年間にわたり秘密として保持する。

第7条（責任の制限）
乙の助言は最善のものを提供するが、結果保証は行わない。
甲が乙の助言に基づき意思決定する責任は甲にあり、結果について乙は責任を負わない。

第8条（管轄）
本契約に関する紛争は、[甲または乙の所在地] を管轄する地方裁判所を第一審の専属的合意管轄裁判所とする。
```

```
[例：YYYY年MM月制作の Web サイトデザイン一式、ロゴデザイン1点 等]

第2条（使用許諾の範囲）
1. 乙は甲に対し、本件成果物について以下の使用を許諾する：
   ・媒体：[Web／印刷物／映像 等]
   ・地域：[日本国内のみ／全世界 等]
   ・期間：[締結日から N 年間／無期限 等]

第3条（著作権の帰属）
本件成果物の著作権は乙に帰属する。
本契約は使用権の付与であり、著作権の譲渡ではない。

第4条（クレジット表記）
甲は本件成果物使用時、可能な範囲で「Design by [乙の氏名]」のクレジット表記を行う。

第5条（修正・改変）
1. 甲は本件成果物の軽微な修正（色味調整・サイズ変更等）を自由に行える。
2. 大幅な改変を行う場合は乙の事前書面同意を得る。

第6条（許諾料）
1. 本許諾の対価は金 [金額] 円（消費税別）とする。
2. 支払いは [本契約締結後7日以内／月末締め翌月末払い 等] とする。

第7条（保証）
乙は本件成果物が第三者の権利を侵害していないことを保証する。

第8条（解除）
```

```
本覚書の対象は、以下の業務とする：
[業務内容と納期の概要]

2. ただし、業務範囲の追加・変更が生じた場合は、双方協議の上、書面にて報酬を見直す。

第3条（支払時期・方法）

   ・着手金：契約締結後7日以内に [N] %

協議で解決できない場合は [甲または乙の所在地] を管轄する地方裁判所を第一審の専属的合意管轄裁判所とする。
```
