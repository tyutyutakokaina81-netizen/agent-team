## プロンプト本文

### 1. 新規顧問契約の提案メール
```
あなたは税理士事務所の代表です。
新規顧問契約の提案メールを作成してください。

【対象】
- 業種：[業種]
- 規模：[従業員N名・年商XXX万円]
- 紹介経路：[紹介者名 / セミナー / WEB問合せ]

【提案内容】
- 顧問料：月額[金額]
- 含まれるサービス：[記帳代行/月次決算/相談など]
- 開始希望時期：[日付]

【トーン】
- 丁寧・落ち着いた言葉遣い
- 押し付けがましくない
- 信頼を感じさせる文体
- 400〜500字
- 末尾に簡潔な料金表を箇条書き
```

### 2. 顧問料改定のお知らせ
```
税理士・社労士事務所として、顧問料改定のお知らせメールを作成してください。

【改定情報】
- 改定理由：[業務量増加 / 人件費高騰 / サービス拡充]
- 現行料金：[月額]
- 新料金：[月額]
- 適用開始：[日付]

【条件】
- 既存顧客への配慮を最優先
- 価格差の合理的な説明
- 既存サービスは維持/拡充されることを明記
- 350〜400字
- 質疑応答の窓口（電話・メール）を明示
```

### 3. 確定申告繁忙期の予約打診
```
税理士として、確定申告期の予約打診メールを作成してください。

【対象】
- 既存顧問先：[個人事業主 / 法人]
- 提出期限：[日付]
- 必要書類：[書類リスト]

【条件】
- 早期予約のメリットを明記（混雑回避）
- 必要書類リストを箇条書き
- 提出期限から逆算した推奨予約日
- 250〜300字
```

### 4. 社労士：労務相談セミナー告知
```
社会保険労務士として、無料労務相談セミナーの告知メールを作成してください。

【セミナー情報】
- テーマ：[例：ハラスメント対策・就業規則・36協定]
- 日時：[日付・時間]
- 形式：[オンライン / 対面]
- 定員：[人数]
- 対象：[経営者 / 人事担当者]

【条件】
- 経営者の悩みに寄り添う書き出し
- 講師（自分）の専門性を1〜2文で示す
- 申込導線をわかりやすく
- 350字程度
```

### 5. 行政書士：許認可手続き案内
```
行政書士として、許認可手続きのご案内メールを作成してください。

【対象案件】
- 業種：[飲食業 / 建設業 / 古物商 / 介護事業 等]
- 必要な許認可：[許認可名]
- 取得期間：[週間〜ヶ月]
- 報酬：[金額]

【条件】
- リスク回避（無許可営業の罰則）に触れる
- 当事務所のサポート範囲を明示
- 必要書類リストを箇条書き
- 400字程度
```

### 6. 顧問先への定期挨拶（季節）
```
士業として、顧問先への季節挨拶メールを作成してください。

【季節】
- [春 / 梅雨 / 夏 / 秋 / 冬 / 年末年始]

【情報】
- 業種特有の注意点：[該当する税務 / 労務 / 法務トピック]
- 本月の重要期日：[日付]

【条件】
- 季節感のある時候の挨拶（堅すぎない）
- 業務に役立つ情報を1つ含める
- 250〜300字
- 末尾に「お気軽にご相談ください」
```

### 7. 紹介依頼メール
```
士業として、既存顧問先へ新規紹介を依頼するメールを作成してください。

【条件】
- 既存サービスへの感謝を冒頭に
- 紹介してほしい理想の顧客像を具体的に
- 紹介特典（あれば）を明記：[例：紹介者向け顧問料1ヶ月割引]
- 強要せず、断りやすい文面
- 300字程度
```

### 8. 料金見積りメール
```
士業として、料金見積りメールを作成してください。

【見積内容】
- 案件種別：[許認可 / 確定申告 / 助成金 / 就業規則]
- 報酬：[金額]
- 着手金：[金額・有無]
- 成果報酬：[金額・有無]
- 期間：[週間〜ヶ月]

【条件】
- 内訳を表形式で明示
- 含まれるもの・含まれないものを区分
- 有効期限を記載
- 400字程度
```

### 9. 契約継続のお願い
```
士業として、契約継続のお願いメールを作成してください。

【契約情報】
- 契約終了日：[日付]
- 現行月額：[金額]
- 継続条件：[現状維持 / 更新値上げ / プラン変更]

【条件】
- 1年間の貢献実績を1〜2文でまとめる
- 来期の予定（法改正・繁忙期）に触れる
- 押し付けず、判断を委ねる文面
- 350字程度
```

### 10. 契約終了のお見送り（円満）
```
士業として、契約終了時の円満な見送りメールを作成してください。

【条件】
- 1年間（または期間）の感謝を冒頭に
- 引き継ぎが必要な書類を箇条書き
- 後任の士業へ伝えるべき情報
- 何かあれば再度ご連絡くださいの一文
- 関係を断たない・敵を作らない文体
- 300字程度
```

### 11. 法改正のご案内（注意喚起）
```
士業として、法改正のご案内メールを作成してください。

【改正情報】
- 法律名：[例：改正電子帳簿保存法 / 改正労働基準法]
- 施行日：[日付]
- 影響範囲：[対象事業者]
- 対応期限：[日付]

【条件】
- 「いつ」「何が変わる」「どう対応すべき」の3点を明確に
- 期限切れのリスクを冷静に伝える
- サポート可能な範囲を明示
- 400字程度
```

### 12. 助成金・補助金のお知らせ
```
士業として、助成金・補助金のお知らせメールを作成してください。

【助成金情報】
- 助成金名：[名称]
- 対象：[要件]
- 金額：[上限]
- 申請期限：[日付]

【条件】
- 「条件に当てはまる可能性がある方へ」のスタンス
- 当事務所のサポート費用を明示
- 失敗事例（受給漏れ）に軽く触れる
- 350字程度
```

### 13. セミナー / 勉強会の招待
```
士業として、セミナー招待メールを作成してください。

【セミナー】
- テーマ：[内容]
- 講師：[自分 / ゲスト]
- 日時：[日付]
- 場所：[オンライン / 会場]
- 参加費：[無料 / 有料金額]

【条件】
- 学べる3つのポイントを箇条書き
- 既存顧問先への謝意
- 参加者特典（あれば）
- 350字程度
```

### 14. 無料相談の予約確認
```
士業として、無料相談予約の確認メールを作成してください。

【予約情報】
- 日時：[日付・時間]
- 形式：[オンライン / 対面]
- ZoomURL or 住所：[詳細]
- 持参・準備物：[書類リスト]

【条件】
- 簡潔・わかりやすい
- 当日の流れ（開始〜終了）
- キャンセル・変更時の連絡先
- 250字程度
```

### 15. 領収書送付のお知らせ
```
士業として、領収書送付メールを作成してください。

【送付情報】
- 対象期間：[日付]
- 金額：[金額]
- 送付形式：[PDF添付 / 郵送]
- 摘要：[サービス内容]

【条件】
- 簡潔・事務的
- 不明点の問合せ窓口
- 150〜200字
```

### 16. 確定申告書類のご案内
```
税理士として、確定申告書類完成のご案内メールを作成してください。

【書類情報】
- 対象年度：[年]
- 提出方式：[e-Tax / 書面]
- 控除額：[金額]
- 還付/納税：[金額]
- 提出期限：[日付]

【条件】
- 還付・納税のいずれかを冒頭で明示
- 確認事項を箇条書き
- 押印・マイナンバー必要書類を案内
- 350字程度
```

### 17. 急な訪問のお詫び
```
士業として、急な日程変更のお詫びメールを作成してください。

【状況】
- 元の予定：[日付・時間]
- 変更後：[日付・時間 / 候補日]
- 理由：[体調 / 緊急業務 / その他]

【条件】
- 冒頭で謝罪
- 理由は簡潔に（詳細不要）
- 代替日程を3案提示
- 250字程度
```

### 18. 担当者変更のご連絡
```
士業事務所として、担当者変更のご連絡メールを作成してください。

【変更情報】
- 旧担当者：[名前]
- 新担当者：[名前・経歴]
- 変更日：[日付]

【条件】
- 旧担当への感謝
- 新担当の経歴を1〜2文で紹介
- サービス品質維持の約束
- 引継ぎ会の打診
- 300字程度
```

### 19. 年末年始の営業案内
```
士業として、年末年始の営業案内メールを作成してください。

【情報】
- 年末最終営業日：[日付]
- 年始開始日：[日付]
- 緊急時の連絡方法：[メール / 電話]

【条件】
- 1年間の感謝
- 簡潔・読みやすい
- 来年の業務予定を一言
- 200字程度
```

### 20. 異業種交流会への誘い
```
士業として、異業種交流会への誘いメールを作成してください。

【交流会情報】
- イベント名：[名称]
- 日時：[日付]
- 場所：[会場 / オンライン]
- 参加費：[金額]
- 参加業種：[業種リスト]

【条件】
- 紹介者であるあなたとの関係性を冒頭に
- 当日のメリット（誰と会えるか）
- 押し付けず、断りやすい文面
- 300字程度
```

---

## 使い方

1. 上記プロンプトの `[ ]` 部分を実情報に書き換える
2. ChatGPT・Claude・Gemini にコピペして送信
3. 出力された文面を確認・微調整して送信
