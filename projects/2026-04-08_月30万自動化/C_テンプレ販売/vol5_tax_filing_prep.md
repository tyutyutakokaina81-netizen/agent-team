## スプレッドシート設計

### シート1：月次収支入力
| 列 | 項目 | 形式 | 備考 |
|----|------|------|------|
| A | 日付 | YYYY-MM-DD | |
| B | 区分 | プルダウン | 収入/経費 |
| C | 取引先・支払先 | テキスト | |
| D | 内容 | テキスト | |
| E | 経費科目 | プルダウン | 後述の科目リストから |
| F | 金額 | 数値 | 税込 |
| G | 消費税区分 | プルダウン | 課税10%/軽減8%/非課税/対象外 |
| H | 支払方法 | プルダウン | 現金/振込/カード/口座振替 |
| I | 領収書有無 | チェックボックス | |
| J | 備考 | テキスト | |

### シート2：経費科目別集計（自動）
```
SUMIFS で月別×科目別に自動集計
=SUMIFS(月次収支!F:F, 月次収支!E:E, "通信費", 月次収支!A:A, ">="&DATE(2026,1,1), 月次収支!A:A, "<="&DATE(2026,1,31))
```

### シート3：年間サマリ（自動）
- 売上合計
- 経費合計
- 所得金額（売上 − 経費）
- 概算所得税（5%/10%/20%/23%/33%/40%/45% の累進）
- 概算住民税（10%）
- 青色申告特別控除（55万 or 65万 e-Tax）

### シート4：経費科目チートシート

| 経費科目 | OK な例 | NG な例 | 注意点 |
|---------|---------|---------|--------|
| 旅費交通費 | 仕事の電車・タクシー・宿泊 | プライベート旅行 | 出張記録を残す |
| 通信費 | 仕事用スマホ・ネット回線 | 100%家庭用 | 按分（事業使用率） |
| 接待交際費 | 取引先との会食 | 友人との飲み会 | 相手・目的を記録 |
| 消耗品費 | 文具・USBメモリ・PC周辺 | 10万円以上のPC | 10万超は減価償却 |
| 新聞図書費 | 仕事関連の本・有料記事 | 漫画・小説 | レシート保存 |
| 会議費 | 打ち合わせのカフェ代 | プライベート利用 | 1人5,000円以下目安 |
| 広告宣伝費 | SNS広告・名刺・LP制作費 | 私的なプレゼント | 業務との関連性 |
| 支払手数料 | 振込手数料・プラットフォーム手数料 | 個人銀行の手数料 | |
| 地代家賃 | 事務所家賃・コワーキング | 全額自宅 | 自宅は事業使用率で按分 |
| 水道光熱費 | 自宅作業の電気・水道 | 全額計上 | 事業使用率で按分（20〜40%目安） |

| 雑費 | どの科目にも該当しない少額 | 高額・恒常的なもの | 全体の5%以内が目安 |

### シート5：青色 vs 白色 切替ガイド

| 項目 | 白色申告 | 青色申告10万円控除 | 青色申告55万/65万円控除 |

| 赤字繰越 | 不可 | 3年 | 3年 |
| 推奨対象 | 副業初年度 | シンプル運営 | 月収20万円以上 |

## 使い方（購入者向け）

### 月初（5分）
1. シート1「月次収支入力」を開く
2. 当月分の取引を入力（収入・経費）
3. 経費科目はプルダウンから選ぶだけ

### 月末（10分）
1. シート2「集計」で当月の科目別経費を確認
2. 異常に多い科目がないかチェック
3. 領収書をスキャンしてクラウド保存

### 確定申告時（30分）
1. シート3「年間サマリ」を確認
2. 国税庁 e-Tax の入力画面に転記
3. シート5「切替ガイド」で青色/白色を判断
