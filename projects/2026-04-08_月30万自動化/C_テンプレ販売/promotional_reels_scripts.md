## 共通仕様

| 項目 | 設定 |
|------|------|

## 撮影・編集の共通テクニック

### 視聴維持率を上げる5つのコツ
1. **冒頭1秒目で動きを入れる**（顔出し or 何かを手で持つ）
2. **テロップは最大15字×2行まで**（読み切れない）
3. **3秒ごとにカット切り替え**（飽きさせない）
4. **音声がなくても伝わる構成**（音オフ視聴対応）
5. **最後の3秒で「保存・シェア」を促す**

### 編集ツール（無料）
| ツール | 特徴 |
|--------|------|
| CapCut（iOS/Android） | テンプレ豊富・自動字幕 |
| InShot | シンプル・初心者向け |
| Canva | テロップデザイン強い |

### BGM選び（著作権フリー）
- Instagram内の音源（リール用・無料）
- YouTube Audio Library
- DOVA-SYNDROME（日本語サイト）

### 撮影機材（最小構成）
- スマホ1台（iPhone推奨）
- 三脚（¥1,000〜）
- リングライト（¥2,000〜）
- 合計：¥3,000〜 で開始可能

## 投稿後の運用

### 投稿時間（2026年のベスト）
- 平日：12:00-13:00、19:00-21:00
- 土日：10:00-11:00、20:00-22:00
- 業種により調整

### 投稿頻度
- 同じ商品の販促 Reels は **週1回まで**（しつこい印象を避ける）
- 商品紹介と関連の有益情報（無料Tips）を3:1の比率で

### KPI測定
| 指標 | 目安（投稿後7日） |
|-----|----------------|
| 視聴回数 | 1,000以上（初心者）/ 5,000以上（中級） |
| 視聴完了率 | 30%以上 |
| 保存数 | 視聴回数の2-5% |
| シェア数（DM送信） | 視聴回数の0.5-2% |
| プロフィールアクセス | 視聴回数の1-3% |
| 外部リンククリック | プロフアクセスの5-10% |
| 商品購入（最終ゴール） | リンククリックの2-5% |

→ 視聴1,000回 → プロフ20件 → クリック2件 → 購入0.1件 = **だいたい10,000視聴で1本売れる**
