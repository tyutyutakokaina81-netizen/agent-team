## スプレッドシート設計

### シート1：月次カレンダー

| 列 | 項目 | 仕様 |
|----|------|------|
| A | 日付 | YYYY-MM-DD |
| B | 曜日 | `=TEXT(A2,"ddd")` で自動 |
| C | 媒体 | プルダウン：Instagram / X / Facebook / 全媒体 |
| D | テーマ・ネタ | 50選から選択 or 自由入力 |
| E | 投稿文（下書き） | 自由入力 |
| F | ハッシュタグ | カンマ区切り |
| G | 画像有無 | チェックボックス |
| H | 投稿済み | チェックボックス |
| I | いいね数 | 数値 |
| J | 備考 | 自由入力 |

**条件付き書式**
- H列がチェック済 → 行全体を薄いグレーに
- A列が今日と一致 → 行全体を薄い黄色に
- C列の媒体ごとに色分け（Instagram=ピンク、X=ライトブルー、Facebook=ブルー）

### シート3：月次サマリー（自動集計）

| 行 | 項目 | 数式例 |
|----|------|--------|
| 1 | 月 | 2026-05 |
| 2 | 投稿予定数 | `=COUNTA(月次カレンダー!D2:D32)` |
| 3 | 投稿完了数 | `=COUNTIF(月次カレンダー!H2:H32, TRUE)` |
| 4 | 完了率 | `=B3/B2` |
| 5 | Instagram投稿数 | `=COUNTIF(月次カレンダー!C2:C32, "Instagram")` |
| 6 | X投稿数 | `=COUNTIF(月次カレンダー!C2:C32, "X")` |
| 7 | Facebook投稿数 | `=COUNTIF(月次カレンダー!C2:C32, "Facebook")` |
| 8 | いいね合計 | `=SUM(月次カレンダー!I2:I32)` |
| 9 | 平均いいね | `=AVERAGE(月次カレンダー!I2:I32)` |
| 10 | 最も反応の良い投稿 | `=INDEX(月次カレンダー!D:D, MATCH(MAX(月次カレンダー!I:I), 月次カレンダー!I:I, 0))` |
