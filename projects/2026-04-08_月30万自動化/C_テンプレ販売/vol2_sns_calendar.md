## スプレッドシート設計

### シート1：月次カレンダー
```
列構成：
A: 日付
B: 曜日（=TEXT(A2,"ddd")で自動）
C: 媒体（プルダウン：Instagram/X/Facebook/全媒体）
D: テーマ・ネタ
E: 投稿文（下書き）
F: ハッシュタグ
G: 画像有無（チェックボックス）
H: 投稿済み（チェックボックス）
I: いいね数
J: 備考
```

### シート3：使い方ガイド（添付）
```
■ 月初の使い方（30分）
1. シート1で当月の日付を確認
2. シート2の50テーマから当月のテーマを選定
3. 媒体（C列）を選び、テーマ・投稿文（D・E列）を埋める
4. ハッシュタグ（F列）を5〜10個設定

■ 投稿当日（5分）
1. E列の投稿文を確認・微調整
2. SNSに投稿
3. H列のチェックボックスにチェック

■ 月末の振り返り（15分）
1. I列にいいね数を記録
2. 反応が良かったテーマを来月に流用
3. シート2のテーマに新しいネタを追記
```
