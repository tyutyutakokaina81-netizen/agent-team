## スプレッドシート設計

### シート1：月次カレンダー
```
列構成：
A: 日付
B: 曜日（=TEXT(A2,"ddd")で自動）
C: 媒体（プルダウン：Instagram/X/Facebook/全媒体）
D: テーマ・ネタ
E: 投稿文（下書き）
F: ハッシュタグ
G: 画像有無（チェックボックス）
H: 投稿済み（チェックボックス）
I: いいね数
J: 備考
```

### シート2：投稿テーマ50選（付録）

#### 自己発信系（1-10）
```
1. 自己紹介・プロフィール

4. よくある質問（FAQ）
5. 業界の豆知識
6. ビフォーアフター
7. 失敗談・学び
8. 日常の一コマ
9. おすすめツール
10. 季節・イベントネタ
```

#### 教育・有益系（11-20）
```

18. 数字で見る業界トレンド
19. 1分で読める豆知識

```
