## シート1：投稿カレンダー（メインシート）

### 列構成（A〜N列）

| 列 | 項目 | 入力方式 | 数式・備考 |
|----|------|---------|----------|
| A | 日付 | 手入力（日付形式） | `2026/06/01` 形式 |
| B | 曜日 | 自動算出 | `=IF(A2="","",TEXT(A2,"ddd"))` |
| C | 媒体 | プルダウン | Instagram / X / Facebook / TikTok / 全媒体 |
| D | 投稿形式 | プルダウン | フィード / リール / ストーリーズ / ツイート / ショート動画 |
| E | テーマ・ネタ | 手入力 | シート2のテーマから選択推奨 |
| F | 投稿文（下書き） | 手入力（長文） | 改行可・最大1000字目安 |
| G | ハッシュタグ | 手入力 | シート3から抽出推奨 |
| H | 画像/動画素材 | チェックボックス | 準備OKでチェック |
| I | 投稿予約 | チェックボックス | Buffer/Later等で予約済 |
| J | 投稿済 | チェックボックス | 公開完了でチェック |
| K | いいね数 | 手入力（数値） | 投稿後1週間で記録 |
| L | コメント数 | 手入力（数値） | 投稿後1週間で記録 |
| M | 保存/シェア数 | 手入力（数値） | 2026年最重要KPI |
| N | 備考 | 手入力 | 学び・改善メモ |

### ヘッダー行（1行目）
```
| 日付 | 曜日 | 媒体 | 形式 | テーマ | 投稿文 | ハッシュタグ | 素材OK | 予約済 | 投稿済 | いいね | コメント | 保存/シェア | 備考 |
```

### データ入力エリア（2〜32行目）
最大31日分の入力欄。月によって29-31日まで使用。

### 主要な数式

#### B列：曜日自動算出
```
B2: =IF(A2="","",TEXT(A2,"ddd"))
```
→ A2に「2026/06/01」と入れると B2 に「月」と表示

#### B列の条件付き書式（土日に色付け）
- 範囲：`B2:B32`
- 条件1：`=B2="土"` → 背景青系（#cfe2ff）
- 条件2：`=B2="日"` → 背景赤系（#f8d7da）

### プルダウン設定（データ入力規則）

#### C列：媒体
- 範囲：`C2:C32`
- 条件：リストから直接（カンマ区切り）
- 値：`Instagram,X,Facebook,TikTok,全媒体`

#### D列：投稿形式
- 範囲：`D2:D32`
- 値：`フィード,リール,ストーリーズ,ツイート,ショート動画,スレッド`

### チェックボックス設定

- H列、I列、J列に「データ → チェックボックス」を適用
- 範囲：`H2:J32`

### 進捗バー（K列右に表示する場合・オプション）
```
オプション列O: =IF(J2=TRUE,"✅完了",IF(I2=TRUE,"⏰予約済",IF(H2=TRUE,"📷準備中","🔲未着手")))
```

## シート4：KPIダッシュボード（自動集計）

### A列：項目 / B列：数値（自動算出）

| セル | 項目 | 数式 |
|------|------|------|
| A2 | 投稿予定数（今月） | =COUNTA(投稿カレンダー!E2:E32) |
| A3 | 投稿済み数 | =COUNTIF(投稿カレンダー!J2:J32,TRUE) |
| A4 | 達成率 | =B3/B2 （%表示） |
| A5 | 媒体別：Instagram | =COUNTIF(投稿カレンダー!C2:C32,"Instagram") |
| A6 | 媒体別：X | =COUNTIF(投稿カレンダー!C2:C32,"X") |
| A7 | 媒体別：Facebook | =COUNTIF(投稿カレンダー!C2:C32,"Facebook") |
| A8 | 媒体別：TikTok | =COUNTIF(投稿カレンダー!C2:C32,"TikTok") |
| A10 | 総いいね数 | =SUM(投稿カレンダー!K2:K32) |
| A11 | 平均いいね数 | =IFERROR(B10/B3,0) |
| A12 | 総コメント数 | =SUM(投稿カレンダー!L2:L32) |
| A13 | 総保存/シェア数 | =SUM(投稿カレンダー!M2:M32) |
| A14 | 平均エンゲージメント率 | =IFERROR((B10+B12+B13)/B3,0) |

### グラフ作成手順（オプション）
1. B5:B8 を選択 → 「挿入 → グラフ」→ 円グラフで媒体別投稿割合を可視化
2. 投稿カレンダー A2:A32 と K2:K32 を選択 → 折れ線グラフで日別いいね数推移

## 実装手順書（オーナー向け・所要30分）

### Step 1：Googleスプレッドシート新規作成（3分）
1. https://sheets.google.com にアクセス
2. 「+新規スプレッドシート」をクリック
3. ファイル名を「SNS投稿カレンダー_テンプレ」に変更

### Step 2：シート1「投稿カレンダー」作成（10分）
1. 1行目に上記ヘッダーをコピペ
2. A2 セルに `=DATE(2026,6,1)` と入力（任意の月の1日）
3. A3 に `=A2+1` を入れて A32 までドラッグコピー
4. B2 に `=IF(A2="","",TEXT(A2,"ddd"))` を入れて B32 までコピー
5. C列、D列にデータ入力規則（リスト）を設定
6. H2:J32 にチェックボックス挿入
7. B列に条件付き書式（土日色付け）

### Step 3：シート2「投稿テーマ50選」作成（5分）
1. 下部の「+」で新シート追加 → 名前を「投稿テーマ50選」
2. 上記50テーマ表をコピペ

### Step 4：シート3「ハッシュタグライブラリ」作成（5分）
1. 新シート追加 → 名前を「ハッシュタグ」
2. 上記100タグリストをコピペ

### Step 5：シート4「KPIダッシュボード」作成（5分）
1. 新シート追加 → 名前を「KPI」
2. 上記数式を A列・B列に入力
3. 円グラフ・折れ線グラフを挿入（任意）

### Step 6：販売用にコピー設定（2分）
1. 右上「共有」→「リンクを取得」→「閲覧者」設定
2. URL末尾の `/edit?usp=sharing` を `/copy` に変更してから販売
   → クリックでコピー作成画面が出る

## 購入者向け使い方マニュアル（PDF用・要約）

```
■ STEP 1：自分のドライブにコピー
```
