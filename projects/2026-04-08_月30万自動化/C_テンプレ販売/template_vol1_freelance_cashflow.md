## スプレッドシート構成設計（Claudeが設計）

### シート1：月次収支

| 項目 | 内容 |
|------|------|
| 日付 | 入力 |
| 案件名/クライアント名 | 入力 |
| 収入金額 | 入力 |
| 請求書番号 | 入力 |
| 入金確認 | チェックボックス |
| 備考 | 入力 |
| 月合計 | 自動計算 |

### シート2：案件管理

| 項目 | 内容 |
|------|------|
| 案件名 | 入力 |
| 単価 | 入力 |
| 稼働時間 | 入力 |
| 時給 | 自動計算（単価÷時間） |
| 開始日・終了日 | 入力 |
| ステータス | プルダウン（進行中/完了/請求済/入金済） |

### シート3：経費管理

| 項目 | 内容 |
|------|------|
| 日付 | 入力 |
| 科目 | プルダウン（通信費/交通費/ソフトウェア/書籍/その他） |
| 金額 | 入力 |
| 摘要 | 入力 |
| 科目別合計 | 自動集計 |

### シート4：年間サマリー（自動生成）

- 月別収入グラフ
- 月別経費グラフ
- 年間収入・経費・手取り
- 最高収入月・最低収入月
