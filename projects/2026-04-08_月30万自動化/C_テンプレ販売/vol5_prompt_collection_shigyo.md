## プロンプト本文（20個 完全版）

### 1. 法改正・税制改正のお知らせ

```
あなたは士業のSNS担当者です。
以下の情報をもとに、Instagramの投稿文を作成してください。

【事務所情報】
- 事務所名：[名称]
- 業種：[税理士／社労士／行政書士]
- ターゲット：[個人事業主／中小企業／個人]

【法改正情報】
- 改正名：[例：インボイス制度の見直し]
- 施行日：[年月日]
- 主な変更点：[箇条書きで3点]
- 影響を受ける人：[対象]
- 推奨アクション：[いつまでに何をすべきか]

【条件】
- 専門用語は初出時にカッコ書きで補足
- 250〜300字
- 「不安を煽らない」「断定的な節税効果を約束しない」
- 最後に「詳しくは個別相談ください」的なCTA
- 絵文字は控えめ（最大3個）
- ハッシュタグ10個
```

### 2. よくある相談事例の紹介

```
士業のSNS投稿を作成してください。
よく寄せられる相談事例を紹介する内容です。

【相談内容】
- 相談者属性：[例：30代個人事業主]
- 相談テーマ：[例：青色申告と白色申告の違い]
- よくある勘違い：[内容]
- 正しい答え：[内容]
- 失敗するとどうなるか：[例]

【条件】
- 「先日こんなご相談がありました」と切り出す
- 個人情報は匿名・特定不能化
- Q&A形式（質問→回答）で構成
- 専門用語は補足
- 250〜300字
- 守秘義務に配慮（具体的金額・社名は出さない）
- ハッシュタグ10個
```

【条件】
