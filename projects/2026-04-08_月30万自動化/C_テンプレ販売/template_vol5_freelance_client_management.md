## 商品コンセプト

**「Notionで作る、フリーランスの自動オペレーション基盤」**

Vol.1（スプレッドシート）は数字管理に特化したが、
日々の案件進行・顧客とのやりとり・納品ステータス・契約書管理は
スプレッドシートでは表現しきれない。

Notion なら：
- データベース機能で案件・クライアントを構造化
- リレーションで案件↔クライアント↔請求を紐づけ
- ステータスごとのカンバン・カレンダー表示
- 添付ファイル（契約書PDF・打ち合わせ議事録）も一元管理

→ **「フリーランスの仕事をすべて見える化する司令塔」** をテンプレとして提供する。

## Notion テンプレ構造（5データベース + 4ビュー）

| プロパティ | 型 | 説明 |

## ビュー設計（4種類）

### ビュー1：今週のダッシュボード（ホーム画面）

```
┌──────────────────────────────────────────────┐
│ 📅 今週納期の案件（カレンダーView）             │
│   - 今週納期の案件を一覧表示                    │
├──────────────────────────────────────────────┤
│ ⏰ 進行中の案件（Board View - ステータス別）   │
│   [作業中] [納品済] [請求済] [入金済]           │
├──────────────────────────────────────────────┤
│ 💰 今月の入金予定（Table View）                │
│   入金予定日が今月のもの                        │
├──────────────────────────────────────────────┤
│ ✅ 今日のタスク（Filter: 期限=今日）            │
└──────────────────────────────────────────────┘
```

### ビュー2：パイプライン（Board View）

ステータスごとの案件をカンバン表示：
- 提案中 → 受注済 → 作業中 → 納品済 → 請求済 → 入金済 → 完了

ドラッグ&ドロップでステータスを更新できる。

### ビュー3：今月の数字サマリー

- 今月の受注件数
- 今月の納品予定金額
- 今月の入金予定額
- 平均単価／平均文字単価／平均時給（計算プロパティ）

→ Vol.1 の月次サマリーへのリンクも併設。

### ビュー4：クライアント分析

- 累計受注額 TOP 10
- 評価★4以上のクライアント一覧
- 6ヶ月以上連絡なしのクライアント（リアクティベーション候補）

## 自動化（Notion 標準機能のみ・無料）

### 自動化1：ステータス変更時のタイムスタンプ

- 「ステータス = 受注済」になったら、開始日を自動入力
- 「ステータス = 入金済」になったら、実入金日を自動入力

### 自動化2：締切リマインダー

- 納期3日前 / 1日前にバナー表示
- 入金予定日を超えても入金されていない場合、警告表示

### 自動化3：契約書更新アラート

- 契約期間終了の30日前に「更新確認」タスクが自動生成
