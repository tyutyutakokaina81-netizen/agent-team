# Vol.2 SNS投稿カレンダー
# Googleスプレッドシート 完全設計書（そのまま作れる）

---

## 作り方（所要時間：約30分）

1. Googleスプレッドシートを新規作成
2. 以下の3シートを作る：「カレンダー」「テーマ50選」「分析」
3. 下記のセル設定・数式・書式・データ入力規則をそのままコピーする
4. 動作確認後、「ファイル → 共有 → リンクを取得」で「閲覧者」に設定し、リンクを購入者へ配布する
5. 購入者は受け取ったリンクから「ファイル → コピーを作成」で複製して使用する

## シート1：カレンダー

### 列構成（1行目：ヘッダー）

```
A1: 日付
B1: 曜日
C1: プラットフォーム
D1: 投稿テーマ
E1: 本文ドラフト
F1: ハッシュタグ
G1: ステータス
H1: 投稿URL
```

### 各列の数式・設定

```
A2 以降: 日付を入力（例 2026/04/12）。書式 → 数値 → 日付（yyyy/mm/dd）

B2: =IF(A2="","",TEXT(A2,"ddd"))
   ※下方向に B201 までフィルダウン。A列が空なら空欄を返す。

C2: プラットフォーム（下記データ入力規則を設定）

D2: 投稿テーマを入力。テーマ50選から選びたい場合は次の数式でも可。
    =IFERROR(INDEX('テーマ50選'!B:B, MATCH(D2,'テーマ50選'!B:B,0)),"")
    ※基本は手入力でOK。

E2: 本文ドラフトを入力（自由記述）

F2: ハッシュタグを入力（例: #フリーランス #SNS運用）

G2: ステータス（下記データ入力規則を設定）

H2: 投稿後にURLを貼り付け
```

### C列：プラットフォーム データ入力規則（プルダウン）

```
範囲: C2:C201
データ → データの入力規則 → 条件「プルダウン」
項目を以下の5つで登録：
  Twitter/X
  Instagram
  TikTok
  YouTube Shorts
  Threads
無効なデータ: 入力を拒否
```

### G列：ステータス データ入力規則（プルダウン）

```
範囲: G2:G201
データ → データの入力規則 → 条件「プルダウン」
項目を以下の3つで登録：
  未投稿
  投稿済
  要修正
無効なデータ: 入力を拒否
```

### G列：条件付き書式（色分け）

```
範囲: G2:G201
表示形式 → 条件付き書式 → 単一色

ルール1: セルの値が完全一致 → "投稿済"
        背景色 #b7e1cd（緑）

ルール2: セルの値が完全一致 → "未投稿"
        背景色 #fff2cc（黄）

ルール3: セルの値が完全一致 → "要修正"
        背景色 #f4c7c3（赤）
```

### A列：今日の行をハイライト（任意）

```
範囲: A2:H201
表示形式 → 条件付き書式 → カスタム数式
=$A2=TODAY()
背景色 #cfe2f3（薄青）
```

## シート3：分析

### 月次集計テーブル

```
A1: 月次サマリー
A3: 投稿予定数（全件）
B3: =COUNTA(カレンダー!A2:A201)

A4: 投稿済
B4: =COUNTIF(カレンダー!G2:G201,"投稿済")

A5: 未投稿
B5: =COUNTIF(カレンダー!G2:G201,"未投稿")

A6: 要修正
B6: =COUNTIF(カレンダー!G2:G201,"要修正")

A7: 投稿達成率
B7: =IFERROR(B4/B3,0)
   書式 → 数値 → パーセント
```

### プラットフォーム別投稿数

```
D1: プラットフォーム別
D3: Twitter/X        E3: =COUNTIF(カレンダー!C2:C201,"Twitter/X")
D4: Instagram        E4: =COUNTIF(カレンダー!C2:C201,"Instagram")
D5: TikTok           E5: =COUNTIF(カレンダー!C2:C201,"TikTok")
D6: YouTube Shorts   E6: =COUNTIF(カレンダー!C2:C201,"YouTube Shorts")
D7: Threads          E7: =COUNTIF(カレンダー!C2:C201,"Threads")
```

### 週次集計（QUERY 関数で月別×ステータス別をクロス集計）

```
A10: 月別 × ステータス別
A12: =QUERY(カレンダー!A2:G201,
       "select month(A)+1, G, count(A)
        where A is not null
        group by month(A)+1, G
        label month(A)+1 '月', count(A) '件数'", 0)

※「month(A)+1」とするのは Google Sheets の month() が 0 始まりのため。
```

### グラフ（任意・ビジュアル化）

```
1. プラットフォーム別投稿数（D3:E7 を選択 → 挿入 → グラフ → 円グラフ）
2. ステータス別件数（A3:B6 を選択 → 挿入 → グラフ → 棒グラフ）
```
