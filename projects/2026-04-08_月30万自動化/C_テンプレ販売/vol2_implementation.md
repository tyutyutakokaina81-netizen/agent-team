# Vol.2 SNS投稿カレンダー 実装仕様

担当：CPO
作成日：2026-04-17

`vol2_sns_calendar.md`（販売ページ草案）の対になる**実装ドキュメント**。
Google Sheets で実際に作る際の関数・データ・セットアップ手順を定義する。

## シート01：月次カレンダー

### 列構成と関数

| 列 | 項目 | 入力形式 | 関数 |
|----|------|---------|------|
| A | 日付 | 手動入力 or `=A2+1` で連番 | — |
| B | 曜日 | 自動 | `=TEXT(A2,"ddd")` |
| C | 媒体 | プルダウン | データ検証：`03_媒体マスタ!A2:A` |
| D | テーマ | プルダウン or 手動 | データ検証：`02_テーマ50!A2:A` |
| E | 投稿文 | 手動 | — |
| F | ハッシュタグ | 手動 | — |
| G | 画像有無 | チェックボックス | データ検証：チェックボックス |
| H | 投稿済 | チェックボックス | データ検証：チェックボックス |
| I | いいね数 | 数値 | — |
| J | 備考 | テキスト | — |

### 条件付き書式

- **H列（投稿済）= TRUE** → 行全体をグレーアウト
  - 適用範囲：`A2:J100`
  - 条件：`=$H2=TRUE`
  - 書式：背景色 `#EEEEEE`、文字色 `#888888`

- **A列の曜日が土日** → 背景を薄黄色
  - 適用範囲：`A2:B100`
  - 条件：`=WEEKDAY($A2,2)>=6`
  - 書式：背景色 `#FFF9E6`

### ヘッダー行（1行目）

```
日付 | 曜日 | 媒体 | テーマ | 投稿文 | ハッシュタグ | 画像 | 投稿済 | いいね | 備考
```

### 推奨セルサイズ

- A列（日付）：90px
- B列（曜日）：50px
- C列（媒体）：100px
- D列（テーマ）：150px
- E列（投稿文）：400px（折り返し有効）
- F列（ハッシュタグ）：250px（折り返し有効）

## シート04：集計

### B2セル：今月の投稿数（媒体別）

```
=COUNTIFS('01_カレンダー'!C:C, "Instagram", '01_カレンダー'!H:H, TRUE)
```

媒体ごとに同様の COUNTIFS を配置。

### B6セル：今月の合計いいね数

```
=SUMIF('01_カレンダー'!H:H, TRUE, '01_カレンダー'!I:I)
```

### B7セル：投稿あたり平均いいね数

```
=IFERROR(B6/COUNTIF('01_カレンダー'!H:H, TRUE), 0)
```

### B8セル：今月の予定投稿数

```
=COUNTA('01_カレンダー'!E:E)-1
```
（ヘッダー行を引く）

### B9セル：消化率

```
=IFERROR(COUNTIF('01_カレンダー'!H:H, TRUE)/B8, 0)
```
セル書式：パーセント表示

## シート00：使い方（PDF/シート両対応）

### セクション構成

```
1. このカレンダーで何ができるか（30秒で読める）
2. はじめに：シートのコピー方法
3. 月次カレンダーの使い方（5ステップ）
4. 投稿テーマ50選の活用法
5. 集計シートの読み方
6. よくある質問
7. 困ったときは（更新情報・お問い合わせ）
```

### 5ステップ（カレンダー使用手順）

```
ステップ1：シート01のA列に今月の日付を入力する
   → A2に「2026/04/17」と入れて下にドラッグで自動連番

ステップ2：投稿する日のC列で媒体を選ぶ
   → プルダウンから Instagram/X/Facebook/全媒体 を選択

ステップ3：D列でテーマを選ぶ
   → プルダウンから50テーマを選ぶ。複数組み合わせもOK

ステップ4：E列に投稿文の下書きを書く
   → AIに「[テーマ]について[業種]向けに[文字数]字で書いて」と頼むと早い

ステップ5：投稿したらH列にチェックを入れる
   → 行がグレーアウトされ、集計シートに反映される
```
