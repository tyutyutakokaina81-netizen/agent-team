## シート1：カレンダー

### 列構成（1行目：ヘッダー）
```
A1: 日付
B1: 曜日
C1: 媒体
D1: テーマ・ネタ
E1: 投稿文（下書き）
F1: ハッシュタグ
G1: 画像有無
H1: 投稿済み
I1: いいね数
J1: コメント数
K1: 備考
```

### 数式・設定
```
B2: =IF(A2="","",TEXT(A2,"ddd"))    （曜日を自動表示）
→ B列全体に下方向へコピー

A2〜A32: 日付を手入力（月初に1ヶ月分を入力）
  ※ ショートカット: A2に開始日を入力 → A3に =A2+1 → 下方向コピー

C列プルダウン設定:
  データ→データの入力規則→リストから項目を選択：
  Instagram,X,Facebook,全媒体,TikTok,LINE

G列: チェックボックス挿入（挿入→チェックボックス）
H列: チェックボックス挿入（挿入→チェックボックス）
```

### 書式設定
```
A列: 書式→数値→日付（YYYY/MM/DD）
E列: 列幅を300pxに広げる（下書き用）
F列: 列幅を200pxに広げる
1行目: 太字＋背景色（#4285F4 青）＋文字色白
```

## シート2：集計（自動計算ダッシュボード）

### レイアウト
```
A1: 📊 月次集計ダッシュボード
A2: （空行）
A3: 項目                  B3: 数値
A4: 総投稿予定数          B4: =COUNTA(カレンダー!D2:D100)
A5: 投稿済み数            B5: =COUNTIF(カレンダー!H2:H100,TRUE)
A6: 未投稿数              B6: =B4-B5
A7: 投稿完了率            B7: =IF(B4=0,0,B5/B4)
A8: （空行）
A9: 媒体別 投稿数
A10: Instagram             B10: =COUNTIF(カレンダー!C:C,"Instagram")
A11: X                     B11: =COUNTIF(カレンダー!C:C,"X")
A12: Facebook              B12: =COUNTIF(カレンダー!C:C,"Facebook")
A13: TikTok                B13: =COUNTIF(カレンダー!C:C,"TikTok")
A14: LINE                  B14: =COUNTIF(カレンダー!C:C,"LINE")
A15: 全媒体                B15: =COUNTIF(カレンダー!C:C,"全媒体")
A16: （空行）
A17: エンゲージメント
A18: いいね合計            B18: =SUM(カレンダー!I2:I100)
A19: コメント合計          B19: =SUM(カレンダー!J2:J100)
A20: いいね平均/投稿       B20: =IF(B5=0,0,B18/B5)
A21: コメント平均/投稿     B21: =IF(B5=0,0,B19/B5)
```

### 書式設定
```
B7: 書式→数値→パーセント（小数点1桁）
A1: フォントサイズ16・太字
A3,A9,A17: 太字＋背景色（薄いグレー #F3F3F3）
```

### グラフ設定
```
媒体別グラフ：A10:B15を選択→挿入→グラフ→円グラフ
  タイトル：「媒体別投稿数」
```

## 販売用マニュアル文（PDFに貼る）

```
【使い方 4ステップ】

STEP1: このリンクを開く
STEP2: 右上「ファイル」→「コピーを作成」
STEP3: 「カレンダー」シートに今月の投稿予定を記入
STEP4: 投稿したらチェックを入れて「集計」シートで進捗確認

【月初30分の使い方】
1. A列に今月の日付を入力（A2に初日→A3に=A2+1→下コピー）
2. 「テーマ50選」シートからネタを選ぶ
3. D列にテーマ、E列に下書きを記入
4. F列にハッシュタグを入力
5. 投稿したらH列にチェック → 集計シートが自動更新

【よくある質問】
Q: スマホでも使えますか？
A: はい。Googleスプレッドシートアプリから使えます。

Q: 3媒体以上を管理できますか？
A: C列のプルダウンにTikTok・LINEも追加済みです。

Q: 1日に複数投稿したい場合は？
A: 同じ日付で複数行を追加してください。
```
