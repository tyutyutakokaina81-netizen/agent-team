## 1. v2 からの方針転換理由

### v2（Claude Code 支援路線）が売れない根本原因

| 問題 | 症状 |
|---|---|
| 顧客不在 | オーナーに Claude Code 指導実績ゼロ、信頼ゼロ |
| オーディエンス不在 | X / note フォロワーゼロ、誰も投稿を見ない |
| 権威性不在 | 「4 日失敗した人」に金を払う人はいない |
| プロダクトアウトの再発 | v1 と同じ罠を別の名前で繰り返していた |
| 現金化までの経路が長すぎ | 記事公開 → 発見 → 信頼 → DM → 契約 → 納品 → 入金 |

### v3 の核心原則

**「需要が既に存在する市場に、既に手元にある商品を出す」**

- メルカリ：月間 2,000 万人のアクティブユーザー、毎日数万件の取引成立
- 女性ジャケット：定番カテゴリ、検索流入多い、中古需要あり
- 手元在庫：オーナーに実物あり（今日から出品可能）

## 2. 新主柱：物販（メルカリ × ジャケット）

### なぜ物販が v2 より優れているか

| 観点 | v2 Claude Code 支援 | v3 メルカリ物販 |
|---|---|---|
| 需要の存在 | 未検証 | **検証済み（巨大市場）** |
| オーディエンス | 必須（0 から構築） | **不要（プラットフォーム済）** |
| 権威性 | 必須 | **不要** |
| 現金化までの時間 | 不明 | **最短 3〜5 日** |
| スキル要件 | Claude Code 熟練 | **写真 + 出品文（テンプレあり）** |
| 初期在庫 | なし | **手元のジャケットを流用** |
| 初期費用 | ¥0 | **¥0**（梱包資材だけ） |
| リスク | 時間浪費 | **売れなくても元本ゼロ** |

## 3. 1 ヶ月目の数値設計

### 目標設定

| 段階 | 純利益 | 出品数 | 成約率 | 平均単価 |
|---|---|---|---|---|
| 🔴 下限 | **+¥100** | 5 着 | 20%（1 着） | ¥2,000 |
| 🟡 標準 | **+¥10,000** | 10 着 | 50%（5 着） | ¥3,000 |
| 🟢 ストレッチ | **+¥30,000** | 20 着 | 60%（12 着） | ¥3,500 |
| 🌙 ムーンショット | **+¥100,000** | 40 着 + 追加仕入 | 70%（28 着） | ¥4,500 |

### 計算根拠（標準シナリオ）

```
売上（5 着成約）
  5 着 × 平均 ¥3,000 = ¥15,000

コスト
  メルカリ手数料 10% = ¥1,500
  送料（らくらくメルカリ便・宅急便コンパクト等）¥500 × 5 = ¥2,500
  梱包資材（OPP 袋・段ボール） = ¥500
  固定費（Claude Pro のみ） = ¥3,000

  計 ¥7,500

純利益 = ¥15,000 − ¥7,500 = +¥7,500（標準シナリオの下限）

+ Vol.1 note 販売（継続）
+ X/note 発信なし（時間を物販に集中）
```

## 4. Day 1〜Day 7 実行計画

- [ ] メルカリアカウント登録（未登録なら）
