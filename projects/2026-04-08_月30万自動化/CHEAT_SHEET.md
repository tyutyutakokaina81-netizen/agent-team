## ⚡ シーン別 即応ガイド

### 「応募したい」
```
1. 案件リスト_応募文.md を開く
2. 送れてない案件を選ぶ
3. URLをタップ→応募文をコピペ→送信
```

### 「返信が来た」
```
1. 返信の種類を見る
   - アンケート→パターン1
   - 見積もり→パターン2
   - テスト作業→パターン3
   - 契約→パターン4
2. クライアント返信テンプレ集.md の該当パターンをコピペ
3. [ ]を埋めて送信
```

### 「受注した」
```
1. 見積書（必要なら）→請求書テンプレート.md
2. 契約書（月5万以上）→業務委託契約書テンプレート.md
3. 作業→workflow.md のフローで進める
4. 納品→テンプレで納品
5. 請求→請求書テンプレート.md
6. 入金確認→ダッシュボード更新
```

### 「営業したい」
```
1. Google Mapsで「地域+業種」検索
2. 営業リスト100社_テンプレート.md に店情報を追加
3. 業種別営業DM10本.md から該当業種のテンプレをコピペ
4. Instagram DM送信
5. 3日後にフォロー→7日後にメール
```

### 「反応がない・落ち込んだ」
```
1. daily_routine.md の最小15分ルーティン
2. 初受注5件の壁を超える戦術.md を読み返す
3. タスク形式の案件に参加→評価を1つでも増やす
```
