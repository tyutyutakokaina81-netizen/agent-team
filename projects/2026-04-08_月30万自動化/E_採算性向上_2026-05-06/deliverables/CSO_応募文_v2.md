# CrowdWorks 応募文 v2（実績差込型）

## 設計意図

旧版（v1）は汎用的すぎて単価¥500-3K帯の「数打ち」案件にしか刺さらなかった。  
v2は **実績・専門領域・サンプル成果物リンク** を差込むことで ¥3-15K帯への進出を狙う。

---

## テンプレ本文

```
{案件タイトル} の件、ぜひお手伝いさせてください。

【専門領域】
{専門領域 例：北陸地方の地域情報・観光・移住}に関する
SEOライティング/取材/データ整理を継続的に手がけています。

【サンプル成果物】
・{サンプル記事URL 例：https://note.com/safe_canna441/n/xxx}
・{サンプル記事URL 例：https://note.com/safe_canna441/n/yyy}
・テンプレート販売（note）：https://note.com/safe_canna441

【今回の案件への対応方針】
1. {案件要件1}について：{具体的な進め方を1-2行}
2. {案件要件2}について：{具体的な進め方を1-2行}
3. {納品形式}にて、{納期}までに納品いたします。

【継続対応の可否】
今回の案件が問題なく完了した場合、月次/週次での継続契約も
ご相談いただけます（単価は要件に応じて柔軟にご相談）。

【ご連絡について】
平日9-18時はチャット即時返信可能です。
不明点があれば事前に確認させていただきます。

どうぞよろしくお願いいたします。
```

## 差込み項目チェックリスト（応募前に必ず埋める）

- [ ] `{案件タイトル}` … 募集要項の正式名称を貼る
- [ ] `{専門領域}` … 案件カテゴリと自分の領域が重なる部分を1行で
- [ ] `{サンプル記事URL}` … 関連性の高い note 記事 を **2本** 貼る
- [ ] `{案件要件}` … 募集要項から要件を **2-3個抽出** し、それぞれに進め方を書く
- [ ] `{納品形式・納期}` … 募集要項の指定があればそれに合わせる

---

## 単価帯別の応募方針

| 単価帯 | 応募スタンス | サンプル本数 | 差別化ポイント |
|--------|-----------|-----------|--------------|
| 〜¥3K | v1（汎用版）でOK、量で勝負 | 不要 | スピード・低価格 |
| ¥3-10K | v2 を使用 | 2本 | 専門領域・継続提案 |
| ¥10-30K | v2 + 提案書（別途） | 3本+ | 課題理解・改善案提示 |
| ¥30K〜 | 直接コミュニケーション必須 | 3本+ポートフォリオPDF | 実績・成果物・人柄 |

---

## 「実績がまだない」場合の暫定運用（最初の30日）

- サンプル成果物として note 記事を **無料公開で5本書く**（30日以内）
- 実績欄には「note にて{ジャンル}記事を継続執筆中（月XX本）」と書く
- 「初回案件は実績作りのため通常単価の80%で対応可能」と明記
- 30日経過後は v2 通常版に切替

## 提案：継続契約への二次提案テンプレ（初回納品時に同送）

```
本日納品の {納品物} につきまして、ご確認のほどよろしくお願いいたします。

もし内容にご満足いただけましたら、
来月以降の継続でのご依頼もご相談いただければ幸いです。

【継続提案】
- 月{N}本の固定発注：単価{XX}円（単発より{Y}%お得）
- 月次定例ミーティング：30分（オンライン）
- 優先対応・即時返信

ご検討いただけますと嬉しいです。
```

→ この二次提案テンプレで **継続率0% → 30%** を目指す。
