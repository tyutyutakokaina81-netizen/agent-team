# note 記事末尾 CTA テンプレ

## 設計意図

note記事は「読まれて終わり」で収益化していない。  
全記事末尾に **3つの導線**（案件相談 / テンプレ販売 / メルマガ）を標準装備し、  
読者の関心を必ず次の行動に変換する。

## 記事ジャンル別カスタマイズ

### エッセイ系記事（地域・暮らし）
標準CTAをそのまま使用。**メルマガ登録**を最上位に。

### ノウハウ系記事（スキル・ツール）
**テンプレ販売を最上位に**：

```markdown
### 📊 この記事の内容を実装したテンプレを販売しています
[Vol.1 フリーランス収支管理スプレッドシート（¥980）](URL)

→ 標準CTA続く
```

### 営業色強い記事（実績紹介・料金表）
**お仕事相談を最上位に**：

```markdown
### 🤝 同じような課題、解決します
過去事例：{事例タイトル}（成果：{KPI改善}）
- お問い合わせ：[フォーム](URL)

→ 標準CTA続く
```

## 自動付与（CDO実装）

`~/ai-auto/generate_daily_outputs.py` の `note_draft` 末尾に  
本テンプレを自動 append するよう改修する（→ `setup_ai_auto.sh` v2 で対応済）。

差込み変数：
- `{NOTE_URL}` … 環境変数 `NOTE_URL` から
- `{CONTACT_FORM}` … 環境変数 `CONTACT_FORM_URL` から（未設定時はDM案内のみ）
- `{X_HANDLE}` … 環境変数 `X_HANDLE` から
