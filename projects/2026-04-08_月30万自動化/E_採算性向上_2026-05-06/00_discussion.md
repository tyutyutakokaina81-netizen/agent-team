# 採算性向上ディスカッション — 2026-05-06

## 議題

現状ペースの自動化（CW応募 + note + Shorts）だけでは月収見込み **¥10K〜¥40K**（目標¥300Kの3-13%）に留まる。  
**月¥300Kへ届かせるために何を変えるか** を5役職で議論する。

---

## 現状診断（CFO）

### 【事実】
- 現状の自動化対象：note 1本/日、X/Reddit投稿、Shorts週3、CW応募1件/日
- 想定コンバージョン：CW採用率5-10%、note→直接案件0%、テンプレ販売Vol.1のみ稼働
- 月収レンジ試算：

| 経路 | 楽観 | 悲観 |
|------|-----|-----|
| note 広告/有料記事 | ¥5K | ¥0 |
| CW 単発¥500-3K × 採用率10% | ¥30K | ¥5K |
| Shorts 収益 | ¥3K | ¥0 |
| テンプレVol.1 | ¥3K | ¥1K |
| **合計** | **¥41K** | **¥6K** |

### 【考察】
目標¥300K に対して 1桁足りない。**「件数を増やす」だけでは届かない**。  
**単価×継続率** を上げる構造変更が必要。

---

## CSO（営業）の見解

### 採算性のボトルネック
1. **応募文が汎用的すぎて単価帯¥500-3Kの「数打ち」案件にしか刺さらない**
2. **単発で終わっており継続契約に繋げられていない**
3. **直接案件（note読者→相談）の導線がゼロ**

### 提案
- **応募文 v2** を作る：実績/サンプル成果物/専門領域を差込み → ¥3-15K帯にシフト
- **継続提案テンプレ** を作る：初回案件納品時に「次月分の予約」を提案
- **note→案件化導線**：全記事末尾にCTA設置（CMO連携）

### 期待効果
- CW単価：¥1.5K → ¥5K（3.3倍）
- 継続率：0% → 30%（毎月3社×¥15K = ¥45K）
- 直接案件：月1件 × ¥30K = ¥30K
- **CSO経由の月貢献 ¥75K → ¥120K**

## CMO（マーケ）の見解

### 採算性のボトルネック
1. **noteが「読まれて終わり」で収益化していない**（広告/有料以外の出口がない）
2. **X/Redditがnoteへの流入装置になっていない**（毎回バラ投稿）
3. **既存テンプレVol.1の販売告知が単発**

### 提案
- **note記事末尾CTAテンプレ** を全記事に標準装備（CSO案件導線 + テンプレ販売 + メルマガ登録）
- **X連投スレッド戦略**：週1で「今週のnote記事まとめ」スレッドを投稿
- **テンプレVol.1 の継続告知**：毎週金曜にX/Reddit でVol.1宣伝（ローテ自動化）

### 期待効果
- note記事 → CTA経由のテンプレ販売：月10本 × ¥980 = ¥9.8K
- note → CSO直接案件：月1-2件 × ¥30K = ¥30-60K
- X連投スレッドからのnote流入：月+500PV
- **CMO経由の月貢献 ¥40-70K**

## CPO（プロダクト）の見解

### 採算性のボトルネック
1. **テンプレ販売がVol.1（¥980）のみ稼働、Vol.2-4は「制作中」のまま**
2. **単品売り切りのみ＝MRR（月次経常収益）がゼロ**

## CDO（自動化・技術）の見解

### 採算性のボトルネック
1. **既存セットアップスクリプトの応募文・note下書きが汎用テンプレ**で、CSO/CMOの上位レイヤー戦略を反映していない
2. **CTA・実績差込みが手作業**で、毎日の生成物に組み込まれていない
3. **KPIダッシュボードがない** → 何が効いているか分からない

### 提案（即実行）
- `setup_ai_auto.sh` の `cw_application` を **CSO応募文v2** に差し替える
- `note_draft` 末尾に **CMO CTAブロック** を自動付与する
- `check_status.py` を拡張し **週次KPI集計** を出力する

### 期待効果
- 上位戦略の自動反映 → 全成果物の質が同時に底上げ
- KPI見える化 → 毎週の改善ループが回る
