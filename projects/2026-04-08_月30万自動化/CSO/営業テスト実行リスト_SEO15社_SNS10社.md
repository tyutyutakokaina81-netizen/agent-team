## 営業パイプライン追跡（Googleスプレッドシート）

### Column 構成

```
A: 日付
B: 企業名
C: サービス（SEO / SNS）
D: ステージ（リード獲得 / DM送信 / 返信待ち / 相談予約 / 相談済 / 提案送付 / 成約）
E: 金額（¥15K or ¥50K）
F: 確度（20% / 50% / 80%）
G: 備考
H: 次のアクション
I: 予定日
```

### ステージ定義

| ステージ | 定義 | 次アクション |
|---------|------|-----------|
| リード獲得 | LinkedIn / X で企業情報収集完了 | DM 送信 |
| DM送信 | DM を送信した | 返信待ち（3日） |
| 返信待ち | 返信を待っている | リマインド DM or 諦め |
| 相談予約 | 返信あり、相談日時が決定 | Zoom 実施 |
| 相談済 | 20 分相談を実施 | 提案資料送付 |
| 提案送付 | 提案資料を送った | 返答待ち（5日） |
| 成約 | 契約書署名完了 | 初月業務開始 |
