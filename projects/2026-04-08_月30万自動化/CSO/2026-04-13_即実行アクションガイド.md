# 即実行アクションガイド

作成日: 2026-04-13

## ステップ1：クラウドワークス登録（15分）

### 手順
1. https://crowdworks.jp にアクセス
2. 「ワーカーとして登録」をクリック
3. メールアドレスで登録
4. プロフィール設定画面で以下を入力:
   - 表示名: [任意の名前]
   - 一行紹介: 「AI×人間レビューで高品質なSEO記事・SNS運用・データ整理を高速納品します」
   - 自己紹介: → `2026-04-13_クラウドソーシング統合プロフィール.md` の「クラウドワークス用」をコピペ
   - スキル: ライティング / SEO / データ入力 / SNS運用
   - 稼働可能時間: 週20時間以上
5. 本人確認書類を提出（任意だが信頼度UP）

## ステップ2：ランサーズ登録（15分）

### 手順
1. https://www.lancers.jp にアクセス
2. 「フリーランスとして登録」
3. プロフィール設定:
   - 自己紹介: → 統合プロフィールの「ランサーズ用」をコピペ
   - 得意分野: ライティング・ネーミング > SEOライティング
   - ポートフォリオ: SEOサンプル記事3本のGoogleドキュメントリンクを登録
4. スキルテスト受験（ライティング系）→ 合格すると上位表示

- カテゴリ: マーケティング・Web集客 > SNS運用

- 価格: 10,000円（初期価格）
- お届け日数: 5日

## ステップ4：案件応募（30分・5件以上）

### クラウドワークスで検索するキーワード
```
SEO 記事 ライティング
ブログ記事 作成
コラム 執筆
データ入力 Excel
リスト作成
```

### 応募のコツ
1. 「AI使用可」「AI歓迎」と書いてある案件を優先
2. 単価 ¥3,000〜¥8,000 の案件を狙う（実績作り優先）
3. 応募文は `2026-04-13_案件応募テンプレート.md` のテンプレAをコピペ
4. 案件の内容に合わせて1〜2文カスタマイズを加える
5. 応募後24時間以内にクライアントからの返信に対応する

### 狙い目の案件タイプ
- 「継続あり」「長期」と書いてある案件 → リピートで安定収入
- 「急募」「本日中」の案件 → 競合が少なく受注しやすい
- 文字数が少ない案件（1,000〜2,000字）→ 最初の実績に最適

## ステップ5：Vol.2 note公開（30分）

### 手順
1. Googleスプレッドシートを新規作成
2. `vol2_sheet1_カレンダー.csv` をインポート → シート名「月次カレンダー」
3. `vol2_sheet2_テーマ50選.csv` をインポート → シート名「テーマ50選」
4. `vol2_sheet3_ハッシュタグ集.csv` をインポート → シート名「ハッシュタグ集」
5. シート4「使い方マニュアル」を手動追加（vol2完成版.mdからコピー）
6. B列に =TEXT(A2,"ddd") の数式を設定
7. C列にプルダウン設定（Instagram/X/Facebook/全媒体）
8. G列・H列にチェックボックス設定
9. スプレッドシートの共有設定 → 「リンクを知っている人が閲覧可」
10. note（https://note.com）にログイン
11. 新しい記事を作成
12. `vol2_SNSカレンダー完成版.md` のnote販売ページ文言をコピペ
13. スプレッドシートのリンクを記事内に貼る
14. 有料設定: 680円
15. 公開
