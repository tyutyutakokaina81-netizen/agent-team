# 営業 変換キット：業種別フック／返信対応／無料テーマ案

**作成日**: 2026-06-22
**役割**: 「30社に撃つ」だけで終わらせない。**返信が来てから受注までを即決させる**ための実物。
**使う場面**: Cowork/オーナーが返信に対応するとき、ここからコピペ。

---

## 1. 業種別フック（targets.csvの industry に合わせて1行差し込む）

送信メールの冒頭に、業種別の一文を足すと刺さる。

| 業種 | 差し込む一文 |
|---|---|
| 飲食店 | 「『富山 ランチ』『高岡 ◯◯（料理名）』で検索する人に、御社の記事を見つけてもらえます」 |
| 美容室・サロン | 「『髪の悩み』『縮毛矯正 持ち』等の記事は検索が多く、新規予約の入口になります」 |
| 整体・治療院 | 「『肩こり 高岡』のような“症状×地域”は検索需要が高く、来院の動線になります」 |
| 工務店・リフォーム | 「『富山 注文住宅』『リフォーム 費用』は単価が高く、記事1本が問い合わせを生みます」 |
| 宿・観光 | 「周辺の見どころ記事で、予約前に検索している人を捕まえられます」 |
| 士業（税理士・社労士） | 「『インボイス 富山』等の実務解説は、信頼構築と無料相談の入口になります」 |

## 2. 返信対応プレイブック（来た返信→即こう返す）

### A.「興味あります／詳しく聞きたい」
```
ご返信ありがとうございます。
さっそく、御社の業種で「検索されやすく、集客につながりそうなテーマ案」を3つお送りします。

（↓ 下の「3. 無料テーマ案」から業種に合う3つを貼る）

このうち1つを、お試しとして1本¥15,000で書きます。
構成案をまずお見せし、OKをいただいてから執筆します。
ご満足いただけなければ料金は頂きません。どれか書いてほしいテーマはありますか？
```

### B.「料金は？」
```
・お試し1本：¥15,000（構成案→本文、修正1回込み）
・継続：1本¥18,000／月2〜4本など、本数でご相談
・SNS運用もご希望あれば月¥50,000〜（投稿20本＋カレンダー＋簡易レポート）
初回はお試し1本だけでOK、合わなければ終了で構いません。
```

### C.「今は間に合っている／不要」
```
承知しました。また必要になったタイミングで構いません。
よろしければ、御社の業種で書けそうなテーマ案を3つだけ置いていきます（無料です）。
（↓ テーマ案3つを貼る）
お役に立てそうなときに思い出していただけたら嬉しいです。
```
→ 断られても関係を残す。半年後の更新需要を取りに行く。

### D.「他社と何が違う？」
```
3点です。
① 富山の“住人視点”で書くので、外注ライターにありがちな「どこにでもある記事」になりません
② 書くだけでなく、検索で見つけてもらう設計（キーワード・構成）まで含みます
③ お試し1本は、ご満足いただけなければ無料。リスクはこちらが持ちます
```

## 3. 無料テーマ案（返信が来たら即送る・業種別に3つ）

> これが「無料で渡す価値」＝受注の決め手。相手は「もう書く中身が見えてる」状態になる。

**飲食店**
1.「地元の人がこの店に通う3つの理由」（指名検索・ファン化）
2.「旬の◯◯（食材）を使った季節限定メニューの裏側」（季節検索）
3.「初めての方へ：人気メニューの選び方ガイド」（来店前の不安解消）

**美容室・サロン**
1.「梅雨・夏に広がる髪、家でできる対策と、サロンでできること」
2.「縮毛矯正の“持ち”を良くする習慣」（悩み検索）
3.「初来店の流れと、失敗しないオーダーの伝え方」

**整体・治療院**
1.「デスクworkの肩こり、自宅ストレッチと“それでも取れない時”」
2.「ぎっくり腰になったら最初にやること・やってはいけないこと」
3.「『高岡で整体を探している人』へ：院の選び方」

**工務店・リフォーム**
1.「富山の家づくり、雪国ならではの注意点」
2.「リフォーム費用の相場と、見積もりで見るべきところ」
3.「実例：◯◯のリフォームでこう変わった（before/after）」

**宿・観光**
1.「◯◯（宿名）から行ける、地元おすすめの朝の過ごし方」
2.「富山・高岡 1泊2日の回り方モデルコース」
3.「観光客が見落とす、地元の人の隠れた名所」

**士業**
1.「インボイス、富山の個人事業主が今やるべきこと」
2.「確定申告、税理士に頼む人・頼まない人の分かれ目」
3.「補助金・助成金、見落としがちな申請のコツ」

## 受注したら → CPO/outputs/2026-06-22_受注後すぐ納品キット へ
```
返信→無料テーマ案→お試し受注 まで来たら、納品キットで「構成案→執筆→納品」を即実行。
スピードと品質が、お試し1本を“継続”に変える。継続こそが月¥30万の正体。
```
