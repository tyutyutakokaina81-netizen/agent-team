# キャッシュフロー分析

## 重要：実態の運用コスト（オーナー申告 2026-05-06）

**月¥20,000/月 が実際の運用コスト**（通信費・サブスク・電気・その他事業費すべて込み）。

旧版で記載していた ¥5,800 は SaaS3点（Claude Pro / Canva Pro / MS365）のみで、
その他の運用コスト（インターネット・スマホ・電気・小物など）が含まれていなかった。
**全ての試算を ¥20K/月 ベースに更新する。**

## ツール別内訳（参考・SaaS分のみ）

| ツール | 用途 | 月額 | 必須度 |
|--------|------|------|--------|
| Claude Pro | 全モデルの作業自動化 | 3,000円 | 必須 |
| Canva Pro | B：SNS画像作成 | 1,500円 | 一時停止検討 |
| Buffer（無料プラン） | B：予約投稿 | 0円 | 無料で可 |
| Microsoft 365 | C：Excelテンプレ制作 | 1,300円 | 一時停止検討 |
| note / BOOTH | C：販売プラットフォーム | 0円 | 無料 |
| クラウドワークス | A：受注プラットフォーム | 0円 | 無料 |
| その他事業費 | 通信・電気・雑費等 | 約14,200円 | 構造的 |

### 運用合計
```
実態運用コスト：20,000円/月
うちSaaS：5,800円（圧縮余地あり）
うちその他：14,200円（構造的・圧縮困難）
```

**初期投資はゼロだが、月額¥20Kの構造的コストがある。**

## 税務上の注意点

- 年間収入が**20万円超**で確定申告が必要
- 月30万×12ヶ月 = 年360万円 → 個人事業主として青色申告推奨
- ツール代・通信費は**経費計上可能**
- Claude ProやCanva Proは経費になる

**→ CFOに確定申告・経費管理の準備を依頼することを推奨**
