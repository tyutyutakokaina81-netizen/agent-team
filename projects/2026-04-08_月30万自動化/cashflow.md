# キャッシュフロー分析｜超超超最強プラン v3

> **方針**：v2の「個人月収最大化」を捨て、**資産形成型エコシステム**へ。
> 受託（短期キャッシュ）→ サブスク（中期MRR）→ プロダクト／コミュニティ（長期資産）→ 法人化＋借入レバレッジ → **EXIT射程**まで一気通貫で設計。
>
> **必達ライン**：5月末プラス転換 → **6月末¥500K** → **7月末¥1M** → **8月末¥2M** → **9月末¥3M** → **Year2 年商¥100M射程**。
>
> **前提空欄**：5/6残高・5月確定入出金・運用口座¥20Kの扱いは§0で投入待ち。

## §10 リスク対策（v3 多層防御）

### 資金リスク
- **6ヶ月運転資金**を生活防衛口座に確保（v2の3ヶ月から倍増）
- **着金ラグ**：CW即払い／請求書ファクタリング（手数料3-5%）も選択肢
- **借入は売上実績後**（売上ゼロでの借入は厳禁）

### 案件リスク
- **単一クライアント依存15%以下**（v2の20%から厳格化）
- **柱別売上比率20%±10%以内**（過度な偏りを抑制）
- **MRR比率を売上の40%以上に**（受託依存の脱却）

### 属人化リスク
- **全業務をSOP化**（Notion or GitBookで管理）
- **AIエージェント＋外注で代替可能性100%**
- **Founder不在時の72時間運営**を6月内に実証

### 法務リスク
- 契約書テンプレ整備（業務委託・秘密保持・利用規約）
- インボイス対応（5月内判断、必要なら6月中登録）
- **顧問弁護士契約**（7月以降、月¥30-50K）

### ピボット条件（早期撤退ライン）
- 5月末で累計手取り < ¥0 → A・Dに資源集中、B営業中止
- 6月末で月収 < ¥250K → 単価戦略を放棄、量で押す＋外注前倒し
- 7月末で月収 < ¥600K → F開講延期、E（顧問）主軸化
- 8月末で月収 < ¥1.2M → 法人化メリット再検証、SaaS開発中断
