# 🟢 ワンパス開店パッケージ — noteメンバーシップ

**これは何**：noteを開いて、上から順にコピペするだけでメンバーシップが開店できる手順書。
**所要**：約30分。**前提**：オーナーのPC/スマホ（note.comにログイン済み）。
**なぜ必要**：私（AI）はnote.comに到達不可（403）。設定はオーナーの手でしか行えない。

> 上から順に、❏ をチェックしながら進めてください。コピペ箇所は ```で囲ってあります。

---

## STEP 1：メンバーシップを作る（5分）

❏ noteにログイン → 右上のアイコン → **「メンバーシップ」**
❏ **「メンバーシップを作成」** をクリック
❏ プラン名に、これを貼る：

```
ひとりで会社をやる人の、考える材料置き場
```

---

## STEP 2：説明文を貼る（3分）

❏ 説明文の欄に、これを貼る：

```
会社をひとりでやっていると、相談相手がいません。

数字を見るのも、値段を決めるのも、断る判断をするのも、ぜんぶ自分。
「これで合ってるのか」を確かめる相手がいないまま、毎日が過ぎていきます。

このメンバーシップは、その「ひとりの判断」を少しだけ軽くするための場所です。

私自身が、富山で個人事業をやりながら、AIを5つの役職に分けて会社のように回しています。
その過程で見つけた「使える判断材料」を、毎週ここに置いていきます。
きれいごとではなく、失敗も数字も、そのまま書きます。

【毎週・深掘り記事】
売上目標から逆算した今月の動き方／現金が苦しいときの順番／単価を上げる会話の言い方

【隔週・テンプレ解説】
請求書を3分で作るプロンプト／数字を整理するスプレッドシート／外注の指示書

【月1・運営ノウハウ】
AIに5役職を渡す実際の分け方／ひとりで孤独にならない仕組み

読むだけで終わらせず、その日から手が動くように、テンプレやプロンプトを配ります。

月額¥980・初月無料・いつでも解約できます。合わなければ初月内に抜けてください。
```

---

## STEP 3：価格を設定（2分）

❏ 月額：**980円**
❏ 初月無料：**オン**（あれば設定。なければ説明文の「初月無料」は外す）
❏ 公開する

---

## STEP 4：初回の有料記事を1〜2本入れる（10分）

> 中身ゼロで開けると価値が伝わらない。まず1〜2本だけ入れて「中身がある」状態に。
> 本文は全て執筆済み。リポジトリの下記ファイルを開いてコピペするだけ。

❏ まず1本目（一番すぐ役立つ＝成功体験になる）：
   `CMO/outputs/有料/2026-06-21_有料03_請求書を3分で作るプロンプト完全版.md`
❏ 2本目（看板記事）：
   `CMO/outputs/有料/2026-06-21_有料01_売上目標から逆算して今月やる3つを決める手順.md`

❏ noteで「新規投稿 → 有料 → このメンバーシップ限定」に設定して、本文を貼る
❏ 残り3本（有料02・04・05）は、今週〜隔週で小出しにする（毎週の接点になる）

---

## STEP 5：無料note読者に告知（5分）

❏ 最新の無料note記事の末尾に、これを貼る：

```
---
📌 もう一歩踏み込んだ話を、メンバーシップで毎週書いています。
「ひとりで会社をやる人の、考える材料置き場」
数字の見方、値段の決め方、断る判断——相談相手がいないまま下している判断を、
少しだけ軽くする場所です。月¥980・初月無料・いつでも解約OK。
→ （メンバーシップのURL）
```

```

```

## 開店後にやること（リテンション・リサーチ反映）

❏ 登録者が出たら、KPIに記録：
```bash

```
❏ 毎週1本、有料記事を出す（＝解約防止の接点。市場リサーチ：最初の7日と定期接点が継続率を決める）
