## 0. 全体方針（GTMの背骨）

【事実】
- 既存の在庫は note記事 約126本（うちクラスタA 高岡富山ローカル 約70本）。これが3商材すべての原資。
- North Star＝海外読者への到達（オーナー方針・STATE.md）。海外強化7施策（英語サブタイトル／英語ハッシュタグ統一／Travel CTA／ローマ字＋英訳／英語要約の適切な長さ／共通語使用／公開時X連携）が既に定義済み。
- 公開は2026-06-12インシデント以降、棚卸し完了まで停止中。GTMは公開再開を起点に動く。

【考察】
- 親プランの結論「ボトルネックは書くことではなく届けること・束ねること」をGTMにそのまま落とす。つまり本計画の中心は **新規執筆ではなく「既存の無料記事を入口にして有料へ運ぶ導線設計」と「告知の運用」**。
- 3商材は読者の温度帯が異なる。冷たい層（初訪問の海外読者）→温かい層（既存フォロワー）→熱い層（事業者B2B）へと、別々のファネルで設計するのが自然。

【提案：3商材を1本のファネルに重ねる】

```
[入口＝無料] 既存note記事（日本語＋英語要約）／X・Reddit・EN記事のクロスポスト
        │  海外読者を集める（North Star）
        ▼
[商材A＝低単価デジタル] note有料マガジン（¥980）  ← 国内・既存フォロワー転換の主力
[商材B＝低単価デジタル] 英語トラベルガイド（Gumroad $12 / note ¥1,500）  ← 海外読者の受け皿
        │  「住人視点の発信」を実物で見せる＝実績ポートフォリオ化
        ▼
[商材C＝高単価B2B] 地域事業者向け英語発信代行（Standard ¥50,000/月）  ← 利益エンジン
```

- A・Bは「商品そのもの」であると同時に、**Cの営業材料（実績の見える化）**を兼ねる。3つは別物ではなく1つの導線。

## 3. 海外読者を増やす施策 × 販売の接続

【事実】海外強化7施策（STATE.md・CAO戦略）は既に定義済み。本計画ではこれを「到達」だけで終わらせず「販売」に接続する。

【考察】海外PVが増えても、その先に買える商品（B英語ガイド）と導線が無ければ売上に変わらない。逆に商品があっても海外到達が無ければ売れない。**7施策の一つひとつを、商材Bの販売動線の部品として再定義する**のが本節の狙い。

【提案：施策→販売の接続表】

| 海外強化施策 | 到達面の効果 | 販売への接続（どう売上に変えるか） |
|---|---|---|
| 英語サブタイトル | 海外読者の記事発見 | サブタイトルを商材Bの章テーマ（食/景観/文化/実用）と揃え、記事末から「このテーマの英語ガイドはこちら」へ誘導 |
| 英語ハッシュタグ統一 | 検索・回遊 | タグ群を固定し、タグ経由で来た海外読者をBガイドのGumroadへ集約（流入経路V6で効果測定） |
| Travel CTA（記事末） | 回遊・再訪 | CTA文言を「次に読む」から「持って旅する一冊（Bガイド）」へ段階的に差し替え。冷たい層には無料記事、温かい層にはガイドを出し分け |
| ローマ字＋英訳 | 海外読者の理解 | 料理・地名のローマ字表記をBガイドと統一し、記事⇔ガイドの語彙を一貫させ「同じ書き手の続き」と感じさせる |
| 英語要約（適切な長さ） | 滞在・離脱低減 | 要約末尾に1行のソフトCTA（毎回別パターン／A6）。例：今回は "There's a longer version of this if you're curious." 次回は別表現 |
| 共通語の使用（umami等） | 海外読者の親近感 | 共通語を入口に、固有の文脈はガイド本編で、という「無料で触り・有料で深掘り」の段差を作る |
| 公開時X連携 | 即時拡散 | 公開と同時に英語ツイート（§2カレンダー）。Bローンチ週はツイートに早期$9リンクを添える |

【接続の具体導線（クロスポスト→販売）】
1. 既存note記事を公開 → 英語要約＋英語タグ＋Travel CTA付き。
2. 同時にX（英語）・EN記事・Reddit（週2回）へクロスポスト＝海外の冷たい層に到達。
3. クロスポスト経由で来た読者を、まず**無料の英語要約記事**で受け、満足した層を**商材B（Gumroad）**へ。
4. Bを買った海外読者のうち、事業者・自治体関係者がいれば**商材C**の潜在リード（B2B）として扱う。

【考察】このファネルは「無料記事で海外到達 → Bガイドで初回課金 → Cで継続B2B」と、温度が上がるほど単価が上がる構造。7施策は全てこのファネルの“上流”を太くする投資として位置づけ直せる。
