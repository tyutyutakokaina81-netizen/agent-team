# プロジェクト：高岡note自動投稿

> **【これは恒久仕様＝引き継ぎ用の「記憶」です】**
> Claudeはセッションをまたいで記憶を保持しません。確実に引き継げるのは
> **Gitにコミットされたこのファイルの内容だけ**です。`context/` は `.gitignore`
> で除外されるため、そこに書いた指示は次回セッションに残りません。
> よって本プロジェクトのオーナー指示はすべてこの brief.md に正式記録します。

## オーナー指示（2026-05-29 受領・確定仕様）

1. **note** に記事を**自動的に公開**する
2. 記事の内容は **高岡市（富山県）の食や文化に関する体験記事**
3. **実写（フォトリアル）のサムネイル画像**も生成して添付する
4. **毎日**、**それぞれ違う内容で 3 本**アップする

## ゴール

毎日3本、高岡市の食・文化をテーマにした体験記事＋実写サムネを
自動生成し、noteへ自動公開するパイプラインを構築・運用する。

## 関与役職

- **CMO**：記事コンテンツ企画・執筆品質・トピック設計
- **CDO**：記事生成／画像生成／note自動公開パイプラインの実装・運用

## パイプライン

```
① トピック選定（毎日3本・重複なし）
   → ② 体験記事生成（高岡市の食・文化）
   → ③ 実写サムネ生成（画像生成API）
   → ④ note へ自動公開
```

## フォルダ構成

```
2026-05-29_高岡note自動投稿/
├── brief.md                  ← このファイル（恒久仕様＝記憶）
├── README.md                 ← 動かし方・必要な認証情報・実行手順
├── articles/                 ← 生成・公開した記事の本文（.md, 履歴として残す）
├── thumbnails/               ← 生成サムネ画像（.png 等／gitignore対象）
├── prompts/
│   ├── article_generation_prompt.md   ← 記事生成プロンプト（毎日違う内容を担保）
│   ├── topic_pool.md                  ← 高岡市トピックの母集団（重複回避用）
│   └── thumbnail_prompt_guide.md      ← 実写サムネ用プロンプト設計
└── pipeline/
    ├── config.example.json            ← 設定サンプル（認証はenv経由）
    ├── 01_select_topics.py            ← 本日の3トピックを選定（重複なし）
    ├── 02_generate_article.py         ← 記事生成（Claude API）
    ├── 03_generate_thumbnail.py       ← 実写サムネ生成（画像生成API）
    ├── 04_publish_note.py             ← note へ自動公開（Playwright）
    ├── run_pipeline.py                ← 毎日3本を通しで実行
    ├── state.example.json             ← 使用済みトピック記録（重複回避の状態）
    └── requirements.txt
```

## 実行に必要なもの（要オーナー準備・要承認）

| 項目 | 用途 | 設定方法 |
|------|------|---------|
| `ANTHROPIC_API_KEY` | 記事の自動生成 | `.env`（gitignore済） |
| 画像生成APIキー | 実写サムネ生成 | `.env`（プロバイダは config で選択） |
| note ログイン情報 | 自動公開 | 初回のみ人手ログイン→セッション保存（`.sessions/`、gitignore） |

> **公開・外部送信・スクリプト実行は会社ルール上「必ず事前承認」。**
> 認証情報が揃い、オーナー承認が出てから初めて実運用（自動公開）を行う。
> それまでは記事・サムネ・パイプラインの「生成と下書き」までを行う。

## 注意事項

- note には公式の投稿用APIがないため、公開は Playwright によるブラウザ自動化で行う
  （既存「柱D」パイプラインと同じ「初回ログインのみ人手」モデルを踏襲）
- 体験記事は**実在の有名スポット・名物**（高岡大仏／瑞龍寺／金屋町／雨晴海岸／
  高岡コロッケ／高岡銅器 など）を題材にし、**架空の店名・価格・日付を捏造しない**
- 機密（note認証・Cookie・APIキー）は**絶対にコミットしない**
