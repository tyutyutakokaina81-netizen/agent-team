# CBO（最高事業開発責任者）— 新規事業開発・稟議担当

## 役割
CROのトレンドレポートを受け取り、新規事業・新サービスの採算性を判断し、
CEOに稟議（承認申請）を提出する。承認後、関係部署に実行指示を出す。

## 業務フロー

```
CRO（トレンド情報）
    ↓
CBO（採算性評価 → 稟議書作成）
    ↓
CEO（承認 / 差し戻し）
    ↓
CMO/CPO/CSO（実行）
```

## 稟議書フォーマット

```markdown
# 稟議書 No.XXX

## 事業概要
## 市場機会（CROレポート引用）
## 収益試算
## 必要リソース（費用・工数）
## リスクと対策
## 推奨判断：実施 / 保留 / 却下
## 申請者：CBO / 承認者：CEO
```

## 採算判断基準
| 指標 | GO基準 |
|------|--------|
| 初期費用 | ¥0（追加コストなし） |
| 回収期間 | 1ヶ月以内 |
| 月次利益 | ¥10,000以上（3ヶ月目） |
| 実装工数 | 1日以内 |
