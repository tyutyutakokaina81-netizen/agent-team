# ランサーズ プロフィール文

> **目的**: クラウドワークスと並行してランサーズでもデータ入力案件を受注する

---

## プロフィール本文

```
データ入力・Webリサーチ専門のフリーランスです。

【得意な作業】
・ExcelやGoogleスプレッドシートへのデータ入力・整形
・Webサイトからの情報収集・リスト化（企業/店舗/商品）
・PDFや紙資料のデジタル化
・スクレイピングツールを使った大量データ取得

【ツール・スキル】
・Excel / Googleスプレッドシート（VLOOKUP・ピボット・関数）
・Python（requests・BeautifulSoup）
・Google Chrome / スプレッドシートの自動化

【仕事の進め方】
① ご依頼内容を確認後、不明点を先に質問してから着手します
② 作業途中でも随時進捗を共有します
③ 納品前に必ず自己チェックを行います
④ 修正対応1回無料で承ります

【実績・得意案件】
・EC商品情報のリスト化（数百〜数千件対応）
・Googleマップ店舗情報の収集・整形
・採用サイトの求人情報収集

丁寧・正確・期日厳守を心がけています。
小規模案件から大量処理まで対応可能です。
まずはお気軽にご相談ください。
```

## スキルカテゴリ設定

```
メインカテゴリ：データ入力・リスト作成
サブカテゴリ：Webリサーチ、Excel/スプレッドシート、スクレイピング
```

## 希望単価設定

```
データ入力：500〜2,000円（件数・難易度による）
Webリサーチ：1,000〜3,000円
スクレイピング：3,000〜10,000円
時給換算：1,000〜1,500円
```
