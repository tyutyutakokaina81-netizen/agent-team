🤖 ドロップイン LLMチャットボット（デモ）

このページの右下のボタンから、実際に会話できるチャットボットを試せます。どんなWebサイトにも <script> 2行で組み込めます。AI自動化サポートの納品物の一つです。

使ってみる： 右下「💬 AIに質問」→ 初回はAPIキー設定を促されます。
OpenAI互換のキーを入れると実際のLLM会話に。キーはこの端末のブラウザ内だけに保存され、外部に送信しません。キー無しでも簡易応答（台本）で動きます。

組み込み方（2行）

<script>
  window.CHATBOT_CONFIG = {
    title: "○○店 サポート",
    system: "あなたは○○店の問い合わせ対応アシスタント。丁寧に、断定せず、最終確認は人が行う前提で答える。",
    greeting: "こんにちは！ご質問どうぞ🤖",
    model: "gpt-4o-mini"
    // proxyUrl: "https://自社サーバ/chat"  ← 本番はキーをサーバ側に置く
  };
</script>
<script src="chatbot.js"></script>

特徴

  • 依存ゼロの単一JS。どのページにも貼れる。
  • 業種ごとに system プロンプトを差し替えるだけで“その店専用”の応答に。
  • APIキーは利用者の自前＝当社にAPIコストが発生しない。
  • キー無しでも台本フォールバックで動く（設定を促す）。

⚠️ 正直な注意

ブラウザにキーを置く方式は「自分用・社内用・小規模」向け。不特定多数に公開する本番では、キーは必ずサーバ側（proxyUrl）に置いてください（キー露出防止）。本番構築は当社で対応します。
