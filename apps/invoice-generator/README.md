# かんたん請求書ジェネレーター

依存ゼロの単一 HTML（`index.html`）で動く、フリーランス向けの請求書作成ツール。
外部API・CDN・ビルド不要。`index.html` をブラウザで開くだけで使えます。

## 特長

- **登録不要・無料・ブラウザ完結**：入力データは外部送信されず、利用端末の `localStorage` にのみ自動保存。
- **リアルタイムプレビュー**：入力に応じて右側（スマホは下側）の請求書が即時更新。
- **PDF保存**：「印刷 / PDF保存」ボタン → ブラウザの印刷ダイアログ →「PDFとして保存」。印刷用CSS（A4・余白調整）を内蔵。
- **入力項目**：請求書番号／請求日／支払期限／消費税率（10%・8%・0%）／登録番号（インボイス・任意）／請求先・請求元／品目（追加・削除可）／振込先／備考。
- **スマホ/PC対応**の日本語UI。

## 使い方

1. `index.html` をブラウザで開く。
2. 基本情報・請求先/請求元・品目・振込先を入力（「サンプルを入れる」で架空例を読み込めます）。
3. 右の（スマホは下の）プレビューを確認。
4. 「印刷 / PDF保存」を押し、出力先を **PDFとして保存** に。
5. 入力は自動保存され、次回開いたときに復元されます。「入力をクリア」で消去できます。

> データはブラウザごと・端末ごとの保存です。別端末や別ブラウザには引き継がれません。
> ブラウザの履歴/サイトデータを消すと入力も消えます。大事な請求書はPDFで保存してください。

## 公開手順（GitHub Pages）

このファイルはそのまま静的公開できます。

- **方法A（Pages設定で公開）**：リポジトリの Settings → Pages で対象ブランチ/フォルダを指定。
  `apps/invoice-generator/` をルートにしたい場合は、その配下をデプロイ対象にするか、独立リポジトリ/サブツリーで公開する。
- **方法B（単体ホスティング）**：`index.html` 1ファイルを任意の静的ホスティング（Pages / Netlify / 任意のWebサーバ）に置くだけ。

## 収益化メモ（控えめ運用）

- プレビュー下に **【商品リンク】**（テンプレ集 / 収支シートへの導線）を1箇所設置。`index.html` 内
  `id="productLink"` の `href="#"` を実URL（note/BOOTH等）に差し替えるだけ。
- 同じ枠に **`<!-- ad slot -->`** を1箇所用意（`id="adSlot"`）。広告タグを入れる場合はここに挿入。
  ※ 当面は無広告でも成立。入れる場合も読者体験を損なわない控えめ運用を推奨。
- 印刷時は商品リンク・広告枠ともに非表示（`print-hide`）になり、請求書本体だけが出力されます。

### 集客の考え方

- 「請求書 テンプレート 無料」「請求書 作成 PDF」等は検索需要が大きい定番ニーズ。軽量・登録不要は離脱を抑えやすい。
- ツール利用者＝個人事業主/フリーランスで、当社のテンプレ集・収支シートと読者層が一致 → 相互送客が狙える。
