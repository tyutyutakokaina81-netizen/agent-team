# 特定商取引法に基づく表記（note / BOOTH 用テンプレ）

作成日：2026-05-21  
作成：CLO  
用途：柱C（テンプレート販売）の note / BOOTH 等での特商法表記

> **【注記】** デジタルコンテンツ販売における特商法表記の一般的なひな形です。
> 個人事業主の住所・電話番号開示には注意が必要です。実際の運用前に
> 行政書士または弁護士に確認してください。

---

## 1. 特商法表記が必要な理由

特定商取引法第11条は、通信販売事業者に対して以下の項目の表示を義務付けている。
note や BOOTH での販売も「通信販売」に該当するため、表記義務がある。

| プラットフォーム | 状況 |
|----------------|------|
| note（有料記事・有料マガジン） | 販売者自身で特商法表記を整備する必要あり |
| BOOTH | プロフィール／ショップ設定で表示 |
| Notion Market | 出品者情報として表示 |

## 3. 「住所・電話番号」の取扱について

特商法上は **原則として住所・電話番号の表示が義務** だが、以下の例外運用が認められている：

### 例外運用（消費者庁ガイドラインに基づく）
> 「営業所等の所在地・電話番号は、購入者の請求に応じて遅滞なく開示する」
> 旨を表示し、実際に請求があれば速やかに開示する運用であれば省略可。

ただし以下の条件が必要：
1. **「請求があれば遅滞なく開示する」旨を明示**
2. **メール等で請求受付窓口を明示**
3. **請求があれば原則 5 営業日以内に開示する運用**
4. 顧客が事業者に容易に連絡できる手段（メール等）が確保されていること

### 推奨運用
- 当面はメール開示型で運用
- 売上規模が拡大したらバーチャルオフィス契約 → 住所を直接掲載に切り替え
- 行政書士・弁護士に確認したうえで運用開始

## 4. 利用許諾・規約セクション（販売ページ別途記載）

特商法表記とは別に、以下を販売ページ本文または「ご利用にあたって」セクションに必ず記載する：

```
■ ご利用にあたって

本テンプレートに含まれる Excel ファイル、Notion テンプレート、
プロンプト集（以下「本商品」）の著作権は販売者に帰属します。

【許諾される利用】
- 購入者ご本人による業務利用・個人利用
- カスタマイズして自身のために使用すること

【禁止される利用】
- 本商品の二次配布・転売・再販売
- 本商品のコンテンツを抜粋して有償・無償で配布すること
- 本商品をテンプレ販売サービスの教材として転用すること

【免責事項】
- 本商品の利用により得られる成果は個人差があり、
  特定の結果を保証するものではありません。
- AIプロンプトの出力結果は、ご利用のAIサービスの仕様により異なります。
- 本商品の利用により生じたいかなる損害についても、
  販売者は責任を負いません。

【サポート】
- ダウンロードに関するご質問は購入後7日以内にお問い合わせください。
- 「カスタマイズ相談30分」が含まれる商品は、購入後30日以内に
  下記アドレスからご予約ください：【相談予約フォームURL】
```
