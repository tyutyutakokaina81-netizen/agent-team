# ふるさと納税 アフィリエイト 表示ガイドライン

**作成日**: 2026-05-10
**担当**: CLO（CMO・CFO 連携）
**目的**: 高岡日和 YouTube 動画の概要欄・SNS 投稿で、ふるさと納税アフィリエイトリンクを **法的に問題なく** 設置する
**コスト**: ¥0
**主に関連する法令**: 景品表示法（ステマ規制）、特商法、消費者契約法

---

## 結論：4 つの遵守事項

1. **「広告」「PR」表示の明示**（ステマ規制対応）
2. **誇大・断定表現の排除**（景表法）
3. **アフィリエイトであることの透明性**
4. **個別商品の効能・品質の保証はしない**

---

## ① ステマ規制（2023 年 10 月施行）の遵守

### 法的要件
- 事業者が第三者を装って広告すること = 違反
- 自己の運営チャンネル・SNS でも、**自社製品でないアフィリ商品** を紹介する場合は明示が必要

### 具体的な表示例

#### YouTube 動画（概要欄）
```
▼ 関連リンク
・高岡市ふるさと納税（楽天）：[アフィリエイトリンク]
  ※本リンクは楽天アフィリエイトプログラムを通じた広告（PR）です。
  ※当チャンネルが利用者から購入手数料を受け取る場合があります。
```

#### YouTube 動画（動画内）
- CTA で言及する際は **動画内テロップで「広告」または「PR」と表示**
- 桐生結のセリフでも一言：「概要欄のリンクは広告（アフィリエイト）です」

#### X 投稿
```
（投稿本文）

[本文]

#PR #高岡市ふるさと納税
※楽天アフィリエイトを利用しています
```

#### note 記事
```
[文中で初めてリンクを出す箇所]
（本記事には楽天アフィリエイトのリンクが含まれます）
```

### NG 例（ステマ違反になる表示）
- 「ふるさと納税で買ったよ！おすすめ」だけ（広告と分かりにくい）
- 「私が選んだ最高の返礼品はこれ」だけ（個人推薦を装う）
- ハッシュタグの隅に小さく「#ad」だけ → 認識されないリスク

## ② 景表法（誇大・断定表現の排除）

### NG 表現の例
- 「絶対損しない」「必ずお得」
- 「最高の返礼品」「日本一美味しい」
- 「健康になる」「免疫力がアップする」（薬機法も）
- 「在庫無くなる前に」（実際の在庫状況を示す根拠がない場合）

### OK 表現の例
- 「私が選んだ返礼品の一つです」
- 「2024 年に話題になった人気返礼品」（事実）
- 「高岡銅器の伝統技法を使った作品」（事実説明）
- 「ふるさと納税は実質 ¥2,000 で返礼品が受け取れる制度です」（制度説明）

---

## ③ アフィリエイト規約の遵守

### 楽天アフィリエイト規約の主要要件
- アフィリエイト ID の適切な管理（不正クリックの禁止）
- リンクの差し替え・偽装の禁止
- 楽天ロゴ・商標の許諾範囲内での使用
- 規約違反時のアカウント停止リスク

### A8.net（さとふる扱い）の主要要件
- 同上 + ASP 別の細則あり
- 月次の報酬振込・税務処理

### 当チャンネルの内部ルール
- 1 動画につきリンク 1 種類（楽天 ふるさと納税 のみ）
- リンクは概要欄の「関連リンク」セクションに統一
- 動画内で「他社 ASP の方が単価が高い」等の比較はしない

## ④ 視聴者への誠実な情報開示

### 透明性のための定型文（概要欄末尾）

```
▼ アフィリエイトについて
本チャンネルは、楽天アフィリエイト・A8.net 等のアフィリエイトプログラムを利用しています。
動画内で紹介した商品・サービスを概要欄のリンクから購入された場合、
当チャンネルが手数料を受け取ることがあります（視聴者様への追加負担はありません）。

紹介する商品・サービスは、当チャンネルが事実に基づいて情報をお伝えするもので、
個別の商品の品質・効果を保証するものではありません。
ご購入は最終的にご自身の判断でお願いいたします。
```

## ⑤ ふるさと納税固有の留意点

### 制度説明の正確性
- 控除額は **個人の収入・家族構成** によって変わる旨を必ず明記
- 「実質 ¥2,000」は **限度額内で寄付した場合のみ** が条件
- ワンストップ特例 vs 確定申告 の選択肢を中立的に説明

### 例：制度説明の中立的文面（コピペ用）

```
ふるさと納税は、応援したい自治体に寄付することで
控除（住民税・所得税）が受けられる制度です。

控除額は寄付者の収入・家族構成によって異なるため、
ご利用前に各サイトのシミュレーターで上限額をご確認ください。

確定申告の手続き（または「ワンストップ特例」の申請）が必要です。
詳しくは総務省のふるさと納税ポータルサイトをご参照ください。
```

### 高岡市特有の注意
- 高岡銅器・高岡漆器は **伝統工芸品** として扱われる返礼品が多い
- 職人の手作業のため在庫数が限られる旨は事実として説明可
- 個別の作家・工房名を紹介する場合は **取材許可** を取る（CSO の取材依頼テンプレ参照）

## ⑥ 違反時のペナルティ

### ステマ規制違反
- 公正取引委員会から **措置命令**（再発防止命令・公表）
- 課徴金（売上の数 % 程度、悪質な場合）
- 社会的信用の失墜

### アフィリエイト規約違反
- アカウント即時停止 → 過去報酬も没収
- 楽天 / A8.net などからの永久 BAN

### 当チャンネルの対応
- 違反指摘を受けた場合、24 時間以内に当該動画を非公開化
- 訂正・修正を反映して再公開
- CLO 主導で原因分析・再発防止策を CLO/research/ に記録
