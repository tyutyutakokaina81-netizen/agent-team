# SNS運用代行 業務委託契約書（テンプレ）

作成日：2026-05-21  
作成：CLO  
用途：柱B（SNS運用代行）専用契約書テンプレ

> **【注記】** ステマ規制・アカウント所有権・成果保証の不存在を明確化した
> SNS運用代行向けの契約書ひな形です。実際に締結する前に弁護士のレビューを推奨します。

---

```
SNS運用代行業務委託契約書

【委託者】（以下「甲」という）：【会社名／屋号／氏名】
【受託者】（以下「乙」という）：【屋号／氏名】

甲及び乙は、甲のソーシャルネットワーキングサービス（以下「SNS」という）
アカウントの運用を乙に委託することに関し、次のとおり契約を締結する。

第1条（業務内容）
1. 乙は甲に対し、次の業務（以下「本業務」という）を提供する。
   (1) 対象SNSアカウント：
       【Instagram: @xxxx】【X: @xxxx】【Facebook: xxxx】
   (2) 月次投稿数：【15】投稿
   (3) 投稿内容：投稿文の作成、ハッシュタグ選定、画像テキスト案
   (4) 予約投稿の実行（Meta Business Suite / Buffer / Later 等の認可ツール経由）
   (5) 月次レポート：フォロワー数、リーチ数、エンゲージメント数の集計

2. 本業務は準委任契約として履行する（成果完成義務は負わない）。

第2条（投稿主体・アカウント所有権）
1. 本契約に基づく投稿の **主体は甲（広告主）** であり、乙は甲の補助者として
   投稿作業を代行するに過ぎないことを甲及び乙は確認する。
2. 対象SNSアカウントの所有権・運営責任は甲に帰属する。
3. 乙は甲から提供されたID・パスワード等の認証情報を厳重に管理し、
   契約終了時には速やかに返却・削除する。
4. 乙は本契約期間中、甲のアカウントを甲の事業目的以外に使用しない。

第3条（ステマ規制等への対応）
1. 乙は景品表示法その他関連法令を遵守して本業務を遂行する。
2. 甲が第三者（インフルエンサー等）への投稿依頼を本業務に含める場合、
   乙は当該投稿に「#PR」「#広告」「#提供」等の表示を行うか、または
   甲に対し当該表示の必要性を助言する。
3. 乙は次の依頼を受託しない：
   (1) 第三者に「自発的な感想」を装って投稿させる依頼
   (2) 架空のお客様の声を作成する依頼
   (3) 景表法上の優良誤認・有利誤認表示に該当する投稿の作成依頼
4. 規制業種（医療・薬事・士業・金融等）の場合、甲は該当業法を遵守した
   投稿内容のチェックに協力する。

第4条（投稿内容の確認）
1. 乙は投稿予約前に甲に投稿案を提示し、甲の承認を得るものとする。
2. 甲は投稿案受領後【3】営業日以内に承認・修正指示を行う。
3. 上記期間内に応答がない場合、承認したものとみなす（みなし承認）。
4. 投稿後に甲から削除依頼があった場合、乙は速やかに対応する。

第5条（NG事項）
甲は契約締結時に次の事項を乙に通知する：
- 投稿してはならないキーワード・トピック
- 競合他社の表記
- トーン（カジュアル／プロフェッショナル）
- 既存顧客・取引先の情報の取扱

第6条（成果保証の不存在）
1. 乙は次の成果について保証しないことを甲は了承する：
   (1) フォロワー数・リーチ数・エンゲージメント数の特定数値の達成
   (2) 売上・問い合わせ件数・予約数の特定数値の達成
   (3) SNSアルゴリズムの変動による表示順位
2. 乙は誠実に業務を遂行する義務（善管注意義務）を負う。

第7条（契約期間・解約）
1. 契約期間：契約締結日から【3】ヶ月間。期間満了の【1】ヶ月前までに
   いずれからも書面通知がなければ、同条件で【1】ヶ月単位で自動更新する。
2. 甲又は乙は、【1】ヶ月前までに書面で通知することにより本契約を解約できる。
3. 重大な契約違反があった場合は、催告のうえ即時解約できる。

第8条（料金・支払）
1. 月額料金：【50,000】円（税込）
2. 支払時期：毎月【末日】までに翌月分を乙指定口座に前払い
3. 初回お試し価格：契約開始月の月額を【25,000】円とする
   （次月以降は通常価格）
4. 振込手数料は甲の負担とする。

第9条（権利帰属）
1. 投稿に使用された画像・テキスト等のコンテンツの著作権は、乙が
   創作した部分については乙に帰属し、甲は契約期間中およびその後も
   非独占的に利用する許諾を受けるものとする。
2. 甲が提供した素材の著作権は甲に帰属する。
3. 甲は乙に提供する画像・テキスト・顧客情報等が適法に取得・保有された
   ものであることを表明保証する。
4. 第三者画像（フリー素材等）を使用する場合、乙は商用利用可能な
   ライセンスのものに限定して使用する。

第10条（顧客個人情報の取扱）
本業務遂行のため、乙が甲の顧客個人情報（DMの返信先、リスト等）を
取り扱う場合、乙は個人情報保護法に従い、目的の範囲内でのみ使用し、
適切な安全管理措置を講じる。

第11条（秘密保持）
（汎用契約書テンプレ第8条と同様）

第12条（解除・反社条項・損害賠償・準拠法）
（汎用契約書テンプレ第12条〜第16条と同様）

【日付】YYYY-MM-DD
甲：
乙：
```

## 使用上の注意

- **第2条と第3条が本契約書の核**：投稿主体の明確化、ステマ規制対応
- 「初回お試し価格」は **景表法上の二重価格表示** に注意。継続的に通常価格で
  販売実績があってから「割引」を表示するのが本来。それまでは「お試し価格」と
  名乗ることでリスクを下げる
- 第6条（成果保証の不存在）は **必須**。フォロワー◯倍などの口頭約束をしない
- 規制業種クライアントの場合は別途業種別 NG リストを契約附則として添付

> 正式な法律意見ではありません。実際に締結する前に、弁護士など
> 有資格者によるレビューを受けてください。
