# 反社会的勢力排除に関する基本方針

作成日：2026-05-21  
作成：CLO  
対象：オーナー（個人事業主としての対外掲示・契約添付用）

> **【注記】** 一般的な反社会的勢力排除方針のひな形です。
> 正式公表前に行政書士・弁護士のレビューを推奨します。

---

## 公表用テンプレ（事業ドメイン・契約書末尾添付用）

```
反社会的勢力排除に関する基本方針

【屋号／氏名】（以下「当方」という）は、市民社会の秩序や安全に
脅威を与える反社会的勢力に対して断固たる態度で臨み、
取引関係を一切持たないことを基本方針として表明します。

1. 取引先・顧客に対して、反社会的勢力でないことを表明・保証することを求めます。
2. 反社会的勢力であることが判明した場合、直ちに取引を停止します。
3. 反社会的勢力からの不当要求には毅然と対応し、必要に応じて警察・
   弁護士などの専門家と連携します。
4. 反社会的勢力との取引を行わないため、契約締結時に反社条項を盛り込みます。

【制定日】2026-05-21
【屋号／氏名】
```

---

## 契約書条項テンプレ（業務委託契約書に添付）

```
第○条（反社会的勢力の排除）
1. 甲及び乙は、自己又は自己の役員・従業員・関係者が、暴力団、暴力団員、
   暴力団準構成員、企業舎弟、総会屋、社会運動標榜ゴロ、特殊知能暴力集団等
   （以下「反社会的勢力」という）に該当しないこと、及び反社会的勢力と
   一切の関係を有しないことを表明し保証する。

2. 甲又は乙は、相手方が前項に違反していることが判明した場合、
   催告なく本契約を解除できる。

3. 前項により本契約が解除された場合、解除された当事者は相手方に対して
   損害賠償を請求できない。

4. 解除した当事者が損害を被った場合、解除された当事者は当該損害を
   賠償する義務を負う。
```

※既存契約書テンプレ（汎用／SNS運用代行／スクレイピング／NDA）には反社条項が
含まれているため、新規作成時はそのまま使用可。

## 取引開始時の確認フロー

1. **契約締結時**：反社条項を含む契約書に署名（または電子署名）
2. **不審な兆候があった場合**：
   - 取引相手の登記情報を確認
   - 警察庁「全国暴力追放運動推進センター」相談（無料）
   - 必要に応じて弁護士相談
3. **判明した場合**：
   - 速やかに取引停止
   - 取引履歴・連絡記録を保全
   - 警察・弁護士と連携

---

## 配置場所

- 事業ドメインのフッターからリンク（公開時）
- 業務委託契約書の末尾（契約締結時）
- 名刺・営業資料（必要に応じて）
