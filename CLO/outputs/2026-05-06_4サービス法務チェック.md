## ① ココナラ「AI 活用 SEO 記事ライティング 24 時間納品」

### 判定: GO（軽微修正あり）

| チェック項目 | 結果 | コメント |
|-------------|------|---------|
| ココナラ ToS | ✅ | ライティング系は推奨カテゴリ |
| AI 利用の明示 | ✅ | 出品文に明記済（推奨） |
| 著作権譲渡 | ⚠️ | 「納品後譲渡」は OK だが、AI 生成物の権利は別途明記推奨 |
| 薬機法 | ⚠️ | 「医薬品・健康食品系記事は別途相談」と免責記載推奨 |
| 景表法 | ⚠️ | 「プロライターが校正」表現の根拠（経歴）をプロフィールに記載推奨 |
| 特商法 | ✅ | ココナラがプラットフォームとして対応 |

### 修正提案

出品文の「お願い」セクションを以下に置換：

```
【お願い】
・ご依頼前に「キーワード」「文字数」「想定読者」「参考 URL」をお知らせください
・薬機法・景表法に抵触する表現は対応不可です（医薬品・健康食品・ダイエット等）
・著作権は納品後にお客様に完全譲渡します（AI 生成部分も含めて譲渡可能な範囲）
・ステルスマーケティング目的の記事はお受けできません（2023 年 10 月の規制以降）
```

## ② ココナラ「飲食店向け SNS プロンプト集 個別カスタマイズ」

### 判定: GO

| チェック項目 | 結果 | コメント |
|-------------|------|---------|
| ココナラ ToS | ✅ | マーケ系で適合 |
| 著作権 | ✅ | プロンプト自体に著作物性は限定的、納品物は譲渡可 |
| 景表法（飲食関連） | ⚠️ | 「絶対集客できる」等の効果保証表現は出品文から排除（現状 OK） |
| 薬機法 | ⚠️ | 飲食店でも「健康・病気予防」訴求は薬機法対象、プロンプト集にも免責推奨 |

### 修正提案

「お願い」セクションに追加：

```
・健康効果・病気予防を訴求するプロンプトはお作りできません（薬機法）
・「絶対集客できる」等の保証表現は含みません（景表法）
・お店の実績・口コミは事実ベースのみ反映します
```

## ③ ココナラ「Excel データ入力代行」

### 判定: CAUTION（スクレイピング部分の明文化必要）

| チェック項目 | 結果 | コメント |
|-------------|------|---------|
| ココナラ ToS | ✅ | データ入力は適合 |
| **スクレイピングの合法性** | ⚠️ | サイトごとに ToS 確認必須、無条件 OK は危険 |
| 個人情報保護法 | ⚠️ | 顧客から預かるデータに個人情報が含まれる場合、取扱いを明記 |
| 著作権 | ⚠️ | スクレイピング元データに著作権がある場合、転載は違法 |

### 必須修正（NO-GO 回避のため）

「スクレイピング不可サイト」の例示が曖昧。以下に書き換え：

```
【スクレイピングについて】
公開情報の収集・転記は、以下の条件をすべて満たす場合のみお受けします：
✓ 対象サイトの利用規約でスクレイピング・自動収集が禁止されていない
✓ 著作権侵害にあたらない範囲（事実情報のみで創作的表現は転記しない）
✓ アクセス頻度を抑え、サーバー負荷をかけない範囲
✓ 個人情報を含む場合は事前にご相談（同意取得が必要なケースあり）

以下は対応不可：
✗ クラウドワークス・ランサーズ・ココナラ等のクラウドソーシング案件情報
✗ Amazon・楽天等 EC サイトの商品情報の大量収集
✗ X（Twitter）等 SNS の投稿の機械的収集
✗ ログインが必要なサイト
```

### 弁護士確認推奨
スクレイピング案件を継続的に受ける場合は、判例（リクナビ事件・図書館事件等）の動向を弁護士確認推奨。

## ⑤ X 投稿 7 本

### 判定: 投稿 1〜3, 5, 6 = GO ／ 投稿 2 = CAUTION ／ 投稿 4, 7 = GO（PR 表示推奨）

### 個別判定

| 投稿 | 内容 | 判定 | 修正点 |
|------|------|------|--------|
| 1 | 確定申告で詰む | ✅ GO | なし |
| 2 | 経費漏れ年 8〜15 万 | ⚠️ CAUTION | 「らしい」表現が景表法上甘い、根拠記載 or 「経験上」「個人事業主の実感として」に修正推奨 |
| 3 | 入金タイミング | ✅ GO | なし |
| 4 | Vol.1 紹介 | ✅ GO | 自社商品のため PR 表示は不要、ただし URL に `?utm_source=x` 等付ければアフィリと混同される懸念ゼロ |
| 5 | Before/After | ✅ GO | なし（個人体験談なら景表法対象外） |
| 6 | 副業 20 万円ルール | ✅ GO | 税法は正確、税理士監修注記があれば尚良し |
| 7 | DM が 3 件きた | ✅ GO | 事実なら問題なし、虚偽だと景表法違反 |

### 投稿 2 の修正案

```
個人事業主の経費漏れ、私の周りでは年 8〜15 万円が体感ベースで多い印象。

・電車代の Suica チャージ
・コンビニで買った文房具
・サブスクの年払い
・取引先との 1 杯

「あとでまとめる」が一番高くつく。
週 1 で 10 分入力するだけで、節税になる。

#フリーランス #個人事業主
```

「らしい」を「私の周りでは…体感ベース」に修正することで、誇大表現リスクを下げる。

### ステマ規制（2023 年 10 月以降）について

- 自社商品の宣伝（投稿 4・7）は **広告主自身の発信** なので「PR」表示は法的に不要
- ただし第三者に依頼して投稿してもらう場合は **必ず「#PR」「#広告」明示**
- アフィリエイトリンクを貼る場合は要 PR 表示

## 弁護士相談を推奨する領域

1. **スクレイピング受託の継続運用**（判例動向の把握）
2. **ライティングで医療・健康・金融分野を扱う場合**（薬機法・金商法）
3. **海外顧客対応開始時**（GDPR・各国広告規制）
4. **SNS 運用代行で 5 社以上抱えた段階**（業務委託契約の整備）

弁護士相談の費用ゼロ手段：
- 法テラス（収入要件あり）
- 中小企業庁の無料法律相談（年数回）
- 商工会議所会員の無料相談（会費別途）
- 各 SaaS 法律相談の初回無料枠
