### 2026-05-13
- **学び**: ChatGPT は請求書 PDF までは生成できないが、Claude Code はデザイン込みで PDF 出力まで一貫して可能。
- **背景**: 杉田氏の動画事例。「請求書作って」とだけ伝え、会社情報・取引先・単価感を学習済みの Claude Code が PDF を即時生成（納期3日→30秒）。
- **示唆**: AI ツール選定では「どこまでアウトプットを完成形で出せるか」が分水嶺になる。チャット型 AI と CLI 型 AI の役割分担を当社のツール棚卸しに反映する。
