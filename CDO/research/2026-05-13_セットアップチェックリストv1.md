# CDO セットアップチェックリスト v1（手元PC実機検証用）

## 目的
本日5役職分の試作メモ・v1具体物を**オーナーの手元PCで実際に動かす**ための、CDO 監修の最短手順チェックリスト。サンドボックスでは検証不可、本ドキュメントは「動かす順序」のみ示す。

## 前提
- 手元PC：Claude Code が起動できる環境
- Google アカウント：Gmail 連携を試す場合に必要
- 試作段階：すべて**自分宛・ダミー値**でのみ実行
