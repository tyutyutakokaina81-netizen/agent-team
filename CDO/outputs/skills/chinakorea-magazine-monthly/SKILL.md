# SKILL: 中国語・韓国語マガジン 月次自動作成

**スキル名**：chinakorea-magazine-monthly  
**担当役職**：CMO（コンテンツ作成）+ CDO（自動化管理）  
**トリガー**：毎月1日 / または「マガジン作って」「次号作って」「magazine」と言われたとき  
**保存先**：`/Users/apple/agent-team/CMO/outputs/`

---

## このスキルが行うこと

毎月1日に、高岡インサイダーズ月刊マガジンの新号を自動作成する。

1. 季節テーマを自動決定（月に応じた内容に更新）
2. 中国語セクション（6記事）を執筆
3. 韓国語セクション（6記事）を執筆
4. Word ドキュメントとして出力
5. CMO/_index.md を更新
6. Gumroad にアップロードすべきファイルのパスを報告

---

## 実行手順

### Step 1：前号の確認

```
CMO/outputs/ 内の最新号を確認：
- 最新ファイル名：Takaoka_Insiders_Monthly_Issue**.docx
- 前号のテーマ・取り上げたスポット・食材をリスト
- 同じ内容の重複を避ける
```

### Step 2：当月テーマの決定

以下の月別テーマカレンダーに従い、今月号のメインテーマを決定する。

| 月 | メインテーマ | 季節キーワード |
|----|------------|--------------|

| 2月 | 伝統行事と節分 | 伝統工芸・地酒・甘酒 |
| 3月 | 春の訪れ・梅と北前船 | 梅・白えび解禁・春の準備 |
| 4月 | 桜と高岡の祭り | 高岡御車山祭・桜・春野菜 |
| 5月 | 新緑と伝統工芸体験 | 若葉・ホタルイカ最終期・連休 |
| 6月 | 梅雨の高岡 | 鮎・梅・萤・雨の金屋町 |
| 7月 | 海と漁師の町・新湊 | 内川・夕焼け・海産 |
| 8月 | 夏の縁日と高岡花火 | 万葉まつり・ホタル・夏野菜 |
| 9月 | 秋の実り・新米と地酒 | 五百万石新米・さんま・紅葉前 |

| 12月 | 年末の高岡と越中の食 | 年越し蕎麦・ぶり・師走の生活感 |

### Step 3：コンテンツ生成ルール

#### 中国語セクション（6記事）
各記事はてつさん（高岡市在住の一人称視点）で書く。

1. **编辑来信**（200字）：当月の季節感・その月に高岡で感じたこと
2. **本月焦点**（500字）：メインテーマの深掘り（観光客が知らない一次情報）
3. **秘密景点：本地居民才知道的3个地方**（600字）：地元民しか知らない3スポット
4. **美食情报：本月当季食材**（400字）：旬の食材3種（具体的な名前・特徴・食べ方）
5. **实用贴士**（400字）：当月の実用ガイド（交通・注意点・持ち物など）
6. **下月预告**（100字）：来月号のテーマをひとこと予告

#### 韓国語セクション（6記事）
中国語版と内容は対応するが、表現は独自に書く（翻訳ではない）。

1. **편집장의 말**（200字）
2. **이달의 특집**（500字）
3. **현지인 비밀 스팟 3선**（600字）
4. **이달의 제철 먹거리**（400字）
5. **실용 가이드**（400字）
6. **다음 달 예고**（100字）

#### 品質基準（必須遵守）
- ❌ 「Xには、Yがある」型テンプレートフック禁止
- ❌ 同号内での結論重複禁止
- ❌ 根拠のない「日本唯一」「世界最高」禁止
- ✅ てつさんの一次観察・体験を必ず1ヶ所以上含める
- ✅ 具体的な地名（市レベルまで）・食材名・交通情報を含める
- ✅ 中国語と韓国語は独立した文体で書く（翻訳調にしない）

### Step 4：Wordドキュメント生成

`gen_magazine.py` の手順に従い、Python でコンテンツを書き出し、`node create_magazine.js` で DOCX を生成する。

- ファイル名：`Takaoka_Insiders_Monthly_Issue{号数:02d}_{YYYY}-{MM}.docx`
- 保存先：`/Users/apple/agent-team/CMO/outputs/`
- 中国語フォント：SimSun
- 韓国語フォント：Malgun Gothic

### Step 5：_index.md の更新

`CMO/_index.md` の成果物ログに追記：

```markdown
| {YYYY-MM-DD} | Takaoka_Insiders_Monthly_Issue{N}_{YYYY}-{MM}.docx | 中韓マガジン | Issue {N} {月}号 — テーマ：{テーマ} | 完了 |
```

### Step 6：完了報告

```
✅ 中韓マガジン Issue {N}（{YYYY}年{M}月号）完成

📄 Word ファイル：CMO/outputs/Takaoka_Insiders_Monthly_Issue{N}_{YYYY}-{MM}.docx
📋 テーマ：{今月のメインテーマ}
🌏 中国語：{記事タイトル一覧}
🇰🇷 韓国語：{記事タイトル一覧}
🗓️ 次号テーマ予定：{来月のテーマ}

→ Gumroad へのアップロードはてつさんが手動で実施してください。
```

## スケジューラ設定（毎月1日自動実行）

```yaml
# schedule設定例（Cowork スケジュール機能）
name: chinakorea-magazine-monthly
schedule: "0 9 1 * *"   # 毎月1日 午前9時
action: "次号のマガジンを作って"
description: 中国語・韓国語マガジン月次自動作成
```

**CDO による毎月1日の実行プロンプト：**
```
次号（Issue {N}）の中韓マガジンを作成してください。
前号のテーマは「{前号テーマ}」でした。
当月（{YYYY}年{M}月）のテーマカレンダーに従い、以下を実行してください：
1. コンテンツ作成（中国語6記事 + 韓国語6記事）
2. Wordドキュメント生成
3. CMO/_index.md 更新
4. 完了報告
```
