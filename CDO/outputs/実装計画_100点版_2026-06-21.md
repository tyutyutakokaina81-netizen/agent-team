## Phase 3：スケール判定（Month 3 末）

```
Month 3 の成果をもとに、以下を判定：

判定1：複数施策が同時に月¥50K+を生み出してるか？
  ✅ YES（note + テンプレ + 営業 など複合）
     → 各施策のスケール化＆自動化強化へ
  
  ❌ NO（1施策だけが成功）
     → その1施策に集中 + 他は放置

判定2：B2B営業は再現性があるか？
  ✅ YES（3ヶ月で3+件成約）
     → チーム化・外注化を検討
  
  ❌ NO（成約1件以下）
     → B2B営業は一時停止 + 他軸へ

判定3：オーナーの時間は持続可能か？
  ✅ YES（週30時間以下）
     → 現施策を継続＋小規模拡大
  
  ❌ NO（週40時間以上）
     → 自動化・外注・優先順位見直し

---

Month 4-6：スケーリング戦略

  IF 複合施策で成功している場合：
    ├─ note: 有料記事＋メンバーシップ強化（¥50-100K）
    ├─ 営業: 成功パターンの複製（¥100-200K）
    └─ コース：教材化して販売開始（¥50-100K）
    
    期待値：月¥200-400K 到達

  IF 1施策（例：B2B営業）だけ成功の場合：
    └─ その施策にリソース 100% → チーム化
       └─ 営業メール 500通/月 → 営業代行者雇用
       └─ 記事納品 月20本 → フリーランス外注
       └─ 自分は営業＆顧客管理に集中
    
    期待値：月¥300K+達成（6-9ヶ月後）
```

## 成功判定基準（Go/No-Go）

### Go パターン（Phase 2A へ進む）
```
Week 4 末時点で：
  ✅ note有料化: ≥5人登録（月収¥4K+）
  ✅ テンプレVol.2: ≥1件販売
  ✅ Mailchimp: 本番フロー安定稼働
  ✅ 合計月収: ¥20K以上推定
  
  → 「基礎が固まった」と判定
  → B2B営業テスト開始してOK
  → 同時に note/テンプレは「ルーティン化」（自動）
```

### No-Go パターン（Phase 2B へ移行）
```
Week 4 末時点で：
  ❌ note有料化: 0-2人登録のみ
  ❌ テンプレVol.2: 0件販売
  ❌ Mailchimp: 設定不具合 or テスト段階
  ❌ 合計月収: ¥10K以下推定
  
  → 「基礎が脆弱」と判定
  → 原因分析（内容？価格？マーケティング？）
  → 代替戦略（コース化 or コンサル化）へシフト
  → B2B営業は一時停止
```
