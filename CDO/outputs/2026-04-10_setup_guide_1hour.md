# 1時間セットアップガイド
# ── これだけやれば、あとは永久に触らなくていい ──

**所要時間**: 約60分（1回限り）  
**やること**: 4つのアカウント作成 + パイプライン起動  
**これが終わったら**: AIが自動で稼ぎ続ける

```
STEP 1: CrowdWorks 登録（15分）
```

## STEP 3｜パイプライン ログイン設定（10分）

> これが終わると、翌日からAIが自動で案件を探して応募し始める

```bash
# ターミナルで実行（このリポジトリのフォルダで）
cd ~/agent-team
python3 projects/2026-04-08_月30万自動化/D_エクセル入力スクレイピング/pipeline/00_session_setup.py
```

**実行すると何が起きるか**:
1. ブラウザが自動で開く
2. CrowdWorks のログイン画面が表示される
3. 手動でログイン（ID・パスワードを入力）
4. ランサーズも同様
5. ログイン完了 → セッションが保存される

**これ以降は触らなくていい。**  
毎日自動で案件を検索・応募・納品する。

✅ 完了したら次へ

## よくある質問

**Q: 個人情報は出る？**  
A: CW/ランサーズは作業者名（ニックネーム可）のみ。顔・本名不要。

**Q: 万が一案件でミスしたら？**  
A: パイプラインの `05_review.py` が品質チェックし、問題があれば自動で止まる。

**Q: パイプラインはいつ動くの？**  
A: STEP 3 完了後、以下を1回実行するだけ（その後は放置）
```bash
python3 projects/2026-04-08_月30万自動化/D_エクセル入力スクレイピング/pipeline/run_pipeline.py search
```
または `pipeline_server.mjs` を起動すればiPhoneから操作できる。
