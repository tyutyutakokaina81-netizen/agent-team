# 月次レポート生成ツール

担当：CDO
作成日：2026-05-19
スクリプト：`CDO/outputs/scripts/monthly_report.mjs`

---

## 概要

A受注トラッカー / B営業パイプライン / 経費管理 の3つのCSVを読み込み、
月次サマリを Markdown で生成する依存ゼロのCLIツール。

毎月初に1回実行して `CFO/outputs/YYYY-MM_月次サマリ.md` を生成し、
CFO月次運用フローの起点として使う。

---

## 使い方

```bash
# 標準出力に表示
node CDO/outputs/scripts/monthly_report.mjs 2026-05

# ファイル出力
node CDO/outputs/scripts/monthly_report.mjs 2026-05 \
  --out=CFO/outputs/2026-05_月次サマリ.md
```

### 引数

| 位置 | 必須 | 内容 |
|------|------|------|
| 第1引数 | 必須 | 対象月（YYYY-MM 形式） |
| `--out=` | 任意 | 出力ファイルパス。未指定なら標準出力 |

---

## 読み込むCSV

| 種別 | パス（既定） | 期待される列 |
|------|------------|-------------|
| A受注 | `projects/2026-04-08_月30万自動化/A_ライティング/order_tracker_template.csv` | 受注日 / プラン / 金額 / ステータス / 入金確認 |
| B営業 | `projects/2026-04-08_月30万自動化/B_SNS運用代行/sales_pipeline_template.csv` | 業種 / 初回送信日 / 返信日 / 商談日 / 契約開始日 / 受注金額 / ステータス / 失注理由 |
| 経費 | `CFO/templates/2026-05-19_経費管理シート.csv` | 日付 / 科目 / 金額 / 事業按分 |

> 各CSVが存在しなければそのセクションをスキップして残りを生成する。
> 実運用では templates の代わりに `CFO/outputs/` 配下の実データCSVに差し替えて使う。
> 既定パスを変更したい場合はスクリプト末尾の `sources` 定義を編集する。

---

## 出力内容

1. A：ライティング受注（件数・売上・入金状況・プラン別・ステータス別）
2. B：SNS運用代行 営業パイプライン（送信数・返信率・商談化率・成約率・業種別・失注理由）
3. 経費（科目別内訳・按分後合計）
4. 損益（簡易）（A売上＋B売上 − 経費）

> Cテンプレ販売（note/BOOTH）はAPIアクセスがないためスクリプト対象外。
> プラットフォームのダッシュボードから値をコピーして CFO の月次サマリmdに手動追記する。

---

## CSV仕様の注意点

- 日付は `YYYY-MM-DD` 形式（前方一致で年月を判定）
- 金額は数値のみ（カンマや¥は除去される）
- 事業按分は `100%` `50%` のように % 表記（前後空白OK）
- 入金確認は `TRUE` `true` `1` のいずれか
- 列名は完全一致（半角全角・スペースに注意）

---

## メンテナンス

- スクリプトは依存ゼロ（Node 標準モジュールのみ）
- Node.js 18+ で動作
- 出力フォーマットを変えたい場合は `buildOrderSection` / `buildSalesSection` / `buildExpenseSection` を編集
