# 契約書自動生成スクリプト

担当：CDO（実装）× CFO（運用）
作成日：2026-05-19
スクリプト：`CDO/outputs/scripts/contract_generator.mjs`

---

## 概要

取引先マスターCSVと契約書テンプレートから、クライアント別の契約書を
Markdown ＋ HTML 形式で一括生成する。

HTML はブラウザで開いて「印刷 → PDFとして保存」すれば、
OS標準の日本語フォントを使ったままPDF化可能。
→ **フォント埋め込み・外部ライブラリ不要**で日本語契約書PDFが作れる。

---

## 使い方

```bash
# 単一クライアント
node CDO/outputs/scripts/contract_generator.mjs CL-001

# 全アクティブクライアント
node CDO/outputs/scripts/contract_generator.mjs --all

# カスタムパスを指定
node CDO/outputs/scripts/contract_generator.mjs CL-001 \
  --template=CFO/templates/2026-05-19_業務委託契約書ひな形.md \
  --client-csv=CFO/outputs/2026-05-19_取引先マスター.csv \
  --outdir=CFO/outputs/clients

# 受託者情報（自分）を環境変数で渡す
OUR_NAME="○○ 太郎" OUR_ADDRESS="東京都..." MEDIA="Instagram / X" \
  node CDO/outputs/scripts/contract_generator.mjs CL-001
```

### 引数

| オプション | 内容 |
|----------|------|
| 第1引数 | 取引先ID（`--all` 指定時は省略可） |
| `--all` | アクティブな全取引先で実行 |
| `--template=` | 契約書テンプレ md パス |
| `--client-csv=` | 取引先マスターCSVパス |
| `--outdir=` | 出力フォルダ（既定：`CFO/outputs/clients`） |

### 環境変数

| 変数 | 説明 |
|------|------|
| `OUR_NAME` | 受託者名（乙の名前） |
| `OUR_ADDRESS` | 受託者住所 |
| `MEDIA` | 対象媒体（例：`Instagram / X / Facebook`） |

---

## 置換ロジック

テンプレ内のプレースホルダを CSV の値で置換する。

| プレースホルダ | 置換元 |
|--------------|--------|
| `[甲：クライアント名]` | `取引先名` |
| `[乙：受託者名]` | 環境変数 `OUR_NAME` |
| `月額¥XX,XXX（税込）` | `¥{契約金額（税込）}（税込／税抜¥{税抜}）` |
| `業務開始日：YYYY年MM月DD日` | `契約期間` の開始日（和暦変換） |
| `本契約の有効期間は…` | 開始日〜終了日（和暦変換） |
| `[Instagram / X / Facebook]` | 環境変数 `MEDIA` |
| 末尾の日付 | 当日（和暦） |

## 出力構造

```
CFO/outputs/clients/
└── CL-001_○○株式会社/
    ├── 2026-05-20_契約書_○○株式会社.md
    └── 2026-05-20_契約書_○○株式会社.html
```

- `.md`：エディタで最終調整・差分管理用
- `.html`：ブラウザで開いて印刷→PDF化用（A4・縦・余白なしを推奨）

---

## PDF化の手順（owner 作業）

1. ターミナルでスクリプト実行
2. 生成された `.html` を Chrome / Safari / Firefox で開く
3. `Cmd+P`（Win: `Ctrl+P`）
4. 送信先：「PDFとして保存」を選択
5. 用紙：A4・縦・余白：標準
6. 保存

→ 日本語フォントは OS の明朝（Hiragino Mincho / Yu Mincho 等）が自動で使われる。

---

## 終了コード

- `0`：正常終了
- `1`：引数不足
- `2`：テンプレが見つからない
- `3`：取引先CSVが見つからない
- `4`：対象クライアントが見つからない
