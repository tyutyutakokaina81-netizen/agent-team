# note記事 作成チェックリスト（検索最適化・標準書式）

- **作成日**: 2026-06-29 ／ **担当**: CDO（code）
- **目的**: 今後 note 記事を作るときに **最初から「検索で見つかる」形で生まれる**ようにする恒久書式。
  過去記事を手編集する owner 労力ゼロで、新規が放置で note内検索・Google検索の流入を取れるようにする。
- **位置づけ**: `CMO/outputs/2026-06-29_note閲覧数グロース実行プラン.md`（L1）の標準化版。
  **ローカル発信（高岡・氷見・富山＝北極星）記事に必須適用。** 暮らし/仕事系は §6 のタグ運用のみ。
- **絶対制約の順守**: A4（町名・住所・番地NG＝市レベルまで）／A5（誇張・テンプレ・断定NG）／A2・A3（画像はplaceholder）。

## ✅ 公開前チェック（ローカル記事）

### 1. タイトル（最重要・閲覧数の8割はここ）
- [ ] **前半に「地名（市まで）＋固有名詞」が入っているか**（例：富山／高岡／氷見 ＋ 白えび・金屋町・雨晴）
- [ ] 検索者が打つ語になっているか（叙情だけで固有名詞ゼロはNG）。例：`市場散歩`→`高岡の朝市`
- [ ] 叙情フックは残してよい。型：`〔フック〕。— 富山「〔固有名詞〕」` または `〔地名〕の〔対象〕は、〔フック〕。`
- [ ] 32文字前後まで（note一覧・検索結果で全部見える長さ）
- [ ] A4違反なし（町名・地区名・店の番地を出していない／市レベルで止めている）

### 2. 冒頭2〜3行（検索結果のスニペット＝プレビューになる）
- [ ] 1行目で「これは何の記事か」を名指し（地名＋対象を含める）
- [ ] 「〇〇とは」「〇〇の行き方／値段／旬」など、検索意図に最初の3行で答えている
- [ ] 体言止め・問いかけで引き込みつつ、キーワードを自然に含む

### 3. 見出し（h2／■）
- [ ] 見出しに対象キーワードや疑問形を使う（例：`■ いくらかかる？`『■ いつが旬？』）
- [ ] 検索される質問（値段・行き方・旬・違い・おすすめ）を見出しで拾う

### 4. ハッシュタグ（先頭3つは検索される定番タグ・固定）
- [ ] **食**：`#富山` `#富山グルメ` `#ご当地グルメ` ＋ 対象名（#白えび 等）＋ #北陸
- [ ] **場所**：`#富山` `#富山観光` `#北陸旅行` ＋ 固有名詞（#雨晴海岸 等）
- [ ] **旅行ガイド**：`#富山旅行` `#富山観光` `#北陸旅行` ＋ 対象（#立山黒部 等）
- [ ] **まんが聖地**：`#富山` `#聖地巡礼` ＋ 作品名（#ドラえもん 等）
- [ ] 叙情・心情タグ（#土曜の朝 等）は4つ目以降に。タグは合計10前後。

### 5. 回遊導線（フッター・複利で効く）
- [ ] 末尾に「▼ あわせて読む」＋関連2本（同テーマ）のタイトル＋URL
- [ ] 「フォローすると毎日届きます」等のフォローCTA一文（カテゴリ別ローテで定型回避）
- [ ] 生成は `CDO/outputs/note_footer/gen_footers.py` を使う（自己リンク除外・二重付与防止）
- [ ] その記事が属する**マガジン**に追加する（`CMO/outputs/2026-06-29_note_マガジン編成リスト.md` の6分類）

### 6. 海外読者ブロック（北極星・任意だが推奨）
- [ ] 末尾に `🌏 For English readers` の英語要約（毎回パターンを変える＝A6順守）
- [ ] 該当する英語ガイド（`apps/toyama-guide/en-*.html`）への送客リンク

### 7. 事実・サムネ
- [ ] 末尾に「事実検証ノート」（✅確実／⚠️要確認／諸説あり）
- [ ] サムネ＝みんなのフォトギャラリー検索ワード指定 or 実写生成プロンプト（code は画像を作らない＝A2）
- [ ] 数値・料金・運賃はWebSearchで裏取り、「approximate・公式要確認」明記（A5）

## タイトル型テンプレ（コピーして使う）

| 種別 | 型 | 例 |
|---|---|---|
| 食 | `〔地名〕の〔食材/料理〕は、〔フック〕。` | 富山のころ柿は、軒先で甘くなる。 |
| 食 | `〔フック〕。— 〔地名〕「〔固有名詞〕」` | 縄で縛って持ち歩ける豆腐がある。— 富山「五箇山豆腐」 |
| 場所 | `〔地名〕・〔固有名詞〕は、〔フック〕。` | 高岡・金屋町は、鋳物が始まった石畳の町。 |
| 旅行ガイド | `〔対象〕、〔疑問〕？〔年〕の〔答えの軸〕` | 立山黒部アルペンルート、結局いくら？2026年の料金と回り方 |
| まんが聖地 | `〔地名/施設〕で〔作品〕に〔フック〕。` | 氷見『潮風ギャラリー』には、もうひとりの藤子がいる。 |

## なぜこれが効くか（1行ずつ）
- ローカル×固有名詞の検索は**競合が薄い**＝note記事でも上位を取りやすい。
- 閲覧数は **検索・回遊・フォロー** の3経路でしか増えない。本書式は3つ全部を1記事内で踏む。
- 過去記事の手編集に頼らず、**新規が自動で流入を取る**＝放置(SEO)モデルと整合（owner労力ゼロ）。
