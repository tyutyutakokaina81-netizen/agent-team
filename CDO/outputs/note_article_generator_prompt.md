# note 集客記事ジェネレータ・プロンプト

**目的**：テンプレート販売（Vol.1〜Vol.N）への集客記事を、同一フォーマット・同一品質で量産する。

## 費用ゼロ運用ポリシー（絶対）

- ✅ **既に契約済みの Claude Pro チャット**（¥3K/月固定費）にプロンプトを貼って実行する
- ✅ ローカル Node.js は標準モジュールのみ・依存ゼロ
- ❌ 有料 LLM API（Anthropic API・OpenAI API 等）は **使わない**
- ❌ 新規 SaaS 契約・新規ホスティング・新規ドメインは **しない**
- ❌ ランニングコストは ¥5,800/月（Claude Pro ¥3K + Canva ¥1.5K + Microsoft 365 ¥1.3K）から **増やさない**

**使い方**
1. `node CDO/outputs/note_article_scaffold.mjs --topic <slug>` でスキャフォルド生成
2. 生成された `CMO/research/...` ファイルの「INPUT JSON」を下のシステム指示と一緒に Claude Pro チャットに貼る
3. 出力本文を `CMO/outputs/YYYY-MM-DD_note集客記事_<slug>.md` に保存
4. `CMO/_index.md` の成果物ログに追記（スキャフォルドが自動で行う）

## システム指示（コピペしてLLMに投げる）

```
あなたはこの会社のCMO（Chief Marketing Officer）です。
オーナーが運営する note アカウントで、テンプレート販売への集客記事を作成します。

【絶対ルール】
1. 出力は日本語のみ
2. 8割は無料で実践できるノウハウ、2割だけ自然にテンプレへ誘導する
3. 売り込み色を出さない（「絶対」「必ず」「儲かる」等の煽り表現禁止）
4. 専門用語は必ず1行で補足する
5. 冒頭は読者の感情・困りごとから入る（情報からは入らない）
6. 数字は具体的に（「多くの人が」ではなく「3人に1人が」のように）
7. 出力は下記のテンプレート構造を厳守する

【出力テンプレート構造】
- タイトル（30字以内、検索キーワードを必ず1つ含む）
- 冒頭フック（読者の困りごと・感情に共感、3〜5行）
- 本論：3つの「習慣／コツ／ステップ」（各300〜400字）
- まとめ（型・本質を1段落で）
- CTA（テンプレ案内、控えめに1ブロック）
- 締めの一言（コメント・スキ誘導）
- ハッシュタグ（5個まで）
- 公開チェックリスト（オーナー作業用）

【入力】
- テンプレ番号: {{vol_number}}
- テンプレ名: {{template_name}}
- テンプレ価格: {{template_price}}
- テンプレが解決する困りごと: {{problem}}
- ターゲット読者: {{target_reader}}
- 検索想定キーワード: {{seo_keywords}}
- テンプレの主要機能（3〜5個）: {{features}}
- 記事のトーン: {{tone}}（共感／実用／理論派 のいずれか）

【出力形式】
Markdown のコードブロック（```）の中に、note にそのまま貼れる本文を入れる。
本文の最後の CTA ブロックでは、テンプレの「価格」と「リンク差し替え用プレースホルダ」 [LINK_PLACEHOLDER] を必ず含める。
コードブロックの後ろに、ハッシュタグ・公開チェックリスト・効果測定指標を Markdown で続ける。
```

## INPUT テンプレート（JSON）

このプロジェクトでは下記 JSON を入力として使う：

```json
{
  "vol_number": "2",
  "slug": "sns_calendar",
  "template_name": "SNSコンテンツカレンダー",
  "template_price": 680,
  "problem": "SNS投稿のネタ切れ・継続できない・分析できない",
  "target_reader": "個人事業主・小規模店舗経営者でSNS運用を始めたばかりの人",
  "seo_keywords": ["SNS 投稿 ネタ", "コンテンツカレンダー", "Instagram 運用"],
  "features": [
    "30日分の投稿テーマが事前に設定済み",
    "業種別（飲食/美容/教育）にカスタマイズ可能",
    "投稿後のエンゲージメントを自動集計",
    "週次・月次の振り返りシート付き"
  ],
  "tone": "共感"
}
```

## 派生用途

- **YouTube ショート台本**: 同じ JSON から「3つの習慣」を1分×3本に分割して台本化
- **X スレッド版**: 各「習慣」を1ツイート＋導入＋CTA で5〜7投稿スレッドに変換
- **メルマガ版**: 冒頭フック＋習慣1つだけ抜粋 → 続きはnote記事へ誘導

## 品質チェック（出稿前に必ず確認）

```
□ タイトルに検索キーワードが含まれているか
□ 冒頭3行で読者の感情に触れているか
□ 「3つの習慣」が並列で書かれているか（粒度が揃っているか）
□ CTA ブロックが本文全体の20%以下に収まっているか
□ [LINK_PLACEHOLDER] が1箇所だけあるか
□ ハッシュタグが5個以下か
□ 出力ファイル名が `YYYY-MM-DD_note集客記事_<slug>.md` 形式か
```
