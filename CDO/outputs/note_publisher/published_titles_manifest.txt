# published_titles_manifest.txt
# note に「実際に公開済み」の記事タイトルを 1行1タイトルで列挙する。
# publish_to_note.py がこの一覧とタイトル完全一致（正規化後）で照合し、重複公開をブロックする。
#
# 使い方（オーナー/cowork・Macで）:
#   1. note のダッシュボード（記事一覧）から公開済みタイトルをコピー
#   2. このファイルに 1行1タイトルで貼り付け（# 始まりの行はコメントで無視される）
#   3. 以後 publish_to_note.py が同タイトルの再公開を自動で止める（--force で明示解除）
#
# 注：コンテナ(クラウド)は note に到達できないため、この一覧は Mac 側で維持する。
# 掃除（重複394→約110本）後の正となる公開タイトル一覧をここに置くこと。
#
# ↓ ここから 1行1タイトル（例。実際の note 公開タイトルに置き換える）
