# 効果測定テンプレート（仕組み・自動化のBefore/After検証用）

**作成日**：2026-05-21
**用途**：CDO が導入した仕組み・プロンプト・ツールの効果を定量的に検証する

---

## 1. 対象の特定

| 項目 | 内容 |
|------|------|
| 仕組み名 | （例：CMO用 YouTube台本生成プロンプト v2） |
| 導入日 | YYYY-MM-DD |
| 対象役職 | （CMO / CFO / 全役職 など） |
| 解決したい課題 | （例：台本作成に毎回90分かかる） |

---

## 2. Before（導入前計測）

| 指標 | 値 | 計測方法 |
|------|----|---------|
| 1件あたり所要時間 | ◯分 | 直近5件の平均 |
| 月次発生頻度 | ◯回/月 | 過去3ヶ月の平均 |
| エラー・手戻り率 | ◯% | レビュー時の差し戻し件数 |
| 主観負荷（10段階） | ◯ | 担当役職の自己評価 |

---

## 3. After（導入後計測：1〜4週間後）

| 指標 | 値 | 変化 |
|------|----|------|
| 1件あたり所要時間 | ◯分 | ▲◯分 |
| 月次発生頻度 | ◯回/月 | 同 |
| エラー・手戻り率 | ◯% | ▲◯% |
| 主観負荷（10段階） | ◯ | ▲◯ |

---

## 4. Delta（効果算定）

```
月間削減時間 = (Before所要時間 − After所要時間) × 月次頻度
            = ( ◯分 − ◯分 ) × ◯回 = ◯分/月 = ◯時間/月
```

| 項目 | 値 |
|------|----|
| 月間削減時間 | ◯時間/月 |
| 年間換算 | ◯時間/年 |
| 副次効果 | （例：エラー削減により顧客信頼度↑） |

---

## 5. 判定と次アクション

- [ ] **継続**：効果が想定通り。運用継続・他役職展開を検討
- [ ] **改修**：効果が限定的。原因を分析し v2 を設計
- [ ] **廃止**：効果なし or 負担増。仕組みを撤去し別アプローチを検討
