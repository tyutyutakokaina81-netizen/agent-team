# 自動化プロンプト全書（2026-04-13）

Claudeに貼るだけで全業務が自動化する15プロンプト集。
**毎日の作業を1時間→15分に短縮**。

---

## 🎯 全体像

```
【朝】
  1. 案件探索プロンプト → 5件ピックアップ
  2. 応募文一括生成プロンプト → 5件分の応募文
  3. メール返信プロンプト → 未返信への返信案

【午前】
  4. 記事構成生成プロンプト
  5. 見出し作成プロンプト
  6. 本文執筆プロンプト

【午後】
  7. SNS投稿一括生成プロンプト → 月15投稿分
  8. ハッシュタグ選定プロンプト

【夜】
  9. 振り返りプロンプト
  10. 翌日計画プロンプト

【週次】
  11. 週次レポート生成プロンプト

【月次】
  13. 月次レポート生成プロンプト
  14. KPI分析プロンプト
  15. 改善提案プロンプト
```

```

【検索対象】
- クラウドワークス（crowdworks.jp）
- ランサーズ（lancers.jp）

【条件】
- 単価3,000円以上
- 初心者OK
- 作業内容が明確
- クライアント評価4.0以上
- 応募数10件以下

【出力形式】
### 案件N
サイト：
URL：
案件概要（1行）：
報酬：
納期：
クライアント評価：

---

上記フォーマットで5件抽出してください。分析・比較は不要。条件を満たす最初の5件で即決定してください。
```

## 📍 プロンプト2：応募文一括生成（朝10分）

```
あなたは営業のプロです。
以下の5つの案件に対して、それぞれの特徴を反映した
応募文を個別に生成してください。

【案件情報】
案件1：[案件概要1]
案件2：[案件概要2]
案件3：[案件概要3]
案件4：[案件概要4]
案件5：[案件概要5]

【応募文の条件】
- 丁寧かつ簡潔（200〜300字）
- 案件内容を1文で反映
- 改行あり
- テンプレ感を消す（案件ごとにカスタマイズ）
- 最後に「すぐに作業開始可能」と記載

【応募文の基本構造】
1. 冒頭：挨拶＋自己紹介（1-2行）
2. 本案件への対応内容（2-3行）
3. 強み・スキル（2-3行）
4. 稼働時間・連絡スタンス（1-2行）
5. 締め（1行）

5件分すべての応募文を出力してください。
```

**使い方：** プロンプト1の結果を渡す → 5件分の応募文が即完成

## 📍 プロンプト4：記事構成生成（ライティング案件）

```
以下の条件でSEO記事の構成を作成してください。

【基本情報】
- メインキーワード：[キーワード]
- 文字数：[文字数]字
- 想定読者：[ペルソナ]
- トーン：[丁寧/カジュアル/専門的]

【構成の要件】
1. タイトル案3つ（32字以内・キーワード含む）
2. メタディスクリプション（120字以内）
3. リード文の方向性（300字）
4. H2見出し4-6個
5. 各H2に紐づくH3見出し2-3個
6. まとめのCTA方針

出力は以下の形式で：

# タイトル案
1. ～
2. ～
3. ～

# メタディスクリプション
～

# 記事構成
## H2-1：～
  ### H3-1-1：～
  ### H3-1-2：～
## H2-2：～
（以下続く）
```

**使い方：** 受注直後にこのプロンプト実行 → 構成が10秒で完成

## 💡 プロンプトのカスタマイズ

### 精度を上げるコツ
- 【条件】は具体的に
- 悪い例も提示する
- 出力形式を明示する
- 最初に「あなたは○○のプロです」と役割を与える

### エラーを減らすコツ
- 文字数制限を明記
- 禁止事項を列挙
- 出力形式をテンプレで示す
