## Start Here — 6人のAI役員に仕事を任せる

全部ひとりでやるのをやめて、AIに「役員」を割り当てます。

| 役員 | 担当 | このOSでの道具 |
|---|---|---|
| CMO マーケ | 集客・発信 | SNS/記事ネタ/英語要約プロンプト |
| CPO 商品 | 商品・教材づくり | 商品説明/LP/メニュー文プロンプト |

| CAO 分析 | 振り返り・改善 | 作業ログ分析/週次振り返りプロンプト |
| CDO 技術 | 効率化・自動化 | 要約/整理/文章の自分の言葉化プロンプト |

## The Revenue Loop — お金が生まれる流れ

```
作る（記事・商品） → 出す（SNS・販売ページ） → 売る → 測る（採算） → 改善
```

無料の発信で人を集め、実用の商品（このプロンプト集のような道具）で換金する。
高い固定費も借入もいりません。**売れた分だけ手数料を払う**ところから始められます。

## How to use — 精度を上げる3つのコツ

1. 各プロンプトの「あなたは〜」を消さない（出力が安定します）
2. 「こういうのは避けて」を1行足すと、欲しくない方向を防げます
3. 一度で完璧を狙わず「もっと短く」「もっと柔らかく」と返して調整

# Part 1 — ひとり会社の実務 15本

## 使う前に：精度を上げる3つのコツ

1. **役割を与える**：各プロンプト冒頭の「あなたは〜」を消さない。AIの出力が安定する。
2. **悪い例も渡す**：「こういうのは避けて」を1行足すと、欲しくない方向を防げる。
3. **一度で完璧を狙わない**：出力に「もっと短く」「もっと柔らかく」と返すだけで精度が上がる。

---

## ① 請求書の文面を3分で作る

**用途**：取引先へ送る請求メール本文＋請求項目の整理。

```
あなたは丁寧で正確な事務担当です。以下の条件で、取引先に送る請求メールの本文を作ってください。
- 宛先：【会社名・担当者名】
- 案件：【案件名】
- 金額：【税抜金額】（消費税【10】%）
- 支払期限：【YYYY-MM-DD】
- 振込先：【銀行・支店・口座】
条件：敬語は丁寧すぎず簡潔に。件名も提案。最後に「ご不明点はお気軽に」を一文。
```

**コツ**：金額の最終確認は必ず自分で。AIに計算を丸投げしない。

---

## ② 顧客への返信下書き（言いにくい内容ほど効く）

**用途**：催促・お断り・値上げ連絡など、書き出しに悩む返信。

```
あなたは関係を壊さない交渉が得意なビジネスパーソンです。
次の状況で、角を立てずに用件を伝える返信文の下書きを3案ください。
- 相手：【取引先/顧客】
- 伝えたいこと：【例：支払いがまだ／納期を延ばしたい／値上げのお願い】
- 関係：【継続したい/今回限り】
条件：謝りすぎない。理由は簡潔に。相手の立場への一言を必ず入れる。
```

**コツ**：3案のいいとこ取りで1通にまとめると自然になる。

---

## ③ SNS投稿を1週間分まとめて作る

**用途**：X / Instagram などの週次投稿ネタを一気に。

```
あなたはSNS運用のプロです。【発信テーマ：例 高岡の暮らし／ひとり会社の働き方】について、
1週間分（7投稿）のSNS投稿文を作ってください。
- 文字数：各【140】字以内
- トーン：【親しみやすく・押し売りしない】
- 曜日ごとに切り口を変える（気づき／質問／実例／裏側／おすすめ／失敗談／まとめ）
条件：ハッシュタグを各3個。同じ書き出しを2回使わない。
```

**コツ**：「同じ書き出しを使わない」の一文がテンプレ臭さを消す。

---

## ④ 「営業しない」営業メール

**用途**：売り込み感を出さずに仕事につなげる初回連絡。

```
あなたは押し売りを嫌うベテラン営業です。次の相手に、売り込み感ゼロで信頼から入る初回メールを書いてください。
- 相手：【業種・規模】
- 自分の提供：【サービス内容】
- きっかけ：【相手の投稿を見て／紹介で／など】
条件：最初に相手への具体的な関心を示す。自分の宣伝は全体の2割まで。返信のハードルを下げる質問で締める。
```

**コツ**：相手の発信を1つ実際に引用すると返信率が上がる。

---

## ⑤ 1週間の優先順位を3行に整理する

**用途**：月曜の朝、やることが多すぎて動けないとき。

```
あなたは生産性コーチです。次のタスク一覧を見て、今週やるべきことを「重要かつ緊急」で並べ替え、
今週の最重要3つだけに絞ってください。残りは「やらないことリスト」に回してください。
タスク一覧：
【ここに思いつくまま全部貼る】
出力：①今週の3つ ②各タスクの最初の一歩 ③やらないことリスト
```

**コツ**：絞られた3つだけをカレンダーに先に入れる。

---

## ⑥ 長い文章を「貼れる要約」にする

**用途**：議事録・記事・資料を、人に渡せる短さに。

```
あなたは要約の名手です。次の文章を3つの粒度で要約してください。
①一文（30字以内）②要点3つ（箇条書き）③詳細版（200字）
専門用語が出たら、初めての人にも分かる言い換えを( )で添えてください。
文章：
【ここに貼る】
```

**コツ**：①一文をタイトルや投稿の書き出しに転用できる。

---

## ⑦ 記事・投稿のネタを30個出す

**用途**：「今日何書こう」で止まる時間をゼロにする。

```
あなたは編集者です。【発信テーマ】で、読者が思わず読みたくなる記事ネタを30個出してください。
- 読者：【誰／どんな悩み】
- 切り口を散らす：体験談／ハウツー／意外な事実／比較／失敗／日常の気づき
条件：タイトル案も添える。ありがちで平凡なものは除外し、具体的な切り口にする。
```

**コツ**：30個のうち刺さる3個を残し、リストに貯める。

---

## ⑧ 海外読者向けの英語要約をつくる

**用途**：noteやブログに英語サマリーを併記して海外に届ける。

```
You are a bilingual editor. Summarize the following Japanese text into natural English (about 120 words)
for an international reader who doesn't know Japan well.
- Keep proper nouns, add a short gloss in ( ) the first time.
- Tone: warm, like a local sharing daily life. Avoid travel-brochure clichés.
Text:
【日本語本文を貼る】
```

**コツ**：固有名詞に短い注釈を入れると海外読者の離脱が減る。

---

## ⑨ 値段の決め方を相談する

**用途**：サービスや商品の価格に自信が持てないとき。

```
あなたは価格戦略のコンサルタントです。次のサービスの価格を3パターン（松/竹/梅）で提案してください。
- 内容：【サービス/商品の中身】
- 提供にかかる時間・コスト：【ざっくり】
- 競合のおおよその相場：【分かれば】
出力：各価格の根拠／どんな客に刺さるか／最初に出すべき1つの推奨。
条件：安売りに逃げない理由も添える。
```

**コツ**：推奨された1つを「先行価格」として小さく試す。

---

## ⑩ クレーム・指摘への一次対応

**用途**：感情的な連絡に、冷静かつ誠実に返す。

```
あなたは顧客対応のプロです。次の指摘に対し、火に油を注がない返信を作ってください。
- 相手の指摘：【内容を貼る】
- 事実関係：【こちらに非がある/ない/確認中】
条件：まず受け止める一文。言い訳から入らない。次の具体的アクションと期限を必ず示す。
```

**コツ**：送る前に一晩置く。AIの下書きは「冷静さの担保」に使う。

## ⑪ 契約書・規約の要点を読み解く

**用途**：受け取った契約書の「危ない箇所」を把握する。

```
あなたは契約書を読み慣れた実務家です（法的助言ではない前提で）。次の契約書/規約を読み、
①自分（受注側）に不利な条項 ②曖昧で揉めそうな箇所 ③確認すべき質問 を箇条書きで出してください。
契約書：
【貼る】
注意：最終判断は専門家に相談する前提で、論点の洗い出しに徹してください。
```

**コツ**：重要な契約は、洗い出した論点を持って専門家に相談する。

## ⑫ 1日の作業ログから「時間の使い方」を分析する

**用途**：何に時間を溶かしているかを見える化する。

```
あなたはタイムマネジメントの分析官です。次の1日の作業ログを見て、
①時間を使ったカテゴリの内訳 ②明らかに削れる時間 ③明日の改善案1つ を出してください。
作業ログ：
【時刻とやったことを箇条書きで貼る】
条件：精神論ではなく、具体的な行動レベルの改善を1つだけ。
```

**コツ**：改善案は「1つだけ」にすると実行できる。

## ⑬ ありがちな文章を「自分の言葉」に直す

**用途**：AIっぽい・テンプレっぽい文を、人間味のある文に。

```
あなたは編集者です。次の文章から、ありがちな常套句・誇張・テンプレ的な言い回しを取り除き、
具体と一次体験のある文に書き直してください。
- 残すこと：事実・数字・実体験
- 消すこと：「Xには、Yがある」型／「世界唯一」など根拠の薄い断定／同じ締め方の反復
文章：
【貼る】
```

**コツ**：このプロンプトは⑦で出したネタを記事化したあとの仕上げに効く。

## ⑭ 商品・サービスの説明文（LP用）を書く

**用途**：noteやBOOTHの商品ページ、サービス紹介文。

```
あなたはセールスライターです。次の商品の紹介文を書いてください。
- 商品：【名前と中身】
- 買う人：【誰／どんな悩み】
- 価格：【金額】
構成：①一行で刺さるキャッチ ②この商品が解く悩み ③中身（箇条書き）④買うとどうなるか ⑤背中を押す一言
条件：誇張しない。「誰でも簡単に」など根拠のない言葉は使わない。
```

**コツ**：④「買うとどうなるか」を具体的な場面で書くと売れる。

# Part 2 — 飲食店と小さな食ビジネス 12本

## 使う前に

- 各プロンプトの「あなたは〜」は消さない（出力が安定します）
- 店の雰囲気（カジュアル/上品/家庭的）を一言添えると、文章のトーンが揃います
- 一度で完璧を狙わず「もっと短く」「方言を少し」などと返して調整

---

## ① メニューの説明文を魅力的に書く

```
あなたは飲食店のメニューライターです。次の料理の説明文を、注文したくなる60字前後で書いてください。
- 料理名：【名前】
- 主な食材・産地：【例：地元の○○】
- 推しポイント：【食感/温度/季節/作り方】
- 店の雰囲気：【家庭的/上品/カジュアル】
条件：大げさな美辞麗句より、具体的な五感（音・湯気・食感）で。3案ください。
```

## ② SNS投稿（今日のおすすめ・1週間分）

```
あなたは飲食店のSNS担当です。【店の種類：例 定食屋】の今週の投稿を7本作ってください。
- 推したいもの：【日替わり/季節限定/定番】
- トーン：【親しみやすい】／文字数：各100字前後
条件：曜日で切り口を変える（今日のおすすめ／仕込み風景／お客様の声／豆知識／週末告知／裏側／感謝）。ハッシュタグ各3個。
```

## ③ 良い口コミへの返信

```
あなたは店主です。次の好意的な口コミに、テンプレ感のない返信を書いてください。
- 口コミ：【内容を貼る】
条件：具体的に触れて感謝する。次回また来たくなる一言を添える。短く誠実に。
```

## ④ 厳しい口コミ・クレームへの返信

```
あなたは冷静で誠実な店主です。次の厳しい口コミに、火に油を注がない返信を書いてください。
- 口コミ：【内容を貼る】
- 事実：【こちらに非がある/ない/確認中】
条件：まず受け止める。言い訳から入らない。改善する点があれば具体的に。読んだ他の人にも誠実さが伝わるように。
```

## ⑤ 海外のお客さん向け 多言語メニュー

```
You are a menu translator for a small Japanese restaurant.
Translate the following menu into natural English (and add a short note for ingredients that may need allergy/▲vegetarian info).
- Keep the Japanese dish name in romaji, then the English description.
Menu (Japanese):
【メニューを貼る】
```
※中国語・韓国語が要る場合は「Translate into Simplified Chinese / Korean」に変えるだけ。

## ⑥ 店頭POP・黒板メニューの文言

```
あなたは販促のプロです。【商品/おすすめ】の店頭POPの文言を3パターン作ってください。
- 価格：【金額】／推し：【限定・季節・お得感】
条件：通りすがりの人が3秒で読める短さ。手書き黒板に書ける長さで。
```

## ⑦ 季節・イベント限定メニューの企画

```
あなたはメニュー開発者です。【季節/イベント：例 梅雨・夏・地元の祭り】に合う限定メニューを5案出してください。
- 店の種類：【】／使いやすい地元食材：【あれば】
出力：各案のネーミング案・想定価格帯・推しポイント。仕込みが重すぎないものを優先。
```

## ⑧ 原価・価格の相談

```
あなたは飲食店の経営コンサルです（最終判断は店主が行う前提で）。次のメニューの価格を相談に乗ってください。
- 材料費のざっくり：【】／提供の手間：【】／近隣相場：【分かれば】
出力：推奨価格帯・原価率の目安・「安すぎないか」の指摘。値上げする場合の伝え方も一言。
```

## ⑨ 予約確認・お礼のメッセージ

```
あなたは丁寧な店主です。次の場面のメッセージを作ってください。
- 場面：【予約確認／当日リマインド／来店お礼】
- 予約内容：【日時・人数・要望】
条件：固すぎず温かく。次回予約や口コミ投稿をさりげなく促す一言を最後に。
```

## ⑩ スタッフ・アルバイト募集文

```
あなたは採用担当です。【店名/業態】のアルバイト募集文を書いてください。
- 条件：【時給・時間・曜日】／求める人物：【未経験歓迎 等】／店の魅力：【】
条件：上から目線にならない。働く姿がイメージできる具体描写を入れる。応募ハードルを下げる一言で締める。
```

## ⑪ 常連さんを増やすメッセージ・スタンプカード文言

```
あなたは販促のプロです。リピートに繋がる仕掛けの文言を作ってください。
- 仕掛け：【スタンプカード／次回クーポン／LINE登録】
条件：押し売り感なく。「また来たい」と思える温度感で。特典は具体的に。
```

## ⑫ インバウンド客への一言・店内案内

```
You are helping a small Japanese eatery welcome foreign tourists.
Write short, friendly English phrases for: greeting, "please wait to be seated", recommending today's special, and explaining how to pay.
Keep each under 15 words, easy to read aloud.
```
