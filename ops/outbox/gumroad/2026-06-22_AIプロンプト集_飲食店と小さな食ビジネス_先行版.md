# AIプロンプト集 ― 飲食店と小さな食ビジネスの実務12本（先行版）

> カフェ・食堂・パン屋・お惣菜・ネット販売など、**小さな食ビジネス**のための、コピペで使えるAIプロンプト集。
> ChatGPT / Claude / Gemini どれでもOK。【 】の中だけ自分の店の情報に置き換えてください。
> 海外のお客さん対応（多言語）も入っています。

- **価格（推奨）**: ¥1,280（ひとり会社版¥980とのバンドルで¥1,980も可）
- **対象**: 個人経営の飲食店・食物販・キッチンカー・お菓子/パン・産直・ネットショップ
- **作成**: CPO（プロダクト責任者）

---

## 使う前に

- 各プロンプトの「あなたは〜」は消さない（出力が安定します）
- 店の雰囲気（カジュアル/上品/家庭的）を一言添えると、文章のトーンが揃います
- 一度で完璧を狙わず「もっと短く」「方言を少し」などと返して調整

---

## ① メニューの説明文を魅力的に書く

```
あなたは飲食店のメニューライターです。次の料理の説明文を、注文したくなる60字前後で書いてください。
- 料理名：【名前】
- 主な食材・産地：【例：地元の○○】
- 推しポイント：【食感/温度/季節/作り方】
- 店の雰囲気：【家庭的/上品/カジュアル】
条件：大げさな美辞麗句より、具体的な五感（音・湯気・食感）で。3案ください。
```

## ② SNS投稿（今日のおすすめ・1週間分）

```
あなたは飲食店のSNS担当です。【店の種類：例 定食屋】の今週の投稿を7本作ってください。
- 推したいもの：【日替わり/季節限定/定番】
- トーン：【親しみやすい】／文字数：各100字前後
条件：曜日で切り口を変える（今日のおすすめ／仕込み風景／お客様の声／豆知識／週末告知／裏側／感謝）。ハッシュタグ各3個。
```

## ③ 良い口コミへの返信

```
あなたは店主です。次の好意的な口コミに、テンプレ感のない返信を書いてください。
- 口コミ：【内容を貼る】
条件：具体的に触れて感謝する。次回また来たくなる一言を添える。短く誠実に。
```

## ④ 厳しい口コミ・クレームへの返信

```
あなたは冷静で誠実な店主です。次の厳しい口コミに、火に油を注がない返信を書いてください。
- 口コミ：【内容を貼る】
- 事実：【こちらに非がある/ない/確認中】
条件：まず受け止める。言い訳から入らない。改善する点があれば具体的に。読んだ他の人にも誠実さが伝わるように。
```

## ⑤ 海外のお客さん向け 多言語メニュー

```
You are a menu translator for a small Japanese restaurant.
Translate the following menu into natural English (and add a short note for ingredients that may need allergy/▲vegetarian info).
- Keep the Japanese dish name in romaji, then the English description.
Menu (Japanese):
【メニューを貼る】
```
※中国語・韓国語が要る場合は「Translate into Simplified Chinese / Korean」に変えるだけ。

## ⑥ 店頭POP・黒板メニューの文言

```
あなたは販促のプロです。【商品/おすすめ】の店頭POPの文言を3パターン作ってください。
- 価格：【金額】／推し：【限定・季節・お得感】
条件：通りすがりの人が3秒で読める短さ。手書き黒板に書ける長さで。
```

## ⑦ 季節・イベント限定メニューの企画

```
あなたはメニュー開発者です。【季節/イベント：例 梅雨・夏・地元の祭り】に合う限定メニューを5案出してください。
- 店の種類：【】／使いやすい地元食材：【あれば】
出力：各案のネーミング案・想定価格帯・推しポイント。仕込みが重すぎないものを優先。
```

## ⑧ 原価・価格の相談

```
あなたは飲食店の経営コンサルです（最終判断は店主が行う前提で）。次のメニューの価格を相談に乗ってください。
- 材料費のざっくり：【】／提供の手間：【】／近隣相場：【分かれば】
出力：推奨価格帯・原価率の目安・「安すぎないか」の指摘。値上げする場合の伝え方も一言。
```

## ⑨ 予約確認・お礼のメッセージ

```
あなたは丁寧な店主です。次の場面のメッセージを作ってください。
- 場面：【予約確認／当日リマインド／来店お礼】
- 予約内容：【日時・人数・要望】
条件：固すぎず温かく。次回予約や口コミ投稿をさりげなく促す一言を最後に。
```

## ⑩ スタッフ・アルバイト募集文

```
あなたは採用担当です。【店名/業態】のアルバイト募集文を書いてください。
- 条件：【時給・時間・曜日】／求める人物：【未経験歓迎 等】／店の魅力：【】
条件：上から目線にならない。働く姿がイメージできる具体描写を入れる。応募ハードルを下げる一言で締める。
```

## ⑪ 常連さんを増やすメッセージ・スタンプカード文言

```
あなたは販促のプロです。リピートに繋がる仕掛けの文言を作ってください。
- 仕掛け：【スタンプカード／次回クーポン／LINE登録】
条件：押し売り感なく。「また来たい」と思える温度感で。特典は具体的に。
```

## ⑫ インバウンド客への一言・店内案内

```
You are helping a small Japanese eatery welcome foreign tourists.
Write short, friendly English phrases for: greeting, "please wait to be seated", recommending today's special, and explaining how to pay.
Keep each under 15 words, easy to read aloud.
```
