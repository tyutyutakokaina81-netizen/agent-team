## 2. なぜ「暗黙化」していたか

| 暗黙化の原因 | 解説 |
|---|---|
| CMOが「読みもの」として書いていた | エッセイ型は無料読了で消える。商品化の意識を組み込んでいなかった |
| CTAがない | 各記事末尾に商品リンクがなく、読了→離脱で完了してしまう |
| 検索キーワードが甘い | 「請求書 プロンプト」「ひとり起業 30日」など売れる検索語が記事に埋まっていない |
| まとめ売りしていない | 単発記事が散在し、「これを買えば終わる」体感を作れていなかった |

## 3. 採算性ファースト記事の書き方（CMO向け新ルール）

| 旧ルール | 新ルール（採算性ファースト） |
|---|---|
| 読まれる文章を書く | **読まれた後の行動を設計する** |
| 1記事1テーマ | 1記事1**読了後の購入導線** |
| 高岡シリーズはブランド | 高岡シリーズも**末尾に必ず商品3つCTA** |
| 「いい話」で締める | 「これを使えば即解決」で締め、商品リンクを置く |

具体的に：

- **本文末尾の固定ブロック**（全記事共通）：
  ```
  ─────
  この記事の方法を、すぐ使える形にまとめています：
  🛠 AI役職テンプレ集 ¥4,980 → [リンク]
  📅 営業しない営業 90日 ¥9,800 → [リンク]
  🔥 月30万への全記録（月額 ¥1,980） → [リンク]
  ─────
  ```
- **本文中の自然な誘導**（記事のテーマに合う商品1点を1行で挿入）
