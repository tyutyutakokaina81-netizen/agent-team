## テンプレ1：実演系

### タイトル例
```
【AIで時短】Claudeを使って○○を3分で作る方法
```

### 台本

```
[0:00] オープニング（15秒）
こんにちは、[チャンネル名]です。
今日は「○○をAIで3分で作る方法」を実演します。
普段、これを手作業でやると30分かかるんですが、
今日紹介する方法だと本当に3分で終わります。
チャンネル登録がまだの方、ぜひお願いします。

[0:15] 問題提起（45秒）
皆さん、こんなお悩みありませんか？
・「○○を作るのに毎回30分かかっている」
・「アイデアが思いつかなくて時間ばかり過ぎる」
・「AIを使いたいけど、うまく指示が出せない」

実は、私も独立した当初は同じ悩みを抱えていました。
今日紹介するプロンプト1つで、その悩みが解消されます。

[1:00] 本編Part 1：今日使うプロンプト（1分30秒）
まずは、今日使うプロンプトをお見せします。

【プロンプト全文を画面表示】
このプロンプトのポイントは3つです。
1. ○○を最初に指定
2. △△を条件として入れる
3. 出力形式を明示する
この3つを抑えるだけで、AIの出力が劇的に変わります。

[2:30] 本編Part 2：実演（3分）
実際にClaudeに貼り付けてみます。
（画面操作を実演）
入力したら、Enter。
すると、こんな結果が出てきました。
（出力を画面表示）

[5:30] 本編Part 3：手直しポイント（2分）
AIの出力は完璧ではありません。
ここを3点だけ手直しします。
1. ○○を△△に変更
2. ××の表現を柔らかく
3. ハッシュタグを追加

これで完成です。所要時間：3分。

[8:30] まとめ（1分）
今日のポイントを3つにまとめます。
1. プロンプトの3要素を意識する
2. 一発で完璧を目指さず、手直しを前提に
3. AIは「アシスタント」として使う

このプロンプト、概要欄からテンプレ集をダウンロードできるので、
よかったらどうぞ。

[9:30] エンディング（30秒）
今日はここまでです。
次回は「○○を作る方法」を紹介予定。
チャンネル登録と高評価で応援していただけると嬉しいです。
それではまた次回お会いしましょう、バイバイ。

## テンプレ2：ノウハウ系

### タイトル例
```
【保存版】副業の確定申告で絶対やるべき3つのこと
```

### 台本

```
[0:00] オープニング（15秒）
こんにちは、[チャンネル名]です。
副業を始めたけど、確定申告って何から手をつけていいかわからない…
そんな方のために、今日は「絶対やるべき3つのこと」だけに絞って解説します。

[0:15] 問題提起（45秒）
副業の確定申告で「これだけは絶対やってほしい」3つがあります。
これを知らないと、
・最大10万円以上損する
・税務署から問い合わせが来る
・翌年も同じ失敗を繰り返す
こんなことになります。

逆に、この3つさえやっておけば、初心者でも確定申告は怖くありません。

[1:00] 本編Part 1：①収支を月ごとに記録する（2分）
1つ目は「収支を月ごとに記録する」です。

なぜか？それは、3月の確定申告期に1年分まとめて記録しようとすると、
1ヶ月以上かかるからです。
（カレンダーや実例画像を表示）

私が使っているのが、月別自動集計スプレッドシート。
1行入力するだけで、月別の収支がグラフになります。
このテンプレ、概要欄に貼っておきます。

[3:00] 本編Part 2：②領収書はその場で撮影（2分30秒）
2つ目は「領収書はその場でスマホ撮影」です。

紙の領収書を溜め込むと、確定申告期にこうなります。
（領収書の山の画像）

電子帳簿保存法という法律があって、
スマホ撮影してPDF化すれば、原本は処分してOKです。
（実演：実際にスマホで撮影してドライブにアップする様子）

[5:30] 本編Part 3：③20万円ルールを正しく理解（3分）
3つ目は「20万円ルールを正しく理解する」です。

副業で年20万円以下なら確定申告不要、と聞いたことありませんか？
これ、半分正解で半分間違いです。

正しくは「所得税の確定申告は不要」だけ。
住民税は別途申告が必要です。
（フローチャートを表示）

これを知らずに「20万以下だから何もしなくていい」と思っている人が
多いんですが、後から市役所から問い合わせが来ます。

[8:30] まとめ（1分）
今日のポイントを3つにまとめます。
1. 収支は月ごとに記録（テンプレを使うのが楽）
2. 領収書はその場でスマホ撮影＆PDF化
3. 20万円ルールは住民税には適用されない

詳しい手順は概要欄のテンプレと連動しています。

[9:30] エンディング（30秒）
今日はここまで。
次回は「青色申告 vs 白色申告」を解説します。
お役に立てたら高評価とチャンネル登録お願いします。
それではまた次回。
```

## テンプレ3：事例系

### タイトル例
```
【正直告白】フリーランス1年目の失敗ベスト5
```

### 台本

```
[0:00] オープニング（15秒）
こんにちは、[チャンネル名]です。

全部正直にお話しします。
これから独立する方への注意喚起になれば嬉しいです。

[0:15] 問題提起（45秒）
フリーランスとして独立して3年が経ちました。
今は順調にやっていますが、1年目は本当にひどかったんです。
今日はその失敗5つを、原因と対策セットで紹介します。

[1:00] 失敗#5：価格設定が低すぎた（1分30秒）
5つ目の失敗は「価格設定が低すぎた」です。

最初、自信がなくて記事1本3,000円で受けていました。
これ、今思うと完全に間違いでした。

【失敗の代償】
・月の労働時間：250時間
・月の手取り：12万円
・時給：480円

【教訓】
「最低時給を逆算する」これだけで判断できます。

[2:30] 失敗#4：契約書なしで仕事を始めた（1分30秒）
（同様の構造で続ける）

[4:00] 失敗#3：経費を計上し忘れた（1分30秒）
（同様の構造）

[5:30] 失敗#2：SNS発信を後回しにした（1分30秒）
（同様の構造）

[7:00] 失敗#1：体調管理を軽視した（1分30秒）
（同様の構造）

[8:30] まとめ（1分）
5つの失敗から学んだこと：
1. 時給逆算で価格設定する
2. 契約書なしで仕事を始めない
3. 経費は当月に記録する
4. SNSは独立日から始める
5. 体調管理は仕事の一部

[9:30] エンディング（30秒）
今日はここまで。
あなたが同じ失敗をしないことを祈っています。
チャンネル登録と次回もお楽しみに。

## サムネイル設計ガイド

| 要素 | 仕様 |
|------|------|
| 文字数 | 13字以内（モバイル可読性） |
| 文字サイズ | 画面の20%以上 |
| 色のコントラスト | 背景色と補色（白×ネイビー、黒×イエロー等） |
| 表情 | 自分の顔を使う場合は驚き・困り・笑顔のいずれかを誇張 |
| 視覚要素 | 比較矢印・赤い丸印・数字を1つ |
| ロゴ | 右上または右下に小さく統一配置 |

### タイトルとサムネイルの整合性
- タイトル＝詳しい説明
- サムネイル＝最大のフック
- 同じことを言うのではなく、補完関係に
