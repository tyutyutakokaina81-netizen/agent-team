## エントリー

### 2026-05-13
- **学び**: 1本の動画を 記事 / ホワイトペーパー / LINE / Threads / Instagram / TikTok / メルマガ に横展開する「マルチメディア戦略」が、Threads 単体で1ヶ月29万再生に到達した実例がある。
- **背景**: 杉田氏の動画事例。Claude Code で 2-3時間おきに自動投稿（人間チェック込み）。
- **示唆**: 当社の YouTube・ブログ等の単発コンテンツを 5媒体以上に横展開する仕組みを CDO と連携して設計したい。柱B（SNS運用代行）の差別化要素にもなる。

### 2026-05-13（追加・v1作業の学び）
- **学び**: マルチメディア展開は「コピペ展開」ではなく「**媒体別再設計**」が前提。Threads=共感／X=要約／メルマガ=深掘り、で文体・長さ・CTA を全て差し替える必要がある。
- **背景**: 本日 `2026-05-13_1to3媒体展開ワークフロー_v1.md` を起こす際、同じ動画素材から3媒体の投稿サンプルを書いたが、3本とも別物になった。
- **示唆**: 「1本素材→N媒体」を機械的にやるとブランドの体温が落ちる。媒体ごとの最小再設計テンプレを CMO が運用ルール化することで、人間の従来作業の1/3〜1/2の工数で品質維持できる見込み。柱B 受注後オペレーションの中核に据える。
