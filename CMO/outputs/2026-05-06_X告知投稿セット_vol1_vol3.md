## スレッド版（翌日投稿・5ツイート）

### 1/5（フック）
```
確定申告で慌てるフリーランス1〜3年目に、
たった1つだけ伝えるなら——

「請求日」と「入金日」を分けて記録しろ。

これだけで、来年の2月の自分が泣かずに済みます。

```

### 2/5（習慣1）
```
習慣1｜「請求日」と「入金日」を分ける

12月納品・1月入金の案件は、
"12月の売上"として計上します（発生主義）。

通帳ベースで管理すると、ここが必ずズレる。
2月に税理士から指摘されて慌てる典型パターン。

スプレッドシートに2列作るだけで解決します。
```

### 3/5（習慣2）
```
習慣2｜経費の科目を入力時に決める

レシートを年末にまとめて分類——これが続かない最大の原因。

最低限おさえる科目は6つ：
・通信費 ・交通費 ・消耗品費
・新聞図書費 ・支払手数料 ・ソフトウェア費

入力時にプルダウンで選ぶ仕組みにしておけば、2月の自分が泣かない。
```

### 4/5（習慣3）
```
習慣3｜案件ごとの時給を毎月確認する

単価10万円・20時間 → 時給5,000円
単価5万円・5時間 → 時給1万円

数字で見ると、断る勇気が出ます。

私自身これで、売上は変わらず労働時間が3割減りました。
```

# Vol.2｜SNSコンテンツカレンダー記事の告知

## 単発告知ツイート（公開直後）

```
SNSが続かない理由は、
ネタがないからでもセンスがないからでもなくて、
"毎回ゼロから考える設計"になっているからだった。

月5分で続く3つの習慣を、note にまとめました。

▼[記事URL]

#SNS運用 #個人事業主
```

## スレッド版（翌日投稿・5ツイート）

### 1/5（フック）
```
SNS運用が3日で止まる人に、
たった1つだけ伝えるなら——

「曜日テーマ」を3日分だけ決めろ。

毎日ゼロから考えるから止まる。
最初に型を決めれば、面倒くさがりほど続きます🧵
```

### 2/5（習慣1）
```
習慣1｜毎週「曜日テーマ」を固定する

例：飲食店なら
・月曜：今週のおすすめ
・水曜：お客様の声
・金曜：週末のお知らせ

月曜の投稿は「今週のおすすめは何？」の1問だけになる。
ゼロから考えるより、はるかに楽です。
```

### 3/5（習慣2）
```
習慣2｜投稿前に1行だけメモする

「30代女性の常連さんに、新作スイーツを来週試してもらう」

この1行があるかないかで、
写真の角度・キャプション・タグまで全部変わります。

公開前に消してOK。書く"だけ"が大事。
```

### 4/5（習慣3）
```
習慣3｜月末に5分だけ振り返る

全投稿を分析しようとした瞬間、人は諦めます。

正解は"伸びた投稿トップ3"だけを眺めること。
共通項が必ず見つかる。
来月の投稿はその方向に寄せるだけ。

スマホのカレンダーに登録しておくと続きます。
```

```

記事末尾で紹介しています。
ランチ1回分で、来月のSNSは止まらなくなります。

#SNS運用 #コンテンツカレンダー

出力の精度が体感2〜3倍変わります。

```

「新作の生牡蠣のSNS投稿文を書いて」

迷う／驚く／懐かしい／落ち着く——
