# note記事：夏至前の北陸は、19時を過ぎても明るい。

- **日付**: 2026-06-16（暮らし・季節）

## タイトル
```
夏至前の北陸は、19時を過ぎても明るい。
```

## 本文

```
6月の中旬、夕食の支度をしながら窓の外を見て、毎年すこし戸惑う。

19時を過ぎているのに、外がまだ明るい。

時計と空が合っていない感じがして、何度も時刻を見直す。
一年でいちばん日が長い夏至が、もうすぐそこまで来ている。

---

■ 仕事のあとに、もう一区切りある

ひとりで仕事をしていると、終業時間は自分で決めるしかない。
冬は16時半に薄暗くなりはじめて、そこで気持ちも店じまいになる。

いまの季節は逆だ。
18時に仕事を終えても、外にはまだ2時間ちかく「昼の続き」が残っている。

この時間に散歩に出るようになった。
日中の暑さが抜けて、田んぼの水面が夕方の空をそのまま映す。
水を張ったばかりの田は、この季節だけの巨大な鏡だ。

冬の北陸の重い空を知っているから、
この明るさが「貸し出された時間」みたいに思えて、無駄にしたくなくなる。

---

■ 季節で仕事の形を変えていい

会社員の頃は、季節がどうであれ、働く時間は同じだった。
いまは違う。

6月は朝を少し早くして、夕方の明るい2時間を散歩と考えごとに使う。
12月は逆に、暗くなったら机を離れて、夜の作業を朝に回す。

日の長さに合わせて仕事の形を変えるのは、
たぶん電気のない時代にはみんなやっていた、ごく普通のことだ。
フリーランスになって、それが一周回って戻ってきた感じがある。

---

■ 夏至は「折り返し」の合図

夏至を過ぎると、日はまた少しずつ短くなっていく。
つまり、いまのこの夕方は、一年でいちばん延長された夕方ということになる。

今年の上半期にやれたこと、やれなかったこと。
明るい19時の道を歩きながら考えるのが、ここ数年の6月の習慣になった。

大げさな振り返りの儀式はいらない。
日の長さが、勝手に「そろそろ折り返しだよ」と教えてくれる。
```
