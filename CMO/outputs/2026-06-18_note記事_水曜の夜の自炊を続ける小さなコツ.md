# note記事：水曜の夜、自炊を続けるための「3つの軽くする」。

- **日付**: 2026-06-18（生活／ひとり仕事）
- **媒体**: note
- **テーマ**: 平日のど真ん中、水曜夜の自炊を「やめないために」軽量化する方法

## タイトル
```
水曜の夜、自炊を続けるための「3つの軽くする」。
```

## 本文

```
平日のいちばん疲れる夜は、水曜だと思う。
月曜のスタートダッシュも終わり、
金曜のゴールも見えていない、ちょうど真ん中の夜。

そういう夜、自炊を続けるのはハードルが高い。
「もうコンビニでいいか」「Uberでいいか」と思う日もある。

ここ数年、「水曜は自炊を軽くする」と決めて、
やめないかたちで続けている。
今日はそのコツを、3つだけ書いておく。

---

■ コツ1：洗い物を減らす

水曜の自炊は、「フライパンひとつ」を厳守する。

ご飯は冷凍ストックを電子レンジで温める。
味噌汁はインスタント。
メインだけ、フライパンで作る。

豚肉と野菜を炒めて、塩、胡椒、最後に醤油をひと回し。
皿に盛ったら、フライパンは洗剤と熱湯ですぐ洗う。

洗い物が、フライパン1個、皿1枚、お椀1個、箸1膳。
これくらいなら、食後5分で終わる。
食後の片付けが軽いと、自炊を「やる気」のハードルが下がる。

---

■ コツ2：献立を考えない

「今日は何を作ろう」と冷蔵庫を開けてから考えると、
それだけで20分のスタミナを使う。

週末にスーパーで買い物するときに、
「水曜は豚肉と野菜炒め」と決めておく。
食材を買うときに「水曜分」と頭の中でラベルを貼る。

これだけで、水曜の夜に「考える」という工程が消える。
冷蔵庫から取り出して、切って、焼く。
頭が止まっていても手が動く状態にしておく。

献立は、つくる前にだいぶ勝負がついている。

---

■ コツ3：完璧を諦める

水曜の夜のご飯は、味の完成度を求めない。
「だいたい食べられればOK」のレベルで終わらせる。

味付けがちょっと薄かったら、卓上のしょう油でいい。
焼きすぎたら、ご飯と一緒に食べる。
家族や子どもがいる家庭はもう少し気を遣う必要があるけど、
それでも「水曜は手数を減らす日」と家族内で合意しておくと、
気が楽になる。

完璧を求めると続かない。
60点を続けるほうが、長い目で見て自分の体にずっといい。

---

■ 外食じゃなく自炊を続ける、ちいさい理由

なぜ水曜まで自炊にこだわるのか。
お金の節約もあるけど、いちばんは「リズム」だ。

自炊を1日休むと、翌日も外食、翌々日もコンビニ、と
ずるずる流れることが、よくある。
水曜の自炊を、軽くしてでも続けると、
木曜・金曜も自然と自炊で乗り切れる。

おしゃれな料理を作る日ではなく、
「自炊のリズムを切らさない日」と思って、
水曜の夜は淡々とフライパンを温める。
それだけで、平日のご飯のリズムが守られる。
```
