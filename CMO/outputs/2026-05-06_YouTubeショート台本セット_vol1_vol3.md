# YouTube ショート台本セット：note 集客記事 Vol.1〜Vol.3

**用途**：各 note 記事の3つの習慣／コツを、それぞれ1分の YouTube ショートに変換。1記事あたり3本（合計9本）。
**運用方針**：1記事につき3本連投で投稿し、説明欄から note 記事へ誘導。
**費用**：¥0（既存 YouTube アカウント・スマホ撮影／既存 Canva で字幕作成）
**目安**：1本あたり50〜60秒（YouTube ショートは60秒以内）

## 共通フォーマット（各台本に適用）

| 区間 | 秒数 | 役割 |
|------|------|------|
| フック | 0〜5秒 | 視聴者の手を止める一文 |
| 結論 | 5〜10秒 | 「答え」を先に言う |
| 理由 | 10〜30秒 | なぜそうなのか |
| 具体例 | 30〜50秒 | 行動レベルに落とす |
| CTA | 50〜60秒 | 概要欄／固定コメントへ |

# 撮影・編集の運用

- 自然光のある室内 or デスクライト1灯

## 編集ツール（すべて既存・追加費用ゼロ）

- 字幕：Canva の動画機能（既存契約 ¥1.5K/月内）
- 短いカット編集：CapCut（無料版で十分）
- サムネイル：Canva（既存）

## 投稿運用ルール

| 項目 | 内容 |
|------|------|
| 投稿曜日 | 月・水・金（週3本ペース） |
| 投稿時刻 | 19〜21時（夜の視聴ピーク） |
| ハッシュタグ | 各動画に5〜7個 |
| 概要欄 | note 記事URL＋テンプレ販売URL を必ず記載 |
| 固定コメント | 「note 記事はこちら ▶ [URL]」 |
