## タイトル
```
1玉で家族が数日困る、入善ジャンボ西瓜という夏の贈り物
```

## 本文
```
七月に入ると、うちの玄関に大きな段ボールが届く。持ち上げると腰にくるくらい重い。開けると、細長い緑の塊がひとつ、藁のような緩衝材にくるまれて横たわっている。入善ジャンボ西瓜だ。富山の東のほう、海に近い砂地で育つ、規格外に大きい西瓜である。

[写真①]

ふつうの西瓜が丸いのに対して、これは俵のように長い。抱えると赤ん坊を抱くような格好になって、両腕がふさがる。もらった瞬間はうれしい。「今年も来たか」と声が出る。でも、うれしさは五秒くらいで、次に来るのはもっと現実的な感情だ。これ、どこに置く。

台所に運んで冷蔵庫を開けてみる。当然、入らない。野菜室の扉に横向きに当ててみても、閉まらない。切り分けてから冷やすにしても、切った西瓜を入れる容器がまた足りない。結局、その日は西瓜を床に置いたまま、家族で「どうするか会議」が始まる。毎年やっているのに、毎年、答えが出ていないのが面白い。

[写真②]

我が家の暫定的な結論は、玄関の三和土に新聞紙を敷いて転がしておく、というものだ。夏でも玄関はいくらか涼しい。ただし切ったあとは話が別で、冷やさないと甘さがぼやける。だから包丁を入れる前に、まず食べる人数と、その日の予定を計算する。全部いっぺんに切ると、確実に食べきれない。かといって少しずつ切ると、断面から先に傷んでいく。この「切る量の見積もり」が、住人にとっての本当の勝負どころだ。

味のことも書いておかないと不公平だ。見た目のインパクトが強いので、大味なんじゃないかと思われがちだが、食べると裏切られる。しゃりっとした食感で、水っぽくならず、甘さがまっすぐ来る。砂地でよく水はけがいいところで育つと、こういう身が締まった味になるらしい、と近所の年上の人が言っていた。真偽は分からないが、少なくとも実感とは合っている。

困るのは味ではなく、量と大きさの物理だ。二日目、三日目になると、冷蔵庫の一段が完全に西瓜のタッパーで埋まる。牛乳や作り置きの居場所がなくなって、家族から苦情が出る。「西瓜のせいでプリンが入らない」と子どもがふくれる。四日目あたりで、ようやく西瓜が減ってきて、冷蔵庫に平和が戻る。この一連の流れが、うちにとっての夏の始まりの合図みたいになっている。

[写真③]

考えてみると、この西瓜は「食べ物」であると同時に「事件」なのだと思う。届いた瞬間から数日間、家の中の話題と段取りをぜんぶ持っていく。置き場所に悩み、切る量で悩み、冷蔵庫の陣地取りで揉める。でもその面倒くささが、たぶん贈り物として記憶に残る理由でもある。小さくて上品に冷蔵庫に収まる果物では、こうはならない。

もし夏にこの西瓜をもらうことがあったら、ひとつだけ助言がある。受け取る前に、冷蔵庫の中を一段空けておくといい。届いてから慌てるのが、毎年うちの失敗だから。
```
