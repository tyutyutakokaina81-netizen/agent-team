# メルカリ出品プレイブック — 女性ジャケット特化

**作成日**: 2026-04-12
**担当**: CMO
**用途**: 物販 Day 1 から使える出品マニュアル
**対象商品**: 女性向けジャケット（アウター全般に応用可）

---

## 1. 出品前の検品・状態判定

### 状態ランクの定義（メルカリ準拠）

| ランク | 定義 | 価格影響 |
|---|---|---|
| 新品、未使用 | タグ付き・未着用 | 定価の 60〜80% |
| 未使用に近い | 数回着用・ほぼ新品 | 定価の 40〜60% |
| 目立った傷や汚れなし | 使用感軽微 | 定価の 25〜40% |
| やや傷や汚れあり | 一般的な使用感 | 定価の 15〜25% |
| 傷や汚れあり | 明らかな使用感 | 定価の 10〜15% |
| 全体的に状態が悪い | ジャンク寄り | 定価の 5〜10% |

### 検品チェックリスト（各着ごと）

- [ ] ボタン・ファスナー・裏地・ポケットの破れ・ほつれ
- [ ] 襟・袖口・脇の汗染み・黄ばみ
- [ ] 毛玉・毛羽立ち
- [ ] シワ（アイロン可能か）
- [ ] ペット毛・タバコ臭
- [ ] シミ（落とせるもの vs 落ちないもの）
- [ ] 肩の色褪せ
- [ ] タグの切れ・かすれ
- [ ] サイズ表記の確認

## 2. 写真の撮り方（7〜10 枚）

メルカリで写真が売上の **70% を決める**。以下を必ず撮る：

| # | 内容 | 注意点 |
|---|---|---|
| 1 | 全体正面（メイン写真） | 明るい場所、白壁背景、ハンガー or トルソー使用 |
| 2 | 全体背面 | 同じ条件 |
| 3 | 畳んだ状態 | 購入者が発送サイズをイメージできるように |
| 4 | タグ（ブランド名・サイズ・素材） | 文字がクッキリ見えるように |
| 5 | 襟周り | 汚れがある/ないを明示 |
| 6 | 袖口 | よくある汚れポイント |
| 7 | 裏地・ポケット内部 | 質感と汚れの両方 |
| 8 | ディテール（ボタン・刺繍など） | 特徴があれば |
| 9 | 汚れ・傷があればその箇所を正直に | 隠すと低評価の最大原因 |
| 10 | 着用イメージ（任意） | トルソー or 自撮りで雰囲気を伝える |

### 撮影のコツ
- **自然光推奨**（窓際、午前 10〜14 時）
- **フラッシュ禁止**（色が飛ぶ）
- **手ブレ厳禁**（スマホを両手で固定）
- **白 or 薄グレー背景**（商品が映える）
- **色補正なし**（実物と違うと苦情の原因）

## 3. タイトルの型（40 字以内）

```
[ブランド] [アイテム名] [サイズ] [色] [特徴] [状態]
```

### 例
```
✅ GU ノーカラージャケット M ベージュ オフィス 美品
✅ ユニクロ テーラードジャケット S ネイビー 春秋 新品タグ付き
✅ ZARA ツイードジャケット 38 ホワイト フォーマル
✅ 無印良品 デニムジャケット L インディゴ 定番
```

### NG 例
```
❌ 可愛いジャケット売ります！！（情報ゼロ）
❌ お得♪♪♪即決歓迎♪♪♪ジャケット（絵文字過多）
❌ 【美品】【送料無料】【即購入OK】【値下げ不可】【最終値下げ】（括弧過多）
```

### タイトルに必ず入れる 5 要素
1. ブランド名（不明なら「ノーブランド」）
2. アイテム種別（ジャケット/ブレザー/コート/カーディガン）
3. サイズ（S/M/L/XL or 号数）
4. 色
5. 状態キーワード（美品/新品/タグ付き）

```
ご覧いただきありがとうございます。

▼ 商品詳細
・ブランド：[ブランド名]
・アイテム：[ジャケット / ブレザー / コート 等]
・サイズ：[S / M / L / XL / XXL / 号数]
・色：[色名]
・素材：[素材（タグ記載通り）]
・季節：[春秋 / 冬 / オールシーズン]

▼ サイズ実寸（平置き）
・着丈：[cm]
・肩幅：[cm]
・身幅：[cm]
・袖丈：[cm]
※素人採寸のため 1〜2cm の誤差はご容赦ください

▼ 状態
[美品/使用感あり/タグ付き新品] です。
[具体的な状態を 1〜3 文で正直に記述]
[例：数回着用後クローゼット保管でした。目立つ汚れや傷はございません。]

▼ 発送方法
らくらくメルカリ便（匿名配送・追跡あり）を予定しております。
コンパクトに畳んで発送いたします。

▼ お願い
・自宅保管品にご理解いただける方のみご購入ください。
・ペット・喫煙者は[おりません/おります]。
・即購入 OK です。
・コメントなしのご購入歓迎いたします。

ご質問があればお気軽にコメントください。
```

### 説明文のコツ
- **嘘をつかない**（傷・汚れは明記）
- **実寸を書く**（サイズトラブル防止の最大要素）
- **ペット・喫煙の有無**を明記（アレルギー対策）
- **即購入 OK** と明記（購入率が上がる）
- 絵文字は **使わない**（女性向けでも情報性を優先）

## 5. 価格設定のロジック

### ステップ 1：相場調査（必須）

1. メルカリアプリで **同ブランド + アイテム名** を検索
2. **「売却済み」フィルタ**で絞り込み
3. 直近 10 件の平均価格を算出
4. 自分の商品の状態ランクに応じて ±10〜20%

### ステップ 2：初期価格の設定

```
初期価格 = 相場平均 × 1.15（15% 上乗せ）
```

**理由**：メルカリの文化的に値下げ交渉がある前提。最初から底値にすると交渉余地ゼロで逆に売れにくい。

### ステップ 3：下限価格を決めておく

```
下限価格 = 相場平均 × 0.80（売れなくて下げる時の底）
```

コメントで「〇〇円にできますか？」と来たら、下限以上なら応じる。

### 例

```
ZARA テーラードジャケット（美品）
相場平均：¥2,500
初期価格：¥2,875 → ¥2,900（切りよく）
下限：¥2,000

発送：らくらくメルカリ便（宅急便コンパクト）¥450
手数料：¥2,900 × 10% = ¥290
送料：¥450
手取り：¥2,900 − ¥290 − ¥450 = ¥2,160
```

### 送料別・価格ライン早見表

| 販売価格 | 手数料 10% | 送料（小） ¥175〜¥450 | 純利益レンジ |
|---|---|---|---|
| ¥1,500 | ¥150 | ¥175〜¥450 | ¥900〜¥1,175 |
| ¥2,500 | ¥250 | ¥450 | ¥1,800 |
| ¥3,500 | ¥350 | ¥450〜¥700 | ¥2,450〜¥2,700 |
| ¥5,000 | ¥500 | ¥700 | ¥3,800 |

## 6. 発送方法の選び方

| 商品 | 推奨発送 | 送料 | サイズ上限 |
|---|---|---|---|
| 薄手ブレザー・カーディガン | ネコポス | ¥210 | 31.2×22.8×2.5cm |
| 一般的なジャケット | 宅急便コンパクト | ¥450 | 専用 BOX（¥70 別） |
| 厚手コート・ダウン | 宅急便 60 サイズ | ¥750 | 3 辺合計 60cm |
| 大きいコート | 宅急便 80 サイズ | ¥850 | 3 辺合計 80cm |

**らくらくメルカリ便**を選ぶ理由：
- 匿名配送（住所を相手に知られない）
- 全国一律料金
- 追跡あり
- トラブル時にメルカリがサポート

## 7. 出品後のオプション設定

- **購入申請**：OFF（即購入で OK）
- **オファー機能**：ON（値下げ交渉を受け付け）
- **カテゴリ**：レディース > ジャケット/アウター > 該当項目
- **配送の目安**：2〜3 日で発送

---

## 8. 出品時間帯

メルカリは **新着順** で流入する。狙い目の時間：

- **平日 20:00〜23:00**（仕事帰り、就寝前）
- **土日 10:00〜12:00**、**14:00〜17:00**
- 避ける：平日昼間、深夜 2〜7 時

---

## 9. 売れない時の対処

### 3 日経過しても「いいね！」が 5 未満
- 写真を撮り直す（1 枚目の印象を変える）
- タイトルの順序を変える

### 7 日経過しても売れない
- 価格を 10% 下げる
- 説明文の季節キーワードを調整（春→春夏に切替など）

### 14 日経過しても売れない
- 価格を 20% 下げる（下限価格を割る覚悟）
- カテゴリを変更してみる
- もしくは一度削除して再出品（新着復活）

### 30 日経過しても売れない
- そのジャケットは市場に合わない → 塩漬けにせず処分検討

---

## 10. 初回出品の絶対 NG

- ❌ 嘘・誇張表記（「新品」なのに使用済みなど）
- ❌ 送料を無視した値付け（手数料込みで赤字）
- ❌ 他人の写真の転載
- ❌ ブランド不明なのに「〇〇風」と書く（商標トラブル）
- ❌ 連絡なしの遅延発送（低評価直結）
