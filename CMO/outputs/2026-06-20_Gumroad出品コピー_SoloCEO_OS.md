## 4. 説明文（本文エディタにそのまま貼る）

```
■ ひとりなのに、会社のように回す

事業を一人で始めると、企画も、数字も、営業も、発信も、ぜんぶ自分に乗ってきます。
タスクが増えるほど判断が雑になり、「とりあえず手を動かす」だけの日が続く——
これは能力の問題ではなく、役割が分かれていないことの問題です。

Solo CEO OS は、その一人に「役員チーム」を持たせるための運営テンプレート一式です。
あなたは社長（CEO）として方針だけを決め、実務は6人のAI役員に渡します。


■ 6人のAI役員

・CDO（技術）   … プロンプト整備・自動化・仕組みづくり
・CFO（財務）   … 見積・請求・契約・経費の整理
・CMO（集客）   … 発信・コンテンツ企画・ランディングページ
・CPO（商品）   … 教材・テンプレート・サービス設計
・CSO（営業）   … 顧客対話・提案・パイプライン管理
・CAO（分析）   … データ整理・仮説・振り返り

各役員には「担当領域・人格・手順・やらないこと」を定義したプロンプトが付属。
役職をまたぐ仕事は projects/ に切り出す——という連携ルールまで含まれています。


■ このOSの中身

1. 運営ルール一式（company.md / CLAUDE.md）
   会社の意思決定・ファイル管理・報告フォーマットを言語化。チームが迷わない土台。

2. 6役員のプロンプト
   コピーして使える役割定義。今日から各役員に仕事を振れます。

3. 記憶の仕組み（STATE.md）
   セッションが変わっても文脈を失わないための永続メモ設計。
   「前回どこまで進んだか」を毎回ゼロから説明する必要がなくなります。

4. 自動実行プロトコル（「実行して」）
   一言で、その日の決まった作業を最後まで走らせる運用ルール。
   毎回の指示出しを定型化し、判断の往復を減らします。

5. 新しい役員を増やすルール
   既存の役職に当てはまらない仕事が増えてきたら、新役員を足す判断基準つき。
   事業の成長に合わせてチームを拡張できます。


■ こんな人に

・やることが多すぎて、一つひとつが中途半端になっている
・AIを使ってはいるが、毎回ゼロから指示していて積み上がらない
・「会社っぽい仕組み」を、一人でも持っておきたい
・副業・フリーランスを、属人作業から仕組みへ移したい


■ 始め方

1. ダウンロードした一式を作業フォルダに展開
2. Claude もしくは Claude Code で開く
3. まず company.md と各役員のプロンプトに目を通す
4. 一番渋滞している領域の役員（多くはCMOかCSO）から仕事を渡す

特別なツールや有料サブスクは前提にしていません。
無料で使えるAIから始めて、必要になったら拡張してください。


■ 注意

・本製品はテンプレートとプロンプトの一式です。AIの利用環境（Claude等）はご自身でご用意ください。
・成果はご自身の事業・運用によって変わります。収益や結果を保証するものではありません。
・購入後に改訂した場合、同じリンクから最新版をダウンロードいただけます。
```

- サイズ：1000×1000px（600×600以上の正方形ならOK）、書き出しはPNG
- 背景：濃紺〜黒のグラデーション（管理画面のダークトーンと統一感）
- 中央大見出し：`Solo CEO OS`（白・太字・サンセリフ）
- サブ：`ひとりで、会社のように。`（薄いグレー）
- 下部に小さく6つのバッジを横並び：`CDO / CFO / CMO / CPO / CSO / CAO`

- 文字は画像端から十分マージンを取り、サムネ縮小時も `Solo CEO OS` が読める大きさに
