# 英語 travel writing 執筆テンプレ（CMO・2026-06-07）

海外読者向け。翻訳ではなく独立した travel writing（150–250 words）。

## 構造
1. Hook：意外性のある一文
2. From Kanazawa：金沢からの所要時間＋手段（JR氷見線／北陸新幹線／新高岡）
3. 中身：見られる/食べられるものを具体1–2点
4. Honest line："skip the crowds" / "barely any foreign tourists"
5. Soft CTA：立ち寄る価値を一言

## フレーズbank（記事ごとに変える＝反復NG）
- from Kanazawa, about NN minutes by ...
- skip the crowds / barely any foreign tourists
- This is the Japan most tourists never see
- worth the detour / a quiet alternative to ...

## 必須 / NG
- 必須：アクセス情報、固有名詞はローマ字＋英訳、食記事は末尾にじゃらんPR表記＋リンク（affiliate/links.md）
- NG：直訳／同一フレーズ反復／誇張・断定
