# 英語 travel writing 執筆テンプレ（CMO・2026-06-07）

海外読者向け。翻訳ではなく独立した travel writing（150–250 words）。

## 構造
1. Hook：意外性のある一文
2. From Kanazawa：金沢からの所要時間＋手段（JR氷見線／北陸新幹線／新高岡）
3. 中身：見られる/食べられるものを具体1–2点
4. Honest line："skip the crowds" / "barely any foreign tourists"
5. Soft CTA：立ち寄る価値を一言

## フレーズbank（記事ごとに変える＝反復NG）
- from Kanazawa, about NN minutes by ...
- skip the crowds / barely any foreign tourists
- This is the Japan most tourists never see
- worth the detour / a quiet alternative to ...

## 必須 / NG
- 必須：アクセス情報、固有名詞はローマ字＋英訳、食記事は末尾にじゃらんPR表記＋リンク（affiliate/links.md）
- NG：直訳／同一フレーズ反復／誇張・断定

## ★「世界に刺さる英語」基準（2026-06-07 オーナー目的・最優先）

目的は要約ではなく、**海外読者の心に刺さり、シェアされ、検索で見つかる英語記事**。

### 刺さる5条件（全て満たす）
1. **1行目フック**：意外性 or 感情。`Most travelers rush past...` `Forget Kyoto for a day` のように、続きを読ませる。
2. **発見の角度**：`hidden Japan` / `the side of Japan tourists never see` / `skip the crowds`。観光地の逆を突く。
3. **具体の感覚描写**：固有の一点（音・温度・匂い・数字）。一般論を書かない。
4. **シェアされる1文**：引用したくなる一節を必ず1つ（`This is the Japan most tourists never see.` 等・記事ごとに変える）。
5. **発見可能性（SEO）**：海外が検索する入口語を自然に含める＝**Doraemon / Kanazawa / Hokuriku Shinkansen / Noto / hidden gem / day trip**。

### 構造（150–250語）
Hook → from Kanazawa＋アクセス → 具体1–2点（感覚） → 逆張りの本音 → 行く価値（soft CTA）

### NG（刺さらない英語）
- 直訳・要約調／毎回同じフレーズ反復／一般論（"famous for its nature"）／誇張断定。

### セルフチェック（公開前）
- 海外の見知らぬ読者が **1行目で止まるか？** 引用したい1文があるか？ 検索入口語が入っているか？

## ★外国人が「読む理由」＝5つの興味ドライバー（2026-06-07 調査反映）
海外読者が記事を開く/読み切る動機。各記事は最低2つ、できれば全部を満たす。
1. **驚き・へぇ（quirky/意外）**：「えっそうなの?」の一点（数字・逆説・珍）。1行目フックに最適。
2. **住人の日常（what locals do）**：観光ガイドに無い一人称の生活描写＝我々の最強資産。
3. **アニメ・ポップ（Doraemon等）**：感情・ノスタルジア。世界の入口語。
4. **穴場・本物（authenticity/no crowds）**：「自分だけの発見」「混雑なし」。
5. **背景の物語（why）＋心が動く瞬間**：成り立ちの文脈＋emotional payoff。
> 海外で刺さる「本物の体験」の式＝**great attraction × backstory（why）× no crowds**。3点が揃うと読まれる&シェアされる。
