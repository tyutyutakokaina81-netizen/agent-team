## 無料部分

```
金沢まで来て、そのまま帰ってしまう人が、本当に多い。
兼六園を歩いて、市場で海鮮を食べて、それで北陸はおしまい。
もったいない、といつも思う。金沢から電車で少し東へ行くだけで、
富山・高岡・氷見という、まったく空気の違うエリアが待っているからだ。

私は高岡に住んでいる。
この記事は、その住人が「友人が金沢から遊びに来たら、実際にこう案内する」
という前提で組んだ、2泊3日のモデルコースだ。

■ 誰向けのコースか

・金沢観光にプラスして、北陸をもう少し深く歩きたい人
・有名観光地の行列より、落ち着いた町歩きと海の幸が好きな人
・電車とバスと徒歩で回りたい人（レンタカーなしで全行程を組んでいます）

■ このコースの設計思想

富山側は、金沢ほど観光地化されていない。
言い換えると、行列も、観光価格も、作られた「映え」も少ない。
国宝の寺が静かに建っていて、路面電車が日常として走っていて、
市場では地元の人がふつうに買い物をしている。
このコースは、そういう「観光地ずれしていない北陸」を、
無理のない移動でつなぐことだけを考えて組んだ。
詰め込まない。1日の柱は2〜3つ。あとは歩く時間と、食べる時間。

旅行サイトのモデルコースと違うのは、順番と時間帯に理由があること。
たとえば雨晴海岸は「朝」に置いてある。金屋町は「夕方の光」で歩いてほしい。
住んでいると分かる、その場所がいちばんよく見える時間帯で並べてある。

■ 3日間の全体像

・Day1：高岡 —— 国宝の寺と、鋳物の町並み（この日は移動が少ない）
・Day2：雨晴・氷見・新湊 —— 海の日。車窓、魚、運河、帆船
・Day3：富山市 —— 水辺とガラスの町を歩いて、金沢へ戻る

金沢を朝に出れば、初日の昼前から動き出せる。逆に、東京・大阪方面から
富山入りして最後に金沢へ抜ける逆回りでも、そのまま使える構成にしてある。

■ Day1の午前だけ、先にお見せします（このコースの歩き方の見本です）

金沢駅から高岡へ。新幹線なら新高岡駅まで十数分ほど、
在来線（IRいしかわ鉄道→あいの風とやま鉄道）でも40分ほどで高岡駅に着く。
運賃はどちらも片道数百円〜千円台が目安（時期・経路で変わるので要公式確認）。

高岡駅に荷物を預けたら、まず瑞龍寺へ。駅から歩いて10分ほど。
加賀前田家ゆかりの禅寺で、山門・仏殿・法堂が国宝に指定されている。
これだけの伽藍を、行列もなく静かに見られる。ここが富山側の第一印象になる。

瑞龍寺から駅の反対側へ抜けて、高岡大仏へ。こちらも駅から徒歩圏。
奈良や鎌倉のような巨大さではないけれど、銅器の町・高岡の職人が造った大仏で、
商店街のすぐ横、町の暮らしの中に立っているのがいい。参拝は気軽にできる。
柵の向こうに鎮座しているのではなく、通学路の風景の一部として立っている。
「町が大仏を持っている」感じは、実際に前に立つとよく分かる。

——ここまでで、まだDay1の午前中。
歩く距離は控えめで、それでも「国宝」と「銅器の町の象徴」を両方見ている。
この密度と余白のバランスで、残りの2日半も組んである。

■ ここから先は、保存版の本体です

・Day1午後〜Day3の全行程（午前／昼／午後／夜の時間帯別）
・各スポットに「住人目線で、なぜ行く価値があるか」を一言ずつ
・雨の日の差し替え案（各日1つ）
・概算予算表（交通・食事・宿のレンジ）
・氷見線・万葉線・バスの「移動の注意」と持ち物メモ

旅行前に1回読んで、旅行中にもう1回開く。
そういう使い方を想定した保存版です。

## 有料部分

```
【Day1 午後】町歩きの高岡 — 山町筋と金屋町

●昼
高岡駅周辺か、山町筋方面へ歩きながら。
高岡は「コロッケの町」としても知られていて、惣菜店やパン屋で見かけたら
1個つまむのも高岡らしい昼になる。しっかり食べたい日は、
細めの手延べうどん「氷見うどん」を出す店を駅周辺で探すのがおすすめだ。

●午後①：山町筋（やまちょうすじ）
駅から徒歩15分ほど、または路面電車の万葉線で数分。
明治の大火のあとに建てられた、黒い土蔵造りの商家が並ぶ通り。
→ 住人目線：観光客で混まないので、重厚な建物を独り占めするように歩けます。

●午後②：金屋町（かなやまち）
山町筋から歩いて15分ほど。千本格子の家並みが続く、高岡鋳物発祥の町。
石畳には銅片が敷き込まれていて、足元まで鋳物の町らしい。
→ 住人目線：夕方の斜めの光で格子の影が伸びる時間帯が、いちばんきれいです。

余裕があれば、金屋町のすぐそばを流れる千保川へ。
鋳物の町は、この川のそばで始まった。町と川をセットで歩くと成り立ちが見える。

もう1か所足せるなら、高岡古城公園。前田利長の城跡がそのまま公園になっていて、
入るのは自由。堀と緑のあいだを、地元の人が散歩している。
→ 住人目線：「観光」というより「町の日常に混ざる」時間として、夕方前に向いています。

●夜：高岡泊
夕食は駅周辺で「富山湾の魚＋地酒」を軸に。
白えび、ホタルイカ（春）、ブリ（冬）など、季節の魚を黒板に書く店が多い。
店選びは「地魚」「地酒」を掲げている店なら大きく外さない、というのが住人の感覚。

---

【Day2 午前】雨晴海岸 — このコースのハイライト

朝は早起きして、高岡駅から氷見線に乗る。雨晴駅まで20分ほど（目安）。
晴れて空気の澄んだ日は、海の向こうに雪をかぶった立山連峰が立ち上がる。
海越しに3000m級の山並みがはっきり見える場所は、世界的にも多くない。
→ 住人目線：毎日見えるわけではありません。会えたらラッキー。
　ただ、見えなくても氷見線は線路のすぐ外が海で、乗ること自体が景色です。

雨晴駅の近くには道の駅（雨晴）があり、海を眺めながら休憩できる。
海岸には「女岩（めいわ）」という小さな岩の島が浮かんでいて、
山が見える日は、女岩と立山連峰を重ねる構図がこの海岸の定番になっている。
→ 住人目線：山狙いなら、空気の澄む寒い季節の「よく晴れた朝」がいちばん確率が高い。
　夏は山に雲がかかりやすいので、海と車窓を主役に考えておくと落胆しません。

【Day2 昼】氷見 — 魚の町

雨晴から氷見線でさらに終点の氷見へ。
昼は「ひみ番屋街」（道の駅）へ。氷見の魚を使った食事処と物販が集まっていて、
海鮮系の昼食はここで完結する。氷見うどんの店もある。
→ 住人目線：氷見といえば冬の寒ブリが有名ですが、それ以外の季節の地魚も十分うまい。
　買い物をするなら氷見の干物。朝食が変わるレベルの土産で、保冷バッグがあると安心。

食後は、氷見市潮風ギャラリーへ。
氷見はまんが家・藤子不二雄Ⓐ氏の出身地で、ここが作品世界の拠点施設。
商店街にはキャラクターのモニュメントが点在している。
→ 住人目線：まんがに詳しくなくても、港町の商店街歩きとして楽しめます。

【Day2 午後〜夜】新湊内川と海王丸パーク

氷見から高岡方面へ戻り、万葉線（路面電車）に乗り換えて新湊エリアへ。
内川は、家並みのあいだを川が抜け、漁船が係留されている水辺の町。
「日本のベニス」と紹介されることもあるが、実際に流れているのは
静かな漁師町の日常で、それがいちばんの見どころだと思う。
→ 住人目線：橋ごとにデザインが違うので、橋めぐりだけで小一時間歩けます。
　観光船で川から眺める楽しみ方もある（運航日・料金は要公式確認）。
　小腹がすいたら、屋内市場の「新湊きっときと市場」へ。昼を軽くした日は
　ここで海鮮丼にしてもいい。紅ずわいがに（秋〜冬が漁期）の産地でもある。

続いて海王丸パークへ。白い帆船・海王丸が係留され、
新湊大橋と富山湾をまとめて眺められる。夕方の光の時間に合わせられると気持ちいい。

●夜：2泊目は富山市内泊がおすすめ（Day3の行程が楽になる。高岡泊のままでも可）。
夕食は富山市中心部で。黒いスープのご当地ラーメン「富山ブラック」を試すか、
引き続き地魚か。刺身を昆布で締める「昆布じめ」は、このあたりの居酒屋の定番だ。

---

【Day3】富山市 — 水辺とガラスの町

●午前：富岩運河環水公園
富山駅から歩ける運河沿いの公園。朝の散歩にちょうどいい。
→ 住人目線：地元の人がランニングや犬の散歩をしている、生活の中の公園です。

●午前〜昼：富山市ガラス美術館
図書館と同居した複合施設（隈研吾氏設計）の中にあり、建物自体も見どころ。
→ 住人目線：展示に加えて、吹き抜けの木の内装を見上げるだけでも入る価値があります。

●昼：ます寿司
富山の駅弁として全国区だが、市内には製造元が多く、
店ごとに酢の加減も鱒の厚さも違う。駅や百貨店で数種類買って食べ比べると楽しい。
→ 住人目線：お土産にするなら日持ちに注意（当日〜翌日が目安。要確認）。

●午後：富山城址公園を軽く歩いて、富山駅へ。
新幹線なら金沢まで20分強、在来線でも1時間前後で戻れる（いずれも目安・要確認）。

●お土産の考え方（駅でまとめて買えるもの）
・ます寿司：食べ比べた中で気に入った1本を。日持ちは短いので帰宅日向き。
・白えびせんべい：軽くて配りやすく、日持ちもする。ばらまき土産の定番。
・昆布製品：高岡・富山は昆布の消費文化が根強い町。とろろ昆布は軽くてかさばらない。
→ 住人目線：迷ったら「魚系は自分用、乾き物は配り用」で分けると失敗しません。

【行程の組み替え早見】

・1泊2日に圧縮するなら：Day3（富山市）を丸ごと落とす。
　高岡＋雨晴・氷見・新湊の2日間だけでも、このコースの核は残る。
・逆回り（富山側から入って金沢へ抜ける）：Day3を初日に置き換えるだけで成立する。
・冬に行くなら：雨晴の立山連峰がいちばん見えやすい季節。ただし列車の乱れも
　起きやすいので、Day2の午後（新湊）を「行ければ行く」扱いにして余白を作る。
・子ども連れなら：高岡おとぎの森公園（藤子・F・不二雄氏ゆかりの公園）を
　Day1午後の町並み歩きと差し替える手もある。

【移動の注意（ここで失敗しやすい）】

・氷見線は本数が限られる（時間帯によっては1時間に1本程度）。
　雨晴・氷見へ行く日は、朝のうちに帰りの時刻まで確認しておくと安心。
・交通系ICカードが使えない路線・車両がある。小銭・現金を必ず用意しておく。
・万葉線は路面電車。高岡駅前から新湊方面へ、のんびり走る。急ぐ移動には
　向かないが、乗ること自体がこの旅の一部と考えるとちょうどいい。
・新湊〜氷見のように、エリア間を直接つなぐ公共交通は限られる。
　迷ったら「いったん高岡に戻って組み直す」前提で考えると迷子にならない。
・路線バスは本数が少ない区間が多く、日祝ダイヤでさらに減ることがある。
　このコースが鉄道と徒歩を軸にしているのは、それが理由でもある。
・荷物は高岡駅・富山駅のコインロッカーを基点に。宿を移る日（Day2夜）は、
　朝のうちに荷物を預けてから動くと、海沿いを身軽に歩ける。
・冬（12〜2月）は天候で列車が乱れる日がある。行程に余白を残しておく。
