# 単価を上げる会話、実際の言い方5パターン

*※ メンバーシップ限定記事（深掘り③）*

---

無料の記事で「営業しないで仕事が来る仕組み」を書きました。
仕事が来るようになると、次にぶつかるのが**単価**です。

ひとりでやっていると、値上げの話を切り出すのが一番こわい。
「高いと言われたら」「断られたら」「関係が悪くなったら」。
この恐怖で、私も長いあいだ、最初に決めた安い単価のまま走り続けました。

でも、単価を上げないと、件数で稼ぐしかなくなります。
件数には体力の限界があります。だから、どこかで単価の会話は避けて通れません。

ここでは、私が実際に使った（そして通った）言い方を5パターン、場面別に書きます。
精神論ではなく、**そのまま言えるセリフ**として。

---

## パターン1：新規の見積もりで、最初から相場で出す

一番ラクなのは、既存客の値上げではなく、**新規に最初から高く出す**こと。

新規には過去のしがらみがありません。あなたの「安い時代」を知らない。
だから、相場の単価をそのまま提示できます。

```
「今回のご依頼ですと、◯◯円でお受けしています。
内容には△△と□□が含まれます。」
```

ポイントは、値段の後に**含まれる中身**をセットで言うこと。
値段だけ言うと高く聞こえ、中身を添えると「その値段の理由」になります。

新規を相場で取りながら、既存客は段階的に、が現実的な順番です。

---

## パターン2：既存客に「次回から」と予告する

今やっている案件の途中で値上げするのは、印象が悪い。
**次回・次契約から**にすると、ぐっと言いやすくなります。

```
「今の内容で続けていただいていて、本当にありがたいです。
ひとつご相談で、次回ご依頼分から、料金を◯◯円に見直させてください。
理由は△△で、品質を保つためのお願いです。」
```

「見直させてください」と、お願いではなく決定として、でも丁寧に。
そして必ず理由を添える。理由のない値上げは、ただの値上げ。理由のある値上げは、相談になります。

---

## パターン3：「できることが増えた」を理由にする

値上げの理由で一番納得されやすいのが、**提供価値が増えたこと**。

```
「最近、△△（分析レポート/追加の対応）も合わせてお出しするようにしていて、
その分、料金を◯◯円に調整させていただいています。」
```

値段だけ上げると抵抗されますが、「中身が増えたから値段も」は、商売として自然です。
だから、値上げのタイミングに合わせて、小さくてもいいので提供物を一つ足す。
これが一番きれいな値上げです。

---

## パターン4：キャパを理由に、選んでもらう

仕事が詰まってきたら、これが使えます。

```
「ありがたいことに依頼が増えていて、
今お受けしている分から、料金を見直させていただこうと思っています。
もし今の料金が前提でしたら、無理にとは申しません。」
```

「無理にとは申しません」が効きます。
これは、相手に「続けるか・抜けるか」を選んでもらう言い方。
本当に必要としてくれている客は残り、安いから頼んでいただけの客は離れます。
そして、離れた分のキャパで、相場の新規が取れる。**入れ替え**です。

こわいですが、安い客で埋まったキャパは、相場の客を取る余地を奪っています。

---

## パターン5：相手から「お願いされる」状況を待って、応じる

一番強いのは、自分から切り出さず、**相手が頼ってきたタイミング**で条件を整えること。

「急ぎでお願いできないか」「これも追加でできないか」。
こういう相談が来たときが、自然に条件を更新できる瞬間です。

```
「もちろん対応します。
その分、今回からは◯◯円でお願いできればと思いますが、いかがでしょう。」
```

相手が「助けてほしい」状態のときは、値段より「対応してくれること」の価値が上がっています。
高圧的にではなく、さらっと条件を添える。これが一番、関係を壊さずに単価が上がります。

---

## 全パターンに共通する1つの原則

値上げの会話で一番大事なのは、**自分が先に納得していること**です。

自分が「この値段は高すぎる」と思っていると、声のトーンに出て、相手も不安になります。
逆に、自分が「この内容ならこの値段は妥当だ」と腹落ちしていれば、淡々と言えます。

だから、言い方の練習より先に、**自分の仕事の値段を自分で正当だと思えるか**。
ここが整っていないと、どんなセリフを使っても、語尾が弱くなります。

値上げは交渉技術である前に、自己評価の問題です。

---

## まとめ

| 場面 | 言い方の軸 |
|------|----------|
| 新規 | 最初から相場 + 中身をセットで |
| 既存 | 「次回から」と予告する |
| 価値増 | 「できることが増えた」を理由に |
| キャパ | 選んでもらう「無理にとは申しません」 |
| 受け身 | 相手が頼ってきた瞬間に条件を添える |

5つのどれを使うかより、**自分がその値段に納得しているか**が先。
そこさえ整えば、あとはこの5つから場面に合うものを選ぶだけです。
