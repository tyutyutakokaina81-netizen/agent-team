# AIに5役職を渡す、実際の分け方と指示文

*※ メンバーシップ限定記事（運営ノウハウ①）／指示文テンプレ付き*

---

無料の記事で「AIに5つの役職を渡してみた話」を書きました。
反響が大きかったのですが、「で、具体的にどう分けて、どう指示するの？」という質問が一番多かった。

ここでは、私が実際に使っている**5役職の分け方**と、**それぞれに渡している指示文の冒頭**を公開します。
概念ではなく、明日から真似できる形で。

---

## なぜ「ひとり」なのに役職を分けるのか

ひとりで全部やると、頭の中で「書く自分」「数字を見る自分」「売る自分」が同時に喋り出します。
これが思考の渋滞です。

役職を分けるのは、**一度にひとつの帽子だけかぶる**ため。
今は数字の帽子、次は発信の帽子。AIにその帽子をかぶせて相談相手にすると、自分も一緒にその役になりきれます。

ポイントは、AIを「賢い1人の万能アシスタント」として使わないこと。
**5人の専門家**として、相談先を切り替える。これだけで、出てくる答えの質が変わります。

---

## 5役職の分け方（私の場合）

| 役職 | 担当 | 相談する場面 |
|------|------|------------|
| CDO（技術） | 自動化・プロンプト・効率化 | 「この作業、もっと速くできない？」 |
| CFO（数字） | 売上・原価・現金・価格 | 「この値段で利益出てる？」 |
| CMO（発信） | 記事・SNS・集客 | 「これ、どう発信すれば届く？」 |
| CPO（商品） | 教材・テンプレ・商品設計 | 「これを商品にするなら？」 |
| CSO（営業） | 提案・顧客対応・断り方 | 「この客にどう返す？」 |

あなたの事業に合わせて、役職名は変えていい。
大事なのは数（多すぎると切り替えが面倒）と、**守備範囲が重ならないこと**。

---

## 各役職への指示文（冒頭テンプレ・コピペ可）

AIに役を与えるとき、私は会話の冒頭にこれを置きます。

### CFO（数字）に相談するとき
```
あなたは私の会社のCFOです。
数字の正確さと現実性を最優先し、希望的観測を排除してください。
私が見落としている費用やリスクがあれば、必ず指摘してください。
楽観的な励ましは不要です。事実と数字で話してください。
```

### CMO（発信）に相談するとき
```
あなたは私の会社のCMOです。
読み手が誰で、その人が何を感じるかを起点に考えてください。
テンプレ的な表現（「〜には〜がある」の連発など）は避け、
具体と一次体験を重視してください。
```

### CSO（営業）に相談するとき
```
あなたは私の会社のCSOです。
顧客との関係を長期目線で守ることを最優先してください。
売り込みすぎず、かといって安売りもせず、
相手が断りやすい余白も残した提案を考えてください。
```

CPO・CDOも同じ要領。
**「最優先すること」と「やってほしくないこと」をセットで書く**のがコツです。
これがないと、AIはどの役でも同じ無難な答えを返してきます。

## 役職を「会議」させる上級テク

ひとつの判断に迷ったとき、私は2役職を**対話させます**。

```
この件について、CFOとCMOで意見を出し合ってください。
CFOは数字の観点、CMOは集客の観点から。
意見が割れたら、その対立点を明確にしてください。
私はその対立を見て決めます。
```

たとえば「この記事を無料にするか有料にするか」。
CFO（有料化で収益を）とCMO（無料で読者を増やす方が長期的に得）が議論すると、
自分ひとりでは見えていなかった**トレードオフの中身**が言語化されます。

答えをAIに出させるのではなく、**判断材料を整理させて、決めるのは自分**。
これが、ひとり経営でAIを使う一番健全な形だと思っています。

---

## やってみて分かった注意点

1. **役職を増やしすぎない**
   7つ8つに増やすと、どの帽子をかぶればいいか自分が迷う。5つが上限。

2. **AIの答えを「上司の決定」にしない**
   AIは優秀な相談相手であって、決定者ではない。最終判断は必ず自分。
   ここを混同すると、自分の事業なのに他人事になる。

3. **役の冒頭テンプレを使い回す**
   毎回ゼロから役を説明すると疲れる。テンプレを保存して貼るだけにする。

## まとめ

- AIを「万能アシスタント1人」でなく「専門家5人」として使う
- 各役には「最優先」と「やってほしくないこと」をセットで指示
- 迷う判断は2役職を議論させ、対立点を見て自分が決める
- 役職は5つまで、決定者は常に自分

ひとりは、孤独です。
でも、頭の中を5つに分けて、それぞれに相談相手を持つと、
ひとりなのに、会議ができるようになります。
