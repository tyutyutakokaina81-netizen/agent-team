# note タイトル最適化バッチ（owner 編集用）

- **作成日**: 2026-06-29 ／ **担当**: CMO（code）
- **使い方**: note の各記事「編集」→ タイトルを「改善案」に差し替え→ 更新（**URLは変わらない＝既存評価を維持**）。
  ハッシュタグも「推奨タグ」に寄せる（先頭3つは検索される定番タグ）。
- **原則**: 叙情フックは残しつつ **タイトル前半に「地名（市まで・A4順守）＋固有名詞」** を入れる。
  町名・住所・店の所在地番地は書かない（市レベルまで）。
- **★印** = L5 で先に編集して効果検証したい上位10本。

## 暮らし・仕事（別ペルソナ＝マガジンで分離）

これらは地名キーワードより**テーマタグの定番化**が効く（検索より note 内回遊・フォロー狙い）。
タイトル変更は不要、タグだけ定番化：

| 記事系統 | 推奨タグ（先頭固定） |
|---|---|
| フリーランス/ひとり会社 | #フリーランス #ひとり起業 #個人事業主 #働き方 |
| AI/仕事術 | #AI活用 #ChatGPT #業務効率化 #仕事術 |

> ※ この系統は「富山・高岡の発信（北極星）」とは読者が別。マガジンを分けて混ぜない
> （`note閲覧数グロース実行プラン.md` L3 参照）。

## 適用手順（owner）

1. まず **★10本** のタイトルを差し替え＋タグを推奨形に（15分程度）。
2. 1〜2週間後、その10本の閲覧数を見て **伸びた型**を code に共有。
3. 効いた型に合わせて、残りの食・場所記事を一括最適化（code が次バッチを作成）。
