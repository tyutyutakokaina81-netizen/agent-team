- **★印** = L5 で先に編集して効果検証したい上位10本。

## 適用手順（owner）

1. まず **★10本** のタイトルを差し替え＋タグを推奨形に（15分程度）。
2. 1〜2週間後、その10本の閲覧数を見て **伸びた型**を code に共有。
3. 効いた型に合わせて、残りの食・場所記事（バッチ2含む）を一括最適化。
4. **今後の新規記事は最初から最適化**：`CDO/outputs/2026-06-29_note記事_検索最適化チェックリスト.md` に沿って作る
   （過去記事の手編集に頼らず、新規が自動で流入を取る＝放置モデルと整合）。
