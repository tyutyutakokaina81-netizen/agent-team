## 本文
```
十二月に入ると、富山の家の冷蔵庫には、たいてい「かぶら寿司」が入っている。名前に「寿司」とつくけれど、酢飯の握りとは別物だ。塩漬けにした厚切りのかぶに切り込みを入れ、そこへブリを挟み、米の糀（こうじ）でじっくり漬け込んで発酵させる。分類でいえば「なれずし」の仲間、漬物に近い保存食だ。

[写真①]

高岡に住んでいると、これが冬の風物詩になっている。輪切りのかぶは雪みたいに白く、切ると中からブリがのぞく。ひと切れ口に入れると、糀のやわらかな甘みと、かぶのしゃきっとした歯ざわり、そして発酵ならではのほのかな酸味が重なる。塩漬けのブリの旨みがそこへ溶けていく。ごはんにも合うけれど、私はやっぱり日本酒と一緒にやりたい。

なぜ冬なのか。北陸は雪に閉ざされる季節が長く、昔から越冬の蓄えが要った。ブリは大漁の日もあれば、まるで獲れない日もある。獲れたぶんを糀で漬けて発酵させておけば、日持ちがして、しかも旨くなる。理にかなった、雪国の知恵だと思う。

もとをたどれば加賀藩の食文化と地続きらしい。富山の西のあたりも、かつては加賀に属していた縁がある。今でも年の暮れが近づくと、店先にかぶら寿司が並びはじめる。「これがないと正月が来た気がしない」という人も、まわりに少なくない。

家ごとに味が違うのも、この食べ物の面白いところだ。糀の効かせ方、漬ける日数、挟む魚。作る人の数だけ、少しずつ違う正月の味がある。
```

## 事実検証ノート
```
✅確実：かぶら寿司は塩漬けした厚切りのかぶ（蕪）に切り込みを入れ、塩漬けのブリ等の魚を挟み、米糀（甘酒麹）で発酵させた保存食で、酢飯を使う握り寿司とは異なる「なれずし」の一種（漬物に近い）。北陸（石川・富山）の冬〜正月の郷土料理として定着している。北陸は積雪期が長く越冬の保存食を要したこと、ブリが安定して獲れないことから、発酵で保存性と旨みを両立させた背景がある。富山県西部はかつて加賀藩に属した縁があり、加賀の食文化と地続きとされる。年末になると店頭に並ぶ。（出典：農林水産省「うちの郷土料理」富山県/石川県、Wikipedia「かぶら寿司」ほか）
⚠️要確認：挟む魚はブリが代表的だがサバを使う地域・家庭もある。漬け込み日数は7〜10日程度が一例だが、レシピ・家庭により差がある。糀の効かせ方や味付けは作り手ごとに異なるため、本文の味の描写は一般的傾向＋住人の体感。
諸説あり：起源には金沢の漁師が正月の行事食として食べ始めた説、藩主が温泉宿で振る舞われた説などがあり定かでない。少なくとも江戸時代には食べられ、宝暦7年（1757年）頃に年賀の客へ出した記録があるとされる。
```
