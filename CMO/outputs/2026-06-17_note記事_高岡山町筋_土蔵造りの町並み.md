# note記事：高岡には、土蔵造りの商家が並ぶ通りが残っている。— 山町筋

- **日付**: 2026-06-17（文化）

## 本文

```
高岡というと銅器や大仏の話になりがちだけど、町並みそのものが見どころの場所もある。
山町筋は、防火のために建てられた土蔵造りの商家が並ぶ通りで、
重要伝統的建造物群保存地区に選ばれている。

明治の大火のあと、商人たちが「次は燃えないように」と、
黒い漆喰や煉瓦を使った重厚な建物を競うように建てた。だから一棟一棟が少しずつ違う。
歩いていると、ふつうに人が住み、店を営んでいる「生きている町並み」だと分かる。
観光のために保存された書き割りではなく、日常の延長に古い建物がある。

地元の人間にとっては通学路や買い物道だったりするので、特別感はあまりない。
でも他県の人を案内すると、たいてい「これが普通にあるのがすごい」と言われる。
銅器の工房も近く、高岡の「ものづくりの町」としての厚みが、町並みからも伝わってくる。
```

## English (travel writing)

```
Yamachosuji — Takaoka's street of fireproof merchant houses

Takaoka is known for bronze and its Great Buddha, but one of its quiet
highlights is just a street. Yamachosuji is lined with dozo-zukuri merchant
houses — thick, fire-resistant townhouses built after a great fire, now a
nationally designated preservation district. Each one is slightly different,
because the merchants rebuilt competitively so it would "never burn again."

What strikes visitors is that people still live and run shops here — a living
streetscape, not a museum set. From Kanazawa it's about 20 minutes by
Shinkansen to Shin-Takaoka. Pair it with the nearby bronze workshops to feel
how deep this craft town really runs.
```
