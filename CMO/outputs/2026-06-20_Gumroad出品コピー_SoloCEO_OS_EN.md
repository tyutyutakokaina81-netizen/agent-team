## 4. Description (paste into the body editor)

```
■ One person, run like a company

When you start a business alone, everything lands on you — planning, numbers,
sales, marketing, support. The more tasks pile up, the rougher your decisions get,
and you end up just "moving your hands" all day.

That isn't a skill problem. It's a roles problem: nothing is divided.

Solo CEO OS gives that one person an executive team. You stay the CEO and decide
direction; the day-to-day work goes to six AI executives.


■ The six AI executives

· CDO (Tech)       … prompts, automation, building the machinery
· CFO (Finance)    … quotes, invoices, contracts, expenses
· CMO (Growth)     … publishing, content, landing pages
· CPO (Product)    … courses, templates, service design
· CSO (Sales)      … customer dialogue, proposals, pipeline
· CAO (Analytics)  … data, hypotheses, weekly review

Each executive ships with a prompt that defines its scope, personality, workflow,
and what it must NOT do. Cross-role work gets split into a projects/ folder — those
collaboration rules are included too.


■ What's inside

1. The operating rules (company.md / CLAUDE.md)
   Decision-making, file management, and reporting format, written down so the team
   never has to guess.

2. Six executive prompts
   Copy-and-use role definitions. You can start delegating today.

3. A memory system (STATE.md)
   Persistent notes so context survives across sessions. No more re-explaining
   "where did we leave off" from scratch every time.

4. An auto-run protocol
   One phrase runs the day's fixed work end to end. It turns repeated instructions
   into a routine and cuts the back-and-forth.

5. A rule for adding new executives
   When work appears that none of the existing roles fit, there's a clear test for
   adding a new one — so the team scales as the business grows.


■ Who it's for

· You have too many tasks and each one ends up half-finished
· You already use AI, but you instruct it from zero every time and nothing compounds
· You want a "company-like system," even as a team of one
· You want to move a side-project or freelance work from ad-hoc to systematic


■ How to start

1. Unzip the kit into a working folder
2. Open it in Claude or Claude Code
3. Read company.md and the executive prompts first
4. Hand work to the most-jammed role first (often CMO or CSO)

No special tools or paid subscriptions are assumed. Start with a free AI and expand
when you need to.


■ Notes

· This product is a set of templates and prompts. You bring your own AI environment (e.g. Claude).
· Results depend on your own business and operation. No revenue or outcome is guaranteed.
· If the kit is revised, you can re-download the latest version from the same link.
```
