- **テーマ**: 月曜の朝、自分のタスク一覧をAIに渡して並べ替えてもらう小さな習慣

## タイトル
```
月曜朝、AIに「今日の順番」を並べ替えてもらう。
```

## 本文

```
月曜の朝、私はだいたい机に向かってしばらく動けない。
週末の頭がまだ抜けないし、やることが多すぎて、何から手をつければ
いいのか分からなくなる。

去年からずっと続けている習慣がある。
タスクを箇条書きにして、AIに渡して、「並べ替えてください」とだけ言う。

これだけで、月曜朝の動けない時間が、だいぶ短くなった。

---

■ どうやるか

朝、コーヒーを淹れたら、メモ帳に思いついたタスクをそのまま書く。
順番はめちゃくちゃでいい。

「請求書送る、ブログ書く、画像差し替え、A社の返信、洗濯機回す、
親に電話、Vol.2のスライド整理、銀行ATM、晩ごはんの買い物」

これをコピーして、AIにそのまま貼る。
プロンプトは1行だけ。

「これを、今日の効率がいい順に並べ替えてください。理由も短く。」

これだけ。
そうすると、AIは「外出系はまとめて」「集中系は午前」「他人を待つ
返信系は早く投げて待ち時間を有効化」みたいなロジックで
並べ替えて返してくる。

---

■ 「自分で考えない」価値

このやり方の本質は、AIの判断が完璧だから、ではない。
「自分で順番を考えなくていい」ことに、価値がある。

月曜の頭は、判断のスタミナが少ない。
そのスタミナを「並べ替え」みたいな軽い意思決定で使うのは、もったいない。
重要な判断、つまり「Vol.2のスライドをどう構成するか」とか、
「請求金額をどうするか」みたいな、自分にしか決められないことに、
頭のスタミナを温存したい。

並べ替えは、機械的にやれる種類の仕事だ。
だから機械にやらせる。
人間がやるべき仕事だけを、人間が引き受ける。

---

■ ちょっとだけ「気分も整う」

副作用として、並べ替え結果を見るだけで気分も少し整う。

頭の中ではぐちゃぐちゃに見えたタスクが、
箇条書きで番号がついて、見出しで分類されて返ってくる。

それを見ると、「あ、こんなもんか」と思える。
気づくと、リストの1番から手を動かしている。

月曜の朝、5分でこれをやるだけで、
午前のスタートが30分は早くなる。
派手じゃないけど、地味に効いている習慣だ。
```
