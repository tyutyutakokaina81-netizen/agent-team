## Day3（型＝数字・小さな実例）

```
経費のレシート、申告前に床に広げて探す。
毎年それをやってました。

入れるのを「その日のうち」に変えただけで、これが消えた。
変えたのは気合いじゃなくて、入力の軽さ。

#フリーランス #確定申告
```

## Day4（型＝失敗談・自己開示）

```
収支テンプレ、何度ダウンロードして放置したか分かりません。

項目はそろってた。足りなかったのは「毎日入れたくなる軽さ」のほう。
続かない原因、たいてい機能じゃないんですよね。

#フリーランス #働き方
```

## Day5（型＝手順・持ち帰れるTips）

```
お金の管理を続けるコツ、ひとつだけ挙げるなら。

入力項目を減らすこと。
金額とざっくりした分類だけにして、集計とグラフは自動に寄せる。
「あとでまとめて」をなくすと、続きます。

#フリーランス #時間管理 #個人事業
```

## Day6（型＝比喩・視点の提示）

```
忙しさを成果の代わりにしてた時期がありました。

時給という数字を見るようになって、見る場所が「動いた量」から
「1時間あたりの価値」に変わった。同じ1日でも、形が変わります。

#フリーランス #経営
```

## Day7（型＝読者の声・締め）

```
案件別の時給が見えると、「これは割に合わない」を早く判断できる。

値上げも、仕事の取捨選択も、感覚じゃなく根拠でできるようになる。
1週間、数字を見る話を続けてきました。読んでくれてありがとう。

#フリーランス #buildinpublic
```
