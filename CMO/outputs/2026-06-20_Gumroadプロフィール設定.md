# Gumroad プロフィール設定 — そのまま貼れる一式

> 用途：Gumroad の「Profile（プロフィール）」「Settings（設定）」ページの空欄を埋めるための完成稿。
> 入力作業はオーナーが手動で行う（私は外部送信不可）。
> PIIルール遵守：具体的な地区名・町名・住所は書かない（富山県／高岡市レベルまで）。
> 作成：CMO / 2026-06-20

## 3. 自己紹介（Bio）

**短い版（推奨・1〜2文。プロフィール上部用）：**
```
富山発、ひとりで会社のように事業を運営する人。6人のAI役員と一緒に、企画・発信・営業・数字を回しています。その仕組みとテンプレートを置いていきます。
```

**やや長い版（説明欄に余裕がある場合）：**
```
富山でひとり、会社のように事業を運営しています。

「全部自分でやる」をやめて、CDO・CFO・CMO・CPO・CSO・CAO という6人のAI役員に役割を渡す——そんな運営のしかたを実験しながら発信しています。

ここには、その実験から生まれた運営テンプレートやプロンプト集を置いていきます。同じように一人で動く人が、少しでもラクに前へ進めるように。

日々の記録や暮らしのことは note に書いています。
```

**英語版（海外読者向け・併記用）：**
```
Running a one-person business like a company, from Toyama, Japan.
Instead of doing everything myself, I hand each role to one of six AI executives
(CDO/CFO/CMO/CPO/CSO/CAO). Here I share the templates and prompts that came out of
that experiment — for anyone else building solo.
```

## 4. プロフィール画像 / カバー画像

私（AI側）は画像生成不可。オーナーがweb（Canva等）で作成する前提。

**プロフィール画像（アイコン・正方形）：**
- 写真風で統一（オーナー方針）。顔出ししない場合は、無地背景＋イニシャル or シンプルなロゴでも可。
- Canvaで作る場合：濃紺背景に白文字で `Solo CEO OS` の頭文字、またはシンプルな円形ロゴ。

**カバー画像（横長バナー）：**
- 商品カバーと世界観を統一（濃紺〜黒、ミニマル）。
- 中央に `ひとりで、会社のように。` ＋小さく `CDO / CFO / CMO / CPO / CSO / CAO`。
- 詳細指示は `2026-06-20_Gumroad出品コピー_SoloCEO_OS.md` §6 を流用。

## 5. リンク（Social / Links）

プロフィールに載せる外部リンク（お持ちのものだけ・空欄可）：

| 種類 | 入れるURL | 状態 |
|---|---|---|
| note | あなたのnoteプロフィールURL | ⬜要記入 |
| X（旧Twitter） | あればXのURL | ⬜任意 |
| メール登録（任意） | Substack等を作っていればそのURL | ⬜任意 |

> リンクは「集客の入口→Gumroad」「Gumroad→note」の相互送客になる。最低でも note は載せておくと回遊が増える。

## 6. 商品の並び（Featured / Sections）

商品が増えてきたら、プロフィール上での見せ方：

1. **一番上（Featured）**：Solo CEO OS（主力・看板商品）
2. 以降：テンプレート類（Vol.1 フリーランス収支管理スプレッドシート 等）を価格の安い順 or 用途別に。
3. セクション分けできる場合の例：
   - `運営OS・プロンプト`（Solo CEO OS ほか）
   - `テンプレート`（スプレッドシート・カレンダー等）

> 今は商品が少ないので、まずは Solo CEO OS を Featured に置くだけでOK。
