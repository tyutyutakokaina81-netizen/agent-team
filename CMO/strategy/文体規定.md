# 文体規定（人間らしさ・役割分担・裏取り）

> 動画戦略の「2:8の法則」と「人間らしさの追求」を当社の執筆規定に固定する。

---

## 1. 2:8の役割分担
- **人間（オーナー・2割／AIに出せない部分）**
  - 週テーマの決定
  - 一次情報の提供（実体験・失敗談・独自ノウハウ・現地写真）
  - 最終チェック（事実関係・トーン・公開可否）
- **CMO＝AI（8割）**
  - 一次情報をベースに各媒体へ翻訳・展開・形式最適化
  - 記録更新（`_index.md`／`STATE.md`）

## 2. 人間らしさルール
- **AI定型句は禁止語**：
  - 「いかがでしたでしょうか」「させていただきます」の多用
  - 「まとめると」「ポイントは3つです」のテンプレ連発
  - 中身のない総括・無難な締め
- **あえて崩す**：論理を少し飛ばす／変な例え／脱線（ノイズ）を意図的に入れる。
  → 完璧な優等生文章より、ファンがつく人間味を優先。
- **固有名詞と具体**：抽象論を避け、実在の地名・数字・体験で語る（高岡シリーズで実証済み）。

## 3. 裏取り（必須）
- AIが生成した**数字・事例・引用は公的統計／公式リリースで裏付ける**。
- 裏が取れないものは「推測」と明記するか、出さない。
- 機密（顧客名・PII）は出力に混ぜない。センシティブファイルはGit非コミット（`.gitignore`）。

## 4. 事実と意見の分離（company.md準拠）
報告・記事内で断定する際は区別する：
```
【事実】… 【考察】… 【提案】…
```
