# バズ設計ルール（高岡市観光コンテンツ）

## 核心戦略: 「知られていないだけ」の繰り返し

高岡市の価値は本物。足りないのは認知だけ。
すべてのコンテンツはこの1点を軸にする。

## X投稿 設計ルール

### フックの型（最初の1行で止まる）

| 型 | 例 |
|---|---|
| 数字の意外性 | 「日本三大仏のひとつが、無料」 |
| 比較・逆張り | 「京都より高岡に行けばよかった」 |
| 旅費開示 | 「高岡日帰り、全部で¥2,450だった」 |
| 問いかけ | 「高岡大仏が無料なの、知ってた？」 |
| 外国語フック | 「Japan's best kept secret」 |

### 必須ハッシュタグ（用途別）

**日本語投稿（3〜4個）:**
```
#高岡市 #富山観光 #隠れた名所 #旅行好きな人と繋がりたい
```

**英語投稿（3〜4個）:**
```
#HiddenJapan #JapanTravel #TakaokaToyama #JapanTrip
```

**バズ狙い追加タグ:**
```
#穴場 #知らなかった #旅行 #日本三大仏 #国宝
```

### 投稿ルール
- 最初の1行に全てを賭ける（展開ボタンを押させる）
- 箇条書きで読みやすく（スキャン読みに対応）
- URL は本文の末尾（前置きしない）
- 1投稿140文字以内（スレッドで情報量を増やす）

### 週次投稿配分
| 曜日 | 内容 |
|-----|------|
| 月 | 日本語・バズ狙い（比較・数字） |
| 火 | 英語投稿 |
| 水 | グルメ・体験系 |

| 金 | フォロワー向け（役立つ情報） |

## YouTube Shorts 設計ルール

### 3部構成（必須）

```
① フック（0〜3秒）: 黒背景・黄文字・90pt
   → 「え？」と思わせる1行
② 本題（4〜50秒）: 情報を箇条書きで
   → 1スライド1メッセージ
③ CTA（最後3秒）: フォロー・チャンネル登録
```

### フックキーワード（バズ実績あり）
- 「無料で」「知らなかった」「意外と」
- 「日本三大〇〇」「国宝」「¥○○で」
- 「外国人が来ない」「貸し切り」

### Shortsのタイトル規則
```
【知らなかった】+ メインキーワード + サブ説明
例: 【知らなかった】高岡大仏が無料な理由 #高岡市 #Shorts
```

### アップロード最適化
- タイトルに `#Shorts` を含める
- 説明文に全スポットの名前を入れる（SEO）
- 公開直後の1時間が最重要（この間にコメント・いいねが集まる）

## note記事 SEOルール

### タイトルの型

| 型 | 例 |
|---|---|
| 【保存版】+ キーワード | 【保存版】高岡市 完全観光マップ |
| 数字 + 具体的内容 | 高岡市 日帰り旅費¥2,450の全内訳 |
| 逆張り | 京都より高岡に行けばよかった |
| 無料強調 | 日本三大仏が無料で見られる場所 |

### 検索キーワード（記事内に含める）

**高ボリューム:**
- 富山観光, 高岡市観光, 北陸旅行
- 日本三大仏, 高岡大仏
- 瑞龍寺, 国宝

**ロングテール:**
- 高岡市 日帰り 観光
- 富山 穴場 観光スポット
- 外国人 日本 隠れた名所

### 記事構成規則
1. 最初の3行でSEOキーワードを自然に含める
2. 見出しH2に検索キーワードを入れる
3. 実用情報（住所・時間・料金）を表形式で
4. 末尾に関連記事リンク（note内でのPV循環）
