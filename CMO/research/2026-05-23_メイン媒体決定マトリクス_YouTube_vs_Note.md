# メイン媒体決定マトリクス：YouTube vs Note

**作成日**: 2026-05-23
**目的**: AI CMOマルチメディア戦略の「メイン媒体1つ選び」に対する判断材料の整理
**前提**: 最終決定はオーナーが行う（CPOメソッド「人間が握る3判断」の延長）

---

## 結論先出し（推奨）

### 第1推奨：**Note** ＋ YouTube は3ヶ月後に追加検討

**理由**：
1. 既にVol.1販売実績（フリーランス収支管理スプレッドシート ¥980）が note にある
2. オーナーの **作業時間制約（1日30分目安）** とYouTubeの動画編集負荷がミスマッチ
3. テンプレート販売（柱C）との **資産連動性** が note の方が高い
4. SEO評価が長期に効くため、3ヶ月で12本の note 記事は「資産として」蓄積される

### 別案：「音声 + 文字起こし」のハイブリッド

スマホ録音→Whisper文字起こし→Claude整形→noteのフロー。
動画編集を回避しつつ「音声系の1次情報密度」のメリットだけ取れる。

## 評価軸と採点（5点満点）

| 軸 | YouTube | Note | 重み |
|---|--------|------|------|
| 1次情報の出やすさ | 5（話せば良い） | 3（書く負荷あり） | 高 |
| 制作時間（オーナー） | 2（撮影＋編集2h以上） | 4（執筆1h想定） | 高 |
| 派生展開しやすさ | 5（音声→文字起こし→各媒体） | 3（文字→各媒体） | 高 |
| 既存資産との連動 | 1（ゼロから） | 5（Vol.1実績あり） | 中 |
| SEO・長期資産化 | 3（検索流入は限定的） | 5（Google検索対象） | 高 |
| 月30万自動化との連動 | 3（柱A/B/Cへの導線弱） | 4（柱C＝テンプレ販売と直結） | 中 |
| 学習コスト | 2（サムネ・編集・SEO） | 4（執筆スキルがあれば即可） | 中 |
| **合計（単純加算）** | **21/35** | **28/35** | |

> 重みは「高=2倍／中=1倍」で再採点すると：
> - YouTube：5×2 + 2×2 + 5×2 + 1 + 3×2 + 3 + 2 = **31**
> - Note：3×2 + 4×2 + 3×2 + 5 + 5×2 + 4 + 4 = **43**
> Note が明確に優位。

## シナリオ別の推奨

### シナリオA：執筆が苦にならない／既存note資産を活かしたい

→ **Note 一択**。週1記事 × 12週で、3ヶ月後に12本の長期資産。

### シナリオB：話す方が圧倒的に楽／喋り続けられる

→ **音声+note ハイブリッド**：
- スマホ録音（15分）
- Whisperで文字起こし
- Claude Code で note 記事化（3,000字）
- X用に3投稿派生

**メリット**：YouTubeの動画編集を回避しつつ「音声系の1次情報濃度」を得る。

### シナリオC：絶対にYouTubeで顔出ししたい／影響力構築を最優先

→ **YouTube**。但し以下を覚悟：
- 動画編集に週3〜4時間
- 最初の3ヶ月は登録者ほぼ伸びない（チャンネル評価期間）
- サムネイル作成スキルが追加で必要

---

## 各媒体に「決まったら」即やること

### Note を選ぶ場合

1. `CMO/outputs/Note/` ディレクトリ作成
2. 週テーマ運用カレンダーを `projects/` に作成
3. note のプロフィール・ヘッダー更新（自社のサービス導線を明示）
4. **3ヶ月分（12週）のテーマ草案** を1日で出す（オーナーの一次情報1日分を投入）
5. 第1週目の記事を **必ず本日中に1本下書きする** （動かないことが最大の失敗）

### YouTube を選ぶ場合

1. `CMO/outputs/YouTube/` ディレクトリ作成
2. 撮影機材（最低スマホ三脚＋マイク）と編集ツール決定
3. サムネテンプレ作成（Canvaで5パターン）
4. **チャンネル開設＆初回動画を3日以内に出す**
5. 出さなければ意思決定の意味がない

### ハイブリッドを選ぶ場合

1. Whisper APIの利用環境整備（CDO）
2. 録音 → 文字起こし → note公開の3工程パイプラインを `/multimedia` コマンド化
3. 初回録音を **本日中に実施** （15分でいい）
