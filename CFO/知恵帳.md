# CFO 知恵帳

> 毎日1つ、財務・契約・事務・税務に関する学びを残す。短くてよい。

## 運用ルール
- 1日1件、日付プレフィックス付きで追記
- 数字・出典は可能な範囲で明示する
- 個社・個人を特定する情報はここに書かない（`research/` 側で管理）

## エントリー

### 2026-05-13
- **学び**: 請求書作成は AI で月3万円→0円・納期3日→30秒に短縮可能。インボイス番号・振込先・請求書番号の自動採番までカバーできる。
- **背景**: 杉田氏の動画事例。Claude Code で会社情報・取引先・単価感を学習済みにし、口頭指示で PDF を生成。
- **示唆**: 事務代行に月3万円払っているのは見直し余地が大きい。当社の CFO 業務（請求書発行）をプロトタイプとして AI 化候補に挙げる。
