## ⏰ 提出期限

### 開業届
- **開業日から1ヶ月以内**
- 過ぎても罰則はないが、早めに出すこと推奨

### 青色申告承認申請書
- **開業日から2ヶ月以内**、または **申告したい年の3月15日まで**
- 過ぎたら来年分からしか青色申告できない
- **← これが重要。今日出さないと、来年の確定申告で損する**

## 🚀 最速ルート：e-Taxでオンライン提出（推奨・所要15分）

### 必要なもの
- マイナンバーカード
- スマホ（ICカードリーダー代わり）
- パソコン

### 手順

```
1. https://www.e-tax.nta.go.jp/ にアクセス
2. 「e-Taxソフト（WEB版）」を選択
3. マイナンバーカードでログイン
4. 「提出書類の作成」→「個人事業の開業・廃業等届出書」
5. フォームに入力（下記参照）
6. 電子署名→送信
7. 同様に「青色申告承認申請書」も作成→送信
```

### 入力内容（下書き）

#### 開業届
```
納税地：自宅の住所（または事業所）
氏名：本名
生年月日：YYYY年MM月DD日
マイナンバー：12桁の番号
職業：個人事業主 / ライター / コンサルタント 等
屋号：なくてもOK（後で付けてもよい）

届出の区分：開業
開業日：YYYY年MM月DD日（今日でOK）
事業の概要：Webライティング / データ入力 / コンサルティング等

開業・廃業に伴う届出書の提出の有無：
 「青色申告承認申請書」→ 有
 「消費税に関する届出書」→ 無

事業所等を新増設、移転、廃止した場合：（該当なければ記入不要）
廃業の事由：（該当なし）
```

#### 青色申告承認申請書
```
納税地：開業届と同じ
氏名：開業届と同じ
職業：開業届と同じ
屋号：開業届と同じ

平成・令和YYYY年分以後の所得税の申告は、
青色申告書によりたいので申請します。

1. 事業所又は所得の起因となる資産の名称及びその所在地
   名称：自宅 / 所在地：自宅住所

2. 所得の種類：事業所得

3. いままでに青色申告承認の取消しを受けたこと：無

4. 本年1月16日以後新たに業務を開始した場合、その開始した年月日
   → 開業日を記入

5. 相続による事業承継の有無：無

6. その他参考事項：
   (1) 簿記方式：複式簿記 ← 重要！「簡易簿記」だと控除が10万円になる
   (2) 備付帳簿名：総勘定元帳、仕訳帳、現金出納帳、売掛帳、買掛帳、
                  経費帳、固定資産台帳
```

### 必要なもの
- プリンター
- 切手（84円×2枚）
- 印鑑（認印でOK）

### 手順

```
1. 国税庁サイトから書式をダウンロード

   青色申告：https://www.nta.go.jp/taxes/tetsuzuki/shinsei/annai/shotoku/annai/07.htm

2. PDFをダウンロード→印刷

3. 手書きで記入（上記の入力内容参照）

4. 税務署に郵送
   管轄の税務署を調べる：https://www.nta.go.jp/about/organization/access/map.htm
```

**→ 控えも必要：2部印刷して、1部に返信用封筒を同封→控えが戻ってくる**

## 💰 青色申告のメリット（なぜやるべきか）

### メリット1：最大65万円の控除
```
収入：360万円（月30万×12ヶ月）
経費：72万円
青色申告特別控除：65万円 ← これが使える
   ─────────────
課税所得：223万円

vs 白色申告の場合：
課税所得：288万円（青色控除なし）

差額：65万円
→ 税金で約20万円お得（所得税+住民税の節税効果）
```

### メリット2：赤字を3年繰越可能
- 初年度が赤字でも、翌年の黒字から差し引ける
- **フリーランス初年度は赤字が多い → これが効く**

### メリット3：家族への給与が経費になる
- 配偶者や家族に業務を手伝ってもらう→給与を経費計上
- 世帯全体の節税になる

### メリット4：30万円未満の資産を一括経費化
- PC・机・椅子などを買ったらすぐ経費に
- 通常は減価償却が必要

## ⚠️ 注意事項

### 複式簿記 vs 簡易簿記
- **複式簿記**：65万円控除（e-Tax電子申告）・55万円控除（紙）
- **簡易簿記**：10万円控除のみ

**→ 必ず複式簿記を選ぶ。freeeやマネーフォワード使えば難しくない。**

### 事業所得 vs 雑所得
- **事業所得**：青色申告OK・各種控除OK
- **雑所得**：青色申告不可・控除も少ない

**→ 継続的な事業として行うなら「事業所得」で申告**
**→ 2022年の改正で、一時的な副業は「雑所得」扱いになることも**
**→ 月10万以上の継続収入があれば、堂々と事業所得でOK**

## 📚 提出後にやること

### Step 1：事業用口座の開設
```
□ 事業用のゆうちょ銀行口座を作る（屋号口座）
  またはネット銀行（GMOあおぞら・住信SBIなど）
  → プライベート資金と分けるため
```

### Step 2：会計ソフト契約
```
□ freee / マネーフォワード / やよい青色申告 から選択
  → 月1,000〜1,500円程度
  → 1月無料お試しあり
```

### Step 3：国民健康保険・国民年金の手続き
```
□ 会社を辞めた場合は、国民健康保険に加入
□ 国民年金に切り替え
→ 市区町村役場の窓口で手続き
```

### Step 4：小規模企業共済に加入（節税対策）
```
□ 月額1,000〜70,000円を積立
→ 全額所得控除（最大年84万円）
→ 退職時に退職金として受取
```

## 💡 よくある質問

### Q: 屋号は必要？
A: なくてもOK。後から付けられる。付けるなら業務内容がわかる名前が理想。

### Q: 会社員でも開業届は出せる？
A: 出せる。副業として個人事業主を兼務できる。

### Q: マイナンバーは必須？
A: 必須。持ってない場合は通知カードでもOK（ただし不便）。

### Q: 税理士は必要？
A: 月30万程度なら不要。freee/マネーフォワードで十分。

### Q: 印鑑は必要？
A: 2021年以降、押印不要になった。
