# 経費管理ルール（2026-04-13）

フリーランス・個人事業主が確定申告で楽をするための経費管理ルール。

---

## 💰 経費にできるもの・できないもの

### 経費にできる（フリーランス・在宅ワーカーの場合）

| 科目 | 具体例 | 家事按分 |
|------|--------|---------|
| 通信費 | Wi-Fi・スマホ代 | 50〜80%（仕事用途） |
| 地代家賃 | 自宅家賃 | 20〜30%（仕事スペース分） |
| 水道光熱費 | 電気代 | 20〜30% |
| ソフトウェア | Claude Pro・Canva Pro等 | 100% |
| 書籍・研修 | 業務関連の本・講座 | 100% |
| 消耗品費 | 文房具・プリンタインク | 100% |
| 旅費交通費 | 取材・打合せの交通費 | 100% |
| 接待交際費 | クライアントとの打合せ | 100% |
| 新聞図書費 | 業務関連の新聞・雑誌 | 100% |
| 会議費 | オンラインミーティングツール | 100% |
| 広告宣伝費 | SNS広告・名刺・ポートフォリオ | 100% |
| 支払手数料 | プラットフォーム手数料・振込手数料 | 100% |

### 経費にできない

- 所得税・住民税・国民健康保険料（**所得控除**で対応）
- 個人的な飲食・娯楽
- スーツなど私用でも使う衣服
- 健康診断・医療費（**医療費控除**で対応）

## 📊 月次で記録する経費スプレッドシート

### 列構成（既存のVol.1収支管理と同じ形式）

```
A: 日付
B: 科目
C: 金額
D: 支払方法（現金/クレカ/振込）
E: 摘要（何に使ったか）
F: レシートあり（チェックボックス）
G: 家事按分率（%）
H: 経費計上額（自動計算：C列×G列/100）
```

### 入力例

```
2026/04/15  通信費    6,000  クレカ  スマホ代4月分  ✓  70  4,200
2026/04/20  ソフトウェア  3,000  クレカ  Claude Pro 4月分  ✓  100  3,000
2026/04/25  書籍・研修  2,200  現金  ライティング本  ✓  100  2,200
```

## 📅 記録のタイミング

### 基本ルール
- **使ったその日に記録**（溜めると忘れる）
- **領収書・レシートは必ず保管**（7年間）
- **クレカ明細・銀行明細は毎月確認**

### 楽にするコツ
- 業務専用のクレカを作る（freee・マネーフォワード連携）
- 事業用銀行口座を分ける
- 家計簿アプリと経費アプリを使い分け

## 🧾 領収書・レシートの管理

### 保管期間
- 青色申告：7年
- 白色申告：5年

### 保管方法
```
□ 紙の領収書：月ごとに封筒にまとめる
□ 電子レシート：Googleドライブに月フォルダで保管
□ PDFで受領：領収書_YYYYMM_項目.pdfの形式で保管
```

### 電子帳簿保存法対応
- 2024年1月から電子取引のデータ保存が義務化
- 電子メール添付のPDFレシートは**PDFで保管が必須**
- 印刷して紙だけで保管はNG

## 💡 節税になる経費計上のコツ

### コツ1：忘れがちな経費を漏れなく計上

**意外と見落とす経費**
- SNSの有料プラン（Twitter Blue等）
- 有料ニュースレター・note定期購読
- クラウドストレージ（Google One等）
- 業務用のBGMサブスク
- 業務用アプリ（Notion有料・Raycast等）
- 振込手数料（クライアントから自分への入金時）
- クラウドソーシング手数料（報酬から引かれる分）

### コツ2：家事按分をきっちりする

**家事按分とは：** 家事（私用）と事業で共用しているものを、事業割合で経費化する考え方。

**家事按分の合理的な基準**
- 家賃：仕事スペースの面積÷総面積
- 通信費：仕事で使う時間÷総時間
- 電気代：仕事で使う時間÷総時間

**記録例**
```
自宅家賃 10万円/月
仕事部屋 5畳 / 全体 20畳 = 25%
→ 経費計上：2.5万円/月
```

### コツ3：小規模企業共済に加入する

- 月1,000〜7万円を積立（全額所得控除）
- 最大年84万円の節税
- 廃業時に退職金として受取
- フリーランスのセーフティネット

### コツ4：ふるさと納税を活用

- 所得税・住民税の節税
- 返礼品で実質食費を浮かせる
- 年間上限は所得による（目安：年収500万なら6万円程度）

## 📆 確定申告スケジュール

### 1年間の流れ

```
1月〜12月：帳簿を毎月つける（月末までに〆る）
1月：源泉徴収票の確認
2月：確定申告書作成
2月16日〜3月15日：確定申告書提出
3月〜4月：所得税の納付
5月〜6月：住民税の通知が来る
```

### 青色申告のメリット
- **最大65万円の青色申告特別控除**
- 赤字を3年繰越可能
- 家族への給与が経費になる（専従者給与）

### 青色申告するには
- 開業届と青色申告承認申請書を税務署に提出
- 開業から2ヶ月以内に提出（過ぎたら翌年から）
- e-Taxで電子申告すると65万円控除（紙は55万円）

## 🎯 月30万達成時の税務シミュレーション

### 年収360万円（月30万×12ヶ月）の場合

```
売上：3,600,000円
経費：720,000円（20%と仮定）
青色申告特別控除：650,000円
   ─────────────
課税所得：2,230,000円

所得税（5%）：  111,500円
住民税（10%）： 223,000円
国民健康保険：  約200,000円
国民年金：      約200,000円
   ─────────────
税・保険料合計：約734,500円

手取り：約2,145,500円（約179,000円/月）
```

### 節税対策後（小規模企業共済月7万円加入）

```
小規模共済控除：840,000円
   ─────────────
課税所得：1,390,000円（実質）

税金合計：約267,000円（約60万円の節税効果）
小規模共済積立：840,000円（将来の退職金として戻る）

→ 実質手取り：約2,493,000円（約207,000円/月）
```
