**用途**：年間収益が ¥48万円超（または副業収益が ¥20万円超）になる前提での確定申告準備。
**対象期間**：2026年1月1日〜2026年12月31日（2027年2〜3月に申告）
**前提**：青色申告（65万円控除）を最初から目指す
**費用**：¥0（既存ツール内で完結。会計ソフト導入は任意）

# 1. 開業届の提出

## 必要なもの

```
□ 個人事業の開業・廃業等届出書（国税庁HPで無料DL）
□ マイナンバーカードまたは通知カード+本人確認書類
□ 印鑑（認印でOK）
```

## 提出方法（3つ・いずれも費用ゼロ）

| 方法 | 費用 | 所要時間 | 推奨度 |
|------|------|---------|------|
| e-Tax（電子申請） | ¥0 | 30分 | 🥇 推奨 |
| 税務署へ郵送 | 切手代のみ | 60分 | 🥈 |
| 税務署へ持参 | ¥0 | 60〜90分 | △ |

## 記入のポイント

```
■ 屋号
　なくてもOK。あれば「○○ライティング」等
　屋号付き口座を作るなら必須

■ 事業の概要
　「インターネット上での記事執筆・コンテンツ制作・テンプレート販売」
　など複数業務を含めて記載してOK

■ 事業所
　自宅住所でOK（個人事業主の標準）

■ 開業日
　遡って記載可能（最大1ヶ月前まで）。本プロジェクトは 2026-04-08
```

# 2. 青色申告承認申請書の提出

## なぜ重要か

| 申告方法 | 控除額 | 年間税額への影響（年収300万） |
|---------|--------|--------------------------|
| 白色申告 | なし | 約60万円の所得税・住民税 |
| 青色申告（10万） | 10万円 | 約2万円節税 |
| **青色申告（65万）** | **65万円** | **約13万円節税** |

→ 提出するだけで **年間13万円の節税効果**。やらない理由がない。

## 提出期限

- 事業開始から **2ヶ月以内**
- 本プロジェクトの場合：**2026-06-08まで**

## 65万円控除の要件

```
□ 複式簿記での記帳
□ 貸借対照表・損益計算書の提出
□ e-Tax での電子申告 または 電子帳簿保存
```

→ 会計ソフト（freee/マネーフォワード/弥生）を使えば自動対応可能。
→ または自前のスプレッドシート（monthly_financial_summary.md 参照）でも可能だが、複式簿記のため難易度が上がる。

## 推奨：会計ソフト導入は「年間売上 ¥120K 超えたら検討」

| 月額 | 機能 | 推奨タイミング |
|------|------|--------------|
| freee（スターター） | ¥1,180/月 | 月10万円超えてから |
| マネーフォワード（パーソナル） | ¥1,408/月 | 同上 |
| 弥生青色申告オンライン（セルフ） | ¥1,100/月 | 同上 |

→ 売上が出るまでは **自前のスプレッドシート** で帳簿管理。出始めたら会計ソフトに移行。

# 3. 事業用銀行口座の開設

## なぜ必要か

- プライベート口座と混在すると経費の按分が困難
- 確定申告時の入出金確認が圧倒的に楽になる
- インボイス番号取得時にもスムーズ

## おすすめ口座（手数料無料・月額無料）

| 銀行 | 月額 | 振込手数料 | 開設時間 |
|------|------|----------|---------|
| 住信SBIネット銀行 | ¥0 | 月5回まで無料 | 1〜2週間 |
| 楽天銀行 | ¥0 | 一定条件で無料 | 1〜2週間 |
| GMOあおぞらネット銀行 | ¥0 | 個人事業主専用「法人プラン」あり | 1〜2週間 |

→ いずれも個人事業主の屋号付き口座が無料で作れる（屋号があれば）。

# 4. 月次帳簿のスタート

## 最低限の記帳項目

各月、下記4種類のデータを蓄積する：

```
1. 売上（請求日と入金日を分ける）
2. 経費（科目別）
3. プラットフォーム手数料
4. 銀行残高
```

→ 詳細は `CFO/outputs/monthly_financial_summary.md` の構造を参照。

## 月次5分ルーティン

```
毎月末日：
1. 通帳記入 or オンライン明細を確認（1分）
2. 当月の売上をスプレッドシートに転記（2分）
3. 経費の領収書を月別フォルダに振り分け（2分）

→ 累計月60分で年間の確定申告準備が完了する
```

# 5. 領収書の保管ルール

## 保管期間

- **青色申告：7年間**（一部書類は10年間）
- **白色申告：5年間**

## 保管方法（電子帳簿保存法 対応）

```
■ 物理レシート
  □ 月別フォルダ（紙）に保管
  □ ファイル名：YYYY-MM_領収書.zip 等

■ 電子レシート（Amazon、各種SaaSの請求書PDF等）
  □ Google Drive または Microsoft OneDrive に保存
  □ ファイル名：YYYY-MM-DD_店舗名_金額.pdf
  □ フォルダ：受信した月のフォルダに振り分け
```

## 経費にできるもの・できないもの（再掲）

### 経費にできる代表例

```
✅ 通信費：仕事用スマホ・プロバイダ・ドメイン代（家事按分対応）
✅ 交通費：取引先への移動・出張費
✅ 消耗品費：文具・10万円未満のPC周辺機器
✅ 新聞図書費：業務関連の書籍・有料記事
✅ 支払手数料：振込手数料・決済サービス料
✅ ソフトウェア費：Claude Pro・Canva Pro・Microsoft 365
✅ 接待交際費：取引先との会食・お中元お歳暮
✅ 地代家賃：自宅兼事務所の家事按分
```

### 経費にできない例

```
❌ 個人的な食費・日用品
❌ プライベートの旅行・娯楽費
❌ 家族へのプレゼント
❌ 健康診断・人間ドック（業務無関係なもの）
```

# 6. インボイス制度への対応判断

## 結論：年間売上1,000万円超になったら登録

| 売上規模 | 登録判断 | 理由 |
|---------|---------|------|
| 〜1,000万円 | 様子見 | 免税事業者でOK・取引先の状況による |
| 1,000万円超 | 登録必須 | 課税事業者になるため |

## 免税事業者でも登録した方がいいケース

```
□ 取引先がほぼ法人（インボイスがないと先方が損する）
□ クラウドワークス・ランサーズの一部案件で「適格請求書発行事業者のみ」指定がある
□ 継続契約のクライアントが多い（先方の経理を考慮）
```

# 9. 確定申告で慌てないための3つの習慣

（既に Vol.1 集客記事で詳細解説済み。CFO 視点での再掲）

```
1. 「請求日」と「入金日」を分けて記録する
   → 発生主義での売上計上が確定申告で必須

2. 経費の科目を入力時に決める
   → 年末まとめ作業が10分の1になる

3. 不明点は税務署に電話で聞く
   → 無料・確実・1月中なら空いている
```
