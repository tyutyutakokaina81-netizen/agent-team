# 月次財務サマリーテンプレート

**用途**：毎月初に前月の事業数字を集計し、翌月の意思決定に使う。
**運用**：Google スプレッドシート（無料）に下記の構造を作る。月初3営業日以内に更新する。
**対象柱**：A SEOライティング ＋ C テンプレ販売（柱B SNS運用代行は対象外）
**費用**：¥0（既存ツール内で完結）

---

# シート構成

```
スプレッドシート名：2026_月次財務サマリー

├─ シート1: Dashboard（全体ダッシュボード）
├─ シート2: A_Writing（柱A 詳細）
├─ シート3: C_Templates（柱C 詳細）
├─ シート4: Expenses（経費明細）
└─ シート5: Cashflow（キャッシュフロー）
```

# シート1：Dashboard（全体ダッシュボード）

# シート2：A_Writing（柱A 詳細）

## 列構成

| 列 | 項目 | 入力 |
|----|------|------|
| A | 年月 | 2026-05 |
| B | 受注本数 | A_ライティング/templates/application_tracker から転記 |

| 時給換算 | 判断 |
|---------|------|
| ¥3,000 以上 | このまま維持・拡大 |
| ¥2,000〜3,000 | 単価交渉 or 効率化検討 |
| ¥2,000 未満 | 案件内容を見直す（評価ルール強化） |

# 月初ルーティン（30分）

```
1. 各プラットフォームの売上明細をエクスポート（5分）
   - クラウドワークス → 売上管理 → CSV
   - note → クリエイターダッシュボード → 売上一覧
   - BOOTH → 売上明細

2. シート4（Expenses）にツール費等を記入（5分）

3. シート5（Cashflow）の入金予定/実績を更新（5分）

4. シート2・3（柱別詳細）に数字を転記（10分）

5. シート1（Dashboard）が自動更新されるのを確認（5分）

6. 結果を CSO・CDO へ Slack/LINE で共有
   - 売上達成率
   - 柱別の改善メモ
```

# データ保存ルール

- **Git にコミットしない**：本シートには金額・取引先名が含まれるため `.gitignore` 対象
- **バックアップ**：月初に CSV エクスポートを `CFO/research/_revenue_data/`（gitignore対象）に保管
- **共有範囲**：自分のみ。万一クライアントに見せる場合は別途集計版を作成
