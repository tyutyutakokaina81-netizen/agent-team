## ステップ1：開業届・青色申告承認申請（最優先）

### 必要書類
- [ ] 個人事業の開業・廃業等届出書（開業届）
- [ ] 所得税の青色申告承認申請書
- [ ] 青色事業専従者給与に関する届出書（家族雇用する場合のみ）

### 期限
- 開業届：開業から1ヶ月以内（実質的な遅れはOKだが早めに）
- 青色申告承認申請：**開業から2ヶ月以内 or 適用したい年の3月15日まで**

### 提出方法
- e-Tax（マイナンバーカード必須）
- 所轄税務署へ郵送
- 所轄税務署へ持参

### freee や マネーフォワード で作成可（無料）

## ステップ2：日々の記帳（複式簿記）

### 青色申告65万円控除の要件
1. 複式簿記による記帳
2. 貸借対照表＋損益計算書の作成
3. e-Taxまたは電子帳簿保存

→ 控除を最大化するため freee / マネーフォワード / 弥生 等のクラウド会計ソフトを利用する（年¥10,000程度・経費計上可）。

### 推奨ソフト比較

| ソフト | 月額 | 特徴 |
|--------|------|------|
| freee | ¥1,180〜 | 初心者向け・銀行連携自動 |
| マネーフォワードクラウド | ¥1,408〜 | 操作中級・連携サービス豊富 |
| 弥生青色申告オンライン | ¥980〜（初年度無料） | 老舗・操作シンプル |

### 月次タスク（月初30分）

- [ ] 銀行口座の入出金を会計ソフトに取り込み

## ステップ5：申告・納税（2/16〜3/15）

### 提出方法
- **e-Tax（推奨）**：マイナンバーカード＋ICカードリーダー or スマホ
  - 65万円控除の要件
  - 24時間提出可能
  - 還付金が早い
- **郵送**：消印有効
- **税務署窓口**：混雑

### 納税方法
- 口座振替（4月下旬引き落とし・延滞利息なし・推奨）
- e-Taxからクレジットカード払い
- 銀行・税務署窓口で現金納付

## 想定納税額シミュレーション（2026年分）

### 前提
- 年間売上：¥3,600,000（月30万×12ヶ月）※達成シナリオ
- 経費：¥360,000（月¥30,000）※ツール代＋按分等
- 所得：¥3,240,000
- 青色申告特別控除：¥650,000
- 基礎控除：¥480,000
- 国民健康保険・国民年金：約¥400,000

### 課税所得目安
```
¥3,240,000 - ¥650,000 - ¥480,000 - ¥400,000 = ¥1,710,000
```

### 所得税（推定）
```
¥1,710,000 × 5% = ¥85,500（復興税込で約 ¥87,300）
```

### 住民税（推定）
```
¥1,710,000 × 10% = ¥171,000
```

### 国民健康保険＋国民年金
```
約 ¥400,000／年（自治体により異なる）
```

### 合計負担
```
所得税 ¥87,300 + 住民税 ¥171,000 + 社保 ¥400,000 = 約 ¥658,000／年
```

→ 月収¥300K達成しても、手取りベースで月¥245K程度。納税は月¥55K相当を毎月積立てる感覚で。

| iDeCo | 個人型確定拠出年金 | ¥816,000 |

## やってはいけないこと

- [ ] 売上の一部を申告しない（脱税）
- [ ] プライベートな支出を経費計上する（事業按分なし）
- [ ] 領収書がない経費を計上する（後で証明できない）
- [ ] 確定申告期限を過ぎる（無申告加算税15〜20%）
- [ ] 納税期限を過ぎる（延滞税）
