# オーナー指示：富山初 × 事業採算性 の日々調査＆チャレンジ

**記録日**: 2026-04-11
**起点**: オーナー直接指示（本リポジトリでの会話中）

## 指示の原文

> 事業採算性と富山初みたいなキーワードで日々調査してチャレンジして

## 解釈と戦略的意味

この一言は、自立型AIの "日々やるべきこと" を方向づける **standing directive（常駐指示）** として扱う。

### キーワード1: 事業採算性

- Claude API 費用対効果の継続的モニタリング
- 各チャレンジ案件の損益分岐試算
- 既存柱（A/B/C/D/E）の採算性再評価
- 新規アイデアの ROI 試算（時間／金銭／リスクの3軸）
- Claude API 価格・料金体系の変動追跡

### キーワード2: 富山初

- **オーナーの所在地は富山県** である（← この一文で判明）
- 「富山県内初」というポジショニングは：
  - **ニュース価値**：地方紙（北日本新聞・富山新聞）、地方局（KNB・チューリップ・BBT・TUT）に取り上げられる可能性
  - **希少性**：競合が極めて少ない（都市圏に比べ AI 専業事業者がまだ少数）
  - **補助金親和性**：富山県・北陸先端科学技術推進機構・各種公的支援との相性
  - **地域密着型の信頼性**：地元事業者への営業で「富山の会社」という肩書が効く
- 想定される "富山県内初" 候補:
  - 富山県内初の AI 完全自律運営企業
  - 富山県内初の Claude Code 常駐型業務改善提案サービス
  - 富山県内初の AI 議事録／経理／SNS 運用 BPO
  - 富山県内初の地方飲食店向け AI 口コミ返信代行（既に柱E で素材あり）
  - 富山県内初の AI 生成 × 職人・伝統産業支援（高岡銅器・井波彫刻・越中和紙 等）
  - 富山県内初の AI によるシニア向け生活支援情報自動生成サービス

### キーワード3: 日々調査してチャレンジ

- 単発の調査ではなく **毎日ループに組み込む** 宿題
- 調査 → 仮説 → 最小チャレンジ → 記録 → 次の仮説 のサイクル
- 失敗も資産として記録（`research/` に蓄積）

## 自立型AIループへの組み込み方

各役職の日次ターンで、以下を **最優先で検討** するよう prompt を強化する：

| 役職 | このキーワードでやること |
|------|-------------------------|
| CDO | 自立型AI基盤そのものが「富山県内初」になる設計・技術検証 |
| CFO | 各施策の採算性試算（Claude API 費用対効果・BEP 計算） |
| CMO | 富山県内メディア・広報ルートのリサーチ、地方発コンテンツ戦略 |
| CPO | 富山県特化プロダクト（地場産業・観光・福祉 × AI）の企画 |
| CSO | 富山県内の潜在顧客リサーチ、地場 SMB リスト化、補助金情報収集 |
