# COWORK_HANDOFF — リモートAI ⇄ Cowork 連携ボード

> **これが連携の唯一の窓口。** リモートAI（403でネット遮断）とCowork（オーナーMac・ネット有効）は
> 直接は通信できない。**共有できるのはこのGitリポジトリだけ**。だからやり取りはこの1ファイルに集約する。
>
> **使い方**：
> - リモートAIが「Coworkへの依頼」を書く → push
> - Cowork起動時に `git pull` → このファイルを読む → タスク実行 → **結果欄に記入して commit & push**
> - 次にリモートAIが `git pull` して続きを進める
> - ⭐ Coworkに「起動時：pullしてこのファイルを最初に読む」を標準指示にすると連携が自動化される
