# オーナー専用：KDP初期設定 30分チェックリスト（黒字の最後の壁）

> これはAIが代行できない唯一の作業＝**あなたの本人確認・振込口座・税情報**の登録。
> 一度きり・無料。これが済めば、完成済みの3冊(＋英語1冊)を worker が自動でAmazonに出品します。
> 原稿・表紙仕様・リスティング（タイトル/説明/キーワード/価格¥680）は**全て作成済**。あなたは"アカウントを作るだけ"。

## 事前に手元に用意するもの（5分）
- [ ] 本人確認：氏名・住所（Amazonアカウントの登録情報でOK）
- [ ] **振込先の銀行口座**（普通預金・支店名/口座番号）※ロイヤリティの入金先
- [ ] **マイナンバー**（税務情報＝源泉徴収の登録に必要。日本居住者は必須）
- [ ] メールアドレス（既存のAmazonアカウントでログインでOK＝新規作成不要な場合あり）

## 手順（約25分）
1. [ ] https://kdp.amazon.co.jp を開く → 既存のAmazonアカウントで**サインイン**（無ければ無料で作成）
2. [ ] 初回のみ「**アカウント情報**」の登録を求められる：
   - [ ] 著者/出版社名（ペンネーム可。当社の note 名義と揃えたいなら統一）
   - [ ] 住所・電話番号
3. [ ] 「**支払いの受け取り方法**」→ 上の銀行口座を登録
4. [ ] 「**税に関する情報**」→ 案内に沿ってマイナンバー等を入力（米国税の質問はガイドに従えばOK＝日本居住者は源泉軽減の手続き）
5. [ ] ここまでで**アカウント設定は完了**。本の登録は worker が自動で行うので、あなたはここで止めてOK。
6. [ ] （任意）そのブラウザ（KDPログイン済み）を worker が使うMacで開いたままにしておくと、次の便が即アップロードに進めます。

## 済んだら
- [ ] このセッションに「**KDP設定した**」と一言送るだけ。
- → 次の worker 便が 3冊（AI副業入門／AI時短仕事術／フリーランス確定申告入門）を提出まで完遂し、ASIN/管理URLを報告します。
- → 英語の富山本（18,500語）は第2弾として続けて出せます。

## 安全（重要）
- worker（AI）は**あなたのパスワード・2FA・マイナンバー・口座番号を絶対に入力しません**。上記の認証系はすべて**あなた自身が**入力してください。
- AIが代行しないのは手抜きでなく、なりすまし・規約違反・あなたのお金と責任を守るためです。

## これで何が変わるか
- Amazonの購買トラフィックに載る＝当社唯一の弱点「観客ゼロ」を回避。
- 黒字の目安が **3〜6ヶ月 → 8月末〜9月（4〜8週）** に短縮。差は「原稿を作る時間」ではなく「アカウントがあるか」だけ（原稿はもう完成済）。
