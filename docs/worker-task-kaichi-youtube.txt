【単発タスク】かいちのAI大学 YouTube を文字化して取り込む（owner「全部読み取り」）
あなたは cowork（ブラウザ操作可）。codeはA1でYouTube取得不可のため、あなたが代行する。

手順:
1. https://www.youtube.com/@kaichi_gpt/videos を開く。
2. 公開日が新しい順に動画を開き、各動画で「…もっと見る」→「文字起こしを表示(Show transcript)」を開く。
   ※字幕が無い動画はスキップ。
3. 各動画の文字起こし全文を、次の形式で保存:
   CAO/research/kaichi_youtube/<公開日YYYY-MM-DD>_<動画タイトルの安全化>.md
   先頭に: # タイトル / URL / 公開日 / （本文=文字起こし全文）
4. **まず最新20本**を取得（多すぎるので初回は20本上限）。取得できた本数と一覧を報告。
5. git add CAO/research/kaichi_youtube/ && git commit -m "cowork: かいちYouTube文字起こし取り込み(最新N本)" && git push origin main
6. 報告を ops/outbox/ へ（取得本数・スキップ本数・字幕なしの動画数）。
制約: 転載は"自社の学習・分析目的の内部資料"に留める（公開・再配布しない＝著作権配慮）。ブラウザは最後に閉じる。
