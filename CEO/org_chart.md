## 組織図

```
オーナー（意思決定・最終承認）
    │
    ▼
 CEO（統括マネジャー）
    │
    ├─── CRO（トレンド調査部隊）
    │         毎日のトレンド監視・競合分析・ネタ提供
    │         ↓ ネタ供給
    │         ├→ CMO（コンテンツ化）
    │         ├→ CSO（案件ジャンル・単価相場）
    │         └→ CPO（商品化ネタ）
    │
    ├─── CMO（マーケティング・コンテンツ企画）
    │         note記事・X投稿・YouTube台本の企画と生成
    │
    ├─── CPO（プロダクト・立案）
    │         テンプレ・教材・サービスの企画・制作
    │
    ├─── CSO（営業専任）
    │         案件発掘・応募・提案・成約まで
    │         成約後 → COOに引き渡す
    │
    ├─── COO（運用・総務専任）
    │         受注案件の実行・納品・品質管理
    │         システム・スケジュール監視
    │
    ├─── CFO（会計・財務）
    │         収支管理・請求書・確定申告
    │
    ├─── CDO（技術・自動化）
    │         スクリプト整備・プロンプト最適化
    │
    ├─── CCO（コマース）
    │         BOOTH/note出品・価格戦略
    │
    └─── CGO（グロース・新規開拓）
              新規クライアントDM・見込み客管理
```

## 情報フロー

```
【コンテンツ生成フロー】
CRO → CMO → CCO（販売）
CRO → CPO → CCO（出品）

【収益化フロー】
CSO（成約）→ COO（運用・納品）→ CFO（請求・入金）

【技術支援フロー】
COO（依頼）→ CDO（実装）→ COO（確認）

【週次報告フロー】
全部署 → CEO → オーナー
```
