# 事業壁打ちフロー（30分・5ステップ）

> AI-CPOが代行する7業務（アイデア出し／ターゲット言語化／プライシング／収益試算／差別化整理／Go-NoGo／カリキュラム設計）を、この5ステップに圧縮して回す。

---

## 事前：4つの入力要素（これが無いと精度が出ない）
| # | 要素 | 例 |
|---|------|----|
| 1 | 業種 | 製造業／飲食業／税理士事務所 |
| 2 | 規模感 | 10店舗チェーン／個人事業主 |
| 3 | やりたいこと | AI導入支援の新サービス立ち上げ |
| 4 | ゴール | 半日で事業評価まで／月◯円の数値目標 |

＋ **一次情報**（任意だが強力）：顧客アンケート、会食で得た悩み、面談文字起こし 等。

---

## ステップ1：ターゲット選定（〜5分）
- ターゲットを**3パターン書き分け**る。
- 各パターンで「**誰に・どんな悩みを・いくらで**」と「提供価値」を整理。
- 出力：3案の比較表。

## ステップ2：プライシング（〜5分）
- 市場相場に加え「**社長が払って惜しくない金額**」視点を必ず入れる。
- 根拠を付ける：例「半年で投資回収できるか」。商談での説得力に直結。
- 出力：価格案＋根拠（回収シミュレーションの一言）。

## ステップ3：収益試算（〜5分）
- 販売数に応じた**累積売上・粗利を表形式**で出す。
- **損益分岐点**（どこまで売れば損しないか）を数字で可視化。
- 出力例：

| 販売数 | 売上 | 原価 | 粗利 | 累積粗利 |
|---|---|---|---|---|
| 5 | | | | |
| 10 | | | | |
| 20 | | | | |

## ステップ4：競合比較とポジショニング（〜5分）
- 競合3社の価格・特徴を調査（※当社コンテナはWeb403のため、**オーナー端末で調べた情報を貼り付け** → CPOが分析）。
- 2軸（例：価格×専門性）で**ポジショニングマップ**を作る。HTML図解も可。
- 出力：競合3社表＋自社の立ち位置。

## ステップ5：Go/No-Go判定（〜10分）
- `スキル_GoNoGo判定.md` を適用。
- **ゴールドマン/マッキンゼー出身の厳格コンサル人格**で6軸採点。忖度禁止。
- 出力：6軸スコア＋総合判定（Go/条件付きGo/No-Go）＋根拠＋リスク。

---

## 完了後
- 結果を `CPO/outputs/YYYY-MM-DD_案件名_壁打ち.md` に保存。
- `壁打ち案件台帳.md`・`CPO/_index.md` に記録。
- **最終判断は人間（社長）**。AIのA評価でも、自社の資金力・リソースで覆してよい。
