## 出品の流れ（1商品あたり10〜20分）
1. **PDF化**：商品ファイルの「Print Content」を Canva（無料）か Google ドキュメントに貼る → A4/US Letterで体裁を整える → **PDFでダウンロード**。
   - ※ デザインは凝らなくてOK。読みやすい表紙＋本文で十分。Canvaの無料テンプレに流すと早い。
2. **サムネ画像**：Canvaで表紙＋中身プレビューを数枚（Etsyは複数画像可）。※code は画像生成不可(A2)のため、ここだけ手作業 or cowork。
3. **Etsyに出品**：Etsyショップ →「新しい商品」→ **デジタル商品**を選ぶ。
   - タイトル・タグ(13個)・説明文・価格は、商品ファイルの「Etsy Listing Kit」を**コピペ**。
   - 作ったPDFをアップロード → 公開。
4. 慣れたら 1日1〜2本ずつ上げて、まず10本そろえる（点数が増えるほど検索に乗りやすい）。
