## Googleスプレッドシート完全設計書

### シート1：月次投稿カレンダー

#### 列構成
```
A列: 日付（手動入力 or 数式で自動生成）
B列: 曜日 → =TEXT(A2,"ddd")
C列: 媒体 → プルダウン[Instagram / X / Facebook / 全媒体]
D列: 投稿テーマ → 自由入力（シート2から選んでもOK）
E列: 投稿文（下書き） → 自由入力
F列: ハッシュタグ → 自由入力（シート3参照）
G列: 画像準備 → チェックボックス
H列: 投稿済み → チェックボックス
I列: いいね数 → 数値入力
J列: 備考 → 自由入力
```

#### ヘッダー行（A1〜J1）
```
日付 | 曜日 | 媒体 | テーマ | 投稿文 | ハッシュタグ | 画像 | 済 | いいね | 備考
```

#### セル設定
```
A列: 書式→日付 (YYYY/MM/DD)
B列: =TEXT(A2,"ddd") を全行に設定
C列: データの入力規則→リスト[Instagram, X, Facebook, 全媒体]
G列: データの入力規則→チェックボックス
H列: データの入力規則→チェックボックス
I列: 書式→数値（小数なし）
```

#### 集計行（最終行）
```
投稿予定数: =COUNTA(D2:D32)
投稿済み数: =COUNTIF(H2:H32,TRUE)
達成率: =COUNTIF(H2:H32,TRUE)/COUNTA(D2:D32)*100 & "%"
いいね合計: =SUM(I2:I32)
```

#### 条件付き書式
```
H列がTRUEの行 → 背景色を薄い緑（#E8F5E9）
今日の日付の行 → 背景色を薄い黄色（#FFF9C4）
```

### シート4：使い方マニュアル

```
━━━━━━━━━━━━━━
SNS投稿カレンダー 使い方ガイド
━━━━━━━━━━━━━━

【セットアップ（初回のみ・5分）】
1. このスプレッドシートを開く
2. ファイル→コピーを作成→自分のGoogleドライブに保存
3. シート名の「2026年4月」を今月に書き換える

【月初の計画（30分）】
1. A列に今月の日付を入力（1日〜末日）
2. シート2「テーマ案50選」を見ながらD列にテーマを入力
3. 週3回以上の投稿を目標にバランスよく配置
4. E列に投稿文の下書きを入力
5. F列にハッシュタグを入力（シート3参照）

【日々の運用（1分）】
1. 投稿したらH列のチェックボックスをON
2. いいね数をI列に記録（翌日でOK）

【月末の振り返り（15分）】
1. 集計行で達成率を確認
2. いいね数が多かった投稿を確認→来月も似たテーマで投稿
3. 翌月のシートをコピーして新しい月を作成

━━━━━━━━━━━━━━
コツ：完璧な計画より「続けること」が大切です。
空欄があっても気にせず、まずは週3回を目指しましょう。
```
