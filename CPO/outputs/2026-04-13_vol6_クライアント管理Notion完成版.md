## Notionテンプレート設計書

### データベース1：クライアント一覧

```
プロパティ:
- クライアント名 [タイトル]
- 会社名 [テキスト]
- 担当者名 [テキスト]
- メールアドレス [メール]
- 電話番号 [電話番号]
- 連絡手段 [セレクト: メール/チャットワーク/Slack/LINE/DM/電話]
- ステータス [セレクト: 商談中/契約中/休止/完了]
- 初回契約日 [日付]
- メモ [テキスト]
- 案件 [リレーション→案件管理]
- 請求 [リレーション→請求管理]
```

### データベース2：案件管理

```
プロパティ:
- 案件名 [タイトル]
- クライアント [リレーション→クライアント一覧]
- 種別 [セレクト: ライティング/SNS運用/データ入力/テンプレ/その他]
- 単価 [数値・円]
- 納期 [日付]
- 進捗 [セレクト: 受注/作業中/レビュー中/納品済/請求済/完了]
- 作業開始日 [日付]
- 納品日 [日付]
- 工数（時間）[数値]
- 時給 [数式: 単価/工数]
- メモ [テキスト]
```

### データベース3：請求管理

```
プロパティ:
- 請求書番号 [タイトル]（例：INV-2026-001）
- クライアント [リレーション→クライアント一覧]
- 案件 [リレーション→案件管理]
- 請求金額 [数値・円]
- 源泉徴収 [数値・円]
- 手取り [数式: 請求金額-源泉徴収]
- 請求日 [日付]
- 支払期限 [日付]
- 入金日 [日付]
- 入金確認 [チェックボックス]
- ステータス [セレクト: 下書き/送付済/入金済/督促中]
```

### ビュー設計

```
ビュー1：今月のタスク（ボードビュー）
- フィルター：納期が今月
- グループ：進捗別
- 表示：案件名・クライアント・納期・単価

ビュー2：未入金一覧（テーブルビュー）
- フィルター：入金確認=false AND ステータス≠下書き
- ソート：支払期限の早い順
- 表示：請求書番号・クライアント・請求金額・支払期限

ビュー3：クライアント別売上（テーブルビュー）
- グループ：クライアント別
- 表示：案件名・単価・ステータス
- 集計：単価の合計

ビュー4：月別売上推移（カレンダービュー）
- 日付：請求日
- 表示：クライアント・請求金額
```
