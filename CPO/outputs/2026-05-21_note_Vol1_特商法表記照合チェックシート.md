## 2. 「住所・電話番号」の取扱

note の特商法表記設定では、住所・電話番号を非表示にできるオプションが用意されている場合がある（noteの仕様変更に注意）。  
非表示にする場合は **必ず以下の運用** を行うこと：

- [ ] 「請求があれば遅滞なく開示します」の文言を入れている
- [ ] メールアドレスの連絡先が機能している
- [ ] 請求があった場合、5 営業日以内に開示できる体制になっている

不安な場合は当面 **バーチャルオフィス契約** で住所を直接掲載するのが安全策。

## 4. 「永久に売れる」など景表法リスク表現の確認

`projects/2026-04-08_月30万自動化/C_テンプレ販売/service_design.md` には
「一度作れば永久に売れる」という社内用の訴求が記載されているが、**対外（販売ページ／LP／SNS）には使用しない**。  
Vol.1 の販売記事本文に同種の断定表現が含まれていないかチェック：

- [ ] 「永久に〜」「一生〜」「絶対に〜」を使っていない
- [ ] 効果効能の断定表現（「収入が必ず増える」「節税できる」等）を使っていない
- [ ] 効果に言及する場合は「※効果には個人差があります」を併記

## 5. 修正手順

不足項目がある場合：

1. **特商法表記ページの修正**（noteの設定ページから）
   - 上記表の項目 1〜12 を埋める
2. **Vol.1 販売記事の修正**
   - 末尾に「ご利用にあたって」セクションを追記
3. **修正後の確認**
   - スマホ表示でも特商法表記が読めるかチェック
   - 「請求があれば開示」文言と連絡先メールが対応しているかチェック
4. **修正完了報告**
   - 完了したら CPO の `_index.md` に記録、CLO に報告
