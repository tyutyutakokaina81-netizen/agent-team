2026年4月。新しいスーツに袖を通して、初めての出社。

おめでとうございます。

でも、1つだけ聞いていいですか。

**初任給、何に使うか決まってますか？**

「とりあえず貯金」「親に何か買う」「自分へのご褒美」——全部正解。

**でも、知らないだけで毎月数千円〜数万円損する落とし穴が、社会人1年目にはたくさんあります。**

- 住民税は2年目から突然引かれる（手取りが急に減る）
- 社会保険料の仕組みを知らないと、残業の仕方で損する
- NISAを1年目から始めるだけで、10年後に数十万円の差がつく
- ふるさと納税は1年目からできる（知らない人が多い）
- クレジットカードは「最初の1枚」で5年間の還元率が決まる

この記事では、**新社会人が4月〜12月にやるべきお金の手続き・判断を、月別チェックリストで全部まとめました。**

「あのとき知ってれば…」を、ゼロにする教科書です。

## 第1章：給与明細の読み方（これを知らないと始まらない）

### 手取り = 額面 ではない

額面25万円の場合の内訳（目安）：

| 項目 | 金額 | 説明 |
|---|---|---|
| 額面給与 | ¥250,000 | 会社が払う総額 |
| 健康保険 | ▲¥12,500 | 医療費の自己負担を3割にする保険 |
| 厚生年金 | ▲¥22,900 | 将来の年金の積立 |
| 雇用保険 | ▲¥1,500 | 失業時の保険 |
| 所得税 | ▲¥5,200 | 毎月の仮払い（年末調整で精算） |
| 住民税 | ¥0 | **1年目はゼロ。2年目から約¥10,000/月引かれる** |
| **手取り** | **¥207,900** | 実際に口座に入る金額 |

### 2年目の罠：住民税

**1年目の収入に対する住民税が、2年目の6月から引かれ始めます。**

つまり：
- 1年目の手取り：約¥208,000
- 2年目の手取り：約¥198,000（▲¥10,000）

**「昇給したのに手取りが減った」** のカラクリはこれです。1年目から知っておけば慌てない。

## 第2章：4月にやること

### ✅ 銀行口座を「生活用」と「貯金用」に分ける

**鉄則：1つの口座でやりくりしない。**

| 口座 | 用途 | おすすめ |
|---|---|---|
| メイン口座 | 給与受取・生活費 | 会社指定の銀行 |
| 貯金用口座 | 手をつけないお金 | ネット銀行（金利が高い） |

給料日に自動振替で **手取りの20%** を貯金口座に移す設定をする。これだけで年間50万円貯まる。

### ✅ クレジットカードを1枚作る

**新社会人は審査に通りやすい。** このタイミングを逃すと数年後に「クレヒスなし」で苦労する。

選び方：
- 年会費無料
- 還元率1%以上
- スマホ決済連携あり

### ✅ NISA口座を開設する

**2024年からの新NISA、1年目から始めた人と3年目から始めた人では、30年後に100万円以上の差がつく。**

今月やること：
1. 証券口座を開設（ネット証券が手数料最安）
2. NISA口座の申し込み（開設に2〜3週間）
3. つみたて投資枠で月¥10,000 からスタート

**投資先は「全世界株式インデックス」1本でOK。** 迷ったらこれ一択。

## 第3章：5月〜6月にやること

### ✅ ふるさと納税の上限額を計算する

1年目でも **前年収入がなければ住民税がゼロ** → ふるさと納税の効果が薄い場合あり。

**ただし、4月〜12月の収入が一定額を超えれば、年末に駆け込みで使える。**

計算方法：
- 年収見込み（月給 × 9ヶ月 + 賞与）を算出
- ふるさと納税シミュレーターに入力
- 上限額が¥10,000以上なら12月に実行

### ✅ 生活費の固定費を最適化する

| 項目 | 新社会人の平均 | 最適化後 | 節約額 |
|---|---|---|---|
| スマホ | ¥8,000 | ¥3,000（格安SIM） | ¥5,000/月 |
| サブスク | ¥3,000 | ¥1,000（厳選） | ¥2,000/月 |
| 保険 | ¥5,000 | ¥0（不要） | ¥5,000/月 |
| **月間節約** | | | **¥12,000** |
| **年間節約** | | | **¥144,000** |

**特に保険：独身の新社会人に民間保険はほぼ不要。** 社会保険で医療費は3割負担、高額療養費制度で月の上限も決まっている。保険の営業が来ても断ってOK。

## 第5章：年末にやること（11月〜12月）

### ✅ 年末調整を正しく提出する

- 生命保険料控除（入っている場合）
- 地震保険料控除（入っている場合）
- iDeCo（加入している場合）

**提出を忘れると数千円〜数万円損します。** 会社から紙が配られたら必ず記入。

### ✅ ふるさと納税の駆け込み

上限額内で12月中に寄付 → ワンストップ特例申請 → 翌年の住民税が減る。

### ✅ 1年間の収支を振り返る

```
1年目の目標チェックリスト：
□ 貯金口座に50万円以上ある
□ NISA口座を開設した
□ つみたて投資を開始した
□ 不要な固定費を削除した
□ クレジットカードを1枚持っている
□ ふるさと納税を実行した（対象なら）
□ 年末調整を正しく提出した
```

7つ中5つ以上チェックできていれば、**同期の上位10%** です。
