## プロンプト本文（全20個）

### 1. How-to記事（〇〇のやり方）
```
以下の条件でSEOブログ記事を作成してください。

【記事タイプ】How-to（やり方・手順解説）
【キーワード】[例：確定申告 やり方 フリーランス]
【想定読者】[例：確定申告が初めてのフリーランス]
【文字数】[例：4,000字]

【構成ルール】
1. タイトル（32字以内・キーワード含む）
2. リード文（300字・読者の悩みに共感→この記事で解決できると宣言）
3. 「〇〇とは」の簡単な定義（読者の前提知識を揃える）
4. 手順をステップ形式で解説（Step1〜Step5程度）
5. 各ステップに具体例・注意点を含める
6. よくある失敗・注意点（2〜3個）
7. まとめ（要点を箇条書き＋行動喚起）
8. メタディスクリプション（120字以内）

【禁止事項】
- 根拠のない断言
- 「簡単です」「誰でもできます」の安易な使用
- 同じ表現の繰り返し
```

### 2. まとめ記事（おすすめN選）
```
以下の条件でSEOブログ記事を作成してください。

【記事タイプ】まとめ・ランキング（おすすめN選）
【キーワード】[例：会計ソフト おすすめ フリーランス]
【紹介する数】[例：5選/7選/10選]
【想定読者】[例：会計ソフトを探しているフリーランス]
【文字数】[例：5,000字]

【構成ルール】
1. タイトル（「おすすめ○選」をタイトルに含める）
2. リード文（この記事で最適な○○が見つかると宣言）
3. 選び方のポイント（3つ程度）
4. おすすめN選（各項目に：名前・特徴・価格・メリット・デメリット・おすすめな人）
5. 比較表（表形式で一覧化）
6. まとめ（用途別のおすすめを再提示）
7. メタディスクリプション
```
