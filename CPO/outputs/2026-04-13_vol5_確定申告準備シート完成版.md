## Googleスプレッドシート完全設計書

### シート1：売上記録

#### 列構成
```
A列: 日付 → 書式:日付(YYYY/MM/DD)
B列: 取引先名 → 自由入力
C列: 案件内容 → 自由入力
D列: 売上金額（税込） → 数値
E列: 源泉徴収額 → 数値（該当する場合のみ）
F列: 手取り金額 → =D2-E2
G列: 入金日 → 書式:日付
H列: 入金確認 → チェックボックス
I列: 備考 → 自由入力
```

#### ヘッダー行
```
日付 | 取引先 | 案件内容 | 売上（税込） | 源泉徴収 | 手取り | 入金日 | 入金済 | 備考
```

#### 集計行
```
年間売上合計: =SUM(D2:D500)
源泉徴収合計: =SUM(E2:E500)
手取り合計: =SUM(F2:F500)
未入金件数: =COUNTIF(H2:H500,FALSE)-COUNTBLANK(D2:D500)
```

---

### シート2：経費記録

#### 列構成
```
A列: 日付 → 書式:日付(YYYY/MM/DD)
B列: 科目 → プルダウン[通信費/消耗品費/交通費/書籍研修費/ソフトウェア/家賃(按分)/水道光熱費(按分)/交際費/雑費]
C列: 内容 → 自由入力（例：Claude Pro月額）
D列: 金額 → 数値
E列: 按分率 → 数値%（按分しない場合は100%）
F列: 経費計上額 → =D2*E2/100
G列: 支払方法 → プルダウン[現金/クレジットカード/銀行振込/電子マネー]
H列: レシート保管 → チェックボックス
I列: 備考 → 自由入力
```

#### ヘッダー行
```
日付 | 科目 | 内容 | 金額 | 按分率(%) | 経費計上額 | 支払方法 | レシート | 備考
```

#### 集計行
```
経費合計（税込）: =SUM(D2:D500)
経費計上額合計: =SUM(F2:F500)
科目別集計:
  通信費: =SUMIF(B:B,"通信費",F:F)
  消耗品費: =SUMIF(B:B,"消耗品費",F:F)
  交通費: =SUMIF(B:B,"交通費",F:F)
  書籍研修費: =SUMIF(B:B,"書籍研修費",F:F)
  ソフトウェア: =SUMIF(B:B,"ソフトウェア",F:F)
  家賃(按分): =SUMIF(B:B,"家賃(按分)",F:F)
  水道光熱費(按分): =SUMIF(B:B,"水道光熱費(按分)",F:F)
  交際費: =SUMIF(B:B,"交際費",F:F)
  雑費: =SUMIF(B:B,"雑費",F:F)
```

### シート3：年間サマリー（全自動）

```
■ 収入
  事業収入合計: ='売上記録'の年間売上合計を参照
  源泉徴収合計: ='売上記録'の源泉徴収合計を参照

■ 経費
  経費合計: ='経費記録'の経費計上額合計を参照
  
  科目別内訳:
  | 科目 | 金額 |
  通信費: ='経費記録'の科目別集計を参照
  消耗品費: 同上
  交通費: 同上
  書籍研修費: 同上
  ソフトウェア: 同上
  家賃(按分): 同上
  水道光熱費(按分): 同上
  交際費: 同上
  雑費: 同上

■ 所得
  事業所得: =収入合計 - 経費合計
  青色申告特別控除: 650,000（65万円・電子申告の場合）
  課税所得: =事業所得 - 青色申告特別控除

■ 概算税額（参考値）
  所得税概算: =課税所得が195万以下なら×5%、330万以下なら×10%-97,500
  住民税概算: =課税所得×10%
  合計概算: =所得税+住民税

■ 月別推移グラフ用データ
  | 月 | 売上 | 経費 | 利益 |
  1月: =SUMIFS(売上記録!D:D, 売上記録!A:A, ">=2026/1/1", 売上記録!A:A, "<2026/2/1")
  ...（12ヶ月分）
```

```

確定申告 準備チェックリスト

□ 売上記録の入金確認が全て完了している

□ 按分率が正しく設定されている

□ 確定申告書B

□ 支払調書（取引先から届く場合）
□ 医療費の領収書（医療費控除を使う場合）
□ ふるさと納税の受領証明書
□ 社会保険料の証明書
□ 生命保険料控除証明書
□ マイナンバーカード or 通知カード

□ 郵送 → 青色申告特別控除55万円

【申告期限】
毎年2月16日〜3月15日
※ 還付申告は1月1日から可能

【納税方法】
□ 振替納税（口座引落）
□ e-Taxで電子納付
□ クレジットカード
□ コンビニ（QRコード）
□ 税務署窓口
```

### シート5：使い方マニュアル

```
━━━━━━━━━━━━━━
確定申告準備シート 使い方ガイド
━━━━━━━━━━━━━━

【初期設定（1回だけ・10分）】
1. このスプレッドシートを開く
2. ファイル→コピーを作成→自分のGoogleドライブに保存
3. シート2「経費記録」の按分率を自分の状況に合わせて確認
   例：家賃の事業使用割合が30%なら、按分率に30と入力

【毎月やること（5分）】
1. 売上が入金されたら「売上記録」に1行追加
2. 経費が発生したら「経費記録」に1行追加
3. レシートをスマホで撮影して保管

【按分の目安】
- 家賃：事業に使っている面積の割合（例：1部屋/3部屋=33%）
- 通信費：事業使用の割合（例：50%）
- 水道光熱費：事業使用の割合（例：30%）
- スマホ代：事業使用の割合（例：50%）

【確定申告時（1〜2月）】
1. 年間サマリーを開く
2. 収入・経費・所得の数字を確認
3. 確定申告書に数字を転記する
4. チェックリストで必要書類を確認

━━━━━━━━━━━━━━
ポイント：毎月5分の入力を続けるだけ。
確定申告の時期に慌てる必要はありません。
━━━━━━━━━━━━━━
```
