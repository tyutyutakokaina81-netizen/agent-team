# 受講者ジャーニーマップ テンプレート

**作成日**：2026-05-21
**用途**：新教材・新プロダクト設計時の体験設計フォーマット。教材コンテンツ着手より先に本マップを埋める。

---

## 0. 対象プロダクト

| 項目 | 内容 |
|------|------|
| プロダクト名 | （例：SNS運用代行スタートアップ講座） |
| ターゲット | （例：副業フリーランス／ライター） |
| ゴール | （受講者がXX出来るようになる） |
| 期間 | （例：4週間） |

---

## 1. 認知ステージ

| 項目 | 内容 |
|------|------|
| 受講者の状態 | 「自分の課題に合うか分からない」 |
| 接触チャネル | （YouTube / X / 紹介） |
| 想定される疑問 | 「自分でも出来る？」「料金は？」 |
| CPO提供物 | LP用 受講者像・成果イメージ素材 → CMO へ |
| 離脱要因 | （例：ターゲットがずれている、成果が伝わらない） |

---

## 2. 検討ステージ

| 項目 | 内容 |
|------|------|
| 受講者の状態 | 「やりきれるか不安」 |
| 想定される疑問 | 「途中で挫折しない？」「サポートある？」 |
| CPO提供物 | 事前FAQ・難易度の正直な提示・先輩事例 |
| 離脱要因 | （価格、難易度の不透明さ） |

---

## 3. 学習開始（最初の30分）

| 項目 | 内容 |
|------|------|
| 受講者の状態 | 「最初の壁にぶつかる瞬間」 |
| 設計の鍵 | **クイックウィン**：10〜30分で1つの成功体験を必ず提供 |
| CPO提供物 | チュートリアル動画 / セットアップ手順 / 最初の演習 |
| 離脱要因 | 環境構築でつまずく・最初の課題が大きすぎる |

---

## 4. 継続ステージ

| 項目 | 内容 |
|------|------|
| 受講者の状態 | 「詰まって止まる」「忙しくて開かない」 |
| 設計の鍵 | チェックポイント・進捗の可視化・つまずきFAQ |
| CPO提供物 | 各章末のチェックリスト・サポート導線 |
| 離脱要因 | 中盤の難易度ジャンプ・モチベ低下 |

---

## 5. 成果実感ステージ

| 項目 | 内容 |
|------|------|
| 受講者の状態 | 「使える」「やれた」 |
| 設計の鍵 | 成果物テンプレ・卒業課題・実績の言語化支援 |
| CPO提供物 | 最終アウトプット用テンプレ・他者紹介できる成果サマリ |
| 期待される行動 | リピート・紹介・事例提供 |

---

## 6. 離脱ポイント優先度（観察ベース）

| ステージ | 離脱率（実測 or 推定） | 改修優先度 |
|---------|----------------------|-----------|
| 認知 | ◯% | 低 / 中 / 高 |
| 検討 | ◯% | 低 / 中 / 高 |
| 学習開始 | ◯% | 低 / 中 / 高 |
| 継続 | ◯% | 低 / 中 / 高 |
| 成果実感 | ◯% | 低 / 中 / 高 |

## 7. 他役職への連携依頼

| 依頼先 | 内容 |
|--------|------|
| CMO | 認知ステージのコピー・LP素材を提供してほしい |
| CSO | 検討ステージの受講者FAQ素材を提供してほしい |
| CFO | 価格・返金条件の整合性確認 |
| CDO | チェックポイント自動化ツールの相談 |
