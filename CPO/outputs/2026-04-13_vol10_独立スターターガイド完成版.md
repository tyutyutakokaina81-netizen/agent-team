### Chapter 1：独立前にやること10選

```
フリーランスとして独立する前に、会社員のうちにやっておくべきことがあります。
退職してからでは遅いものもあるので、必ず確認してください。

□ 1. クレジットカードを作る
   会社員の信用があるうちに作っておく。退職後は審査が通りにくくなります。
   おすすめ：楽天カード（年会費無料）、三井住友カード

□ 2. 事業用の銀行口座を開設する
   プライベートと事業のお金を分けるため。
   おすすめ：住信SBIネット銀行、楽天銀行（振込手数料が安い）

□ 3. 3〜6ヶ月分の生活費を貯める
   収入ゼロの期間を乗り越えるため。
   目安：月の生活費 × 3ヶ月 = 最低ライン

□ 4. 健康保険の切り替え方法を確認する
   選択肢：国民健康保険 or 任意継続（退職後20日以内に届出）
   任意継続の方が安い場合が多い。両方の金額を市区町村で確認。

□ 5. 年金の切り替え手続きを確認する
   厚生年金 → 国民年金に切り替え（退職後14日以内）
   市区町村の役所で手続き。

□ 6. 退職前に副業で1件でも実績を作る
   実績ゼロより実績1件の方が、独立後の営業が格段に楽になる。

□ 7. クラウドソーシングに登録してプロフィールを整える
   クラウドワークス、ランサーズ、ココナラ。
   会社員のうちに登録・プロフィール作成まで済ませておく。

□ 8. 自分のスキルの相場を調べる
   クラウドソーシングで同じスキルの案件がいくらで募集されているか確認。
   自分の価格設定の参考にする。

□ 9. 確定申告の基礎知識を学ぶ
   「売上 − 経費 = 所得」この1つだけ覚えればOK。
   あとは会計ソフトが計算してくれる。

□ 10. 開業届と青色申告承認申請書の書き方を調べる
    次のChapterで詳しく解説します。
```

### Chapter 2：開業届・青色申告の出し方

```
■ 開業届とは
「個人事業を始めました」と税務署に届ける書類。
提出は義務ですが、出さなくても罰則はありません。
ただし、出さないと青色申告ができないので、必ず出しましょう。

■ 提出方法
1. 国税庁のサイトから「個人事業の開業・廃業等届出書」をダウンロード
2. 記入する（所要時間：10分）
3. 税務署に提出（持参 or 郵送 or e-Tax）

■ 記入のポイント
・届出の区分：「開業」にチェック
・所得の種類：「事業所得」にチェック
・開業日：独立した日（過去の日付でもOK）
・職業：「ライター」「Web制作」「コンサルタント」など
・屆出の区分：「青色申告承認申請書も一緒に出す」

■ 青色申告承認申請書
開業届と同時に提出するのがベスト。
メリット：最大65万円の控除（年間の税金が約10万円安くなる）

■ 提出期限
開業日から2ヶ月以内（1月1日〜1月15日に開業した場合は3月15日まで）

■ 必要なもの
・マイナンバーカード or 通知カード
・印鑑（シャチハタ不可）
・身分証明書
```

### Chapter 3：事業用口座・カードの選び方

```
■ なぜ分ける必要があるか
事業用とプライベートの口座を分けると：
・帳簿作成が圧倒的に楽になる
・確定申告がスムーズになる
・お金の流れが一目でわかる

■ おすすめ銀行口座

| 銀行 | 振込手数料 | 特徴 |
|------|----------|------|
| 住信SBI | 月1〜15回無料 | ネットバンク最強 |
| 楽天銀行 | 月3回無料（条件あり） | 楽天経済圏なら◎ |
| PayPay銀行 | 145円〜 | 個人ビジネス口座あり |
| ゆうちょ | 100円 | 全国ATM多い |

■ おすすめクレジットカード

| カード | 年会費 | 特徴 |
|--------|--------|------|
| 楽天カード | 無料 | ポイント還元1% |
| 三井住友NL | 無料 | コンビニ5%還元 |
| freeeカード | 無料 | 会計ソフト連携 |

■ ルール
・事業の支出はすべて事業用カードで払う
・プライベートの支出は絶対に混ぜない
・これだけで帳簿の手間が半分になる
```

### Chapter 5：最初の1件を取る営業術

```
■ 真実：スキルより営業が大事

どんなに良いサービスでも、お客さんに出会えなければ売上はゼロ。
最初の3ヶ月は「営業80%、作業20%」の気持ちで。

■ 営業チャネル3つ

1. クラウドソーシング（即効性◎）
   → 今日登録して、今日応募できる
   → 実績0でもチャンスがある

2. SNS DM営業（中期◎）
   → Instagramで地域の店舗を見つける
   → 投稿が少ないお店にDMを送る
   → 返信率5〜10%（50件送れば3〜5件返信）

3. 紹介・口コミ（長期◎）
   → 最初の1件を全力で納品する
   → 「他にもお知り合いいらっしゃいますか？」と聞く

■ 応募文のテンプレ

---ここからコピペ---
はじめまして。[案件名]に応募させていただきます。

■ 対応可能な理由
・[関連するスキル・経験]
・[納品できるもの]

■ 納期
ご依頼から[N]営業日で納品可能です。

■ 注意事項
AI執筆支援ツールを使用しますが、
品質管理は全て人間が行います。

よろしくお願いいたします。
---ここまでコピペ---

■ 受注後の鉄則
1. 24時間以内にヒアリング開始
2. 納期の50%で初稿を見せる（安心感）
3. 修正は即対応
4. 納品後に「今後も継続できますか？」と聞く
```

### Chapter 6：月次の経費管理

```
■ 月にやることは2つだけ

1. レシートをスマホで撮影する（その日のうちに）
2. 月末に売上を記録する（5分）

■ 経費になるもの一覧

| 科目 | 例 |
|------|-----|
| 通信費 | インターネット、スマホ（按分） |
| 消耗品費 | 文房具、10万未満のPC周辺機器 |
| ソフトウェア | Claude Pro、Canva、会計ソフト |
| 書籍研修費 | 業務関連の本、オンライン講座 |
| 交通費 | 打ち合わせの移動費 |
| 家賃（按分） | 自宅の事業使用部分 |
| 水道光熱費（按分） | 事業使用割合分 |
| 交際費 | 打ち合わせのカフェ代 |

■ 按分のルール
プライベートと事業で共用するものは、事業割合で計算。
例：家賃10万円 × 事業使用30% = 経費3万円/月

■ おすすめ管理方法
・スプレッドシート（Vol.1またはVol.5を使う）
・会計ソフト（年間売上100万円超えたら導入検討）
```

### Chapter 7：確定申告の準備

```
■ いつやるか
毎年2月16日〜3月15日に前年分を申告。
還付申告（払いすぎた税金を返してもらう）は1月1日から可能。

■ 必要なもの
1. 確定申告書B
2. 青色申告決算書
3. 源泉徴収票（会社員から転職した年）
4. 支払調書（取引先から届く場合）
5. 各種控除証明書
6. マイナンバーカード

■ 申告方法
1. e-Tax（おすすめ）→ 65万円控除
2. 税務署に持参 → 55万円控除
3. 郵送 → 55万円控除

■ 初年度の注意点
・年の途中で独立した場合、会社員時代の給与所得と事業所得を合算して申告
・住民税の切り替え（普通徴収に変更）を忘れない
・消費税は売上1,000万円以下なら免税（インボイス登録していない場合）
```
