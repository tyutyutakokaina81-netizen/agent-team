# Vol.3 飲食店向け AI プロンプト集 — 前半 25 本（コピペ即使用版）

## 使い方

1. 該当するプロンプトをコピー
2. 文中の `{中括弧}` 部分を自店の情報で置換
3. AI（Claude / ChatGPT 等）に貼り付け
4. 出力されたテキストの最終 1 行・絵文字だけは手で調整

## カテゴリ A：メニュー紹介文（8 本）

### A-01：季節メニューの告知（汎用）
```
あなたは {業態} のオーナーです。SNS（X / Instagram）に投稿する文章を作ってください。
- 文字数：250 字以内
- 1 行目で読者の足を止める（数字・季節感・限定感のいずれか）
- メニュー名・価格・期間を明記
- 最後の 1 行は温かい呼びかけ
- ハッシュタグ 3 つ：地名 + 業態 + メニュー名
- 「絶対」「必ず」「最高」など景表法に触れる表現は禁止
- 健康効果（〇〇に効く・痩せる等）は薬機法 NG

【メニュー情報】
- メニュー名：{メニュー名}
- 価格：{価格}
- 期間：{期間}
- 特徴：{特徴}
- こだわり：{こだわり}
- 場所：{所在地}
```

### A-02：新メニュー告知
```
{業態} で、新メニュー紹介文を 200〜300 字で書いてください。
- 1 行目で開発の動機（なぜ作ったか）を一言
- 中盤で具体的な味の特徴
- 終盤で「いつまでに食べに来てほしいか」（試験販売期間など）
- 価格は明記
- 「最高」等の表現は避け、客観的な特徴のみ

【メニュー情報】{メニュー名} / {価格} / {特徴} / {開発エピソード}
```

### A-03：看板メニュー再紹介（既存客向け）
```
{業態} の看板メニュー「{メニュー名}」を、リピーターさん向けに再紹介する文章を 200 字で。
- 「みなさんご存じの〜」風の親近感
- 知らない人向けではなく、ファン向けのトーン
- 「久しぶりに〜」「最近〇〇さんから〜」のような小さなエピソード 1 つ
- 価格・営業時間は文末に小さく
```

### A-04：メニュー写真の説明文
```
{業態} の {メニュー名} の写真の Alt テキスト・キャプションを書いてください。
- 視覚障害者にも伝わる詳細な描写を 80 字
- インスタ用キャッチー一文を別途 30 字
- 食材・調理法・色・温度感が分かる表現を使う
```

### A-05：限定メニューのカウントダウン
```
{業態} の {限定メニュー名} の販売終了告知を 3 段階で作ってください：
- 残り 7 日：「あと 1 週間」感を強調・行動喚起
- 残り 3 日：「すぐ動かないと」感
- 当日：「本日最終日です」
すべて 150 字以内、価格・営業時間明記、それぞれ別投稿として独立。
```

### A-06：ドリンクペアリング提案
```
{業態} の {メニュー名} に合うドリンクペアリング提案を 200 字で。
- メニューの味の特徴
- ペアリングの根拠（合う理由）
- 「ぜひ〇〇とご一緒に」の優しい誘導
- アルコール・ノンアル両方提案（業態に応じて）
```

### A-07：弁当・テイクアウト告知
```
{業態} の {弁当 or テイクアウト} メニュー紹介を 250 字で。
- 平日 / 休日 / 雨の日 など利用シーンを 3 つ提案
- 価格・受取時間・注文方法（電話 / LINE / 店頭）を明記
- アレルギー対応の有無（あれば）
```

### A-08：定食・コース紹介
```
{業態} の {定食 or コース名} を、初めての方向けに 280 字で紹介。
- 全体の構成（前菜 → メイン → ご飯物 → 汁物 → デザート 等）
- 何が一番のこだわりポイントか 1 つ
- ボリューム感（1 人前 or シェア向け）
- 価格、所要時間、予約可否
```

## カテゴリ B：SNS 投稿（10 本）

### B-01：朝の挨拶投稿
```
{業態} の朝 7:30 投稿を 100 字で。
- 当日の天気・気温に合わせた一言
- 「今日のおすすめ」を 1 つだけ
- 重くない、軽い読後感
- 絵文字は朝らしいもの 1 個
```

### B-02：ランチタイム告知（11:30）
```
{業態} の 11:30 ランチ開始告知を 80 字で。
- 「本日のランチ」を 1 行で
- 残り席状況（あれば）
- 営業時間
- 11:30〜の活発さを感じる絵文字 1 個
```

### B-03：ディナータイム告知（17:30）
```
{業態} の 17:30 ディナー前投稿を 150 字で。
- 「今晩のおすすめ」or「平日の特典」
- 予約状況
- お一人様 / グループ どちらでも歓迎の文言
```

### B-04：閉店前最後の告知（21:00）
```
{業態} の 21:00 ラストオーダー前告知を 100 字で。
- 「本日まだ間に合います」感
- 入店可能時刻
- 駆け込みでも歓迎する温かさ
```

### B-05：週末告知
```
{業態} の金曜 18:00 投稿で、土日の予約呼びかけを 200 字。
- 「週末は〇〇」の特別感
- 予約方法（電話 / LINE / Google）
- 1 ヶ月前から予約可能等のルール明記
```

### B-06：雨の日メニュー
```
{業態} の雨の日限定メニューやサービス告知を 150 字で。
- 「雨の日はこんなメニューが沁みる」共感
- 雨の日特典（あれば）
- 駅近 / 屋内 / 駐車場 等のアクセス利点
```

### B-07：平日空席アピール
```
{業態} の平日昼の空席アピールを 100 字で。
- 「今ならゆっくり」感
- 平日特典（あれば）
- 時間帯指定の予約呼びかけ
```

### B-08：祝日営業告知
```
{業態} の祝日営業告知を 150 字で。
- 「祝日も営業します」を冒頭で明示
- 通常と違う点（時間 / メニュー）があれば
- 予約方法
```

### B-09：常連さん向け感謝
```
{業態} のお客様への感謝投稿を 200 字で。
- 「いつもありがとうございます」を冒頭で
- 具体的なエピソード 1 つ（個人特定されない範囲）
- 「これからも〇〇していきます」の宣言
```

### B-10：新人スタッフ紹介
```
{業態} の新人スタッフ紹介を 200 字で。
- 名前（下の名前のみ or ニックネーム）
- 担当（ホール / キッチン）
- ひとこと自己紹介（オーナー視点で代弁）
- 個人情報（学校・年齢・住所）は記載しない
```

## カテゴリ C：店主の自己紹介・人柄（5 本）

### C-01：開業ストーリー
```
{業態} のオーナー（{年齢}代・{性別}・{出身地}）の開業ストーリーを 350 字で。
- なぜこの場所で開業したか
- 修行・前職とのつながり
- 開業時の一番の苦労
- 今のお店で大事にしていること
- 個人情報（学歴・実家住所等）は出さない
```

### C-02：休日の過ごし方
```
{業態} のオーナーの休日紹介を 200 字で、親近感が湧くトーンで。
- 食材調達・自己研鑽 / 趣味 / 家族時間 のいずれか
- 完璧過ぎず、人間味のあるエピソード
- 結びは「明日からまたお店に立ちます」系
```

### C-03：好きな食材紹介
```
{業態} のオーナーの「お店では出していないけど好きな食材」紹介を 150 字で。
- なぜ好きか
- どこで知ったか
- お店で出さない理由（コンセプトに合わない等）
- 自己開示で親近感を作る
```

### C-04：失敗談
```
{業態} のオーナーの失敗談を 200 字で、ユーモアを交えて。
- 開業初期の小さな失敗
- 学んだこと
- 今お客様がそれで困ることはない、という安心感
- 個人を貶めない
```

### C-05：今後やりたいこと
```
{業態} のオーナーの「今後やりたいこと」を 250 字で。
- 半年以内の小目標 1 つ
- 1〜3 年の中期目標 1 つ
- 「お客様と一緒に」の関係性
- 過大な公約はしない
```

## カテゴリ D：お客様の声活用（2 本・前半）

### D-01：レビュー紹介（許可取得後）
```
{業態} に頂いた {サイト名}（食べログ / Google / Tabelog 等）のレビューを紹介する投稿を 200 字で作成。
- 「ありがたいお声を頂きました」と前置き
- レビュー文を 50 字以内に短縮（または引用）
- 「許可をいただいてご紹介させていただいています」の一文
- 「ご来店ありがとうございました」の締め
- 投稿主の名前・写真は載せない（プライバシー）
```

### D-02：DM での感想紹介
```
{業態} へのお客様 DM 感想を、許可取得済の前提で 200 字でブログ・SNS 化。
- 「先日 DM でこんなお声をいただきました」と前置き
- 内容の要約（個人特定情報は除く）
- そのお客様への返信メッセージ
- 同じ気持ちの方への呼びかけ
- 個人特定リスクの最小化（年代・性別・地名は伏せる）
```

## NG 表現の自動チェック（プロンプトに含める汎用注意事項）

各プロンプトの末尾に以下を必ず付け足す（汎用 NG ガード）：

```
【出力前の自己チェック】
以下の表現は使わないこと：
- 景表法 NG：絶対・必ず・最高・最強・No.1（裏付けなし）
- 薬機法 NG：健康になる・痩せる・〇〇に効く・免疫力アップ
- ステマ NG：第三者を装った推薦
出力に上記が混入していたら、自分で書き直してから提示してください。
```
