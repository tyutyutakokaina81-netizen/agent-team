## 使い方（3ステップ）

### ステップ 1：プロジェクトフォルダを作る

```bash
mkdir ~/my-project
cd ~/my-project
```

### ステップ 2：自分の業種に合った CLAUDE.md をコピー

例：ブロガー・ライターの方

```bash
cp /path/to/vol_a/01_blogger_writer.md ~/my-project/CLAUDE.md
```

### ステップ 3：Claude Code を起動

```bash
claude
```

以上。CLAUDE.md の内容が自動的にロードされ、あなたの業種に特化した動きをします。

## カスタマイズのヒント

各テンプレート内の `[ここに記入]` 部分を、あなたの実情に合わせて書き換えてください。

例えば `01_blogger_writer.md` の場合：

```markdown
## ターゲット読者
[ここに記入：例「30代のワーキングマザー、忙しくて効率化の情報を求めている」]
```

この部分を具体的に埋めるだけで、Claude の出力品質が大きく変わります。
