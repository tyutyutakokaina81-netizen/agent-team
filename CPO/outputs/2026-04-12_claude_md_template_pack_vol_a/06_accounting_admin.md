# CLAUDE.md — 個人経理・事務担当向け

このファイルは Claude Code に対するプロジェクト指示書です。経理・事務業務の定型作業を AI で効率化します。

## プロジェクトの目的

[ここに記入：例「小規模企業の経理・月次処理・請求書発行・経費精算の効率化」]

## 担当範囲

- 請求書発行
- 経費申請書類の仕訳チェック
- 月次レポート作成
- 取引先別の売掛金・買掛金管理
- 経費科目の分類提案
- 月次・年次の数字チェック

## やらないこと

- **金額の最終確定**（必ず人間が検算する）
- 税法・会計基準の個別判断（税理士の領域）
- 銀行口座・クレジットカード情報の外部送信
- 監査対応の一次対応
- 確定申告書の作成

## 経費科目の標準分類

| 科目 | 該当する支出例 |
|---|---|
| 旅費交通費 | 電車・バス・タクシー・出張宿泊費 |
| 通信費 | 携帯電話・インターネット・郵送料 |
| 消耗品費 | 文具・印刷用紙・トナー |
| 会議費 | 取引先との打ち合わせ飲食 |
| 接待交際費 | 取引先接待・お中元・お歳暮 |
| 外注費 | 業務委託・フリーランス支払い |
| 広告宣伝費 | 広告・チラシ・HP制作 |
| 租税公課 | 印紙代・自動車税・固定資産税 |
| 保険料 | 損害保険・労災 |
| 減価償却費 | 10万円以上の備品 |
| 新聞図書費 | 書籍・定期購読 |
| 支払手数料 | 振込手数料・決済手数料 |

## 数字チェックのルール（必須）

すべての数字に対して：

1. **桁数の確認**（¥10,000 と ¥100,000 の混同が最多ミス）
2. **税込/税抜の区別**
3. **前月比較**（急増・急減があればアラート）
4. **取引先名のスペル確認**
5. **合計と内訳の一致確認**

## 請求書の必須項目

- 発行日
- 請求書番号
- 宛名（法人名・敬称）
- 発行者情報（名前・住所・連絡先）
- 取引内容（品目・数量・単価）
- 小計・消費税・合計
- 振込先（銀行名・支店名・口座種別・口座番号・名義）
- 支払期限
- 備考

## 月次ルーチン

### 月初
- 前月の請求書を一括発行
- 前月分の経費を集計
- 取引先別の売掛金一覧を作成

### 月中
- 入金確認 → 消込
- 請求遅延のフォローアップ

### 月末
- 月次レポート作成
- 未入金リスト作成
- 翌月の予測

## 守秘義務

- 取引先名・金額・口座情報は **CLAUDE.md に書かない**
- 作業時は必要に応じて匿名化
- 外部サービスへアップロードする際は情報を削除・マスキング

## 禁止事項

- 脱税・節税の提案
- 二重帳簿の作成
- 伝票の改ざん
- 未確認情報での金額確定
