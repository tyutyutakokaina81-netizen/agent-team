### 無料で公開する設定 5 つ

#### ✅ 1. バッテリー充電上限を 80% に設定

設定 → バッテリー → 充電 → **80%制限** を選択

バッテリーの化学的劣化を大幅に抑えられます。2年後のバッテリー持ちが全然違います。

#### ✅ 2. 自動ロックを「1分」に変更

設定 → 画面表示と明るさ → 自動ロック → **1分**

初期設定は長すぎます。ポケットに入れたまま画面が付きっぱなし → バッテリー浪費の最大原因。

#### ✅ 3. カメラコントロールを「ダブルクリック」に変更

設定 → カメラ → カメラコントロール → **ダブルクリック起動**

iPhone 17 の側面ボタン、初期設定だと触れただけでカメラが起動します。ポケットの中で勝手に写真が撮れる事故を防止。

#### ✅ 4. 適応型電力制御（Adaptive Power Mode）をON

設定 → バッテリー → **適応型電力制御 ON**

iOS 26 の新機能。AIがあなたの使い方を学習して、バックグラウンド処理を自動最適化。これだけでバッテリー持ちが体感20%改善。

#### ✅ 5. Wi-Fi アシストをOFF

設定 → モバイル通信 → 一番下 → **Wi-Fiアシスト OFF**

Wi-Fiが弱いと自動でモバイル通信に切り替わる機能。便利そうだけど、気づかないうちにギガを大量消費します。

## バッテリー最適化（6〜10）

#### ✅ 6. バックグラウンド更新を選択制に

設定 → 一般 → Appのバックグラウンド更新

全アプリONではなく、**本当に必要なアプリだけON**にする。
推奨ON：LINE、メール、天気
推奨OFF：SNS、ゲーム、ニュース

#### ✅ 7. 位置情報を「使用中のみ」に一括変更

設定 → プライバシーとセキュリティ → 位置情報サービス

各アプリを開いて「**使用中のみ許可**」に変更。「常に」は原則不要。地図アプリだけ「常に」で可。

#### ✅ 8. 画面の明るさ自動調整ON + 手動でやや暗めに

設定 → アクセシビリティ → 画面表示とテキストサイズ → **明るさの自動調節 ON**

さらに手動で少し暗めにするとバッテリーが長持ち。

#### ✅ 9. メールのプッシュを「15分ごと」に変更

設定 → メール → アカウント → データの取得方法 → **15分ごと**

「プッシュ」は電池を食います。15分ごとのフェッチで十分。

#### ✅ 10. 不要なウィジェットを削除

ホーム画面を長押し → 使っていないウィジェットの「−」をタップ

ウィジェットは常にバックグラウンドで更新している。使わないものは消す。

## セキュリティ・プライバシー（11〜16）

#### ✅ 11. Face ID を設定（未設定なら）

設定 → Face IDとパスコード → Face IDをセットアップ

マスク対応済み。設定所要時間1分。

#### ✅ 12. 「盗難デバイスの保護」をON

設定 → Face IDとパスコード → **盗難デバイスの保護 ON**

盗まれた場合にパスコードだけではApple IDやパスワードの変更ができなくなる。

#### ✅ 13. Safariのトラッキング防止をON

設定 → Safari → **サイト越えトラッキングを防ぐ ON**

広告があなたのネット閲覧履歴を追跡するのを防止。

#### ✅ 14. Apple Intelligence のデータ共有設定を確認

設定 → Apple Intelligence と Siri → **Apple Intelligenceの改善 OFF**

あなたの音声データをAppleに送信するかどうかの設定。OFFが無難。

#### ✅ 15. パスワードの自動入力をON

設定 → パスワード → パスワードオプション → **自動入力 ON**

iCloudキーチェーンと連携。サイトごとに違うパスワードを自動生成・保存。

#### ✅ 16. 2ファクタ認証が有効か確認

設定 → [自分の名前] → サインインとセキュリティ → **2ファクタ認証 ON**

最も重要なセキュリティ設定。未設定なら今すぐ。

## カメラ・写真（17〜19）

#### ✅ 17. 写真フォーマットを「互換性優先」に

設定 → カメラ → フォーマット → **互換性優先**

HEIF/HEVCだとWindowsやAndroidに送ると開けないことがある。

#### ✅ 18. Live Photos をデフォルトOFF

設定 → カメラ → **設定を保持 → Live Photos ON** にして、カメラアプリでLive Photos OFFに

容量節約。必要な時だけONにする。

#### ✅ 19. グリッド線をON

設定 → カメラ → **グリッド ON**

写真の構図が劇的に良くなる。三分割法の補助線。

---

## 通知・集中モード（20〜22）

#### ✅ 20. 通知プレビューを「ロック解除時のみ」に

設定 → 通知 → プレビューを表示 → **ロック解除時のみ**

ロック画面でLINEの中身が丸見えになるのを防止。

#### ✅ 21. 集中モード「おやすみ」を設定

設定 → 集中モード → おやすみモード → **23:00〜7:00**

睡眠中の通知を自動ミュート。

#### ✅ 22. 不要アプリの通知をOFF

設定 → 通知 → 各アプリ

ゲーム、ショッピング、ニュースアプリの通知は基本OFF。通知が必要なアプリだけ残す。

---

## ストレージ・その他（23〜24）

#### ✅ 23. iCloud写真を有効化

設定 → [自分の名前] → iCloud → 写真 → **iCloud写真 ON**

本体ストレージの圧迫を防ぐ。5GBの無料枠で足りなければ50GB（¥130/月）が最安。

#### ✅ 24. 「iPhoneを探す」をON

設定 → [自分の名前] → 探す → iPhoneを探す → **ON**

紛失・盗難時に位置特定＋リモートロック＋リモート消去が可能。
