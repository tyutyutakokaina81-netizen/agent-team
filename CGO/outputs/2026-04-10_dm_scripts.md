# 初回DM スクリプト集（業種別）

> **使い方**:
> 1. Instagram の候補アカウントを開く
> 2. 業種に合ったスクリプトを選ぶ
> 3. [ ] 部分を書き換えて DM 送信
> 4. 送信したら prospect_list の「接触日」を記入する

---

## 共通ルール

- 文字数：100〜150字（読まれやすい長さ）
- 売り込みではなく「問い合わせ・提案確認」のトーン
- 最初のDMで価格を言わない
- 返信がなければ7日後に1回だけフォローアップ

---

## 飲食店向け（カフェ・レストラン・居酒屋）

```
はじめまして。
[お店の名前]様のお料理の写真、とても美味しそうで拝見しました。

SNS投稿の代行を専門にしています。
毎月の投稿作りが大変でなければ問題ないのですが、
もし「更新が追いつかない」「ネタが思いつかない」と
感じることがあれば、お力になれるかもしれません。

少しだけお話を聞いていただけますか？
```

---

## 美容院・ネイルサロン向け

```
はじめまして。
[サロン名]様のInstagram、施術のクオリティがよく伝わってきて素敵ですね。

SNS運用のお手伝いをしています。
「定期投稿したいけど撮影・文章が大変」という声をよく聞くのですが、
もし共感していただけるようなら、一度ご相談させていただけませんか？

お気軽に返信いただければ嬉しいです。
```

---

## 整体院・整骨院・鍼灸院向け

```
はじめまして。
[院名]様のInstagramを拝見し、患者さんへの丁寧な発信が
印象的で連絡しました。

SNS投稿の代行専門です。
「毎日投稿したいが時間がない」とお感じなら、
月額でまるごとお任せいただける仕組みがあります。

ご興味があればお気軽にどうぞ。
```

---

## コーチ・コンサル・スクール向け

```
はじめまして。
[名前/屋号]様の発信内容、すごく刺さる言葉が多くていつも参考にしています。

SNS運用の代行を専門にしています。
発信の「質」は高いのに「頻度」だけが課題という方のお手伝いが多いです。

もしそういった状況があれば、ご相談だけでも喜んで。
```

## 士業（税理士・社労士・行政書士）向け

```
はじめまして。
[事務所名]様のSNSを拝見しました。
専門知識をわかりやすく発信されていて勉強になります。

SNS投稿の代行をしています。
先生の知識・実績をもとに投稿文を作成し、
見込み客へのアプローチを継続するお手伝いができます。

少しだけお話を聞いていただけますか？
```

## フォローアップDM（7日後・返信なし）

```
先日ご連絡した[名前]です。
突然のメッセージ失礼しました。

もしご興味なければ全く問題ありません。
もし少しでも気になっていただけていたら、
30分だけオンラインでお話しできれば嬉しいです。

引き続きInstagram楽しみにしています！
```

---

```
返信いただきありがとうございます！
オンラインで30分ほどお話しできますか？

日程候補を3つほどお知らせします：
・[日付・時間]
・[日付・時間]
・[日付・時間]

Zoom / Google Meet どちらでも対応します。
```
