# 見込み客プロスペクトリスト（B2B SNS運用代行）

> **目的**: SNS運用代行の初回契約を1社獲得する（¥25,000 初月半額）
> **ターゲット**: フォロワー 500〜5,000 の地域密着型アカウント
>
> **探し方（オーナー実施・10〜15分）**:
> 1. Instagram を開く
> 2. 検索タブ → 地域名＋業種で検索（例:「渋谷 カフェ」「新宿 美容院」）
> 3. フォロワー数 500〜5,000・最終投稿が2週間以上前のアカウントを探す
> 4. 下の表にアカウント名・URL を記入してから DM を送る

---

## ターゲット探し方のコツ

```
✅ 良い見込み客の条件
- フォロワー: 500〜5,000（0や超大手は除く）
- 最終投稿: 2週間〜3ヶ月前（更新が止まっている）
- プロフィールに「ご予約は電話で」など→ SNS 活用しきれていない証拠
- ローカルビジネス（近所の人が客）

❌ 外す条件
- すでに月15投稿以上している → 担当者がいる
- フォロワー1万超 → すでにプロ運用
- 大手チェーン店
```

## 確率の目安

```
20件 DM 送信
  → 返信率 20〜30%（4〜6件返信）
  → 相談承諾率 50%（2〜3件相談）
  → 成約率 30〜50%（1件成約）

初月半額 ¥25,000 × 1社 → 目標達成
```

> 1社取れれば来月以降は継続 ¥50,000/月 になる

---

## DM 送信スケジュール

```
Day 1〜3:   5件/日 × 3日 = 15件送信
Day 4〜7:   残り5件送信・返信対応
Day 8〜14:  フォローアップ（7日経過した未返信に1回だけ）
Day 15〜:   返信があった相手と相談・提案
```
