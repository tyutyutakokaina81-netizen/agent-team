## 【提案】iPhone での毎朝の流れ

### 朝7時〜8時（誰でも実行可能）

1. **GitHub アプリ**を iPhone で開く
2. リポジトリ `tyutyutakokaina81-netizen/agent-team` → `daily/<最新日付>/` を開く
3. `note_draft.md` をタップ → **Raw** → 全選択 → コピー
4. **note アプリ**を開く → 新規記事 → 貼り付け → 公開
5. 完了

**所要：1分**

### Reddit を有効化する（任意・5分の初回設定）

1. https://www.reddit.com/prefs/apps で「create another app」→ script タイプを選択
2. リポジトリの Settings → Secrets and variables → Actions → New secret で以下を登録：
   - `REDDIT_CLIENT_ID`
   - `REDDIT_CLIENT_SECRET`
   - `REDDIT_USER_AGENT`（例：`ai-auto:v1.0 by /u/yourname`）
   - `REDDIT_USERNAME`
   - `REDDIT_PASSWORD`
3. Settings → Secrets and variables → Actions → Variables で `ENABLE_REDDIT_AUTO_PUBLISH=true`
4. （任意）`REDDIT_SUBREDDIT` 変数を設定（既定: SlowLiving）
5. 翌朝から Reddit 投稿が完全自動化される

## 【提案】3段階の自動化レベル

| Level | 内容 | mac | iPhone | リスク | 工数 |
|------|------|-----|--------|--------|------|
| **L0**（現在）| GitHub Actions ドラフト生成のみ・公開は iPhone コピペ | 不要 | 1分/日 | ゼロ | 0分 |
| **L1**（推奨）| L0 + Reddit を GitHub Actions で完全自動公開 | 不要 | 1分/日 | 極小 | 5分（初回） |
| **L2**（任意）| L1 + mac で Plan B 全自動投稿（note/X/CW も自動） | 必要 | 不要 | 中〜高 | 30分（初回） |

L1 が時間対効果最大。L2 は BAN リスクを承知で長期運用したい場合のみ。

詳細：`projects/2026-05-05_AI自動収益化引き継ぎ/deploy/iphone_shortcuts.md`
