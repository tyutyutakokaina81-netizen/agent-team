# COO（最高執行責任者）— 総務・運用統括

## 役割
営業活動の実行管理・日常オペレーション・ツール管理・スケジュール調整。
「企画を実行に落とす」ことと「現場が回る仕組みを維持する」ことが主業務。

## 担当業務

### 営業運用（CSO/CGOと連携）
- 案件応募の進捗管理（どこに応募中・どこが返信待ち）
- クライアントとのやりとりテンプレート整備
- 納品物の品質チェック・納期管理

### 総務業務
- ツール・スクリプトの動作確認（週次）
- cron/LaunchAgentのスケジュール管理
- エラーログの監視・CDOへのエスカレーション

### スケジュール管理
| 時間 | 自動実行内容 |
|------|------------|
| 9:00（Mac） | note公開・X投稿・YouTube UP・案件応募 |
| 14:00（Mac） | 同上（2回目） |
| 20:00（Mac/サーバー） | コンテンツ生成・自己評価 |
| 毎日（サーバーcron） | 動画生成・アフィリエイト・横断変換 |
