# 面談文字起こし テンプレート

**作成日**: 2026-05-23
**用途**: CPOメソッドの一次情報インプット（顧客の生の声）
**運用**: 1面談 = 1ファイル、`CSO/research/YYYY-MM-DD_面談_[相手].md` に保存

---

## 使い方

1. 面談前にこのテンプレを `CSO/research/YYYY-MM-DD_面談_[相手略称].md` にコピー
2. 録音 → 文字起こしツール（Whisper / Notta 等）で本文を流し込み
3. 面談直後10分以内に「3行要約」と「キーセンテンス」だけは手動で埋める
4. `/business` コマンド実行時に Read で参照させる

---

## テンプレート本文（ここから下をコピーして使用）

```markdown
# 面談文字起こし：YYYY-MM-DD ◯◯様

## メタ情報

| 項目 | 内容 |
|------|------|
| 日時 | YYYY-MM-DD HH:MM〜HH:MM |
| 相手 | 氏名／会社名／役職 |
| 業種 | … |
| 規模 | … |
| 媒体 | 対面／Zoom／電話 |
| 関係 | 既存顧客／見込／紹介／会食 |
| 録音 | あり／なし（あればファイルパス） |

## 3行要約（面談直後に必ず記入）

1. 一番印象に残った発言：
2. 相手の主訴（一番困っていること）：
3. 次アクション：

## キーセンテンス（直接引用・5〜10件）

> 「…」  — 相手の発言を一字一句

> 「…」

## 文脈タグ（複数可）

- [ ] 課題発見
- [ ] 価格反応
- [ ] 競合の話題
- [ ] 成功体験
- [ ] 失敗体験
- [ ] 業界トレンド
- [ ] 紹介可能性

## 全文文字起こし

（Whisper / Notta などの出力をそのまま貼り付け）

…

## CPOメソッド連携メモ

| 連携先 | 抽出したい情報 | 該当箇所 |
|-------|------------|---------|
| `/business` ステップ1 | ターゲット像・悩み | キーセンテンス第○ |
| `/business price` | 予算感・価格反応 | キーセンテンス第○ |
| `/business compare` | 競合への不満 | 全文 L○○ 付近 |

## 個人情報の取扱

- [ ] 氏名・連絡先は別管理（Gitコミット禁止）
- [ ] 引用の社外共有は本人承諾済み or 匿名化済み
```

---

## 運用ルール

1. **面談直後10分以内**に「3行要約」だけは必ず書く（記憶の鮮度が命）
2. 全文文字起こしは後日でOK
3. キーセンテンスは **5〜10件** を目安に（多すぎるとAIが拾いきれない）
4. 個人情報（氏名・連絡先）は別ファイルに分離し `.gitignore` で除外
5. 匿名化したものだけを `CSO/research/` 配下に置く
