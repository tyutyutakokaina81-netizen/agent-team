# 顧客アンケート設計テンプレート

**作成日**: 2026-05-23
**用途**: CPOメソッド（5ステップ）の一次情報インプット
**運用**: 100人規模の定性コメント収集を想定

---

## 1. 使い方

1. 配信ツール（Googleフォーム／Notion／LINE公式）に下記設問を移植する
2. 配信後、回答をCSVエクスポート
3. CSV を `CSO/research/YYYY-MM-DD_アンケート_[テーマ].csv` に保存
4. `/business` コマンド実行時に **一次情報** として参照させる

---

## 2. 設問テンプレート（コピペ用）

### A. 属性（必須）

```
Q1. 業種を教えてください（自由記述）
Q2. 事業規模：[ ] 1人  [ ] 2〜5名  [ ] 6〜20名  [ ] 21名以上
Q3. 年商規模：[ ] 〜500万  [ ] 〜3000万  [ ] 〜1億  [ ] 1億超
Q4. お住まいの地域（都道府県）
```

### B. 課題ヒアリング（CPOメソッド ステップ1の素材）

```
Q5. 今、業務で一番時間を取られていることは何ですか？（自由記述・100字程度）
Q6. それを解決するために、月いくらまでなら払いますか？
   [ ] 〜5,000円  [ ] 〜30,000円  [ ] 〜50,000円  [ ] 〜100,000円  [ ] それ以上
Q7. 過去に類似サービスを使ったことがありますか？あれば名前と感想を教えてください（自由記述）
```

### C. 価格感度（CPOメソッド ステップ2の素材）

```
Q8. このサービス（A：◯◯）が月¥XX,XXXだったら申し込みますか？
   [ ] 即申込  [ ] 検討する  [ ] 高い  [ ] 興味なし
Q9. 「高い」と感じた場合、いくらなら申し込みますか？（自由記述）
Q10. 申し込まない理由をすべて教えてください
   [ ] 価格  [ ] 効果が不明  [ ] 信頼性  [ ] タイミング  [ ] その他（自由記述）
```

### D. 自由記述（鬼コンサル人格への重要素材）

```
Q11. このサービスに「これが入っていれば絶対買う」という機能・特典はありますか？
Q12. 逆に「これが入っていたら買わない」という地雷は何ですか？
Q13. 普段、同業他社のどんな点に不満を感じていますか？
```

---

## 4. 集計方針（CPOメソッド連携）

| アンケート設問 | 連携先 |
|--------------|------|
| Q5・Q11・Q12 | `/business` ステップ1（ターゲット選定の一次情報） |
| Q6・Q8・Q9 | `/business price`（プライシング論拠） |
| Q7・Q13 | `/business compare`（競合差別化の素材） |
| Q11・Q12 | `/business score`（事業評価リスク軸の根拠） |

---

## 5. 注意

- 個人情報（氏名・連絡先）を集める場合は別シートで分離管理し、Git にコミットしない
- 100人規模を1回で集めるより、10人×10波の継続収集が有効
- 自由記述は文字数制限を緩めに（最低100字を確保）
