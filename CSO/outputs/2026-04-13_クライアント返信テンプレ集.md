# クライアント返信テンプレ集（2026-04-13）

応募後に来る返信パターン別の即コピペ返信文。

---

## パターン1：アンケート・ヒアリング依頼

### スキル・経験について
```
■ 使用可能なツール
Excel / Googleスプレッドシート / Word / Googleドキュメント
Canva / ChatGPT・Claude
基本的なPC操作全般

■ 作業経験
データ入力・リスト作成・コピペ作業を中心に対応してきました。
Excel/スプレッドシートでのVLOOKUP・ピボットテーブル等の関数操作も可能です。

■ 得意分野
正確性を重視した丁寧な作業
ダブルチェックによるミス防止
マニュアル通りの安定した作業
```

### 稼働時間・対応可能時間
```
平日：3〜5時間（主に夕方〜夜）
土日：対応可能
連絡：即日〜数時間以内に返信可能
納期厳守を最優先にしております。
```

### 今後の展望
```
継続的にお仕事をお受けし、
長期的な信頼関係を築いていきたいと考えております。

データ入力だけでなく、リサーチやSNS運用など
業務の幅も広げていきたいです。
```

## パターン2：見積もり・単価交渉

### 提示された単価を受け入れる場合
```
ご提示いただいた単価で問題ございません。
丁寧かつ正確な作業を徹底し、
納期厳守でお納めいたします。

まずはテスト的に少量からでも対応可能ですので、
ご都合の良い案件数からスタートできればと思います。

よろしくお願いいたします。
```

### 単価について質問する場合
```
ご連絡ありがとうございます。
作業内容について確認させてください。

・1件あたりの入力項目数：
・想定される1件あたりの作業時間：
・継続案件の場合、月間の件数目安：

上記が確認できましたら、
ご提示の単価で問題なく対応可能です。

よろしくお願いいたします。
```

## パターン3：テスト作業・試用依頼

### 受諾する場合
```
テスト作業のご依頼ありがとうございます。
喜んで対応させていただきます。

・納期：[指定された日時]
・納品形式：[Excel/スプレッドシート/CSV等]
・注意点：[クライアントが指定した内容を復唱]

上記の認識で間違いないでしょうか。
作業開始後、不明点がございましたら随時ご確認いたします。

よろしくお願いいたします。
```

### テスト提出時
```
お世話になっております。

■ 作業内容
・[実施した内容を箇条書き]

■ 作業時間
・[実際にかかった時間]

■ 気づき・ご確認事項
・[作業中に気づいた点があれば記載]

ご指摘・ご修正があればお知らせください。
引き続きよろしくお願いいたします。
```

## パターン5：質問・確認への返信

### 不明点を質問する
```
お世話になっております。

【質問内容】
[具体的な質問]

【該当箇所】
[どのタスク・どの箇所か明示]

お手すきの際にご確認いただけますと幸いです。
よろしくお願いいたします。
```

### クライアントからの質問に答える
```
ご連絡ありがとうございます。
ご質問の件についてお答えいたします。

[回答内容を簡潔に]

他にもご不明点がございましたら
お気軽にお知らせください。

よろしくお願いいたします。
```

## パターン8：怪しい案件・要注意返信

### 以下のような返信が来たら無視または辞退

- 事前に登録料を要求される
- LINE/外部サービスへの誘導
- 個人情報を過剰に要求される
- 「簡単に月100万」等の非現実的な話
- 内容が曖昧で具体性がない

### 丁寧に辞退する場合
```
ご連絡ありがとうございます。
検討させていただきましたが、
今回は見送らせていただきます。

他のお仕事でご一緒できる機会があれば
よろしくお願いいたします。
```

## 使い方

1. クライアントから返信が来たらパターンを特定
2. 該当するテンプレをコピペ
3. [ ] の箇所を実情に合わせて書き換え
4. 送信

迷うな。貼れ。送れ。
