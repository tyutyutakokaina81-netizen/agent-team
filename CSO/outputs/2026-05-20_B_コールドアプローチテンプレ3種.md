- 用途：見込み顧客への初回接触（コールドアプローチ）

---

## 共通設計方針

1. **最初の1通は「売り込み」ではなく「観察と質問」** に徹する（返信ハードルを下げる）
2. 自分が **相手の発信を実際に見た証拠** を1行入れる（コピペ営業対策）
3. クロージングは「30分相談」ではなく **「1問だけ質問」** にする
4. AI訴求は **一切しない**（中小事業者は警戒する）
5. 価格・プラン提示は2通目以降に回す（初回は出さない）

## テンプレ①：Instagram DM 用（飲食店・美容院向け／200字以内）

### 用途
- 地域の飲食店・美容院・整体院・パーソナルジムなど BtoC 個人事業店舗
- Instagram の DM 機能で直接送信
- 1日10件まで（連投スパム判定回避）

### 文面

```
○○様、はじめまして。
SNS運用のサポートをしている[氏名]と申します。

[店名]の○○の投稿、[具体的に見た要素：例「テリーヌの断面写真」]が
とても綺麗で、思わずプロフィールから過去投稿まで拝見しました。

ひとつだけ伺っても良いでしょうか。
投稿の頻度をもう少し上げたいというお気持ち、ありますか？

もしお手伝いできることがあればと思い、ご連絡しました。
（営業ではなく、まずは現状だけお聞きできればと思っています）
```

### カスタム必須箇所
- `[店名]`
- `[具体的に見た要素]`（必ず1分でも実際にプロフィールを見る）

### 送信ペース
- 1日10件まで（曜日問わず）
- 同じ業種に同日連投しない
- 返信があったら24時間以内に必ず一次返答

## テンプレ②：お問い合わせフォーム用（士業向け／400字以内）

### 用途
- 税理士・社労士・行政書士・司法書士の事務所サイト
- Web 公開されている「お問い合わせ」フォームから送信
- 件名行が用意されている場合は別途記入

### 件名
```
ご相談：先生のSNS発信について（営業ではなく1問お伺いしたく）
```

### 文面

```
○○先生

突然のご連絡失礼いたします。
SNS運用のサポートをしております [氏名] と申します。

先生のサイトと、リンクされていた[媒体名：X/Facebook等]の発信を
拝見しました。
[具体的に見た投稿の話題：例「最近の電子帳簿保存法の解説投稿」]は
わかりやすく、こうした情報を待っている方は多いと感じました。

ひとつだけ伺いたいことがあり、ご連絡しました。
**「SNS発信の頻度をもう少し上げたい」というお気持ち、
ございますでしょうか？**

もしあれば、業務を妨げない形でお手伝いできる方法を
お話しできればと思っております。
（本メールは情報提供のみで、即時のご返答や契約を求めるものではありません）

ご多忙のところ恐縮ですが、一行でもご返信いただけますと幸いです。

[氏名]
[メールアドレス]
```

### カスタム必須箇所
- `[媒体名]`（サイト上で公開されている SNS リンクを確認）
- `[具体的に見た投稿の話題]`（必ず実際に1〜2投稿読む）

### 送信ペース
- 1日5件まで（士業はメール文化なので過密だと逆効果）
- 月〜金の午前中（10〜11時）に送信
- 返信があったら **必ず先方の都合に合わせる**（24時間以内に一次返答）

## テンプレ③：クラウドワークス応募文用（SNS運用代行の公募案件向け／400字程度）

### 用途
- クラウドワークスで「SNS運用代行」「Instagram 運用」「投稿代行」等のキーワードで公募されている案件に応募
- A の応募運用ルールに準じて日次10件まで送信可能
- A の応募ログとは **別カラムで管理**（テンプレ：`B-CW` と表記）

### 文面

```
ご担当者様

[案件タイトルの主要キーワード]の運用代行案件、拝見しました。
SNS運用サポートをしております [氏名] と申します。

【ご提案できること】
・ターゲット設定・投稿テーマ設計・投稿文作成・予約投稿まで一貫対応
・初回ご返信は24時間以内、ご納品物は事前にトンマナ確認後に着手
・月次レポート（フォロワー数・リーチ・エンゲージメント）付き

【対応媒体】
Instagram / X（Twitter）/ Facebook
※ご指定の媒体に絞った運用も可能です

【お試し条件】
初回1ヶ月のみ、通常 50,000円 のところを 25,000円 で承ります
（ご評価コメントのご記入が条件・先着3社限定）

【ご質問】
1点確認させてください。
[案件文面から想定したターゲット読者像、または投稿頻度の希望]で
合っていますでしょうか？
合っていれば、本日中に1ヶ月分の投稿テーマ案ドラフトをお送りすることも可能です。

ご検討よろしくお願いいたします。
[氏名]
```

### カスタム必須箇所
- `[案件タイトルの主要キーワード]`
- `[案件文面から想定したターゲット読者像 / 投稿頻度の希望]`

### スタートアップ枠管理
- 「先着3社限定」の進捗を `CSO/research/応募ログ/2026-05.md` のスタートアップ枠とは **別管理**
- 3社達成 or 2週間ゼロで終了判定（A と同様）

## 運用優先順位（推奨）

1. **テンプレ③（CW応募）から開始** — A の応募運用基盤がそのまま流用できる
2. CW で返信が出始めた段階で **テンプレ①（Instagram DM）** を1日10件試行
3. 両方の返信率を見て **テンプレ②（士業フォーム）** に展開

→ いずれも A の本格運用が安定（5/27 レビュー以降）してから着手するのが推奨。
今週はテンプレ整備のみで実運用は来週から。
