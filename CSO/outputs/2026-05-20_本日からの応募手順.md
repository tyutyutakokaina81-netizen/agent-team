## 本日（5/20）のタスク（所要 約30分）

### Step 0：パイプラインのセッション準備（初回のみ・5分）
```bash
cd projects/2026-04-08_月30万自動化/D_エクセル入力スクレイピング/pipeline
python run_pipeline.py setup
```
- セッションが無ければ `python 00_session_setup.py` を実行してクラウドワークス・ランサーズに手動ログイン

### Step 1：パイプライン実行（10分）
```bash
python run_pipeline.py search --category writing
```

自動で実行されること：
- ライティング案件をキーワード5種で検索（SEO記事／ライティング／Webライター／ブログ記事／記事執筆）
- 応募禁止ルール該当案件を即 NO-GO 判定
- スコアリング（テーマ／信用／採算／明確性 各25点）と GO/CAUTION/NO-GO 判定
- 推奨テンプレ（CSO 5パターンから自動選択）と応募文の自動生成
- **応募フォーム自動入力モード起動の確認**（y で続行）

### Step 2：応募フォームを開いて自動入力させる（15分／5件分）

応募フォーム自動入力モード（`03b_autofill.py`）が起動すると、案件ごとに：

1. ブラウザが案件ページを自動で開く
2. **オーナーが「応募する」ボタンを手動でクリック**
3. 応募フォーム画面への遷移をパイプラインが自動検出
4. **本文欄が自動入力される**（CSO 5テンプレから選ばれた応募文）
5. オーナーが内容を確認し、必要なら手動編集
6. **オーナーが応募確定ボタンを手動でクリック**（送信は人手・規約遵守）
7. ターミナルで Enter を押すと次の案件へ

> 上位 5 件に対して順次実行されます。途中で Ctrl+C で中断可能。

### Step 3：応募ログ記録（5分）
- `CSO/research/応募ログ/2026-05.md` に送信した案件を1行ずつ追記
- スコアと推奨テンプレも記録（後でABテストの素材になる）
