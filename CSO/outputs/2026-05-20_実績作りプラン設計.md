# 実績作りプラン（5件限定）設計書

- 日付：2026-05-20
- 担当：CSO（営業）× CMO（コピー）
- 用途：クラウドワークス受注ゼロ突破のための初動限定プラン
- 関連：`CSO/research/2026-05-20_クラウドワークス受注ゼロ要因分析.md`（要因C）

---

## 1. 要点

実績・評価ゼロの状態で文字単価3.75円帯（スタンダード）に応募しても、
実績ありのライターに価格優位もなく敗北する。
**最初の5件だけ単価を下げて評価を獲得する** プランを併設する。

5件のクライアント評価★4.5以上が貯まれば、その後はスタンダードプランで戦える状態になる。

---

## 2. プラン仕様

### 基本スペック

| 項目 | 内容 |
|------|------|
| プラン名 | スタートアッププラン（実績作り） |
| 文字数 | 2,000字 |
| 価格 | 4,000円（文字単価2.0円） |
| 納期 | 2営業日 |
| 修正 | 2回まで無料 |
| 含まれるもの | 構成案／本文／メタディスクリプション |
| 上限 | **先着5件まで**（達成後は通常プランへ） |
| 条件 | クライアントからの評価コメント記入を依頼 |

### 適用ルール（裏側で運用）

- このプランで応募するのは「実績不問」「初心者歓迎」明記の案件のみ
- 文字単価が募集側で1.5円以上明示されている案件は **通常プランで応募**（値下げしない）
- 5件達成時点で本プランは取り下げ、サービスページからも削除する
- 5件のうち2件以上で★4未満が出た場合 → プランを停止し、品質側の見直しに切り替え

---

## 3. クライアントへの提示文（応募時に添える）

応募文テンプレ③（初心者歓迎系）末尾に、以下の差し替えブロックを追加する。

```
【スタートアッププラン（5件限定）】
ご評価をいただくため、最初の5名様限定で
通常 8,000円のライトプラン（2,000字）を、
4,000円でお受けしております。

ご検討の条件として、
納品後にご評価コメントをお願いできれば幸いです。

※すでに残り[X]枠です（応募時に更新）
```

---

## 4. 経済性チェック

| 項目 | 金額 |
|------|------|
| 5件の売上 | 20,000円 |
| 機会損失（本来15,000円×5件） | ▲55,000円 |
| 期待効果：その後の成約率2倍想定 | スタンダード換算 月+10万円〜 |

**5件分の値引き = 55,000円の自己投資** として位置づける。
キャッシュフロー上は2026-06末の入金として計上。

---

## 5. プラン終了の判定基準

以下のいずれかを満たした時点で本プランは終了し、サービスページから削除する。

- A：5件受注完了（成功シナリオ）
- B：★4未満が2件以上（品質課題のため見直し）
- C：2週間以上応募ゼロ・受注ゼロ（プラン以外の要因が大きいと判断）
