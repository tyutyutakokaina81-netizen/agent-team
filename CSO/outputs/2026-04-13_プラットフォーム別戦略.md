# プラットフォーム別受注戦略（2026-04-13）

5つのプラットフォームで同時展開し、リスクを分散する。

---

## 1. クラウドワークス（既存・応募中）

**モデル：** 応募型（こちらから応募する）
**単価：** 1,000〜1,400円/時間
**手数料：** 20%
**強み：** 案件数が最多、初心者案件が豊富

**攻略法：**
- 「初心者歓迎」「タスク形式」の案件から入る
- 評価5件貯めてからスタンダード案件を狙う
- 本人確認を必ず済ませる

**URL：** https://crowdworks.jp/public/jobs/category/52

---

## 2. ランサーズ（既存・応募中）

**モデル：** 応募型
**単価：** 1,000〜1,500円/時間
**手数料：** 16.5%
**強み：** クラウドワークスより単価が若干高い

**攻略法：**
- 「データ収集・入力・リスト作成」カテゴリに絞る
- 認定ランサー目指す（月10万以上で認定）
- プロフィールに「リスト作成」「コピペ」のスキルを登録

**URL：** https://www.lancers.jp/work/search/task/collect

---

## 3. ママワークス（新規・推奨）

**モデル：** 応募型（業務委託中心）
**単価：** 1,226〜1,450円/時間（**他サイトより高単価**）
**手数料：** 0円（企業が直接雇用）
**強み：** 主婦特化・在宅前提・継続案件多い

**攻略法：**
- 週3日×3.5時間の継続案件を狙う
- 早朝枠（5:00-9:00）は競争率が低い
- 商品情報調査・商品登録が取り組みやすい

**URL：** https://mamaworks.jp/list?lrj%5Brecruitmentjob_id%5D%5B0%5D=200

**推奨応募文（コピペ可）：**
```
はじめまして。
継続的な在宅ワークを希望しております。

■ 稼働可能時間
週3日以上、1日3.5時間以上
早朝枠・午前枠ともに対応可能

■ 使用ツール
Microsoft Excel / Googleスプレッドシート
ChatGPT・Claude等のAI補助ツール

■ 作業スタンス
・マニュアル通りの正確な作業
・ダブルチェックによるミス防止
・納期厳守・連絡迅速
・長期継続希望

商品情報の調査・登録作業は得意分野です。
まずはテスト的に少量からでも対応可能です。

ご検討のほど、よろしくお願いいたします。
```

---

## 4. ココナラ（新規・推奨）

**モデル：** 出品型（こちらから商品を出す→依頼主がオファーしてくる）
**単価：** 自由設定（1,000〜10,000円/件）
**手数料：** 22%〜
**強み：** 待ちの営業・寝てても売れる

**攻略法：**
- 他のプラットフォームとは違い「出品型」
- データ入力・リスト作成を商品として登録
- 最初は相場より安く設定して評価を貯める

**出品タイトル案（3つから選ぶ）：**
```
1. 【1日以内納品】データ入力・コピペ作業を丁寧に対応します
2. 【初心者歓迎】Excelデータ入力・リスト作成を承ります
3. 【1,000円〜】AIも活用した高速・高精度なデータ入力
```

**出品説明文：**
```
■ サービス内容
Excel/Googleスプレッドシートへのデータ入力、
リスト作成、コピペ作業、簡単なWebリサーチを承ります。

■ 価格
・データ入力：1件20円〜（量と難易度による）
・リスト作成：1件30円〜
・コピペ作業：1件10円〜
・最低料金：1,000円〜

■ 納期
・通常：2〜3日
・短納期（1日以内）：追加料金あり

■ 対応範囲
・Excel/スプレッドシートへの入力
・Webからの情報収集
・簡単なデータ整理・クレンジング

■ 対応できないもの
・医療・法律・金融分野の専門データ
・個人情報を含む作業
・違法・脱法的な内容

■ 事前にお知らせください
・データ量（件数）
・入力項目
・参照元（URL・ファイル等）
・希望納期

まずはご相談からお気軽にどうぞ。
```

**URL：** https://coconala.com/

## 5. シュフティ（新規・保留）

**モデル：** 応募型
**単価：** 低め（500〜800円/時間）
