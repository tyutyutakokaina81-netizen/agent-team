# クラウドワークス プロフィール文・実績欄

- 日付：2026-05-20
- 担当：CSO
- 用途：クラウドワークス公開プロフィールに掲載

---

## 1. プロフィール本文（350字版・推奨）

```
SEOライティングを中心に、Web記事の企画・構成・執筆を行っております。

【対応領域】
・SEO記事（ブログ、オウンドメディア）
・コラム、まとめ記事、解説系記事
・LP原稿、サービス紹介文

【強み】
・執筆前に必ず構成案（H2/H3）をご提示し、方向性合意後に本文へ進みます
・初回ご返信は24時間以内を厳守
・修正は2回まで無料
・公的データ・公式情報を必ずソース併記でき、信頼性の高い記事を納品

【執筆スタイル】
読者の検索意図を起点に、「悩み→解決→次の行動」の流れで
読者がストレスなく読める文章を心がけています。

【ご連絡】
ご相談・お見積りは無料です。
お気軽にメッセージください。
```

## 3. スキル・対応ジャンル欄

**対応ジャンル（公開）**
- ビジネス・ライフスタイル・ITツール解説・教育・趣味全般

**裏で管理（応募しないジャンル）**
- 医療・法律・金融（YMYL）／成人向け／投資勧誘／薬機法該当領域

---

## 4. 完了させる本人確認・登録項目（チェックリスト）

オーナーが本日〜1週間以内に完了すべき項目：

- [ ] 本人確認書類提出（運転免許証またはマイナンバーカード）
- [ ] 電話番号認証
- [ ] NDA（秘密保持契約）同意
- [ ] 振込口座登録
- [ ] スキル検定：ビジネス事務検定（初回無料）
- [ ] スキル検定：ライティング技能検定（任意・推奨）
- [ ] プロフィール画像設定（顔出し不要・抽象画像可）
- [ ] 自己紹介ムービー（不要・任意）

## 5. ポートフォリオ作成方針

サンプル記事3本を **Googleドキュメント公開リンク** で提示する。

| 番号 | ジャンル | 文字数 | テーマ案 |
|------|---------|--------|---------|
| 1 | ビジネス | 2,000字 | リモートワーク導入企業が失敗する3つの理由 |
| 2 | ライフスタイル | 2,000字 | 一人暮らしの食費を月2万円に抑える買い物術 |
| 3 | ITツール | 2,000字 | 無料で使えるタスク管理アプリ徹底比較 |

作成依頼先：CDO（Claude プロンプト経由で初稿生成 → CSO レビュー → 公開）

## 6. 単価表記の注意

- プロフィールに固定単価を **書かない**（案件ごとに応相談で柔軟対応）
- ただし「○○円〜」の最低ライン表記もしない（敷居を下げる）
- 質問された場合は応募文／個別メッセージで提示する
