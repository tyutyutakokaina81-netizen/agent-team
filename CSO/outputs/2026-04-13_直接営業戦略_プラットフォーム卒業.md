# 直接営業戦略：プラットフォーム卒業ガイド（2026-04-13）

クラウドワークス・ランサーズの手数料20%を**0%**にする。
実収入を**1.25倍**に増やす上級者向け戦略。

---

## 🎯 目的：プラットフォーム手数料ゼロへ

### 現状（プラットフォーム経由）
```
売上：30万円
手数料（20%）：▲6万円
実収入：24万円
```

### 直接契約後
```
売上：30万円
手数料：0円
実収入：30万円
```

**→ 同じ売上で実収入+6万円/月 = 年間+72万円。**

---

## 🚀 直接契約までの3段階

### Phase 1：プラットフォーム内で実績を作る（1-2ヶ月目）
- 評価10件以上・受注5件以上の実績
- ポートフォリオURL公開
- リピート案件で信頼構築

### Phase 2：プラットフォーム経由で関係を深める（3ヶ月目）
- 受注案件で期待値以上の納品
- クライアントとの関係を築く
- 問題解決・改善提案で差別化

### Phase 3：直接契約を打診する（4ヶ月目〜）
- プラットフォームの規約を確認
- タイミングを見計らって提案
- 直接契約に移行

**メリット：**

## 💰 直接契約の料金設計

### 推奨：プラットフォーム価格より10-15%OFF

**理由：**
- クライアントにとってメリットを明示できる
- 手数料分が自分の取り分になる
- 長期契約のインセンティブになる

### 料金比較表

| プラットフォーム経由 | 直接契約（推奨） |
|--------------------|-----------------|
| 1記事5,000円→実収入4,000円 | 1記事4,500円→実収入4,500円 |
| 月額50,000円→実収入40,000円 | 月額45,000円→実収入45,000円 |

**→ クライアントは10%お得、自分は12.5%増収。Win-Win。**

## ⚠️ プラットフォーム卒業の注意点

### プラットフォーム規約違反に注意

**クラウドワークス規約（抜粋）：**
> 「本サービスを利用して知り合った方と、本サービスを介さずに直接取引を行うことは禁止されています」

**→ プラットフォーム内で知り合った相手との直接契約は規約違反**

### 安全な方法

**方法1：** プラットフォーム案件を1年以上継続してから
- 1年の継続取引後は規約の縛りが緩くなるケースが多い
- クライアント側から「直接やりませんか」と打診してもらう

**方法2：** クライアントの既存取引先になる
- 最初はプラットフォーム経由
- 別案件（プラットフォーム非経由のもの）で関係を広げる

**方法3：** 新規クライアントは最初から直接
- プラットフォーム非経由のクライアント（Wantedly・Indeed等）
- 規約違反にならない
