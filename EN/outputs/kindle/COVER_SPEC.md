# Cover Spec — *Toyama: Japan's Uncrowded Coast*

Kindle KDP eBook cover. Build this with free tools (Canva free tier, GIMP, or Photopea) — no paid assets required. All photo candidates below are **real photographs only** (per A2/A3: no AI-generated or illustrated cover art; ¥0 = free-stock or owner's own photos).

---

## 1. Canvas & export

| Item | Value |
|---|---|
| Dimensions | **1600 × 2560 px** (KDP ideal, 1.6:1 ratio) |
| Resolution | 72 dpi is fine for eBook; 300 dpi if reused for print later |
| Format | JPEG (quality ~90) or PNG under 50 MB |
| Color profile | RGB (sRGB) |
| Safe margin | Keep all text ≥ **120 px** from every edge; thumbnail-legibility is the priority (many buyers see this at ~160 px tall) |

---

## 2. Text layout (top-to-bottom)

The cover is a **vertical thirds** layout: photo fills the frame, a darkened band top and bottom carries the type.

```
┌───────────────────────────┐  ← 1600 × 2560
│  [TOP BAND ~ upper 38%]    │
│                            │
│   MAIN TITLE (2 lines)     │  ← largest element, must read at thumbnail size
│   TOYAMA                   │
│   Japan's Uncrowded Coast  │
│                            │
│   — thin rule —            │
│   SUBTITLE (1–2 lines)     │
│                            │
├───────────────────────────┤
│                            │
│   [MIDDLE ~ 40%]           │
│   hero photo, clean focal  │  ← the Alps-over-the-sea shot lives here, uncovered
│   point, no text overlap   │
│                            │
├───────────────────────────┤
│  [BOTTOM BAND ~ lower 18%] │
│   A Local's Guide to       │
│   Takaoka · Himi · Toyama  │
│   ── author line ──        │
│   TETSU                    │
└───────────────────────────┘
```

### Text content (use exactly this wording)

- **Main title:** `TOYAMA` (very large) with `Japan's Uncrowded Coast` underneath (medium).
  - Alt one-line treatment if the stacked version feels crowded: `TOYAMA` big, subtitle below.
- **Subtitle / descriptor (top band, under a thin rule):** `The Sea-of-Japan side of the Alps — seafood, mountains & no crowds`
- **Bottom band, small caps:** `A Local's Guide to Takaoka · Himi · Toyama`
- **Author line:** `TETSU` (or `by TETSU — a Toyama local`)

> Keep the wording consistent with the book's title page. Do **not** add superlatives like "world's best" or "#1" (A5: no exaggeration).

### Type sizing (relative, for 1600 px width)

| Element | Approx size | Weight |
|---|---|---|
| `TOYAMA` | 150–190 px cap height | Bold / Extrabold |
| `Japan's Uncrowded Coast` | 70–85 px | Regular / Medium |
| Subtitle descriptor | 40–48 px | Regular (letter-spaced) |
| Bottom "A Local's Guide…" | 38–44 px | Medium, small-caps, letter-spaced |
| Author `TETSU` | 46–54 px | Medium, letter-spaced |

## 3. Photo candidates (real photos only, free sources)

Pick **one hero image** for the full-frame background. Priority order:

1. **Amaharashi / Himi coast with the Tateyama range rising over Toyama Bay** *(first choice — this is the book's signature image).*
   - Ideal composition: calm bay in the lower third, snow-capped Alps floating on the horizon, cool morning light. The lone pine-rock off Amaharashi is a bonus but not required.
   - Search terms (free stock — Pexels/Unsplash/Pixabay, or owner's own): `Amaharashi coast Tateyama`, `雨晴海岸 立山`, `Toyama Bay Alps`, `Himi sea mountains snow`.
2. **Toyama glass art (富山ガラス)** — a single glowing, colourful glass piece on a dark background. Strong, modern, high-contrast; works well if a clean coast shot can't be sourced. Search: `Toyama glass art`, `富山ガラス`, `glass sculpture blue`.
3. **Himi fishing port / boats at dawn (氷見の海)** — weathered boats, calm water, soft light. Evokes the "real, lived-in" angle. Search: `Himi fishing port`, `氷見 漁港`, `Japan Sea of Japan harbour dawn`.

**Secondary / spine or back-cover accents (optional):** white shrimp sushi (shiroebi), a snow-walled Alpine Route bus, Gokayama thatched roofs under snow.

> Licensing: use only images cleared for commercial use (Pexels / Unsplash / Pixabay licenses, or the owner's own photos). Record the source URL + license for each image chosen, since KDP can query cover rights.

## 4. Colour & mood

- **Palette:** cool, calm, "Sea of Japan" tones — deep indigo/navy, slate blue, and a crisp near-white for snow and type. One warm accent only (a thin gold/amber rule or the author line) to lift it.
  - Suggested hexes: background dark band `#12283C` (deep navy) → `#1E3A52`; snow/type white `#F5F8FA`; accent gold `#D9A441`.
- **Bands:** apply a top-and-bottom gradient overlay (transparent in the middle → ~70–80% opacity navy at top and bottom edges) so text stays legible over any photo. In Canva: add a rectangle, fill with the navy, set to a linear-gradient transparency; in GIMP/Photopea: a black/navy layer with a gradient layer mask.
- **Contrast check:** the title must stay readable when the whole cover is shrunk to ~160 px tall and viewed in greyscale. If it blurs, increase the band opacity or the title size.

---

## 5. Fonts (free / open-license)

Use a **serif display for the title + clean sans for the rest**, or an all-sans modern look. All suggestions are free for commercial use:

| Role | Font (free) | Notes |
|---|---|---|
| Title `TOYAMA` | **Playfair Display** (bold) or **Cormorant Garamond** | Elegant, travel-editorial feel |
| — alt all-sans title | **Montserrat** (Extrabold) or **Archivo** | Cleaner, more modern |
| Subtitle / body lines | **Montserrat** / **Lato** / **Inter** | Letter-space the small caps lines slightly (tracking +50–100) |
| Author line | same sans as subtitle | Uppercase, letter-spaced |

> Pair at most two type families. Keep the title in one family and everything else in the other.
