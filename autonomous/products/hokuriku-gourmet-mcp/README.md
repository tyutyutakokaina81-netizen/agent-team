# hokuriku-gourmet-mcp

北陸三県（富山・石川・福井）のグルメ・郷土料理・B級グルメ・和菓子・地酒・伝統食情報を AI エージェントに提供する MCP サーバー。

**位置づけ**: autonomous AI 会社の収益商品。toyama-local-mcp と同じ wallet アドレスで運用可。

## 何をする？

- 39 品の北陸グルメ情報を収録（海鮮・郷土料理・B級グルメ・和菓子・地酒・伝統食）
- 県（富山/石川/福井）・カテゴリ・季節・予算でフィルタ検索
- 季節・予算・人数・シーンに応じた食事プランレコメンド
- 富山湾の月別旬魚カレンダー連動
- 料理同士のペアリング情報（一緒に食べたい組み合わせ）
- 有名レストラン・店舗情報付き
- Claude / ChatGPT / その他 AI エージェントから直接呼び出せる
- x402 protocol で自動決済（$0.02 〜 $0.08 USDC）

## 料金表

| エンドポイント | 料金 | 説明 |
|----|----|----|
| `GET /` | 無料 | HTML ランディング |
| `GET /health` | 無料 | ヘルスチェック |
| `GET /info` | 無料 | メタ情報・旬魚カレンダー |
| `GET /free/dish` | 無料 | 1品ランダム表示（お試し）|
| `POST /search` | **$0.03** USDC | カテゴリ/県/季節/予算で検索 |
| `POST /recommend` | **$0.08** USDC | 食事プランレコメンド |
| `POST /detail` | **$0.02** USDC | 1品の詳細取得 |
| `POST /mcp` | **$0.03** USDC | MCP JSON-RPC |

## MCP ツール

| ツール名 | 説明 |
|---------|------|
| `search_hokuriku_food` | カテゴリ・県・季節・予算でグルメ検索 |
| `recommend_meal_plan` | シーンに合わせた食事プランレコメンド（旬魚カレンダー連動） |
| `get_food_detail` | 1品の詳細情報（ペアリング・レストラン・旬判定付き） |

## ターゲットユーザー

- 北陸旅行で何を食べるか調べたい旅行者
- 旅行ブログ・コンテンツ作成者（AI で食事プランを組ませたい）
- 飲食店・旅館のコンシェルジュ bot
- インバウンド向け北陸グルメガイド
- 地方創生・食文化系の AI エージェント

## デプロイ

toyama-local-mcp と同じ手順：

```bash
cd autonomous/products/hokuriku-gourmet-mcp/
npm install
npx wrangler secret put SERVER_ADDRESS  # 同じ Coinbase Wallet address
npx wrangler deploy
```

deploy 後の URL: `https://hokuriku-gourmet-mcp.<your-subdomain>.workers.dev`
