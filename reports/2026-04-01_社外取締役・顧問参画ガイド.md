# 社外取締役・非常勤役員・スタートアップ顧問 参画ガイド
**対象**: 富山県高岡市在住・55歳・北陸電力送配電勤務・事務職・年収800万円・歩行困難・在宅完結希望  
**作成日**: 2026年4月1日

---

## 概要・結論

現職在籍中でも**会社の許可を得れば**社外取締役・顧問として活動できる。  
リモート完結案件が主流になっており、在宅勤務との親和性は高い。  
複数社掛け持ちで月額合計15〜60万円（年間180〜720万円）の副収入が現実的な射程圏内。  
ただし北陸電力送配電の**就業規則の個別確認と事前申請が最優先事項**。

## 1. 社外取締役・非常勤役員の報酬相場

### 企業規模別の月額・年額目安

| 企業区分 | 年額報酬の目安 | 月額換算 | 備考 |
|---|---|---|---|
| **中小企業（非上場）** | 120〜300万円 | 10〜25万円 | 明確な基準なし。関与度・実績次第 |
| **スタートアップ（シード〜シリーズA）** | 60〜200万円 | 5〜17万円 | 現金少なめ＋SO付与が多い |
| **スタートアップ（シリーズB〜上場前）** | 200〜480万円 | 17〜40万円 | IPO準備で報酬上昇 |
| **プライム上場（中央値）** | 約840万円 | 約70万円 | デロイトサーベイ2022 |
| **大企業（売上1兆円超、中央値）** | 約1,440万円 | 約120万円 | デロイトサーベイ2023 |

### 実務的な拘束時間
- 月1〜2回の取締役会出席（オンライン参加可が主流）
- 準備含め月10〜20時間程度が目安
- 顧問契約は月2〜4時間のミーティング＋メール対応のみのケースも多い

### 報酬が高くなる条件
- IPO準備中（上場経験、財務・ガバナンス知識を持つ人材）
- 電力・エネルギー・インフラ分野の専門知識
- コーポレートガバナンス、社内規程整備、リスク管理の経験
- 大企業勤務経験そのもの（中小・スタートアップには希少性が高い）

## 2. 現職在籍中に社外取締役になれるか

### 法的整理：問題なし（ただし条件あり）

- 会社法上、社外取締役は**委任契約**であり雇用契約ではない
- 職業選択の自由（憲法22条）が保障されており、**副業自体に法的障害はない**
- 問題は法律ではなく「**就業規則・内規**」と「**競業避止義務**」

### 実務的チェックリスト

| 確認事項 | 内容 | 優先度 |
|---|---|---|
| **就業規則の確認** | 兼業・副業禁止条項の有無 | ★★★ 最優先 |
| **事前申請・承認** | 許可制の場合は申請が必要（多くの大企業は許可制） | ★★★ |
| **競業避止義務** | 電力・エネルギー分野の同業他社でないか | ★★★ |
| **秘密保持義務** | 本業の機密情報を副業先に流さない | ★★★ |
| **利益相反回避** | 本業と顧問先に取引関係が生じないか | ★★ |
| **確定申告** | 年20万円超の副収入は確定申告必要 | ★★ |
| **住民税の普通徴収** | 確定申告時に「自分で納付」選択で副業バレ防止 | ★★ |

### 電力会社の副業動向
- **九州電力**は2021年に副業解禁（許可制、本店勤務2,300人対象）
- 電力各社で段階的な副業容認の流れが続いており、「許可制」が標準的な形式
- 北陸電力送配電の場合、**人事部門への個別確認が必須**
- 電力業界特有の災害対応・緊急業務への支障がないことが許可の前提条件になりやすい

### 推奨する進め方
1. 就業規則を確認（兼業・副業に関する条項）
2. 人事または上長に非公式に相談（制度上の可否を把握）
3. 可能であれば「副業申請書」等の正規手続きを踏む
4. 競合・エネルギー関連以外の業種から始める

## 4. ストックオプション付き顧問契約の実態

### 仕組みと特徴

- スタートアップが現金報酬を抑える代わりにSOを付与するのは標準的慣行
- **税制適格SO**（無償）が使えれば：
  - 付与時・行使時は課税なし
  - 売却時のみ**譲渡所得20.315%**の税率
  - 通常の給与所得課税（最高55%）より大幅に有利

### 令和6年度改正後の付与上限（税制適格SO）

| 会社の状況 | 年間行使額上限 |
|---|---|
| 設立5年未満 | **2,400万円/年** |
| 設立5〜20年未満または上場後5年未満 | **3,600万円/年** |

### 顧問・社外取締役への付与相場

| 関与度・ポジション | 付与比率の目安 | IPO時の想定リターン |
|---|---|---|
| 重要顧問・社外取締役（中核的関与） | 0.1〜0.5% | 数千万〜1億円超 |
| 一般的な顧問 | 0.01〜0.1% | 数百万〜数千万円 |
| 形式的な社外役員 | 0.005〜0.05% | 数十万〜数百万円 |

### 注意点
- **IPOしなければ紙切れ**になる可能性が高い（シード期の失敗率は70%超）
- 行使価格と現在の株価の差が大きいほど価値が高い（アーリー期参画が有利）
- 付与時に契約書で「行使条件・期限・会社売却時の取り扱い」を必ず確認
- 非上場株式のSOは売却先が限られるため流動性リスクがある

### 現実的な期待値の持ち方
- **現金報酬メイン＋SOをボーナス扱い**が安全な考え方
- SOをメインに据えるのは財務的余裕がある場合のみ

## 6. 在宅完結のためのポジション選定戦略

### 推奨する切り口

1. **電力・エネルギー系スタートアップの顧問**
   - 再エネ、省エネ、EVインフラ、系統安定化などの分野
   - 北陸電力勤務の経験は希少価値が高い
   - ただし競業避止義務に注意（社内確認が前提）

2. **非エネルギー系中小企業の社外取締役（ガバナンス強化目的）**
   - 大企業の事務職経験（コンプライアンス・稟議・規程整備）は中小に価値がある
   - 月1回のオンライン取締役会出席が中心
   - 競業リスクが低くなりやすい

3. **DX・業務改善コンサルとしての顧問**
   - 事務職×Excel・業務フロー改善の知見
   - IT化・ペーパーレス化を進めたい中小企業への助言
   - 完全在宅でのオンライン顧問が成立しやすい

4. **北陸・新潟エリアの地方スタートアップ顧問**
   - 同じ地域文化・商習慣を理解した顧問として価値
   - 富山大学・金沢大学発スタートアップのエコシステムに注目

- [ ] SCOP TOYAMAや富山県創業支援課のイベントにオンライン参加

- [ ] SOを付与された1〜2社のIPO・M&A成果を期待値として保有
- [ ] 早期退職・独立の選択肢として社外取締役業を主軸に切り替え検討

## 8. リスクと留意事項

| リスク | 内容 | 対策 |
|---|---|---|
| 本業への支障 | 副業が本業の能率・信頼を損なう | 拘束時間を週末・夜間に限定。本業優先を明確化 |
| 就業規則違反 | 無許可副業で懲戒処分リスク | **必ず事前申請・承認を取る** |
| 法的責任 | 社外取締役として損害賠償請求を受ける可能性 | D&O保険（役員賠償責任保険）の付保を条件に交渉 |
| SOの無価値化 | スタートアップ倒産でSOが紙切れに | 現金報酬をメインに設計し、SOはボーナス扱い |
| 過重負荷 | 複数社の取締役会・課題対応が重なる | 最初は1〜2社から始め、徐々に拡大 |
| 情報漏洩 | 本業の機密情報を顧問先で活用するリスク | 守秘義務契約を締結し、情報は厳格に分離 |
