## 自動化サーバー（補助）

依存ゼロの Node.js JSON API サーバーが付属しています。

```bash
node server.mjs           # http://127.0.0.1:3000
PORT=8080 node server.mjs # ポート変更
```

エンドポイント：

| メソッド | パス | 説明 |
|--------|------|------|
| GET | `/health` | ヘルスチェック |
| GET | `/version` | バージョン |
| POST | `/echo` | JSON エコー |

iPhone 操作用パイプラインサーバー（柱D）：

```bash
PIPELINE_TOKEN=secret node pipeline_server.mjs
# http://[サーバーIP]:3001
```
