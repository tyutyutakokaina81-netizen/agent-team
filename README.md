## 概要

本リポジトリは、5名のC役員（CDO/CFO/CMO/CPO/CSO）を AI で運営するオーナー専用の経営運用ツールです。
詳細な役職定義・運用ルールは `CLAUDE.md` および `company.md` を参照してください。

## 自動化サーバー（補助）

依存ゼロ（Node.js 標準モジュールのみ）の JSON API サーバー：

```bash
node server.mjs           # http://127.0.0.1:3000
PORT=8080 node server.mjs # ポート変更
```

| メソッド | パス | 説明 |
|--------|------|------|
| GET | `/health` | ヘルスチェック |
| GET | `/version` | バージョン |
| POST | `/echo` | JSON エコー |
